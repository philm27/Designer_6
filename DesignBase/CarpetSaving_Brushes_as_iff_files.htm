Saving Brushes as IFF files

 

The quickest and simplest way to save a brush is to use the Brush->Save menu option.

 

  1. Cut a brush if you have not already done so.

  2. Select menu option Brush->Save. The "Save from Shelf" dialog will be displayed.

  3. Save as for a normal design.

 

To load a brush back into a design, simply import it into a shelf then click on the shelf item with the left mouse button to capture it as a custom brush.
