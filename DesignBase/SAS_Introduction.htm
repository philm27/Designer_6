SAS Mode Introduction

 

What is it?

SAS mode or Small Active Section Mode is a way of working on a small section of a larger design so that the rest of the design is totally unaffected by the drawing operations.

This is useful for 3 main reasons.

(1) It is a simple means of protecting the rest of the design from colour fills, colour changes, mirror, recentre, insert/delete duplicate end or row etc etc

(2) The program will respond quicker as there is effectively a much smaller design section "active".

(3) A large amount of memory might be saved, this in itself will mean faster responses on very large designs.

How do I change from one section to another?

The current section can be changed in one of two ways:

(1) Use Select Section on the Power Tools Menu (as illustrated above). This will display a dialog where you can select a section visually or you can enter the location and size of a section.

(2) You can use the Preview Window (use function key <f7> or select menu option View->Design Preview). This window has a pop-up menu and features which allow a section to be selected for viewing but also a menu option "Edit This Section" can be used to switch to that section in the main window. You can simply double click the preview window to edit that section as the main design.

 

How big can a section be?

The section can be as big or as small as you like except that it cannot exceed the size of the main design. It can cross the edges of the main design. If you try to select a section that is larger that the warning size for loading designs (see above), a warning will be displayed.

 

How do I design whilst in SAS mode?

Designing in SAS mode is no different to normal mode. Whilst in SAS mode, the section is treated as a Rug by the software which means that you cannot cross the edges of the section.

 

What if I want to do several pattern fills in different sections which are all the same area?

All Pattern Fill operations are aligned relative to the edges of the whole design, not the section, therefore fills using the same pattern in different sections should repeat perfectly.

 

Can I save the design whilst in SAS mode?

Yes, you can save the design as normal whilst in SAS mode - the whole design will be saved.

 

What about things like Mirror, Redraft etc

Some features can be used only on the whole design, others on either the section or the whole design and others on only the section.

For example the following can only be used on the whole design: Specification Dialog, Redraft, Size, Rotate, Create Self Match.

The following can be used on either the main design or the section: Mirror, Wheel, Recentre.

When SAS mode is in use, the above routines will display a message similar to the following:

Can I get out of SAS Mode?

You can return to normal mode at any time by selecting Exit SAS on the Power Tools menu.

 

How does it affect shelf items?

Shelf items cannot use SAS mode. If you are editing a shelf item the "Select Section" option will be disabled.

Similarly if you copy the main design to any other shelf item, it will be converted to normal mode. In this case you will be asked whether you want to copy the whole design or just the section.

 

How does it affect Frame Plans for Carpet users?

You cannot display a frame plan whilst using SAS mode. The frame plan will be preserved however, if there is one, and may be displayed on return to normal mode.

 

How does it affect apply weaves for Jacquard users?

You cannot apply weaves whilst in SAS mode.

 

How does it affect Printing

The print layout software will always print the whole design - it does NOT simply print the current section.

 

Can I undo and and redo section changes?

You can use undo and redo normally whilst using SAS mode and the software will keep a record of all changes to all sections. This means that when doing an undo, the section may be switched back to a previous one. If you want the software to warn you when this happens you can set a preference - see the topic Undo Preferences.
