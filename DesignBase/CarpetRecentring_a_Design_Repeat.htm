Re-centering a Design Repeat

 

Note there is also a visual way of re-centering described here.

You can re-centre the design by selecting a new centre end or a new centre row (pick). If it is easier to nominate a new End 1 or Row (pick) 1 then you can choose to do that instead.

To re-centre a design repeat, select one of the four options from the Design->Re-centre sub menu:

New Centre End

New Centre Row

New End 1

New Row 1

Your cursor will change to a vertical arrow for end-related changes, or a horizontal arrow for row (pick) related changes.

Simply click on the new end or row using the mouse.

It is often useful to work with the grid on and at an increased zoom level so that you can accurately select the new end or row.
