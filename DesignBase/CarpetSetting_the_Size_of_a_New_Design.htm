Setting the Size of a New Design

 

This dialog is only displayed as a result of changing a design size in the specification dialog.

When you have set your design size, click on OK. The "Dimension" dialog will be displayed. This allows you to specify whether the design is to be re-scaled or where borders are to be added or removed.

Dimension Dialog

Example Dimension dialog.

Smart Draw

If you have an existing design you may wish to re-scale it up or down to he new size, in which case you can select Scale using Smart Draw.

Simple Re-Scale

Again, if you have an existing design, you can re-scale it using this option. This will normally be quicker than using Smart Draw.

Add/Remove Borders

Select this option to add or remove borders from the edges of the design. If changing the width you can add the new space to (or cut from) the left and/or right. If changing the length you can add the new space to (or cut from) the top and/or bottom. When adding to both, the new space is evenly divided between the two sides. Similarly when cutting from both sides, an equal sized area is cut from each of the two sides. When a size is being reduced the corresponding wording will change to for example "Cut from Left" etc.

When starting a new design, this may be unimportant, so just click on OK. The dimension window only becomes important when you alter the size of a design which you have already worked on.

You can immediately see your new design size, both in ends/rows and in design measurements, displayed in the status bar.

Colour

Note also that when increasing the design size, the current background colour will be used for any new area being added on.
