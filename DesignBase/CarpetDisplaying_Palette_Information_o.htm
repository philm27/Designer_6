Palette Information

 

Display the Palette Information Window by selecting menu option Palette->Information, or use the keyboard shortcut key F8.

The window can be left on screen as you work, and will automatically update its information as you edit your design providing the Automatic Refresh option is selected in Preferences (General).

Current Left and Right Button Colors

The "tuck" on the corner of a swatch indicates the currently selected left and right button color.

You can also select a new left or right hand button color by clicking on one of the lines with the left or right mouse button as appropriate.

Transparency

You can also double right click on a color to cause the current brush, if any, to be re-made with the selected color as the new background color. Note this cannot be done whilst using the transparency feature.

 

 

The information given is:

Pal#

A swatch of the colour and its palette number shown in brackets. Note that if a "Zero Color" is present, numbering will begin at (0) rather than (1).

Code

The code number of the shade if used

Description

The Shade Description as specified in the Shade File (i.e. Colors and yarns)

Tuft Count

Number of occurrences of the colour in the design

Usage

The percentage usage for carpet users (i.e. based on a "tuft" count)

RGB

The RGB value

Bobbin Count

The occurrences in warp direction of the colour. This indicates how many bobbins are required in the creel for this colour in each repeat widthwise (Carpet Designer only).

 

To close the information window, de-select it in the menu, or click on the close button in the top right-hand corner of the Palette Information window, or press <F8> on the keyboard.

To refresh the data in  the window you can use the keyboard shortcut <I> (letter I not "one"). This will only be necessary when Automatic Refresh is suppressed in preferences.

You can also minimise the window to the taskbar click on the minimise button in the top right-hand corner.
