Importing to Shelf

 

Importing a design to the shelf is a convenient way to load an image that can be used subsequently as a brush in the main design and by this means, several different designs from different files can be combined into one. The files do not need to be the same type.

Some of the options described here allow several shelf items to be imported at a time. In those cases, the chosen images are loaded in to consecutive locations but if there is an existing image it will not be overwritten, the import will simply continue at the next available free position.

There are now 4 ways to import an image to the shelf. You can import from....

  • Other Projects (i.e. current projects - either the main design or any shelf item)

  • Designs from a disk file

  • One or more items from a shelf in disk file including .bks files

  • Recently used list

You can also use copy and paste on the shelf.

To import a shelf item, right click an empty shelf box. The Shelf Pop-up menu will be displayed. Select the Import menu item and an import pop-up menu will be shown listing the four import types listed above.

An example of the Shelf Pop-up menu showing the Import->Shelf... option selected

From Other Project

Example of the project import dialog. The user has selected 2 shelf items from one project and 2 from another. The items in the list with a grey background are the main designs in that project. Clicking OK will import those 4 items to the shelf in the current project.

The following buttons and check boxes are available:

  • Shelf Items - Check this option to show shelf items otherwise they are hidden - only the main designs are shown.

  • Large Icons - Show only a set of large icons instead of the "Report View" as in the above example.

  • Report View - Show a "Report View" as in the above example.

  • Select All - select every item

  • Select None - de-select every item

  • OK - close the dialog and import the selected items

  • Cancel - close the dialog ignore any selections

When using this dialog, the following features are available:

  • <ctrl> key - Select an item, press and hold the <ctrl> key and select or de-select further items to retain current selections.
  • <shift> key - Select an item, press and hold <shift> key and select another item before or after the original. The two items are selected together with all items in-between.
  • Pop-up menu - right lick in the list to show a pop-up menu allowing a switch from Report View to Large Icon view.

Simply select the required project in order to insert the design. Note that it is possible to insert the main design from the current project using this method.

 

Import from a Disk File

Select the import option "Design..." and a standard dialog will be displayed allowing a file from a local hard drive or other media, or from a network location to be selected.

An example of the "Design" Import to Shelf Dialog.

Select the design in the usual way and click on Open.

 

Import Shelf...

Using this option the user can import a whole shelf or partial shelf from a disk file. After selecting the Shelf... option from the pop-up menu a normal "Open File" dialog will be displayed as in the previous method but then, after selecting a file, a further dialog will list the shelf items in that file as shown below.

Example of the Shelf Import Selector dialog - the user has clicked “Select All” and then de-selected the first item (main design). Note that the dialog will always show the main design in addition to all shelf items if the list fills the window, a scroll bar will appear. Note that you can also re-size the dialog by using the mouse to drag any one of the 4 sides.

You can use the following standard procedures:

  • Click on an item with the left mouse button to select it.

  • Hold <ctrl> key to retain existing selections and use the mouse to select or de-select additional items.

  • Select an item then hold <shift> key and click on a second item in order to select a range of items.

  • Click on the “Select All” button to select all items. You can then, optionally, de-select one or more items by holding the <ctrl> key and clicking on those items.

Simply select the required items and click on OK. The software will display the “Do you want to add all colors” message, as is normal when importing to the shelf. The imported items will be placed in consecutive locations on the shelf if possible but all existing shelf items will be retained.

Although this method is intended primarily for .prj files, you can actually select any file type by choosing the “*.* All Files” filter and selecting for example a BMP file though it will of course only contain 1 image item.

 

Open from Recent List

Any design on the "Recent" (or "History") list can be selected to be inserted onto the shelf. The history list shown in the Shelf Import option is the same one as is shown on the main File menu. It is simply a list of the most recent designs saved or loaded using the file menu. This will only import the main design from the chosen file.

 

Import using the main Shelf Menu

Select the menu item Shelf->Import. This will display the following dialog in which you should set the shelf item number (it will default to he first empty position) then click on OK, the above "File Open" dialog will then be displayed.

 

Import using Keyboard Shortcut

Hold down the <shift> key and click with the left mouse button on an empty position on the shelf. An open file dialog will be displayed.

Palette Control

When an item is added to the shelf from any of the sources described above, the software must convert it to use the current project palette, and/or, must extend the current palette to include the colors from the new image.

This can be done in one of two different ways. The following dialog will be displayed allowing the user to select the required method.

 

Click on YES or NO to proceed or CANCEL to abandon the import:

    • YES - If you want to force the colors in the new image to be added to the palette whether or not the RGB values are already in the existing Palette then select YES. You can always merge them later if you wish. Choosing YES keeps the colors completely separate allowing you to change the new colors independently of existing design colors.

    • NO - If you want to use existing palette colors where they are the same RGB value, select NO. Any RGB values in the new image which are not in the existing palette will be added. Using this option will keep the palette size to a minimum.*

* Note that you can specify a tolerance when matching colors and you can control whether a color matches another one if they have the same shade code using a preference. (Menu item Shelf->Properties on the Shade Matching tab).

Quality

You may also see the following dialog when importing to the shelf:

Example of the dialog shown when the imported end and rows/picks per inch is different to the main design. In this case the main design is 7x8 and the incoming design is 8x8.4

You can choose to:

  • Redraft the imported design using Smart Draw to make it the same as the current project

  • Do a normal re-draft

  • Simply modify figures which may result in some slight distortion of the incoming image

  • Leave the quality as it is. This option is useful when the intention is to place the incoming image using Place/Fill optionally at an angle or to place it using Regions as an Outfill etc

  • You can check the box "Do not Ask Again" and the same selection will be used next time there is a shelf  import.

Click OK to proceed with the import.
