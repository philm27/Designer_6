History List

 

The History List can be found at the bottom of the File menu. It is a list of the last 10 designs or projects that you have either saved to a folder or opened from a folder. The list also tells you which folders the designs are saved in. The most recently used design file will be at the top of the list.

If you want to recall a design from the History List, simply select it from the menu in the normal way.
