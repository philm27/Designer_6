Removing all Windows

 

It is possible to remove all the secondary windows from your screen (toolbox , toolbar , shelves , status bar , palette etc.) so that you can view/work on your design without obstructions! The menu bar will remain on screen. Simply select menu option View->All Windows Â– the windows will disappear. Recover the windows by reselecting the same menu option a second time.

Alternatively, use the keyboard shortcut key F12 to hide or redisplay windows.
