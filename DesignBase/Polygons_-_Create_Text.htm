Polygons - Create Text

You can use this feature at any time to create text using any of the available fonts. You can choose to add the polygons created to the list of those that already exist, or to simply render them into the current design.

All polygons, if retained, are stored in the design file.

As with all polygons - they do not have to be rendered into the current design. They can simply be used as Guide Lines for drawing.

Also, all polygons, once created, can be re-sized, moved, rotated dynamically and can also be copied using the polygon toolbar and menu.

The shapes created complete text will be grouped together, with each character being a separate group and can optionally be given a name. The user can ungroup individual characters if necessary either using polygon mode or using the Polygon List.

To use this feature select menu option Polygons->New->Text, or select the "New Text" button on the Polygon Tool bar.

Text

Type the text into this box any printable characters can be used.

Font

Either type the font name or click on the Browse button to display the standard font selector.

Size

You can specify the font size in any of the standard units. Simply choose the units from the drop down list (inches in the example above). Then key in the required size. The size in this case is from the bottom edge of the lowest point in any character to the top edge of the highest point. This figure is NOT the rotated height. In the example above, the text height will be 23.5 inches regardless of the angle of rotation.

Using the Resulting Polygons

All standard features of polygons are available for the text.

To enter polygon mode you can select Polygon Mode on the main menu or click on the tool bar button.

Then:

Use Move to move the text to another location.

Use Resize to re-size it interactively.

Use Rotate to rotate it interactively.

At any stage you can draw it normally (as specified in the properties), or draw the outline or fill the shapes with a solid color or a brush.

For more information about all the ways to edit polygons see the topic "Polygons - Editing"
