Shade Library Introduction

The Shade Library allows users to access shades, Color Ranges and Collections in the Colors and Yarns database and/or to access Plain shades in a text file held anywhere in the system or network.

 

Jacquard Designer

For Jacquard Designer users, when using the Yarn Tab, the thickness, ply and twist of the yarns used will affect the cloth images produced. Also, of course, the yarn design will affect the cloth image for all types for example, Mixtures, Twists, Space dyed etc.

Also, for Jacquard Designer users, the Ranges and Colorways option will cause two tabs to be displayed - a Warp Range and Weft Range tab.

Carpet Designer and Textile Designer

For Carpet Designer users, the current version will only use an average color in prints and screen images.

To Access Shade Library

Select menu option "Palette->Shade File" or use the keyboard shortcut <shift>+S.

Text Files

A set of different text files can easily be accessed and if "Text File" is selected on the Library Tab the contents of the appropriate text file will replace the normal colors in the Plain Shades tab.

The selection can be changed as many times as is necessary.

Of course when a design is loaded it will use the current settings to get the latest RGB value and shade description for each yarn/shade code so it may not find a shade if the settings are now different. However, since the RGB value and shade description are stored in the iff or prj file this may not be important. Again, this only applies to Plain shades.

Note any Yarns, Mixtures, Twists etc used in the design which are based on a Plain Shade will always refer to the database, never to a text file.

These text files are initially created by means of an Export from Colors and Yarns. They can be modified, with care, using a text editor.

Note however that shades selected from a text file can be used in the design but they cannot be used in a Range.
