Create New Shelf Item

 

In addition to copying brushes and files onto the shelf, it is also possible to start a new design. It will have default specification details, but will of course use the palette of the current project.

This is a useful function if you are creating a design with many components, working with a common colour palette. For each new component you can start a new design on the shelf and save all the components together as a PRJ  remember to save your work frequently.

Empty Shelf Item

To start a new design in an empty position on the shelf:

  1. Click with the right-hand mouse button on an empty shelf item. The Pop-up shelf menu will be displayed.

  2. Select New from the menu. The default Specification dialog will be displayed

  3. Edit the size or any other details.

  4. Click on the OK button

The new shelf item will be created at the specified size and will become the edit box  for you to start designing. The shelf item should be outlined in red indicating that it is the "current design"

Used Shelf Item

You can replace a design on the shelf with a new one  follow the above steps, clicking on an unwanted shelf item rather than an empty one. You will be asked to confirm that you do not need the existing shelf item.

Using the Main Menu

You can also start a new shelf item using the main menu. Select menu item Shelf->New. A dialog will be displayed in which you can specify the shelf item number.
