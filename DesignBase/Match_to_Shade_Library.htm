Match to Shade Library

Match All Palette Entries

The menu option Palette->Match All Shades allows the user to quickly match or rematch all shades to the current shade file.

This can be useful if you have developed a design from scratch using RGB values, possibly in Third Party software.

To use this feature click on the menu option Palette->Match All Shades. The software will look at the current list of Plain shades, whether that is in the C+Y database or possibly an external text file, and will match the RGB values in the design those in the shade file.

After the process is complete, the software will display the following dialog before the new shades are set into the design palette:

Example of the Shade Matching "Approve Changes" dialog. In his case, there is a subtle difference between the Before and After images which may or may not be acceptable - click on OK to accept or Cancel to reject.

After accepting the changes,  the shade names and descriptions will be set into the palette. These changes can, of course, be undone using the Palette Undo.

 

Match Individual Palette Entries

Using the pop-up palette menu you can match an individual shade in the current palette. To display the pop-up menu either hold <ctrl> while right-clicking on a palette color, or, right click in an empty area to one side of the color (usually below for a horizontal palette, to the right for a vertical palette).

Select "Match to Plain Shades" to match an individual color.

(You can also select Match All Shades" on this menu).

As for the main menu option you can undo any changes using Palette Undo.
