Wacom Tablet / Digitizer Pressure Settings

 

If you work with a pen and digitizer rather than a mouse, you can use Pen Pressure Control in drawing functions. By applying greater or less pressure with the pen on the digitizer, your brush thickness will increase or decrease  rather like using a paintbrush.

You are unlikely to want to have this facility selected as a default  but will just use it for certain styles of design-work.

To set the Digitizer Pressure Settings, follow these steps:-

 

  1. Select menu option Edit->Preferences to display the Preferences Dialog.

  2. Select the Pressure tab to reveal the following window.

You can click on the image below to find out more.

 

 

Switching the feature On or Off

Enable Digitizer Pen Pressure controls if the pen pressure feature is used. If it is not selected all the following controls are disabled.

 

Pressure Threshold

You can set the minimum pressure that will count as a button press. This value is based on a scale of 0 (lightest pressure) to 100 (heaviest) and determines how much pressure must be exhorted on the pen to count as a button click. If this value is too high (100) you could find it impossible to exhort enough pressure to register the control size and thickness in the other settings.

Experiment with different figures until you find a level thats right for you.

 

Other Settings

The remaining options allow you specify whether you want the pen pressure to control thickness, size or speed of various drawing tools. The greater the pressure exerted above the minimum pressure value specified the thicker, larger or faster the controlled value will be.

Pressures below the threshold specified above will be treated as a normal drawing operation (i.e. the same as a normal mouse drawing operation).
