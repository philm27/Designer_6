Catch Cords & Running Cords

 

This allows an alternative style of line drawing as illustrated below. It is switched on by pressing <ctrl> on the keyboard whilst drawing a line.

Normal Line - <ctrl> key NOT pressed

Same Line drawn left to right with <ctrl> key pressed

Same Line drawn right to left with <ctrl> key pressed

 

There may be small variations depending on which direction the line is drawn. In the example above there is a slight difference when drawing from right to left as opposed to left to right.

Note that it is possible to switch Catch cord lines on permanently using the Line Properties dialog.

Note that using the <ctrl> key applies also to circles and ellipses and bendy (curved) lines when drawn 1 pixel wide.
