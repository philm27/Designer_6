Increasing/Decreasing Design Size

 

Design size can be modified in any of the following ways...

Method

Description

Change size in the specification

Add or remove borders, or re-scale design after Specification change.

Insert or delete ends/rows

Increase or decrease the width or length of the design using a straight horizontal or vertical line to define insert or delete point.

Insert or delete special

Increase or decrease the width or length using a "crazy" line.

Duplicate ends/rows

Increasing the width or length by duplicating all or part of the design.

Resize

Alters the width and/or length of a design by stretching or compressing in one or both directions i.e. making the whole thing bigger or smaller.

Redraft

This function actually keeps the design size the same in terms of physical woven size but will increase or decrease the number of ends and/or rows/picks in the design.
