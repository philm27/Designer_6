Creating/Editing a Custom Colour Palette

 

When you create a new design it will be set up with the default colour palette . For most of your design work you will want to create a customised palette.

Access the Palette Change window (in order to customise your palette) using one of these ways:-

 

  • Select Palette Change from the Palette drop-down menu

  • Double click on any palette colour in the palette window

  • Click on the icon, which is located in the toolbox and toolbar

  • Use the keyboard shortcut key F6

 

This is the palette change window. Find out how to edit your palette by clicking on the functions within it:-

 

 

When you select buttons Swap, Move, Range, Copy, Delete or Merge, you will notice that the wording on the button changes to STOP. This means that you can continue to use these functions as many times as you wish – they aren’t just a single operation like Restore or Close. When you have completed your editing with one of these functions, re-click on the button to end the operation – you will notice that the button wording returns. Alternatively, click on a different button.

Also, with these buttons (excluding Delete) you will be asked to first select the palette colour you wish to change FROM, and then the colour you wish to change TO.

If you have unused colours in your colour palette, it is possible to delete them all at once, rather than one at a time as in the ‘palette change’ window. Simply select REMOVE UNUSED COLOURS from the PALETTE drop-down menu.
