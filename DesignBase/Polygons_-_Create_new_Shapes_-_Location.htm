Polygons - Create new Shapes - Location

This is the Location tab on the Create New Shape Property Sheet. Use it to specify either the containing rectangle of the shape, or, the centre of the shape.

For more general information see the topic Polygons - Standard Shapes.

Example showing the Location tab. The user has chosen to centre the figure in the design.

By Containing Rectangle

Select one of the four options Top Left, Bottom Left, Bottom Right, Top Right then specify that corner location in the X and Y box. All locations are relative to the bottom left corner of the design.

By Centre Point

If you wish to locate the figure by its centre, select "Centre" and specify the X and Y coordinate of the centre of the figure. Or, in order to quickly centre the figure in the design, select "Centre in Design". Selecting either of these options will de-select the previous "By Containing Rectangle" option.

Note that the centre used here is the centre of the containing rectangle thus for some figures such as a 5 sided figure, the centre is not the same as the point where the radial lines meet.
