Multi-Color Palette Changes

Using the standard palette change dialog, it is possible to modify several colors simultaneously.

An example of the dialog when the user is in HSV mode and 5 reds from a large palette have been selected to be modified using the sliding controls.

To modify several colors simultaneously:

  1. Click on either All, or Selected. In the former case you need not display the palette list.

  2. Select the required colors in the list if necessary - you may find the "Select All" and "Deselect All" buttons useful when doing this. This is NOT necessary if you have chosen the "All" option. As usual, you can hold <ctrl> to select several individual colors or <shift> to select the first and last in a block of consecutive colors.

  3. The sliders will show the average of the chosen colors.

  4. It is recommended that you select HSV when doing multi-color changes but you can use either RGB or HSV and you may switch from RGB to HSV at any time.

  5. Use the sliders as necessary to modify the chosen colors.

Changes in HSV mode

Hue - the software moves each palette color by the same amount of Hue i.e. if the average hue changes by +0.25, then the hue of each color is increased by 0.25.

Saturation and Value are changed on a percentage basis i.e. if the average saturation is increased by 10% then each saturation is increased by 10%.

Changes in RGB mode

The red green and blue values are changed on a percentage basis i.e. if the average red is increased by 10% then each red value is also increased by 10%.
