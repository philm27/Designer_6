Polygons - Properties - General

You can use this feature at any time to modify the settings including color of any polygon.

To display the Property Sheet, first select a polygon by clicking on one of it's handles then select menu option Polygons->Properties, or select the "Properties" button on the Polygon Tool bar.

Example showing settings for a rotated rectangle which is central in the design. Guide Lines are drawn in green and the polygon is not at present locked in position.

Name

You can enter or modify the polygon name here. The name is only used for convenience when looking at long lists.

Guide Line

If this is checked, the polygon will appear over the top of the main design when View->Guide Lines is selected. In Polygon Mode it will appear in the selected color and can be edited like any normal polygon.

By clicking on the color swatch, you can change the line color of the polygon. (See the topic Color Selector). (This is not the same as the color that the software will use to render the polygon into the design). Note that you can change the line color whether or not the polygon is classed as a Guide Line.

Locked

Check this box to protect the polygon against dynamic edits of any kind using the mouse. You can unlock it again using the properties or the polygon menu.

Preview

The polygon preview, which is present on each page, will always show the current shape as defined on the tabs in this property sheet. The four registration marks (if shown) indicate the edges of the repeat.
