Inserting/Deleting Ends/Rows (Picks)

 

Use this facility if you want to increase or decrease your design size by inserting or deleting ends or rows at specific positions in the design e.g. to increase or decrease the gap between figures in the artwork.

 

Inserting One End or Row

Select menu option

Design->Insert One->End or

Design->Insert One->Row

The cursor will change to an arrow  pointing vertically to insert ends, horizontally to insert rows. Every time you click on your design with your mouse, an end or row will be inserted at that point in your current foreground or background colour (use the left-hand mouse button to insert an end or row in the foreground colour, or the right-hand button for your background colour). This function will remain active until you select a new function  letting you insert as many single ends/rows as you need.

Insert Many Ends or Rows

Select menu option

Design->Insert Many->Ends or

Design->Insert Many->Rows

In this case a dialog will appear allowing you to specify the number of ends or rows to be inserted. Having entered this information (using either the up and down arrows or by swiping the default number and typing the new one), the function works in the same way as for single ends/rows.

You can insert the specified number of ends as many times as you like in different places within the design. If you want to insert using a different number of ends or rows you must select the menu option again in order to display the dialog below.

This function will remain active until you select a new function.

 

Deleting Ends/Rows

Select menu option

Design->Delete->Ends or

Design->Delete->Rows

 

The cursor will change to an upward or horizontal pointing arrow. Click and Drag this arrow across the ends or rows to be deleted, greying out the area. When you release your mouse, you will be asked to confirm the deletion  unless you just delete a single end/row, in which case it will be deleted immediately. This function will remain active until you select a new function letting you delete as many ends/rows as you need.

You can immediately see your new design size, both in ends and rows(picks) and in design measurements, displayed in the status bar .
