Drawing Polygons

To draw a new polygon:

If not already in Polygon Mode, click on the Polygon Mode button on the Polygons Tool bar, or, select menu option Polygon->Polygon  Mode.

  1. Click on the "New Polygon" button, or, select menu option Polygon->New->Polygon, or, press N on the keyboard.

  2. Select either Straight or Curved Lines using the buttons or menu options

  3. Using the left mouse button click on each point that the line should "visit" - you can edit them later if you do not click on the correct position.

  4. To close a figure you can double-click instead of single click when you click on the final point and the software will automatically join that point to the first one.

  5. Alternatively, you can close a figure, if you have already drawn the final point, by right clicking that final point and selecting "Close Polygon".

  6. To finish the figure without closing it, select one of the edit functions to edit points, re-size, move or rotate it as required. r, start a new figure as at 1 above.

After this, the software will automatically go into "New Polygon" Mode again allowing another new figure to drawn immediately if required.

When drawing a series of shapes it can be useful to use the keyboard shortcut N to start each new figure.

 

Drawing Vertical and Horizontal Lines

While drawing, the <shift> key can be held to make another point at exactly the same horizontal or vertical location. This allows exact rectangles to be drawn, or, a figure similar to an ellipse when drawing curves.

Examples of this are shown below.

 

Examples

Some simple examples of different shapes drawn using straight lines or curves.

In each case the basic procedure is as described above.

Example A

A simple triangle - select Straight Lines then click on first two points then double click on third.

Example B

A simple irregular quadrilateral  - select Straight Lines then click on first three points then double click on fourth.

Example C

A simple rectangle  - select Straight Lines then hold <shift> key and click on first three points then double click on fourth then release <shift> key.

Example D

A shape similar to an ellipse - this is created in the same way as Example C above except that curves are used. So, select Curved Lines then hold <shift> key and click on first three points then double click on fourth then release <shift> key.

Note, however,  that mathematically correct Ellipses can be drawn using the Standard Shape feature.
