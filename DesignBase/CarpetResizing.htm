Re-size

 

If you need to increase the physical design size, use the re-size facility. This feature will "blow-up" the design size or reduce it as opposed to adding on additional border space or cropping it.

If you need to change the quality of your design whilst keeping it the same physical size and proportions, then use re-draft rather than re-size.

To use the dialog, select the required units if not already displayed, then enter the new size and Click OK. If you click on Cancel, no changes will be made to the design size.

The original size is shown for reference and the new size is shown in "points", feet and inches and Metric units at the bottom, also for reference.

Units

Select the required units from the 4 options list i.e. Ends/Rows, Feet and Inches, Metres or %.

In the case of %, a value of 100 is "no change" and 110 for example would be 10% bigger and 90 would be 10% smaller.

If you switch between different units after having entered new size values, they will be converted to the new units.

All values except for ends, rows and feet can be entered to 3 decimal places.

Smart Draw

Smart Draw will attempt to smooth out jagged edges when increasing the design size. It has no effect when reducing the design size. Note however that the result is of course computer-generated and may still need a designers' eye to perfect it. For more information about Smart Draw see the topic Redraft.

Insert Blank Lines

Insert Blank Lines does just what it says, the lines being inserted in the background palette colour. If, for example, you re-size your design from 252 x 252 to 252 x 288, your re-sized design will have 36 equally spaced horizontal rows inserted into it. Also, in this example, the new image will start with 7 original rows then an inserted row. This will be repeated 36 times.

This has no effect if the length of the design is being reduced, and also note that Smart Draw cannot be used at the same time and will be greyed out. Similarly, if Blank Lines is selected, Smart Draw cannot be used and will be greyed out.

Percentage Option

Select this feature if you want to enter the new design size as a percentage of the old.

In this case you would enter for example 200% in order to double the design size or 50% to halve it etc..

Keep Proportions

Select this option in order to automatically decrease or increase rows by the same percentage when editing ends and vice versa.

 

Double Design Size

This main menu option (Design->Size->Double) will automatically blow-up the design size to double its original size in both width and length without needing to use the dialog.

Smart Draw will always be used if applicable.

 

Half Design Size

This main menu option (Design->Size->Half) will automatically re-size the design to half its original size.
