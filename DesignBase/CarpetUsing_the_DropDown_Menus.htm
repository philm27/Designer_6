Using Menus

 

The drop-down menus provide a means of accessing the powerful functionality of the design software. To find help on a specific menu option you should look for that subject in the help-file index.

In order to select a menu option from a drop-down menu, simply click on the desired menu title on the menu-bar using the left hand mouse button.

Once you have clicked on the menu bar you can move left and right on the menu bar to display different menus and you can also move up and down any menu and if there are sub-menus as in the example below, you can move sideways to select an option at a lower level.

If there is an arrow to the right of a menu item, this means that there is a sub-menu for that line. Simply hovering the mouse pointer over an item will cause its sub-menu to be displayed.

To make a menu disappear in the event that you do not want to select anything simply click anywhere in the menu bar or anywhere on the screen in a "blank" area (i.e. not in the design and not on a button nor shelf item).

The following example shows how a half-drop match is applied to a design. First the Design menu is selected by clicking on the menu title in the menu bar, then moving down the menu to the Match option will cause the sub-menu to be displayed and finally moving sideways and down to "Half Drop" and clicking the left mouse button to select it will cause the menu to disappear and the half drop option to be immediately applied to the design.

 

 

Using the Keyboard

You can use the <ALT> key to activate the menu. After pressing the <ALT> key you should be able to see that the first menu option (File) is "depressed" and all menu titles have a keyboard shortcut underlined as, in fact, shown in the image above. You can use the cursor keys to navigate right, left up or down menus. Alternatively you can use the keyboard shortcut as indicated by the underlined characters (if any) on menus and menu items.

For example, you can select "Design Specification" by typing the following sequence of keys:

<ALT> D S <enter>

Pressing the <enter> key activates the menu option selected.

Note that as an alternative to the drop-down menus, many of the applications can be quickly accessed using keyboard shortcuts.
