Reduce To Palette

This option will reduce the number of colors in the image, However,  unlike the normal Color Reduce, a particular palette can be used for the final image.

To do a Reduce To Palette, use the main menu option Palette->Reduce To Palette. The following dialog will be displayed.

Reduce To Palette Dialog

In this example, colors 2 and 3 have been copied from the current palette and the remainder have been dragged from the Shade Library. Note that the palette can be applied to an current design.

Constructing a Palette

The palette listed is independent of the current design palette and is stored separately to all current projects. The palette is retained after the dialog is closed and can be used again later. Thus, it could be “applied” to several different designs.

There are several ways in which the palette can be created and modified as described below...

Drag from a Palette

You can drag and drop colors from a current palette (not necessarily the design you want to reduce),

Drag from Palette Information Window

You can drag from a Palette Information window and in this case you can multi-select in the standard way (i.e., hold ctrl key while selecting or hold shift key and select a range). You can then drag several selected colors at a time.

Drag from the Shade Library

You can drag any shade from a Shade Library page including the Range page and Collection page. When dragging from the Range tab in the Shade Library, you can also drag an entire colorway by dragging the colorway code at the top of the list.

Drag from the Colorway Update dialog

This is probably of less interest but you can also drag shades from the Colorway Update dialog. This may be more convenient in some cases.

Open any current Palette file or Design file

Clicking on the +Open button will display a dialog allowing any .PAL file or design file to be opened and the palette colors added to the list. (Note they are added – they do not replace).

Add Current Palette

By clicking on the +Current, the colors from the current palette will be added. As above, the colors do not replace the list – they add to it.

Pick+

Click on the "Pick+" button to select one or more colors from the design or from a palette. You can change designs while doing this. You can "lasso" a set of pixels in a design and the software will add the average of those pixels.

Add Current Palette

By clicking on the +Current, the colors from the current palette will be added. As above, the colors do not replace the list – they add to it.

Re-Order

The list can be re-ordered by dragging and dropping colors to new positions. You can also multi-select when doing this.

Deleting Colors

You can delete colors by selecting them and using the <delete> key on the keyboard. Again, multi-select can be used.

Note that due to the implementation of Drag and Drop across all these windows, it is possible to drag colors FROM this window to any palette or to a Colorway Update dialog.

Saving a Palette

If you wish to save the palette you have created, simply click on the "Save..." button and you can save it as a .pal file in the normal way.

Doing the Reduce

When the palette is complete click on the Reduce button. After processing the Before and After images will be displayed as for the normal color reduce so that the change can be accepted or cancelled.

After a cancel, you can further modify the palette as described above and try again as many times as necessary.

Approve Changes Dialog

This will be displayed after a color reduction.

An example of a reduction from 24 colors to 16. The user can click on OK to approve the change or Cancel to return to the Color Reduce Dialog above.

Undo

Note that even after clicking on OK in the Approve Changes dialog, the user can Undo in the normal way using Design Undo (not palette undo).
