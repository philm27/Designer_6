*Loading Design Files and Palette Size

Note that if you try and open a 24 bit image as a design file (i.e. not in a print layout) the software will ask if you wish to load as a 24 bit image or reduce it to an 8 bit image i.e. the number of colors will be reduced to 255 or less. If the image actually only contains 255 or less colors then no changes to the image are necessary, however, if more than 255 colors are used then the software will use an optimisation routine to create a suitable palette and will change pixel colors where necessary.

If you try to load an 8 bit image containing 256 colors, the software will ask if you wish to reduce it to 255 (i.e. 1 less) as index 255 is now a reserved colour.

24 bit files may be imported into print layouts as either logos or as designs though note that you will not be able to override colors.

** EP and BIN file export feature

File types .EP and .BIN etc may be available through the Export feature for creation of special files for production equipment such as looms or paper punch machines. This facility is an optional extra and may not be enabled on your version of the software. Contact APSO for further details.

*** .DES files

There are now 3 known industry standard files that have a .DES file extension. As well as the two mentioned above there is the Eltex SELVCAM jacquard system. The software will distinguish between these 3 types by examining the file content. At the present time it is not possible to load the Eltex DES file into the design software - it can however be loaded into the Name Edge software.
