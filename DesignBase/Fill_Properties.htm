Fill across edges of Repeat

The first page permits “Fill across edges of Repeat” to be selected this works with all repeat types and will allow a brush fill to match across the corners of a repeat in a fill operation. This will also apply to shapes that cross more than one repeat, though it will not always be possible to make them match and a warning message will be displayed in that case.

Boundary Colors

The second page, shown here on the right, allows colors to be specified that will be bounds colors for a “Fill to Bounds” operation – this is achieved by using Flood Fill and holding the <ctrl> key when clicking in a (possibly multi-colored) area in the design.
