Generate Random Pattern 2 Color

This feature allows the user to create a random pattern.

You may choose a dot size and a number of colors which may be mixed in any proportion.

If the dot size width or length exceeds one you may choose to have the dots aligned.

To use this function the menu option Power Tools->Generate Random Pattern->2-Color.

Color 1 and 2

Use the drop down lists to select the required colors

Percentage

Type a percentage or use the Up/Down arrows to adjust it. The figure is the percentage of Color 1. See below for more examples.

Dot Size

Click on the Dot Size tab and either type a dot size or use the Up/Down arrows to modify the values. You can use values up to 99 at present.

Alignment Checked

If this is checked, then the dots will be aligned according to their size. For example if the dot width is 8 and it is aligned, then the dots will start on ends 1, 9, 17...etc. If the design width is not a multiple of the dot width, then there will be a set of partial dots on the right hand side of the resulting image. Similarly with length of course but in this case any partial dots will be at the top of the image.

Alignment Not Checked

If alignment is not checked, then the dots will be aligned randomly. This may result in some overlapping. It is possible to align in one axis but not in the other as shown in one of the examples below.

OK Button

When this is clicked, the software will generate the image. You can click it several times in succession to try and get an ideal result. If you decide to go back to a previous image you can click the Undo button.

Undo

Use this to go back to a previous image. You do not need to click on Undo in order to generate a new pattern to replace the previous one as the prevous one will simple be overwritten anyway. However, this operates like a normal Undo, so you can go back several steps whether they are drawing operations or Random Patterns.

Close

Use this to close the Property Sheet. If you later decide to undo the Pattern you have created you can use the normal undo procedure.
