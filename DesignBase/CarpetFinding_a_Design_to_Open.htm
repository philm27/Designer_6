Open Design

 

Before opening (loading) a design file, make sure that the design currently on screen is saved (if you want to keep it). If you want to be able to return to your current design later in the day without reopening it from a folder, then open the design in a new project .

Access the folders containing designs, in one of the following ways:-

 

  • Click on the Open button on the toolbar

  • Or, select Open... from the File Menu

  • Use the keyboard shortcut <Ctrl> + O

  • Click with the right hand mouse button outside the normal working area of your design. This will bring up a Pop-up menu. Select Open on the Pop-up menu.

 

An Open window will appear on screen and will look something like this:

Example Open File Dialog showing .iff and .prj files with medium sized icons

Find the folder within which your file has previously been saved. To do this, work your way through the folder family tree. Go to the parent folder by clicking on the button, or to the offspring folder by double-clicking on a folder within the currently selected folder.

When you reach the required folder, make sure that you have selected the file types that you want to view from the drop-down list at the bottom right. You can also select "All Files" if necessary.

To Open a File

Select the file that you want to open from the list  this will copy the file name to the File Name box.

Click Open to load that file. If the load is successful the dialog will be closed and the design displayed.

Alternatively, you can simply double click the file name in the list.

File Type

When you reach the required folder, Make sure that you have selected the file types that you want to view from the 'Files of Type' drop down list since only the selected file types will be listed.

In the above example, only IFF files of type .iff and .prj are being displayed.

PAT Files and CPD files

Note that when loading a PAT file using CarpetDesigner, if there is a corresponding CPD file then the frame plan will be loaded from the CPD file.

For example if there is a file called AB1234.PAT and AB1234.CPD in the same directory then when AB1234.PAT is opened, the software will automatically open the file AB1234.CPD and load the frame plan from it.

In JacquardDesigner, the .CPD file is ignored and the artwork loaded from the PAT file.

PAL files

These files are palette files and can only be loaded using the menu option Palette->Load. See the topic Saving/Loading Palettes as PAL Files .

Dialog Size

The whole dialog can be resized by positioning the cursor over an edge (or a corner) of the dialog and dragging it left, right, up or down as appropriate.

Pane Size

All 3 panes - Favourites/Navigation, file list and preview can be resized by positioning the cursor over the vertical line between them and dragging it left or right.

Favourites (or Quick Access)

You can easily add to the favourites list (now known as QUick Access in Windows 10) by right clicking the Favourites heading to display a Pop-up menu then selecting "Add current location to Favourites" (now shown as "Pin current folder to Quick Access").
