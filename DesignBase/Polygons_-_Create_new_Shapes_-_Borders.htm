Polygons - Create new Shapes - Borders

This is the Borders tab on the Create New Shape Property Sheet. Use it to control the number of, depth and color of borders.

For more general information see the topic Polygons - Standard Shapes and for some examples of different shapes see the topic Shapes.

 

An example of the Borders tab. The border depths are 1, 2, 6, 2 inches (there may be more) and they will be rendered using palette colors 1, 2, 6 and 2. Some guide lines are yellow and some are red.

 

Include Border Lines

You can easily switch off the border lines and just draw the basic shape by unchecking this box. This allows the basic shape to be drawn without having to delete all the lines in the borders list.

The same shape as above but with the "Include Border Lines" box unchecked

Guide

This shows the guide line color. You can click in this column to change it using the Color Selector. These colors are not design colors so can be any RGB value.

Line

This is the Line color for the edge of the polygon. If it is simply shown as an "x" in a white background, it is not being drawn. To allocate a color to the line, click in the color swatch and select a design color from the palette. Also, in this case you may select "No Color" in that dialog.

Fill

This is the fill color for the border and works in the same way as the Line color above i.e. an "x" indicates no fill and use may click in the swatch to select a design color.

Number of Repeats (#)

This is the number of times that the border is to be repeated. Since these repeats will all have the same line color and fill color, it is normally only used when creating Guide Lines and/or when rendering the Line.

Depth

This is the depth of  the border. It can be specified in any of the normal set of units. In the example above, the borders have depths of 1, 2 6 and 2 inches.

Units

This column specifies the units of the border depth on each line. To change it - click in the box and a drop down list will appear. Each line can use different units if necessary.

Move Up Button

Select this to move the current row up one place effectively swapping the position of the selected row with the one above it

Move Down Button

This will move the current row down one place effectively swapping the position of the selected row with the one below it

An example of the Convert Units dialog - the user has chosen mm from the drop down list

Convert All Button

This allows the units of all items in the list to be modified for example from inches to millimetres. A simple dialog is displayed, as shown above, allowing the new units to be chosen from a standard drop down list.

 

An example of the Borders tab in which the user has right clicked line 2 to access the pop-up menu

Pop-up Menu

Right click on any row in the grid to access the Pop-up menu. This contains a number of options for manipulating the list.

Add Row will add a new row to the end of the list. It will initially be set to a duplicate of what was previously the final row.

Insert Row will insert a new row at the current location, it will initially be set to a duplicate of the selected row.

Delete Row will delete the selected row.

Move Up will move the current row up one place effectively swapping the position of the selected row with the one above it

Move Down will move the current row down one place effectively swapping the position of the selected row with the one below it
