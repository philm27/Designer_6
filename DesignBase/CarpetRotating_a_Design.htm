Rotating a Design

 

The entire design can be rotated to any angle.

To use this option select menu option Design->Rotate. The following dialog will be displayed. You can click on the images below to find out more.

Design Rotate dialog with a 30 degree clockwise rotation selected

 

Main Rotation Options

You can click on any of the six main buttons (as listed below) to select the required rotation option.

  1. Rotate 90 Degrees Clockwise

  2. Rotate 90 Degrees Anti-clockwise

  3. Rotate 45 Degrees Clockwise

  4. Rotate 45 Degrees Anti-clockwise

  5. Rotate Any Angle Clockwise

  6. Rotate Any Angle Anti-clockwise

For the "Rotate Any Angle" options you should also enter one of the following:

  • The degrees to be rotated, or,
  • The number of ends by which a top corners is moved, or,
  • The number of rows by which a bottom corner is to be lifted.

For Rotate 90 degree options you can select the "Keep Quality" or "Rotate Quality" option (see below).

Rotate Any

For the "Rotate Any Angle" options, enter your required angle of rotation in degrees or alternatively enter the number of ends or rows by which you want to move the corners.

Keep Quality/Rotate Quality

For 90 degree turns you can also choose to "keep quality" or to "rotate quality". The latter can be useful if you have created a design on its side with the quality (setting) reversed and you wish to rotate the design by 90 degrees back to the correct orientation it will have in production.

If you choose to keep quality the ends and rows (picks) per inch or cm etc will remain the same and the design will be redrafted in order to keep it the same physical rotated size.

The "Rotate Quality" option is only available for 90 degree turns.

Rotate Button

Select this to proceed with the rotation.

Undo Button

Select this to undo the last drawing operation. Note that this will undo any drawing operation not just design rotations just like the main Undo button in the toolbox.

Close Button

Click on this button to close the dialog.

 

Example

Design rotated by 45 degrees

Note that in cases other than a 90 degree rotation, the software will make the resulting design the minimum size possible in order to hold the rotated design and will fill in the additional space using the current background colour as shown in this example. You may be able to use this to your advantage in some cases by selecting the colour you want to use for the additional space before using the rotate option.
