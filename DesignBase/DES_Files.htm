DES Files

 

DES files are an alternative to .PAT files and may be produced by some textile companies.

Note that there is another .DES file layout - the Hardaker .DES format which is designed for punched card production and also sampling machines. The software will distinguish the two by examining the content.

The software can be made to update the Colors and Yarns database by extracting shade information from the .DES file. The program will report on the number of updates after loading the design file. This feature is switched on by selecting the "Replace missing shades" option in Preferences (Shade Library tab).
