Palette Preferences

 

Select a palette default by going to menu item Edit->Preferences. You should then select the Palette tab.

 

Default to Stencil On

If this is checked, the software will always display the Stencil buttons in the palette window for a new design or when a design file is opened.

Default to Transparency On

If this is checked, the software will always display the Transparency buttons in the palette window for a new design or when a design file is opened.

Delete Unused Colors on Load

Any design that has been saved to a folder with unused colours in its palette will be loaded with the used colours only.

Default Palette

This is the palette that is loaded when you create a new design. There are fifteen palettes to choose from.

Zero Colour

If this is checked then all new designs will have a "Zero Color". This is always the first colour in the palette and is intended to be used for any areas in a pile fabric or carpet design that is simply backing.

Also, if checked, all .PAT files, when loaded, will have a zero color whether it is used or not.

Include in Usage

This refers only to the zero color if there is one. If checked, the zero color is included in the Usage%. If not checked then the % figures are calculated on the design area excluding the zero color.

Show Palette Numbers

This check box indicates whether the palette window is to show palette numbers. It is recommended that this is always checked.

Text

On some monitors with very high resolution, the palette numbers in the palette window can be rather small. Modify the font size and/or type here as necessary. A value in the range 7-12 is normally sufficient. You can click on the "Browse" ("...") button to select a font and point size.
