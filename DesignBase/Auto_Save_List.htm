Auto Save List

The Auto Save list is displayed by selecting menu option File->Auto Save Files List. A dialog similar to the following example will be displayed.

For further information on this topic, use the links at the bottom of this page.

Using the Display

You can click on the "+" signs on the left hand side to open up a list of the individual .prjasf files in the Auto Save Folder. The detailed list will show the file name and the date and time it was saved.

To close up a list back to the project name, click on the "-" sign.

In the above example the user clicked on the + sign next to a project to cause a list of 3 files to be displayed.

Open File

If you have lost a file or simply want to go back to an older version in the Auto Save Folder, you can either click on a file to select it (does not need to be "checked"), and click on the Open button.

Alternatively just double click on a file to open it.

You can keep the dialog open and move the design around in the background to check it and then, possibly, double click another design in the list and check that. Eventually you should close the Auto File List when you have finished with it.

You will normally do an immediate "Save As" if you wish to keep a file permanently. This will give the project a name which will then, in turn, also be used in future Auto Saves.

Delete All

To delete all Auto Save files simply click on "Delete All". A warning message will be displayed.

Delete

To delete selected files you should place check marks against all those files to be deleted and then click on Delete. A warning message will be displayed.

Example of the message displayed after Delete button used

Selecting Files

To select files for deletion you can use the following features:

  • Click on a check box next to a file to select or de-select that file

  • Click on a check box next to a project name to select or de-select all files in the project. The list of files can be open or closed for this.

  • To select all but one or two files in a long project list, click on the project check box to select all files. You can then, with the list open, uncheck one or two files that are to be retained.
