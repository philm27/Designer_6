Re-Centre

You can re-centre the design visually using this dialog. To access it select Design->Re-Centre.

 

Example of the Re-Centre dialog. The user has adjusted the rectangle to a new location using the mouse

 

Main Design Preview

Use the mouse to drag the rectangle around when the mouse is pointing anywhere inside of the cropping rectangle. When pointing at the tabs you can make the rectangle larger or smaller.

Design Section

These figures give the size of the current rectangle. These figures cannot be modified in this dialog and will be "Read-Only".

Design Offset

This gives the offset of the bottom left of the selected section from the bottom left of the original design which in the above example is 76 ends and 49 rows. This can be modified manually if necessary.

Design Size

These "Read Only" figures give the original design size in ends and rows, feet and inches, and metres.

Section Size

These "Read Only" figures give the selected section size and will be identical to the design size above.

Reset

Click this button to reset the cropping rectangle to the whole design i.e. one repeat.

1 Repeat

Click on this button to show only one repeat. This is not of much general use since there needs to be 2 or more repeats on screen in order to be able to move the selection rectangle across the edges of the current repeat.

2x2 Repeat

Click on this button to show 2 repeats across by 2 repeats lengthwise. This is the normal view in order to be able to re-centre the design. If the design is a Vertical runner then only one repeat is shown width-wise. Similarly if it is a Horizontal runner then only one repeat is shown length-wise. If the design is a Rug, only one repeat is ever shown and the design cannot be re-centered without changing the repeat type.

3x3 Repeat

Click on this button to show 3 repeats across by 3 repeats lengthwise. The same restrictions apply to Rugs and Runners as for the 2x2 display as noted above.

OK

Click OK to re-centre the design according to the current selections. The dialog will be closed. For Carpet Designer users, any current frame plan will also be re-centred if necessary. For Jacquard Designer users, the weave, if any, will also be re-centred.

The locations of any polygons will be adjusted, and, any regions and their inlays and out-fills will be adjusted so that they relate to the same figure in the design.

Note that the design itself is not modified until OK is clicked. The operation can then be undone if required.

Cancel

This will cause a return to the main design window without modifying the design in any way.
