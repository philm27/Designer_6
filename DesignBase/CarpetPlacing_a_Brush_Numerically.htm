Numeric Paste

 

This function allows a brush to be placed numerically by specifying the co-ordinates at which it should be placed. This can be useful if you are placing a large brush into a large design as it ensures that it is placed accurately.

The brush can be placed singly or can be repeated any number of times.

Select menu option Brush->Numeric Paste and the following dialog will be displayed.

Click on areas of the window below to find out more:-

 

 

Note: You must ensure that you have the correct Brush Handle before selecting Numeric Paste.

Note: Numeric Paste uses the Stencil/Working areas.

Repeat Style

This will only have an effect if you are pasting several repeats of your brush.

You can choose one of the following from the drop down list.

  • Straight Match

Applies the brush with a normal straight match when repeating

  • Brick

Applies the brush using a 'Brick' repeat with a half, third or quarter shift as specified.

  • As specification

Applies the brush using the match and drop as per its specification. To set this place the brush on the shelf, then Edit it (i.e. bring it in as the design being edited) then change the specification. Bring the main design back in and select the shelf item as the brush.

  • Drop

Applies the brush using a normal vertical drop with a half, third or quarter shift as specified

 

Drop (or 'shift')

These options (Half, Third, Quarter) are only enabled when the Repeat Style is Brick or Drop. For a drop, the selection specifies how much the next repeat on the right is 'shifted' downwards.

For a Brick repeat the selection specifies how much the next repeat above is shifted to the right.

 

Location

The End number is only enabled if the corresponding option is Single Repeat or Multiple Repeat. Similarly the Row (pick) number is only enabled if the corresponding option is Single Repeat or Multiple Repeat. It is therefore possible for the End number to be required but not the Row number (or vice versa) depending on the Option settings.

Note that when specifying coordinates here, the Brush Handle is used and that when pasting multiple repeats then the handle applies to the whole thing. For example if you paste a block of 4 repeats horizontally and 5 repeats vertically then the software makes a temporary brush of that size and uses it's centre, top left, top right, bottom left, bottom right according to the brush handle setting to locate it at the specified location.

 

Repeats

These two values (horizontal repeats and vertical repeats) are only enabled if the corresponding option is Multiple Repeats. It is therefore possible for one value to be enabled and the other disabled depending on the Option settings.

Simply enter the number of repeats required when pasting multiple repeats.

 

Options

By using the options drop list you can often save time as many of the common requirements can be quickly selected without calculating the repeats or locations.

The left hand drop down list of options refers to the Ends and repeats widthwise and the right hand list to the Rows (picks) and repeats lengthwise.

Note that this image is for illustrative purposes only - it is not possible in practice to display both drop down lists simultaneously.

Widthwise Options

Single Repeat

This will automatically set repeats to 1 widthwise and the edit box for number of repeats will be disabled. The single widthwise repeat can be placed at any end i.e. Location (End Number) is required.

Multiple Repeats

Similar to above except that several repeats widthwise can be specified.

Full Width

Automatically pastes a number of repeats of the brush to make up a full design repeat of the main pattern widthwise. This may involve pasting a partial repeat.

The repeating pattern will automatically start at end 1 in the main design regardless of the Brush Handle setting . The two edit boxes for Location End and Repeats Widthwise will be disabled.

Centre

Places one horizontal repeat of the brush at the centre of the design widthwise regardless of Brush Handle.

At left edge

Places one horizontal repeat of the brush against the left edge of the design regardless of the Brush Handle Setting.

At right edge

Places one horizontal repeat of the brush against the right edge of the design regardless of the Brush Handle Setting.

Centre on edge

Places one horizontal repeat of the brush so that its vertical centre is on the edge of the design. The left half of the brush will be at the right side of the design repeat and it's right half will be at the left side of the design.

Note that the repeat drop match of the main design will be taken into account when pasting this type of image.

 

Lengthwise Options

Single Repeat

This will automatically set repeats to 1 lengthwise and the edit box for number of repeats will be disabled. The single repeat can be placed at any row (pick) i.e. Location (Row Number) is required.

Multiple Repeats

Similar to above except that several repeats Lengthwise can be specified.

Full Height

Automatically pastes a number of repeats of the brush to make up a full design repeat of the main pattern lengthwise. This may involve pasting a partial repeat.

The repeating pattern will automatically start at row 1 in the main design regardless of the Brush Handle setting . The two edit boxes for Location End and Repeats Lengthwise will be disabled.

Centre

Places one vertical repeat of the brush at the centre of the design lengthwise regardless of Brush Handle.

At top edge

Places one vertical repeat of the brush against the top edge of the design regardless of the Brush Handle Setting.

At bottom edge

Places one vertical repeat of the brush against the bottom edge of the design regardless of the Brush Handle Setting.

Centre on edge

Places one vertical repeat of the brush so that its centre is on the top edge of the design. The top half of the brush will be at the bottom of the design repeat and it's bottom half will be at the top of the design.

Note that if the repeat type of the main design is a brick repeat, then the horizontal shift will be taken into account when pasting this type of image.
