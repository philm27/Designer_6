Ranges and Color Sets for Jacquard Designers

A typical Warp Range tab in the Shade Library might look something like this...

 

An example of the Warp Range tab using the swatch option. The Range code is R500 and each of the colorways (A, B, C etc) has 6 colors. (Note there is no set limit on the number of colors in a colorway and also, each one could have a different number of colors though that will not normally happen. "Recolor Warp" is selected which means that when a color set is selected it will replace the current warp colors.

 

You can display the above Warp Range tab or the Weft Range tab by selecting Shade Library on the Palette menu (or press <Shift>+S ) see the Shade Library Introduction.

Range

Select any range from the drop down list. These range codes are taken from the Colors and Yarns data base.

The Colorways List

The code for each color set is shown as a heading at the top of a column of swatches forming the colorway. If there are a large number of colors in one or more colorways, the software will display a vertical scroll bar and if there are a large number of colorways (7 or more) there will be a horizontal scroll bar as in the example above.

The Range list is used in exactly the same way as a normal list as described for the Color Tab. Thus the list can be scrolled by holding <Shift> and dragging it left right or up and down. Also you can select colors by a Double-Click or by Drag and Drop to the Palette but also you can select a whole colorway at a time as described below.

Recolor Artwork

This additional pair of radio buttons are shown on the Warp Range and Weft Range tabs. If Recolor Artwork is selected,  the software will replace palette colors based on position in the palette.

Recolor Warp or Weft

If this option is selected then color sets will replace the current warp colors (or weft if using the Weft Range tab). This option does not affect the Drag and Drop option for palette update.

Selecting a Colorway

The Range tabs allow a further method of recoloring a whole design (or the warp or the weft) by selecting a complete color set ...

  1. Double Click on the Colorway Title (i.e. the Colorway Number)
  1. Hold <Ctrl>+<Shift> and click on the Colorway Title
  1. Drag and Drop from the Colorway Title to the Palette.

Method 1+2 will replace artwork colors if "Recolor Artwork" is selected (zero color if any is ignored) but will replace Warp colors if "Recolor Warp" is selected. Similarly, on the Weft tab, Weft colors are replaced if "Recolor Weft" is selected. (Replace Warp or Weft is the initial default selection).

Method 3, since it is controlled by the mouse, gives an extra level of control. You can drop on color 1 in the palette to do the same as options 1+2 but you can also replace/add colors further down the palette. In fact the color set could be dropped after the last palette color to add the entire color set to the end of the palette,

Replacing Warp or Weft

When a color set is used to replace warp or weft the entire color set will replace the current warp or weft colors in the design. If there are not enough colors in the color set, then one or more colors in the threading plan will be unchanged. On the other hand if there are more colors in the color set than there are in the threading plan, then the extra colors will be added both to the palette and to the threading  plan.

Selecting a Color

The 4 methods of individual color selection also apply to Ranges/Colorways i.e....

  1. Double Click on a Shade
  1. Hold <Ctrl>+<Shift> and click on a shade
  1. Type a shade code (or click on it once in the list) and press <return>
  1. Drag and Drop from the list to the Palette.

As usual, for methods 1-3, the setting of the radio buttons for Add, Replace, and Replace+Next is important
