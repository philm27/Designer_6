Copy Brush to Another Project

 

It is often necessary to copy brushes from one project to another or even to, or from, a separate piece of software.

This process normally utilises the Clipboard which is an invisible, internal, holding place for text and images used by all software packages.

In general, to use this feature you copy to the clipboard. You can then paste the clipboard into any project or main design, or indeed to 3rd party software.

Copy Brush To Clipboard

To copy a brush FROM third party software you should follow the operating instructions for that software. This will usually involve using <Ctrl> + C.

From the APSO Designer products you can do any of the following:

  • Use menu item Brush->Copy to Clipboard

Copy Shelf Item To Clipboard

If the item it already on the shelf you can:

  • Right click a shelf item to display the pop-up menu

  •  Cut or Copy from the Pop-up menu. The former will delete the item after copying (Cut is not available for shelf item 1 which is the main design).

Paste from Clipboard

Having placed an image or design section on the Clipboard, you can paste it as the main design or as a shelf item in the same or any other project:

  • Select the target project on the project menu if necessary.

  • <Ctrl> + V to paste as the main design, OR...

  • Use menu item Edit->Paste to also paste as the main design, OR...

  • Right click a shelf item and select Paste from the Pop-up menu to paste as a shelf item, OR...

  • Use menu option Shelf->paste to paste as a shelf item.

After pasting onto the shelf you can easily make the item into the current brush by left clicking the shelf item.

Palette Colours

If a paste operation is replacing the main design and there are no other shelf items then the paste will take place immediately and there will be no further warnings.

However if, following a paste, there will be 2 or more shelf items (i.e. main design plus at least one other), the following dialog will be displayed allowing you to decide what to do with the colours in the pasted item.

 

 

Select YES to force the program to add all the colours in the brush to the palette whether they are the same RGB value or not. This may be useful if you wish to keep the colours separate allowing you to re-colour the design later.

Select NO to add colours to the palette from the new image which are not present in the current palette. In some cases the software will not need to add any new colours to the palette. In other cases, every colour in the brush might be a new colour and will be added to the palette. In the latter case the effect of these two options is exactly the same

The new shelf item will become the shelf item being edited.
