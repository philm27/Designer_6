Replace Missing Shades

 

The Shade Library Tab in Preferences includes this option under the heading "Missing Shades"

 

Replace Missing Shades

Select this feature to cause the software to insert any missing shades back into the Colors and Yarns database. They will of course be given the RGB value from the design file. At present this only applies to the .DES NIFF files described above.

After an update has taken place, a summary of successful updates and errors will be displayed. If all shades present in the design file are also present in the database, then no message is displayed.

 

Note that if you have the Colors and Yarns database open and you are currently viewing the Plain yarns then you may have to switch to a different tab, for example Mixtures, and back again order to refresh the list with the new shades.

In order to use this feature the option "Use C + Y database direct". Must also be selected.

For more information see the topic Shade Library Preferences.
