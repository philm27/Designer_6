The Shelf

 

For each design project , there are 99 possible shelf items associated with it they are normally shown down the right-hand side of the screen.

You can choose the height of each shelf item in Shelf Properties.

To scroll the shelf use the scroll bar on the right or use the Mouse Wheel while hovering over the shelf window.

Example design and shelf items on the right hand side. The top item (box 1) in the shelf is the main design. Shelf items 2 to 5 are selected.

Note that the size of the  font used to draw the shelf numbers can be modified - see the Shelf Properties topic for more information

You can use the shelf in any of the following ways:-

  • Store Brushes from the current design for use later in the design process

  • Import files (designs or shelf items) from other design files, for use in your current design

  • Paste Images from the clipboard

  • Save a shelf item to a folder as .iff files

  • Import from another of your current projects

  • Copy a Shelf Item into another of your current projects

  • Edit in the same way as the current design

  • Create a new shelf item to be used as a brush and using the current palette

 

Turn the shelf display on and off by selecting menu item View->Shelf, or alternatively use the keyboard shortcut key <F9>.
