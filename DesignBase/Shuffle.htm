Shuffle

This feature allows you to slide ends up or down randomly by a set maximum number of rows, or, alternatively, rows can be moved left or right randomly by a set maximum number of ends.

To use this feature select he menu option Power Tools->Shuffle. The following dialog will be displayed.

 

An example of the Shuffle Dialog. The shift will be vertical with a maximum shift of+/- 4 rows. The ends will be shifted in 2's (grouped in 2's) and any points lost at the top or bottom will wrap around to the other end. There will also be a wave effect with an amplitude of 8 rows, a wavelength of 84 ends and a further random deviation in amplitude of +- 6 rows.

Horizontal or Vertical

Select the direction - only horizontal or vertical shift is allowed

Group Together

Using this value a user can specify that, for example, ends are to be grouped in 2's and shifted together. A value of 1 indicates that ends or rows are to be randomly moved individually.

Allow Warp Around

This indicates, if checked,  that any points lost at one edge of the design are to be moved to the opposite edge into the vacated space. If this is not selected then any vacated points are replaced with the current background colour.

Do Random Shift

Check this box to cause a random movement in the specified direction. This will be added to to the wave effect if any.

Maximum Shift

This indicates the maximum shift in rows for a vertical shuffle, and similarly the maximum shift in ends for a horizontal shuffle.

Same design after shuffle using the above parameters (but without any wave effect)

 

Wave

Check this to cause a wave effect in addition to any random shuffle as specified above.

Amplitude

This is the height of the wave (if shuffle is vertical). Columns (or rows if horizontal shuffle) will be shifted by this amount at the peak of the wave. Similarly a shift of the same amount will occur in the opposite direction to form the low point of the wave. The figure in the dialog is design rows for a vertical shift and design ends for a horizontal shift.

Wavelength

This is the length of one complete wave cycle from any point to the same point in the next cycle so it can be thought of as the distance from one peak to the next. The figure in the dialog is design ends for a vertical shift and design rows for a horizontal shift.

Maximum Deviation

The Deviation is a random figure that will be added to (or subtracted from) the amplitude in each half of the wave cycle, so the peak height may be different to the trough depth in a single cycle. And, of course, all peaks and troughs may be different height or depth. This can be set to zero for a regular sine wave.
