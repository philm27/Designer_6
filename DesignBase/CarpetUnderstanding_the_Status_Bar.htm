Understanding the Status Bar

 

The status bar is at the bottom of the main window. It enables you to view, at a glance, a summary of the design size and other important information.

You may need to scroll the help window widthwise in order to see all the following example of a status bar.

Current drawing tool is freehand draw

repeat type is normal

brush handle is centre

design size is 28 ends by 28 rows (picks) (4.00 in x 4.00 in). (If you have used a metric quality or setting this will be shown in metres and centimetres).

Quality or setting is 7.0 x 7.0 per inch or cm etc

Zoom level is 10

The mouse is pointing at end 193, row (pick) 131 (* See note below for Carpet users).

Shelf item 2 is being edited (** see note below for shaded corner)

 

* Carpet Users

When you point the cursor over any area of the frame plan, the end and row location information will change to showing Frame number and End as shown in this example.

 

** Resizing Tool

The shaded corner can be used to re-size the main window in either or both the horizontal and vertical by using click and drag.

Turning the Status Bar On and Off

Turn the status bar on and off using the menu option View->Status Bar.
