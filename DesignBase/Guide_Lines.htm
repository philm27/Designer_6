Guide Lines

 

Guide Lines are simply any Polygon Lines or groups of lines that are designated as Guide Lines. They have a special status in that they can be made visible by selecting View->Guide Lines on the menu.

 

Example design showing a 12 foot octagonal rug and the beginnings of the design. Guide Lines have been created using the "Standard Shapes" feature.

 

You can change a polygon to a Guide Line (or back to a normal polygon line) by going into polygon mode and then either:

Use Properties

Right click any point on the Polygon then select Properties and check the "Guide Line" option

You may also select a different colour for the polygon line (this is not the same as the design color when and if the line is rendered into the design). Generally speaking Guide lines will be drawn in a vibrant color that will stand out against all others in the design as here.

Use the Guide Line Button

Select any point on the Polygon using the left mouse button.

Use Tape Measure

Whenever the Tape Measure Tool is used you can check the "Keep Lines as Guide" option. In this case the software actually creates a group of 5 lines since a small cross is created at each end of the main line. The software will automatically give the group a name containing the line length.
