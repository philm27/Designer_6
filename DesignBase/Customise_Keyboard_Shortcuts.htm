Customised Keyboard Shortcuts

You can create your own keyboard shortcuts and override existing ones for most commands as described here.

For a list of standard shortcuts see the topic Keyboard Shortcuts.

An example of the dialog, line #4 is selected and has just been changed so that the spacebar is the shortcut for Grid On /Off. Several custom shortcuts have been specified already here such as B for Zoom and Ctrl+Shift+E for the "Eye-dropper" color selector.

An example of the dialog after an individual set of commands has been selected - in this case the Pan and Zoom commands

Main Features

  • The list is arranged in Menu sequence with the greyed out items being menu headers which cannot be allocated a short cut.

  • You can use the drop down list to select an individual menu or pop-up menu for example "File" "Brush" or "Polygon".

  • The list can be sorted by clicking on column headers so the command column can be sorted into alphabetical order, or, the shortcuts can be listed in alphabetical order.

  • The list can be sorted into default (menu) sequence by clicking on the first column headed "Line#".

  • The overrides in the current list can be cleared by clicking on the Clear button.

  • To accept all changes click on OK.

  • To cancel all changes click on CANCEL.

  • The dialog can be re-sized by dragging any edge using the mouse.

  • There are two ways in which the keyboard shortcut can be changed - either click in the override column and type the new key combination, OR, double click in any of the first 3 columns. See below for details.

Example showing the Curved (Bendy) Line command being changed to Ctrl+Shift+U, immediately before the user presses Return.

Method 1: Change Override by clicking in Override Column

Click in the Override Column on the appropriate line.

An edit box will appear, initially containing the current override key combination if there is one.

Type in the new override and press return. You do not need to type it in full you can abbreviate. For example:

Typed Resulting Override
C+S+A Ctrl+Shift+A
Ctr+sh+B Ctrl+Shift+B
Spacebar Spacebar
Space Spacebar
Left Left
Ri Right
S+numpad1 Shift+Numpad1
   

To remove an existing override by this method simply click on it, erase it and press return.

As shown in the examples above you can distinguish between the keys in the numeric pad and the "normal" keys by typing out in full "Shift+numpad1" for example (see the list below). You can abbreviate where possible where the meaning is not ambiguous but anything starting "numpad" must be typed in full.

If the short cut has been used before or is by default allocated to a particular command, the software will display a warning.

Example showing the software ready to accept the new override key combination for Global Fill

Method 2: Change Override by Double Clicking Columns 1-3

Double Click any column other than the Override column

A small dialog will appear as illustrated in the example above

Press the required key combination by actually using the keys making sure that you press Ctrl, Shift and possibly Alt together first then the finally normal key such as ABCDE... 1234... etc as you normally would when using the shortcut.

As soon as the main key has been pressed, the dialog will disappear and  the text for that short cut will be placed in the Override column.

As for Method 1 above, the software will check if the combination has already been used and will display a warning. The software will not, however, prevent a duplicate occurring. The override will take precedence over any normal, default keyboard shortcut but if two or more overrides use the same shortcut the software will simply use the first in the list.

Some Notes about Override Keys

It is best not to use the Alt key as it is reserved by Windows though it may work in some cases

It is best to stick to normal keys in most cases i.e. 0-9 and A-Z though, again, others will work in some cases.
