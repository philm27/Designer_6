Polygons - Create new Shapes - Shapes

This is the Shape tab on the Create New Shape Property Sheet. Use it to specify the type of shape and whether radial lines are to be drawn.

For more general information see the topic Polygons - Standard Shapes.

 

An example showing a Stretched Polygon shape having 8 sides and with radial lines

Basic Shape

Check one of the options:

  • Circle - a normal circle with optional borders. The circle will have 4 control points at NSE and W together with any formed by radial lines.

  • Ellipse - a normal mathematical ellipse with optional borders. As for the circle, the ellipse will have 4 control points at NSE and W together with any formed by radial lines.

  • Regular Polygon - this is a figure having a number of equal sides and with each side being the same distance from a centre point. If the number of sides is an odd number, the default shape will have a horizontal base line. However, the shape can be rotated.

  • Stretched Polygon -  this is basically a regular polygon but can be stretched in one axis to satisfy design requirements or to fill the repeat etc. It is not necessarily a "lozenge" (see below).

Options

  • Arc - Check this to draw an arc of a circle or ellipse. it will be disabled if the shape is a polygon. The "from" and "to" angle of he arc is set and modified on the Arc/Chord tab.

  • Chord - this is identical to the Arc option above except that a straight line is drawn across the arc joining its start point to its end point.

  • Lozenge - this is only available for a stretched polygon and causes the software to stretch the polygon by simply stretching the opposite sides along its longest axis as in the example in the illustration above. The number of sides must be an even number. If radial lines are specified there will be two separate sets. A lozenge will normally have 8 sides but the software will allow 6, 8 or more.

Radial Lines

  • Sides - Use this figure to specify the number of radial lines but also, in the case of a polygon, the number of sides. If the shape is an Arc or a Chord,  the radial lines are split evenly between the start and finish angle.

  • Draw Radial Lines - check this box in order to draw the radial lines.
