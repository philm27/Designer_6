Export Image

 

This feature enables you to create a standard BMP JPEG or TIFF file which shows one or more repeats of the design. The resulting image file can be viewed using any standard software.

This is particularly useful if you want to create an image of a design to send by e-mail to a customer for example who does not have the APSO software.

To use this feature select menu option File->Export Image.

The image file can easily be viewed using any standard package such as the Microsoft Paint program or JASC Paint Shop Prop or Photoshop etc. Images are adjusted to the correct aspect ratio (i.e. saved images are always "square" set) and will show very little distortion if made quite small since APSO Thumbnail technology is used to create the image.

Note that the file saved will be a 24 bit image in order to give a good quality image when the resolution is being reduced.

 

The Preview Area

The dialog features a preview of the image that is to be saved. This will change as you alter the section or offset figures. Note that you can make a rectangle using the mouse in the preview area in order to "zoom" in on a particular section. The figures in the dialog will automatically adjust in this case.

A useful technique when you want to make an image of a design section that crosses the edges of the repeat is to select 2x2 repeats then make a rectangle in the preview of the section you required. The preview will then change to the selected rectangle.

Example preview after selecting 2x2 repeats and making a rectangle using the mouse. The left mouse button should be down whilst making the rectangle. Image is shown immediately prior to the release of the mouse button.

The same example after the mouse button has been released

You can change the size and offset of the preview image as many times as you like by either making a new rectangle visually in the preview itself or by altering the figures in the Design Offset and Design Section.

 

Other Details

Design Section

You can specify the design section size in repeats, inches or cm. In all cases you can use up to 3 decimal places. The section can obviously exceed one repeat in one or both directions and all possible match types will be taken into account.

Design Offset

This specifies the start position of the section from the top left corner of the design. Note that these can be negative if you want to show a section crossing the edge of a repeat. You can specify these values in either inches or cm.

Image

This specifies the pixels per inch or cm of the saved image and also the scale.

Image Size

This shows the resulting image size in pixels and its "natural" size when printed at the specified dpi (or dots per cm). The size is shown in both inches and cm.

Note of course that an end user to whom you have sent an image can view or print the image at any size they wish and will not be restricted to the size you have specified.

Buttons

1 Repeat

Use this to make the image exactly 1 repeat. The offset will also be reset to zero so that the edges of the repeat will be at the edges of the image.

2 x 2 repeats

Use this to set the image to exactly 2 repeats by 2 repeats. Again, the offset will be set to zero.

Reset

This will reset all values to the way they were when the dialog was displayed.

OK

This will cause a standard file save dialog to be displayed allowing selection of BMP JPEG and TIFF file types.

CANCEL

Use this to close the dialog.
