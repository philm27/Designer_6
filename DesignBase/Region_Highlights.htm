Region Highlights

 

Use this simple dialog to modify the highlight colors for Regions produced by the Color Search feature. You may need to use this dialog to select colors that do not clash with those in your design. You can also use it to switch off one or other of the Highlight methods i.e. to disable either Outlines or Cross Hatching.

The dialog is displayed using the "Highlights..." button in the Color Search dialog.

Example showing default colors but cross hatch de-selected.

Outline Color

Check or un-check this option to enable or disable outlining of regions.

Normal (Outline Color)

Click on this box to display the standard color picker so you can select another color for Normal Regions (i.e. all but the selected one).

Selected (Outline Color)

Click on this box to display the standard color picker so you can select another color for Selected Regions.

Cross Hatch

Check or un-check this option to enable or disable the drawing of a Cross Hatch pattern in regions.

Normal (Cross Hatch Color)

Click on this box to display the standard color picker so you can select another color for Normal Regions (i.e. all but the selected one).

Selected (Cross Hatch Color)

Click on this box to display the standard color picker so you can select another color for Selected Regions.
