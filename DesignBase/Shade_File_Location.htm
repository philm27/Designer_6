Text File List

 

Traditionally, the location of the text shade file was c:\Temp\Shade.txt. However, you can now easily have several different shade files, possibly in different folders which can be used at different times for whatever reason.

To view or edit the current set of text shade files, select the Edit button on the Library tab.

Clicking the edit button will display a dialog like this:

Example of the “Modify Shade Library List” after clicking Edit on main tab

To edit the existing information, click on a position in the grid and re-type that entry.

Using the buttons at the top you can:

  • Add a new line

  • Insert a line in front of the current one

  • Delete a line

  • Move the current line up one

  • Move the current line down one

Note that the list is maintained in the order that you put it - it is not in alphabetical order so you can make sure your most commonly used entry is first in the list.

There is also a pop-up menu as shown here (click right hand mouse button in the grid as usual).

The first 5 options are the same as the buttons but the final one - “Browse” – allows you to browse to a shade file to be associated with the current row in the grid. The file should be a .txt file.

Colors and Yarns

When you use the Colors and Yarns program to Export a Shade File, the location and name is automatically recorded and will be read by the design program, when you first use it. This file name will be the initial default for the above list.

In Colors and Yarns you should use the menu option "File->Change Shade File Directory" in order to change the location of the shade file for an export and "File->Export Shade File" to actually do the export.
