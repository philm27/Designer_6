Randomize

This pair of functions (Horizontal and Vertical) allow the user to randomize a design with some control over the amount of change.

A Vertical Randomize operation will move pixel colors up or down each END so that each END contains the same amount of color as it did before.

Similarly a Horizontal Random Balance will move colors left or right along each ROW in the design.

As for the balancing operations these functions will not move whole shapes around, only individual pixels.

Stencil and Protected Areas

Note that these operations do use the current stencil and Protected areas mask.

 

Example of the Randomize dialog

Randomize Percentage

Modify this by typing a value or by clicking on the "Spin Control" to increase  or decrease the value. It will initially default to 50% but can be set to any value between 0 and 100%. A value of 0 would actually result in no change at all and a value of 100% would result in the maximum number of changes.

OK

Click on the OK button to perform a Randomize operation. After the operation is complete you can choose to Undo it or to Close the dialog or indeed to do another Randomize on top of the previous one at the same or a different percentage.

Undo

Click on Undo to undo the previous operation whether or not that is a randomize operation.

Close

Click on this to close the dialog. You can still undo the randomize operation in the normal way after you have closed the dialog.
