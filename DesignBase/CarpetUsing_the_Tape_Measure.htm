Tape Measure

 

The Tape Measure function allows you to accurately and easily measure distances between stripes or other design features.

To activate this feature select menu option Power Tools -> Tape Measure. On selecting it, the "Tape Measure" window will appear on screen and the cursor will change to a tape measure symbol.

This facility will stay selected until you close the window or until you select a new drawing function from the tool box.

Using Tape Measure

To take a measurement within your design, click with the mouse on your start point and then drag the cursor to the finish point. A yellow line will be drawn as you drag the cursor to the end point. The line will disappear automatically when you draw another line or when you select a menu option or drawing function. You can use Auto Scroll or you can scroll the design using the cursor keys whilst measuring.

Note that you can also go across the edges of the design repeat.

The Tape Measure window displays the following information:-

  • The Horizontal distance in ends, Ft/Inches and M/cm

  • The Vertical distance in rows(picks), Ft/Inches and M/cm

  • The Diagonal distance in Ft/Inches and M/cm.

  • The Angle of the line drawn (with 0 degrees being vertical and 90 degrees being due East etc)

  • The co-ordinates of the start and finish points of the distance measured

Guide Lines

You can keep lines drawn as Guide Lines in the chosen colour. These are useful as they can be switched on and off and can provide a guide when drawing detailed areas at a later stage of the design process.

To create Guide Lines check the "Keep Lines as Guide" box at the lower left of the Tape Measure Dialog.

To view all current Guide Lines select View>Guide Lines.

More sophisticated Guide Lines can be created using the Polygon Drawing Feature.

Tape Measure Colour

You can click on the colour box at the bottom right of the dialog (yellow in the examples above) to modify the line colour. This will only affect future lines drawn, not than any existing ones.
