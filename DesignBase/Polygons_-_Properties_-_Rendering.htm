Polygons - Properties - Rendering

You can use this feature at any time to modify the settings including color of any polygon.

To display the Property Sheet, first select a polygon by clicking on one of it's handles then select menu option Polygons->Properties, or select the "Properties" button on the Polygon Tool bar.

Note that the property sheet does not cause the polygon to be drawn. That will only happen if the polygon is selected and then drawn in the "Normal" colors.

An example in which the polygon is to be rendered using a dark green (color 9 in the palette) line 1 pixel wide and is to be filled with a pale green (color 10 in the palette). The line will be rendered using catch cords and the Border Line flag is not set.

Line

If this is checked, the software will draw the line defined by the polygon, into the design at the current quality/sett.

Line Color

Click on the color swatch to change the line rendering color. The Design Color Selector will be displayed allowing selection of one of the palette colors.

Width

Type in a new width for the line here, or use the Up and Down buttons as usual. You can use various units for this (see below).

Units

Use the drop down list to choose the units for the line width. This can be Pixels, cm, yds, ft, in, mm, metres. Note in all cases that the line thickness will be adjusted for quality. (When using pixels, it is assumed to be the thickness of a vertical line i.e. ends).

Fill Solid

Check this box to cause the polygon to be filled with the specified color.

Fill Color

Click on the color swatch to change the Fill rendering color. The Design Color Selector will be displayed allowing selection of one of the palette colors.

Border Line

Check this to cause the polygon line to be treated as a border. This is not normally essential but may provide a speed improvement for some rendering operations. Note that borders are drawn in the order they are added to the list of polygons and it is possible to modify this sequence in the Polygon List dialog.
