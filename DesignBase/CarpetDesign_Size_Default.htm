Design Size Default

 

Create a design size default in the specification window . Once you have entered all the relevant repeat, design and grid information, click on the "New Default Size" box in the bottom left-hand corner of the window to save the information as a default.

Every time you start a new design, it will automatically default to the information that you have specified including the ends and rows (picks) per inch (or cm etc). You can change the default at any time by setting new information into the specification (or possibly opening a suitable design from disk) and clicking on the "New Default Size" box.
