Shelf Notes

The Shelf Notes are the same as the Notes in the Design Specification. They are saved with the design in a .prj file. They can be edited in the Design Specification while that shelf is the current design.

 

Shelf Notes are displayed whilst hovering over a shelf item (the first line is displayed automatically and cannot be edited)

 

There is an easy way to edit these notes for any shelf item using the Edit Notes feature. To do this select "Edit Notes.." on the shelf pop-up menu. A dialog similar to the following will be displayed.

 

An example of the Shelf Notes text input dialog - at present there is a maximum of 2 lines of 40 characters.

Using the Dialog

Use the keyboard as normal.

Press <return> to start a new line

When you have finished you can press <tab> (to select the OK button) and <return> to close the dialog, or, click on OK to accept or Cancel to cancel the changes.
