Drawing Curved Lines

 

Procedure

To access the curved-line draw function, either:-

Note: For a 'rubberbanding' effect try using the Spacebar Toggle facility.

  • Select the from the toolbox

  • or, use the keyboard shortcut J

 

Line Thickness

You can use the toolbox or the keyboard to select line thickness:

  • Move the mouse over the "Pen Selector" at the bottom of the toolbox and click on one of the 6 preset pen sizes. Alternatively double click this area to display a dialog permitting a larger size to be set.

  • While in Circle Drawing mode press one of the keys 1, 2, 3, 4, 5 or 6 to select line thickness.

 

Drawing the Line

To draw a curved line:-

  1. Click with the mouse button on the start point of the line. Do not release the button pressure.

  2. Keeping the button held down, drag the cursor to the lineÂ’s finish point.

  3. Release the button but now move the mouse and the line will become a curve, bending more as you move further away.

  4. Finally make a second click with the mouse button to fix its position.

Draw with the left-hand mouse button to draw in your foreground colour . Use the right-hand mouse button to draw with your background colour .
