Peg Section

These details are only applicable to users of Carpet Designer.

 

Peg Sections will include a palette strip if you choose either of the first two options - "Show Inside" or "Show Outside". If you choose the former, the resulting peg board size including the palette will be exactly as specified when creating it.

If you choose Show Outside then the total Peg Board length will be increased by the palette size as specified above.

If you choose "None", then peg board sections will not include a palette.

You can also change the size (length) of the palette here (in inches). A palette strip will always run across the full width of the sample so the width of each individual colour sample may vary.

(1 Inch equals 2.54 cm).
