Undo Preferences - General

 

Undo Preferences are displayed by selecting menu option Edit->Undo Preferences. The General Tab looks like this...

 

Maximum Number of Undos

Set this figure to the number of undo operations you would like to retain. When setting this figure you should bear in mind the following:

  • Small edits do not use much memory - editing a single point uses about 1 byte for the point edited plus an overhead. However a fill operation may require that the whole design is saved and this would be 1MB of memory for a design size of 1000 x 1000 ends and rows (picks), though you may choose to compress this - see Advanced Settings.

  • Even editing the palette occupies one entry in the UNDO information.

  • If you have several projects - each one has its own undo buffers.

  • Some operations automatically involve saving the whole design into the undo buffer such as re-sizing, re-drafting, mirroring, changing the specification etc. Again, these design "copies" may be compressed see Advanced Settings.

  • The figure is an overall figure and does not apply to each shelf separately.
