Saving a Shelf Item

 

You can either use the Pop-up shelf menu or the main shelf menu.

The Pop-up shelf menu

Using the Pop-up Shelf Menu

This is the simplest way to save a shelf item.

  1. Click with the right-hand mouse button on the shelf item that you want to save. The pop-up shelf menu will be displayed.

  2. Select Export. The Save dialog will be displayed.

  3. Save as the required file type. Only IFF PAT BMP and DES types are allowed in this case. (E.g. select DES file from the File Type list)

 

Keyboard Shortcut

There is a keyboard shortcut that makes the shelf save even easier. By holding down the <Shift> key and clicking with either mouse button on the shelf, you will get straight to stage 3 in the above process.

 

An example of the the Shelf Selector Dialog - select the required shelf item using the mouse

Using the Main Menu

Alternatively, you can...

  1. Select menu item Shelf->Export. The Export dialog (see above) will be displayed.

  2. Select the appropriate item from the list of thumbnails and and click on the OK button or press <Return>. The Save dialog will be displayed.

  3. In the Save File dialog, navigate to the required folder if necessary and type the file name, select the file type and click on the Save button.
