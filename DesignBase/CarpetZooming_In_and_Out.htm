Zooming In and Out

 

Zoom in and out of your design to work on fine detail, see the design in several repeats etc. The status bar shows the current zoom level. Whole numbers give an accurate picture Â– fractions (i.e. zoom level 1.75) may make some tufts slightly distorted.

There are various ways of zooming in and out:

 

Plus (+) and Minus (-) Keys

This is the easiest way of zooming in and out. Simply use the + (zoom in) and - (zoom out) keys on the keyboard. Using this method you can rapidly zoom in and out several times in a few seconds.

If you have selected Zoom to Cursor in Preferences then the design will be zoomed and panned so that the same design point is under the cursor as you zoom in and out.

Mouse Wheel

Many modern PC's feature a mouse with a "mouse wheel" (which normally also doubles as a third button). If present, this wheel can be used to zoom in by pushing the wheel forward or zoom out by pulling it backward.

The mouse wheel works particularly well with Zoom to Cursor - just point at a tuft and roll the mouse wheel.

Zoom Tool

Click on the zoom button in the toolbox. The cursor will become a magnifying glass and you can then zoom in using the left-hand mouse button, and zoom out with the right-hand mouse button.

You can also select this mode by using the keyboard shortcut <Z>.

Zoom to Specific Level

Simply use the keyboard to type the 3 characters Znn on the where nn is the zoom level.

For example to zoom to level 9 type Z09 or to zoom to level 12 type Z12. Pressing <ESC> or any other character or selecting a different tool from the toolbox will cancel the sequence.

Zoom Rectangle

This feature allows a rectangle in the current design to be designated using the mouse. The software will then zoom into that area and increase the zoom level if possible whilst still showing all of the specified area.

To select this mode click on the Zoom Rectangle button in the toollbox.

[Note that although the two zoom buttons are strictly speaking different functions you will find that both buttons perform as both the normal Zoom Tool and the Zoom Rectangle Tool]

Zoom to Fit

This feature displays one repeat of the whole design at its highest zoom level. It is the quickest way to zoom back from a very high or low zoom level to a point where the whole design can be seen. This funtion will also recentre the design at the same time.

To select zoom to fit:

    • Select menu item View -> Zoom to Fit

    • Click on the Zoom to Fit button in the toolbar.

    • Use the keyboard shortcut <SHIFT> + <Z>
