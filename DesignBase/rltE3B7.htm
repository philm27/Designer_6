Undo/Redo

The Undo/Redo feature allows you to restore the project to a previous state with Redo being a simple way of re-applying those changes if necessary.

 

There are separate buttons for each of the following:

Normal Undo/Redo

This restores changes to the design itself in the current shelf item. To perform undo or redo on a different shelf item, simply switch to that shelf using the "Edit" option on the shelf pop-up menu.

Palette Undo/Redo

This restores changes to the Palette - the Palette is common to all shelf items in a project.

Polygon Undo/Redo

This restores changes to polygons.

 

Undo/Redo Buttons

Use the specific button for the required type of undo or redo as follows..

Shows part of the main tool bar with the “normal” Undo/Redo buttons next to the Palette Undo/Redo button

Polygon Undo/Redo

There is currently no keyboard shortcut for this but if you are in a polygon mode, then the normal <ctrl>Z <ctrl>Y will perform a polygon undo/redo as will the Polygon undo/redo buttons on the Polygon tool bar and menu. However, if a polygon has just been rendered into the design, the normal Undo/Redo buttons (or keyboard shortcuts <ctrl>Z and <ctrl>Y) can be used to undo/redo that design change.

 

Undo Messages

These are possibly the two most common messages, there may be others depending on the circumstances.

As previously you may get messages stating that an undo or redo is not possible because a color in the restored image is now no longer in the palette. In some cases it is now possible for the software to re-instate those colors or change the design so that it uses colors in their new locations in the palette and a message will list the options. For example:

This can be caused by attempting to redo a drawing operation that uses a color that has been removed from the palette. Clicking on OK will restore the missing color to the palette and perform the redo normally

This can be caused, for example, by removing unused colors, then attempting to undo a drawing operation. Unless all the unused colors are at the end of the palette, this will cause the above message. Clicking on OK will remap the restored section of the design to the new palette. Clicking NO will skip this undo and attempt the next one.

 

Palette Undo

This will simply undo/redo Palette changes, without affecting the design (except for palette color swap/merge etc). Again this can be very useful where the user wishes to take the palette back to a previous setting whilst retaining the effect of any actual drawing operations.
