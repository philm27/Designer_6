Repeat Type

 

Set the repeat type in either of the following ways:-

  • Select menu option Design->Repeat Type->type where type is one of the above options.

  • Select the repeat type in the specification dialog .

Note that the status bar always shows the current repeat type.

 

Repeat Types

There are five repeat types Normal, Vertical, Horizontal, Rug, Brick. These are described in more detail below.

 

Normal

Normal Repeat with half drop in display many (section of screen only)

Your design will repeat vertically and horizontally. This is the repeat type chosen for most design work.

You will be able to scroll past all 4 edges of the design repeat.

If the design has a drop match then the next repeat on the right will be "dropped" by the specified amount.

 

Vertical

Vertical Repeat in display many (section of screen only). Note that a Vertical repeat can also have a "drop" or shift if required rather than a straight match as shown here which will have the same effect as a brick i.e. a horizontal shift.

The design will only repeat vertically. Only one repeat will be shown horizontally when "View Many" is selected but as many as possible will be shown vertically.

You will be able to scroll past the top and bottom edges but not the left and right edges.

Choose this if you are designing a vertical border, runner or ribbon etc.

 

Horizontal

Horizontal Repeat in display many (section of screen only). Note that a Horizontal repeat can also have a drop match rather than straight match if required.

The design will only repeat horizontally. Only one repeat will be shown vertically when "View Many" is selected but as many as possible will be shown widthwise.

You will be able to scroll past the left and right edges but not the top and bottom edges.

Choose this if you are designing a horizontal border or runner, ribbon or name tape etc.

 

Rug

Rug repeat type cannot be shown in "View Many". This is the single design repeat on which all these examples are based

The design will not repeat. Choose this if you are designing a rug.

You will not be able to scroll past any of the 4 edges of the design.

 

Brick

Brick Repeat with half drop (or shift) in display many (section of screen only)

This is similar to normal repeat but the drop is applied horizontally. If the drop is zero, then that is equivalent to a straight match and the effect is the same as for a Normal repeat type.

If the design has a drop match then the next repeat vertically will be "shifted" to the right by the specified amount.

 

Viewing in Repeat

If the "View Many" option is selected, Normal and Brick repeats will show as many repeats as possible in each direction. Horizontal repeats will show as many repeats as possible in a Horizontal direction but only one vertically. The opposite is true of a Vertical repeat. A rug type will only ever show one repeat.

 

Placing a Brush

For Normal and Brick repeats, a brush placed over an edge will "wrap" around to the opposite edge. This will only happen left to right for a Horizontal repeat and top to bottom for a Vertical repeat. It will not happen at all for a Rug. For more information about placing a brush see the topic Placing a Brush using the Mouse.
