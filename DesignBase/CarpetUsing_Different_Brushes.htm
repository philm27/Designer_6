Standard Brushes

All drawing functions use either a brush or a line thickness which you should select from the lower panel in the toolbox.

The first 8 standard brushes are simple solid colour squares from a size of 1x1 up to a size of 8x8.

These are followed by 4 small circles having diameters of 3, 4, 5 and one that is a "custom" size.

The final row features 2 moresque patterns, a solid "standard" square (see table below) and a solid custom size square which you can set to any size you like.

The standard square (3rd tool on bottom row) will vary in size according to the current units ("per" value) for the design quality (setting) in the specification according to the following table:

Note that if you have cut a brush from the design or from the shelf then the above standard brushes are replaced by a preview of that brush. However, if the mouse pointer is over the toolbox the above will again be displayed allowing selection of a standard brush.

Custom Size Brush Rectangle

The brush at the lower right is a custom size brush. If you right click it, the following dialog will be displayed allowing you to set the size of brush that you require. Also, if you are currently using a brush other than circular and you right-click the Freehand draw button, this dialog will be displayed.

The size can be set in a variety of units: Ends, Inches, Feet, Yards, mm, cm, Metres.

If the "Square" option is checked then only the width can be set - the height will be disabled

Custom Size Brush Circular

The fourth brush in the row of circular brushes a custom size brush. If you right click it, the following dialog will be displayed allowing you to set the size of circle that you require. Also, if you are currently using a circular brush and you right-click the Freehand draw button, this dialog will be displayed.

Custom Circular Brush Size dialog showing a selected size of 1.5 inches (approx. 38mm).

In addition to the brushes in the toolbox, it is also possible to draw and fill with a Custom Brush.
