View Many

 

You can view and work on your design in either a single repeat or in many repeats (except for a Rug which cannot be viewed in repeat - see the topic Selecting a Repeat Type).

You can select View Many (or switch this feature off) as follows:

  • Select the View Many button in the toolbar.

  • Select the menu option View->View Many.

View 1

If View Many is not selected you will only ever see one design repeat on the screen. However you can scroll that repeat at any time to see the edges of the design. The repeat edge is marked by a red line if the grid is switched on.

Rugs

The software will disable the View Many button and menu options if the design is a rug.

Horizontal and Vertical repeat types

The software will show many repeats in the horizontal and vertical direction respectively for these repeat types but will only show one repeat width in the other direction.
