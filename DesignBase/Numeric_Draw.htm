Numeric Draw

The so-called "Numeric Draw" option allows you to easily draw large figures using numeric information such as radius/diameter and location. This is particularly useful on very large designs when drawing very large figures

To access this feature select menu item Design->Numeric Draw. A dialog similar to the following will be displayed.

 

Drawing a Circle or Ellipse using Radius and Centre position

Example will draw an ellipse with an X radius of 42 ends and a Y radius of 64 rows. The centre is at end 257, row 388.

All lines are drawn using the current foreground color and line thickness.

 

Drawing a Circle or Ellipse using Diameter and Bottom Left Location

Example will draw an ellipse with an diameter of 504 ends and 768 rows. The containing rectangle has a bottom left corner a end 1, row 1.

The new option allows an ellipse to be drawn which has an even number of ends or rows in the diameter. This is not possible when specifying the size using radius.

As above, all lines are drawn using the current foreground color and line thickness.
