Colors and Yarns Database

 

You can use the Colors and Yarns database in two different ways:

  1. Direct - the design software will directly access the database for shade information

  2. Indirect - you must "export" the shade information to a "Shadefile" which is simply a text file and this file is accessed by the design software for shade information.

The former avoids the need to export the text file so is recommended.

You may select the direct option on the Shade Library tab in preferences.

Note that if the software cannot open the database you will be offered the option of trying to load the text file version (as in the next option).

Note also that if you want to use the Replace Missing Shades feature (also on the Shade Library tab in preferences) you must also select the Direct option.

(Applies only to plain shades at present).

Using Colors and Yarns

Note that after an update to the Colors and Yarns database you should select the button "Update database for other applications" in C + Y. This will send a message to the design program (and other APSO software) instructing it to update designs and shade lists. The update will include designs and palettes in print layouts in all projects.
