Pipette

 

Use the pipette (or "Eye-Dropper") to select a new foreground or background color from the design itself.

When using the pipette,  the software will automatically show a small information panel next to the cursor showing the color number over which the cursor is "hovering" as shown here.

Having selected pipette, click on a color in the design using the left or right button to select a new foreground or background color respectively. The software will automatically re-select the previous drawing function.

You can also use the spacebar keyboard shortcut. This feature also has a preference setting which may mean that the Pipette feature is activated by the spacebar or that the color being pointed at is immediately selected.

24 Bit Designs

Note that if the design is a 24 bit design, clicking the design will automatically add that color to he palette (if it is not already there). As detailed above, the color will be associated with whichever mouse button is used.
