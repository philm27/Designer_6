Drawing Tool Preferences

 

Circle Drawing Preferences

Software versions prior to 5.6 had 3 circle drawing options.

A new improved option "APSO custom method " was introduced with version 5.6 which has better symmetry and better line thickness control.

It is therefore recommended that the APSO custom method is selected and the other, older, options are disregarded as they may be removed in a future version.

To select the custom drawing mode choose the menu option edit->preferences and select the "Drawing Tools" tab. The following dialog will be displayed.

 

 

Select APSO custom method, or de-select that in which case the older 3 options "Custom", "Windows", or "Even Stepping" will be available.

It is therefore recommended that the APSO custom method is selected for the best results.

This option, as with all other preferences will be stored and reloaded when you next shut down and restart the application.

 

Flat Buttons

This option will produce flat buttons in the toolbox as opposed to 3D buttons. Buttons will only become 3D when the mouse pointer hovers over them.
