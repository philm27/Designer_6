Line Thickness

If the current drawing function is one of the line or shape drawing functions then the brushes in the toolbox are replaced by a choice of line thickness options.

The Line Thickness options from 1 to 8 and Custom. Size 1 is selected in this example.

 

There is a custom size available if you wish to use a size larger than 8. To set a custom size, double-click or right-click on the drawing tool button itself (e.g. Bendy Line) or on any of the above line thickness icons. The following dialog will be displayed.

Example Custom Line Width dialog showing a chosen line width of 2 inches - for more details see the topic Line Properties

 

If you are working with a pen and graphics tablet, then you can control brush size by applying pressure with the pen on the tablet. You will first need to set the Digitiser Pressure Settings.
