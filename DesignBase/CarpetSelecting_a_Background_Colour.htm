Selecting a Background Colour

 

The background colour is the colour that is "live" in the right-hand mouse button.

You can select a new background colour by selecting it in the palette or by using the Pipette feature.

 

Using the Palette

Select a new background colour by clicking with the right hand mouse button on the desired colour in the palette. The swatch selected will immediately be marked with a "tuck" in the bottom right corner.

Also, the new colour will be shown in the small circular colour swatch at the top of the toolbox.

Palette Information Window

You can also click on a color in the Palette Information Window in order to select it

 

Using the Pipette

You can also use the Pipette (or eye-dropper) to select a background colour.

Select the Pipette tool then click on the colour in the design itself with the right hand mouse button. For more details see the topic Pipette.
