Edit Shelf Item

 

The design currently on your screen is normally the main design - the one in box 1 on the shelf. However, you can make any shelf item the "current" shelf and this enables you to view and or edit them in the normal way.

To make a shelf item the "current" one, use one of the following methods:

  • Click with the right-hand mouse button on the required shelf item to display the Pop-up shelf menu and select "Edit", or,

  • Select main menu option Shelf->edit Box and nominate the required shelf item number in the dialog that is then displayed, or,

  • Click with the right-hand mouse button on the required shelf box whilst holding down the <Shift> key.

 

Copying a shelf item to Box 1

If you want to copy a shelf item to the Main Design in box 1.....

  • Click with the right-hand mouse button on the required shelf item to display the Pop-up shelf menu and select Move->To Box 1, or,

  • Select main menu option Shelf->Copy to Box 1 and nominate the required shelf item number in the dialog that is then displayed.

The original version of the main design in box 1 will be lost but note that you can UNDO this operation.

Your palette will remain the same, no matter which is the current shelf as the whole project has a common palette.

The size etc of the current design can be seen at a glance in the status bar.
