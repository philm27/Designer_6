Zoom To Cursor

This feature is activated by selection in Preferences (General Tab).

When used it will cause the software to zoom while keeping the same design point under the cursor. This will be true even if you zoom right out as far as possible (zoom level x0.01) then zoom back in again to a high zoom level.

Note that it can cause the design to be displayed off centre since if only one repeat is being displayed or if the design is a rug, the software will be forced to display the design off centre as in the following example.

In this example the user has pointed at a location towards the top left of the design in a high zoom level and has then zoomed out. Since the user is not using Display Many, the software has displayed the design at the top left of the screen.

Cursor Outside the Design

If you wish to do a "normal" zoom in/out whilst in Zoom to Cursor mode, you can move the cursor off the design itself and use the + and - keys or the mouse wheel.

This also applies to using the numeric keypad to position the design bottom left, top left, top right etc. - you can move the cursor off the design and use the numeric keypad as normal.
