Line Removal

 

In much the same way as you can remove dots from a design, you can also remove or break up horizontal and vertical lines.

For instance, if you do not want to have any vertical lines greater than 6 points (ends rows or picks) in length in your design, you could request them to be broken up into smaller lengths or removed completely.

The software breaks up lines if required using whatever is the dominant color surrounding the line.

The software also allows you to choose normal lines, L shapes and/or T shapes and to choose all colors or just selected ones.

Note that you can also remove lines at the same time by clicking on the Line Removal Tab and selecting "Do Line Removal".

 

To access this feature, select the menu option Design->Line Removal. The following dialog will be displayed.

Example of the Line Removal Tab, in this case the user is removing lines from 3 to 5 pixels long, they are being removed completely (not just breaking up), normal, L shapes and T shapes are being removed in both horizontal and vertical axis. All colors has been selected.

 

Do Line Removal

Make sure this is selected. Note that you can do both a Dot Removal and Line Removal simultaneously and this check box together with a similar one on the Dot Removal tab allow you to specify whether one or both should be performed.

Line Size

Enter the inclusive range of line lengths to be removed. In the example above all lines having a length from 3 to 5 "points" will be removed.

Remove Completely/Break Up

Choose whether to completely remove the lines found or to just break them up into smaller lengths. These two options are mutually exclusive.

Normal

This should be selected if you want to remove normal straight lines. It is not possible to remove diagonal lines

L Shapes

Select this option if you want to remove or partially remove L Shapes.

T Shapes

Select this option if you want to remove or partially remove T Shapes.

Horizontal/Vertical/All

Select one of these mutually exclusive options. it is not possible to remove diagonal lines.

All/Selected Colors

If "Selected" is chosen, you must click on the Colors Tab (in the same way that it is used for Dot Removal) and specify the line colors to be removed.

 

Note

Note that when breaking up lines, they will be split into shorter lengths chosen at random from the range 1 point to one less than the "Remove Lines From" value and there will be either 1 or 2 points between these new lines.

If a line, L shape or T shape has a diagonal connection to some other shape it will not be removed.

Any lines wider than 1 "point" will be ignored.

For L Shapes and T shapes the line length refers to the individual lines making up the shape and they may be individually removed or broken up according to their lengths. This means that a T shape or L shape will only be completely removed (if "Remove Completely" is selected) if both lines making the shape have a length within the specified range.
