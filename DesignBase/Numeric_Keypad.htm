Using the Numeric Keypad

 

The keys in the numeric keypad (found on the right-hand side of your keyboard) are used by the software as shortcut keys in 3 different ways:

  • <ctrl> + <key>
 Set the Brush Handle
  • <shift> + <key>
 Draw the current Brush
  • <key>
 Pan to a Location

In each of the above <key> is a key on the numeric keypad

Set Brush Handle

The brush handle can be important particularly when drawing the brush using , say, <shift>+<key> as described below. For example <ctrl>+4 will set the brush handle to the middle left edge of the brush etc..

Draw the Current Brush

The Brush can easily be drawn in one of the 9 standard locations using <shift>+<key>. For example <ctrl>+<5> will draw the current brush in the centre of the design. Note however that the software will use the current handle to determine the exact location. For example, in the case of <shift>+<5>, if the brush handle is at the top left, then this point in the brush will be located in the centre of the design meaning that the brush will be located in the bottom right quarter of the design. Conversely if the brush handle were in the centre then the centre point of the brush will go to the centre of the design.

Pan to a Location

This is particularly useful if you are working on a large design or are working at a high zoom level, for example press the <1> key on the numeric keypad to move to the bottom left of the design, click on the <9> key to move to the top left etc.. Additionally you can use the <0> and the <dot> keys to move the corners of the repeat to the centre of the screen as described here.

By pressing any of the keys in the diagram, your design will pan immediately to the corresponding location. If the design is a repeating design then the edge of the design will be about one sixth of the way across the design window. If the design is a rug then the edge will be placed against the edge of the window.

Additionally the <0> key will bring the bottom left hand corner to the centre of the display and the <dot> key will bring the top right hand corner to the centre. If the design is a straight match then these two keys will have exactly the same effect.

 

For other information about scrolling see the topic Scrolling around the Design.

It is not necessary to use the <NUMLOCK> in order to use these shortcuts.
