Shade Library Start Page

Use the Shade Library feature to access shades that you have previously saved in your Colors & Yarns database, or that are saved in special text files.

To access the Shade Library option, select it from the menu (Palette->Shade File...) or press <shift>+S on the keyboard. This will display a Start Page similar to the following.

Library Tab

 

Example of the Shade Library property sheet - in this example 5 shade page options have been checked and are available for selection

The window will initially have only two tabs - Library and Plain and it defaults to this Start Page (the Library tab) when first opened.

Pages to Show - Place a check mark against any option in order to add the "tab" for that page. If you are using Jacquard Designer, the Ranges/Colorways option will add both a Warp and a Weft Ranges tab.

Use C+Y data base direct – select this to access the C+Y data base direct – users of Office 2013 may be able to do this without installation of any additional drivers – however, other users should contact their sales representative for further advice,

Use Text File – select this option to use a text file from the drop down list. This only applies to Plain shades and the shades from the text file will replace the Plain shades from the Colors and Yarns database if used. You can switch from one to the other several times but this setting is used when loading designs and getting the latest shade information so switching can cause problems.

Library - this is simply a name for the library. You can select any one of possibly several text files using the drop down list. The name and path of everything in the list can be modified using the Edit button (see below),

Path - this is just for reference here and is the text file path for the currently selected library. As mentioned above it can be modified using the Edit button (see below).

Shades – just for reference this is the number of shades in the selected file.

Options

Click on this button to show a menu of Shade Library Options.

Edit Button

Clicking on the edit button will allow the list of text files to be edited see the topic Shade File Location.
