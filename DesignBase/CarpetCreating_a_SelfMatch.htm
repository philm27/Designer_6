Creating a Self-Match

 

If you have a match-type other than a straight match e.g. half drop, it is possible to automatically convert it to a straight match version of the design using the menu option "Design->Create Self Match".

To do this the software may have to increase the width or length of the design. For example, if you are working on a quarter-drop repeat, the software will create a new design four times wider and repeat the original design into it four times to make a self match.

After selecting the option from the menu, the following message box will be displayed giving the size of the new design.

 

Example for a half drop design of 480x480 which would become 960x480 on conversion to a straight match.
