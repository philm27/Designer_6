Large Designs

Some special features in the software facilitate the creation of large designs.

Preferences

Uncheck the "Automatic Refresh Options" in Preferences. This will prevent the software re-checking the palette usage after every (possibly small) drawing operation.

See the topic "Automatic Refresh Options" for more details.

Note that if any of these preferences are not unchecked and a large design (>10MB*) is either created or loaded, then a warning will be displayed as shown below.

 

Low Memory Situation

A warning will be displayed if the recommended amount of memory is not available for the design size being created or loaded:

The message shown will vary in severity:

Severity

Condition

Message

1

Design Size Exceeds 25% of available memory

The design size (wid x len) is very large and exceeds the recommended maximum for the current available memory. You should close as many other applications as possible. Do you wish to proceed?

2

Design Size exceeds 25% of physical memory

The design size (wid x len) is very large and exceeds the recommended maximum design size for this machine. You will experience very slow responses. Do you wish to proceed?

3

Design Size exceeds physical memory

The design size (wid x len) is excessively large for this machine and you should not proceed.

 

Response Times Warnings

Irrespective of whether there is sufficient memory or not, warnings will be given if the Automatic Refresh Options are selected in Preferences and the design exceeds 10MB*.

Note that the "Frame Plan" option shown here is only applicable to "Carpet Designer"

You can either uncheck one or more of these options and click OK or you can simply click on CANCEL to dismiss the dialog.

Also you can check the "Do not warn me again" option in order to prevent this dialog reappearing during the current session. This may particularly apply if your PC is very fast with a large amount of RAM and therefore you are able to leave these options checked without degrading performance.

Palette Usage

If this is checked, the software will check the design after every drawing operation in order to establish which palette colours are used and will refresh the flags in the palette and the % figures in the Palette information window. This may cause slow responses on large designs.

Palette Stencil

If a "Palette Stencil" is in use (i.e. one created by selecting palette colours) then this will be refreshed every time a drawing operation is performed. This may cause slow responses on large designs.

Frame Plan

(Carpet Designer only) if this is checked, the software may scan the whole design after a drawing operation in order to refresh the frame plan. Again, this may cause slow responses on large designs.

 

* Design Size is simply width times length (i.e. ends rows or picks) to give the internal design size in bytes so a design of 10,120 ends by 13,440 rows or picks is 1.36MB approximately.
