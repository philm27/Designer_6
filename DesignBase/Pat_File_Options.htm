Pat File Options

 

When saving PAT files, under certain circumstances you may need to save the file with a slightly different palette numbering arrangement. Carpet Designer provides an option to save the file with what is known as a "zero" colour. Basically this means that the palette numbering would start from 0, rather than 1. (e.g. 0,1,2,3,4 as opposed to 1,2,3,4,etc.).

In a PAT file colour zero is a 'null' colour representing areas in carpet that are 'felling' or backing only for example in a circular rug the zero colour would be used in the area outside the circle which is not part of the rug.

For normal designs you should use the preference 'No Zero Colour' i.e. the palette does not contain the zero colour. However, if you have made your own zero colour in the palette however so that you can specify the 'dead' areas in the design you should select the alternative 'Save designs using the first palette entry as the zero colour'. In these cases you should use the first palette colour to indicate the 'dead' areas.

 

No Zero Color

You should use this setting for normal designs. It enables you to use all palette colors normally.

  • Go to menu option Edit->Preferences

  • Select the General tab

  • Tick "Save Designs with no Zero Color"

 

To save a design with Zero Color

If you have used the first palette colour to indicate 'dead' areas in the design

  • Repeat the first 2 steps above.

  • Select the option "Save designs using the first palette entry as the zero colour".

 

Click OK to apply these changes and close the Preferences dialog.
