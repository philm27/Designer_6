Popup Palette Menu

In fact there are 3 variations of the Pop-up menu that can be used by right clicking in the palette window:

 

This menu is displayed by right-clicking in the background of the palette rather than on any of the swatches or stencil/transparency buttons.

Stencil On - Switch stencil feature on or off (the upper row of ticks and crosses)

Transparency On - Switch the Transparency feature on or off (the lower row of colored buttons)

Swap Left/Right - Swaps the colors associated with the left and right mouse buttons

Sort by Usage - Sort the palette into usage order. You can select High to Low or vice versa on the sub-menu

Sort by Color - Sort the palette into color order. You can select Dark to Light or vice versa on the sub-menu

Remove Unused Colors - Note that if a color is used on any shelf item, brush, polygon, selector, threading sequence or frame plan it will not be removed

Refresh - Refresh the usage - this is only really relevant if you have opted not to automatically refresh the tuft counts and usage flags in preferences

Shade File - Displays the Shade File property sheet
