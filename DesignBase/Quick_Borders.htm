Quick Borders

Access his feature using the menu option Design->Quick Borders. It offers a method of creating plain borders by specifying their color and depth as described here.

There are 3 alternatives:

  • Rectangle

  • Circle

  • Ellipse

In each case the figure is assumed to be the same size as the design and the orientation is assumed to be normal. Other than that it is similar to the Create Standard Shape feature in Polygons.

Selecting any of these 3 menu options will display a property sheet having 3 pages as follows.

General Tab

The name can be anything appropriate and is useful if several polygons are to be created. A default name will be given if it is omitted. It is ignored if the figure is not added to polygons

 “Add to Polygons” can be checked to cause the group of polygons to be added to the current polygon set.

“Draw into current design”, if checked, causes the software to draw the borders as defined on the other two tabs – see below.

Colors Tab

Polygon Lines – this is the color that the software will use to draw the polygon lines. This will be disabled if you have not checked “Add to Polygons” as it is not required in that case. To cause the polygon to be designated as “Guide Lines” – simply click on the check box (as in this example), and to change the color click on the existing color (blue in this example) and select a new one from the standard dialog that will be displayed.

The other two colors control the way that the centre panel of the figure is to be drawn. If “Draw Outline” is checked, the software will draw a line round the centre panel in the specified color. If “Fill Panel” is checked, the software will fill the centre panel with the specified color. If either of these is not checked, the color will appear white with an “X” in the centre as for “Draw Outline” in this example. To change a color, click in the color box and select a new color from the resulting dialog. In these two cases the color must be chosen from the current design colors.

Borders Tab

Use this tab to specify each border color.

If you simply want the outline without the borders – literally a centre panel the same size as the design – uncheck the “Include Border Lines…”

Note that the Guide Line column will not appear if you are not adding to polygons.

To change any color, click on that color and select a new one from the dialog that appears. For Guide Lines you may choose any RGB value. For Line and Fill colors you must choose a design color. If you choose “No color” then the color will show as white with an X on it in this grid.

 

Guide – the color of the polygon lines

Line – the color of the “outline” for that border i.e. edge of rug for first border – this will be drawn 1 point wide.

Fill – the main border color

# - the number of times to repeat this border

Depth – the depth of this border – you can specify decimal places e.g. 2.5 if required.

Units – select from the drop down list which includes inches, cm and other standard units

Buttons

“Move Up” to move the current line up one place in the sequence (towards the outside of the figure).

“Move Down” if you wish to move the current line down the list.

“Convert All “ – click on this if you would like to change the units – for example if you wish to change to mm click on his button and select mm in the simple dialog that is then displayed.

 

Finally
