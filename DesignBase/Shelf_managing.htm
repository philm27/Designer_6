Managing the Shelf

 

Removing Unused Shelf Items

To quickly remove all empty shelves select the menu option Delete Empty.

To Delete Selected Items - Method 1

Select the first item to delete (using left, or, right button) so that it has a check mark. Then hold the <ctrl> key and scroll the list up and down (the easiest way is to use the Mouse Wheel), selecting each additional item to delete.

Finally, still holding the <ctrl> key, right click on one of the selected items and select Delete on the pop-up menu. A warning message will be displayed so you can cancel if necessary.

An example showing items 2, 3 5 and 6 selected for delete.

To Delete Selected Items - Method 2

Select the main menu option "Shelf->Delete...". A dialog will be displayed as shown above allowing you to multi-select and scroll through all shelf items.

Items are selected in the standard way by holding the <ctrl> key while multi-selecting. Click on OK to do the delete. A warning message will be displayed so you can cancel if necessary.

Changing Repeat/Match of a Block of Shelf Items

To do this select all the shelf items on the Shelf itself in the same way as for Delete Method1 above. Then right click any one of those items keeping the <ctrl> key down and select the new repeat match information on the pop-up menu.

For example it is often useful to be able to change a block of items to Rugs in which case you would select Repeat Type->Rug on the pop-up menu.
