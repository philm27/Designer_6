Quality or Sett

 

Use the specification dialog to set or check the quality or setting (ends or rows(picks) per standard unit).

Various standard units are available: inch, 3 inch, foot, yard, centimetre or 10 centimetres. The values themselves can be set to one decimal point.

You can change the values and/or the units at any time, although this will obviously distort your design work slightly. To amend the quality whilst keeping your design in the same proportions, you should redraft the design.

Note also that the current settings are always shown in the status bar.

 

Aspect Correct

Note that these settings are taken into account by the software when displaying the design but only if the Aspect Correct feature is selected. For more information see the topic Aspect Correct.
