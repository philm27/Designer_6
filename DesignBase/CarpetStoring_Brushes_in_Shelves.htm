Storing Brushes on the Shelf

 

By far the simplest way to store a brush on the shelf is to create a brush by any method then click on an empty shelf box with the left-hand mouse button.

However, if a box already contains an image, you need to either delete it first, or use the pop-up menu option "Brush To Shelf" which will delete the existing shelf item first if necessary.

The shelf pop-up menu is displayed by clicking on the shelf with the right hand mouse button.

When you want to re-select the shelf item as a brush, either:

  • Click on the appropriate shelf item with the left-hand mouse button

  • Select the pop-up menu option "Brush From Shelf"

  • Select main menu option Shelf->Copy to Brush. A dialog will be displayed in which you can specify the shelf item number.

Both of these operations select the item as the current brush whilst leaving a copy of it in the shelf for future use.

For further details about creating brushes, see the topic Create a custom brush.

 

Duplicate Shelf Item

The simplest way to duplicate a shelf item (i.e. copy it to another shelf item) is to make the item into the current brush by selecting it using the left-hand mouse button and then drop it into one or more empty shelf items, again using the left-hand button in each case.

You can even select the current design as a brush.

It can be useful to duplicate the "edit box" if you want to compare a design before and after an alteration. Duplicate the design before making the alteration, and then if you wish to return to the original version you can simply select your duplicate as the "edit box".

Some easy to use menu options are provided for copying Box 1 (the main design) to another box as this is a common requirement. To use this method, click with the right-hand mouse button on the empty shelf item to display the Pop-up shelf menu and select menu item Copy from Box 1 to copy the main design to that box. Alternatively, select main menu option Shelf->Copy from Box 1 and nominate which shelf box you want to copy to in the resulting dialog.
