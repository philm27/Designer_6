Brush Handle Default

 

Select a brush handle default by going to the menu option View->Preferences->Brush Handle.

Simply choose the handle which you would like to be the initial default for a new design or a design loaded from disk Â– either top left, top right, central, bottom left or bottom right.

You can still make temporary changes to the current handle whilst designing using the menu option Brush->Handle.
