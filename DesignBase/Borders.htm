Borders

 

There are several features available which allow you to create designs containing borders.

 

Ribbons, Runners and Border Designs

A piece of design which is intended as a border in its own right can be created as either a horizontal or vertical border.

This means that when you view the design using View Many, it will only be repeated Horizontally or Vertically. To do this select Horizontal or Vertical repeat in the Specification (repeat Type).

Note that you can also specify a drop match i.e. a Horizontal repeat can have a drop and Vertical repeat can have a drop i.e. shift.

 

Mirror and Wheel

If you have created a Half Rug or Scarf having a border and you need to create the whole design, you can use the design mirror or the design wheel option to do this.

 

Mitre Corners

If you need to create a corner which is mitered, you can use the mitering feature. This will effectively mirror a corner of your design along a 45 degree diagonal line taking into account the ends and rows(picks) per unit length.

 

Circular and Elliptical Borders

You can create a design having a circular or elliptical border using the Circular Border or the Elliptical Border features. All you need to design is one repeat of the border which should be a maximum size of one quarter of the whole thing.

 

Pasting Borders

Borders can, of course, be pasted in using normal brushes but the Numeric Paste feature has options to allow a brush to be repeated along one edge.
