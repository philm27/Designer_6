Moving Shelf Items to Another Project

 

If you are working on several projects at once, you may want to copy a design, brush or shelf item from project A to project B.

There are two ways to do this

Using Cut, Copy and Paste

Right click the shelf item in project A and select either Cut or Copy on the pop-up menu as appropriate.

Move to Project B and right click the empty shelf item where you would like to place the image.

Select Paste on the pop-up menu.

For more information about this process see the topic Cut Copy and Paste.

Use the Import features

In Project B, right click the empty shelf item and select "Import->From Other Project" on the Pop-up menu

You can select a main design or a shelf item using the resulting dialog.

For more information about this method see the topic Importing to Shelf.
