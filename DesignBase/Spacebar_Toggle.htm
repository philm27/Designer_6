Spacebar Toggle (Line Direction)

 

Spacebar toggle is a useful drawing feature which allows you to change the direction of certain drawing operations such as Line Draw.

For example, when drawing a simple short line from one point to another you can force the software to draw the line across the design horizontally so that it connects the start point with the finish point but by a different route across the edges of the repeat.

This feature is particularly useful for drawing a straight line from one corner to the other across a repeat. It is also useful for drawing a rectangle forming a border around the edges of a design. In these cases the operation can be performed whilst the design is zoomed in, with a grid on for accuracy and without the necessity to scroll to the opposite corner as both corners can be on screen at the same time.

Note that the spacebar normally operates the Pipette feature if you are not performing a drawing operation.

Procedure

To activate this feature simply press and release the spacebar whilst the mouse button is still down during a drawing operation. For example whilst the button is still pressed during a line draw operation, press and release the spacebar to force the line to be drawn in a different direction. There are actually 4 different directions (including the normal one) for each line draw operation as illustrated in the example below.

You can press the space bar as many times as you like and the mouse can be moved at any time. The line (or rectangle etc.) will not be fixed until you release the mouse button.

The spacebar can be used during line, circle, ellipse, rectangle or square drawing operations. It can also be used whilst cutting brushes.

 

Example 1 Line Draw

The following is a small section from a large straight match design. The corners of the design have been brought to the centre of the screen using the "0" key on the numeric keypad. A line has been drawn from end 1 row (pick) 1 to a point below and left which is actually at the top right of the design.

(A)

This shows the normal line before the spacebar is pressed.

 

(B)

This shows the line after the spacebar is pressed once. The line still connects the two points but it goes to the right and downwards at a shallow angle so that it eventually connects to the second point.

(C)

This shows the line after the spacebar is pressed a second time. The line still connects the two points but it goes vertically and slightly to the left at a shallow angle so that it eventually connects to the second point.

(D)

This shows the line after the spacebar is pressed a third time. The line still connects the two points but it goes diagonally upwards and to the right from the first point so that it eventually connects to the second point at the top right of the same repeat. As the line is still under mouse control it is easy to move the second point so that it is actually at the exact top right of the design repeat.

Note that if the spacebar is pressed again the line will be back to the normal line in image (A) above. In fact you can press the spacebar as many times as you like to cycle through all 4 variations several times

Note also that you can move the mouse at any time to get the required line. The line will not be fixed until you release the mouse button.

 

Example 2 Rectangle Draw

As above, the following is a small section from a large straight match design. The corners of the design have been brought to the centre of the screen using the "0" key on the numeric keypad. A rectangle, thickness 4 has been drawn from end 1 row (pick) 1 to a point below and left which is actually near the top right of the design.

(A)

This shows the normal rectangle before the spacebar is pressed. The start point is at the top right of the rectangle and the finish point is at the bottom left of the rectangle.

(B)

This shows the rectangle after the spacebar is pressed once. The two points are still at the corners of the rectangle but the start point (end 1 row (pick) 1) is now at the top left and the finish point is as the bottom right.

(C)

This shows the rectangle after the spacebar is pressed a second time. The two points are still at the corners of the rectangle but the start point (end 1 row (pick) 1) is now at the bottom right and the finish point is at the top left of the rectangle.

 

(D)

This shows the rectangle after the spacebar is pressed a third time. The two points are still at the corners of the rectangle but the start point (end 1 row (pick) 1) is now at the bottom left and the finish point is at the top right of the rectangle.

(E)

As the line is still under mouse control it is now easy to move the second point so that it is actually at the exact top right of the design repeat. The button can now be released

 

Note that if the spacebar is pressed again the rectangle will be back to the normal one as shown in image (A). In fact you can press the spacebar as many times as you like to cycle through all 4 variations several times

Note also that you can move the mouse at any time to get the required rectangle. The rectangle will not be fixed until you release the mouse button.
