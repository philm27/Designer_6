Re-sizing a Custom Brush

 

A custom brush can be easily re-sized using this menu option.

Having created a brush (either create a new one or select from the shelf), select menu item Brush->Size then one of the following options from the sub-menu:

Menu Option

Pointer

Procedure

Any

Click and drag with the left-hand mouse button and you will find that the brush is re-sized under the control of the mouse. The aspect of the brush will be retained. Release the mouse button when the brush reaches the size you require. If during this process you wish to vary he aspect ratio you can press and hold the <ctrl> key. You can revert to the original aspect ratio by releasing the <ctrl> key.

Horizontal

The brush starts with its full length but only 1 end in the width. Click and drag left or right with the mouse until the brush reaches the dimensions you require then release the mouse button.

 

Vertical

The brush starts with its full width but only 1 row in the length. Click and drag lengthways using the mouse until the brush reaches the length you require.

 

 

It is useful to turn the Information Panel on when resizing a brush in any of the above ways, as the panel will give you the size of the brush in ends and rows as you resize.

The remaining options on the Brush->size sub-menu do not require use of the mouse to control the new size.

Half

The brush is automatically re-sized to half its original size.

Double

The brush is automatically re-sized to double its original size.

Specify

The "Brush Size Change" dialog will be displayed as shown below. Simply enter the new size and click on OK. The image quality will be improved (when you are increasing the design size) if you select "Enhance Image using SmartDraw". For some examples of the effect of SmartDraw see the topic Resize.

 

Example of the Brush Size Change dialog. The original size in this case is 92 x 89 and the new size is to be 120x120 (all dimensions in ends and rows (picks)). SmartDraw is to be used.
