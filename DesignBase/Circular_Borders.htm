Circular Borders

 

This option enables you to create a circular border using a brush or a shelf item.

You should first of all create a single repeat of the border. You can orient this any way you like i.e. with the outer edge at the bottom, top, left or right.

You may wish to calculate the required length of that outer edge. Fortunately, this calculation is performed by the software and you can display the dialog before finishing the border design in order to check the ideal size for the border element. In fact there are 3 "ideal" calculations performed as detailed below:

Select menu option Power Tools->Border->Circular and the following dialog will be displayed.

You may click on the click on the image below to find out more.

Example Circular Border Dialog set for 3 repeats of shelf item 3 per quarter. The shelf item is oriented so that it's bottom edge is the outer edge of the circle. All quarters are to be rendered. Render Quality is High and the border is to be centered in the main design. The outer radius of the border will be 576 ends i.e. half the main design width (the dialog initially defaults to this figure).

Note that the Ideal radius for a brush of this width and 3 repeats is 550 ends and conversely, the Ideal repeats for a brush of this size and a radiius of 576 ends is between 2.4 and 3.1. The ideal Brush width for 3 repeats and a radius of 576 ends is 302 ends.

Source

Use the drop down list to select either the current brush or a shelf item.

Orientation

Use the coloured button (representing a border element) to select a top, bottom, left or right edge. This is important for most brushes as it tells the software which of the four sides of the brush is to be the outside edge of the circular border.

Repeats

Specifies the number of repeats of the brush to be placed in each quarter. This must be a whole number so the minimum value is 1.

Quarters to Render

These four buttons show an stylised circular border split into four quarters. If you click on any one of them with the left mouse button it will be deselected. Click again to re-select it. If a quarter is not to be drawn it will be shown in shades of grey.

Render Quality

You may set this to High Medium or Low. High will take longer to render but may improve the result.

Radius Type

This drop down list allows selection of Inner or Outer to indicate whether the radius specified is the inner or outer measurement.

Radius

May be specified as Ends or Rows or as two separate figures for feet and inches or metre and centimetres.

Radius Units

This drop down list allows selection of one of four options: Ends, Rows, Feet/Inches or Metres/cm. In the latter two cases two fields are shown for the two figures as in the example above. Only one field is shown if Ends or Rows(Picks) are to be used.

Centre Type

You can select either "Centre in Design" in which case the software will automatically set the centre of the circle to the centre of the design. For this option the next two items are not necessary and are disabled. Alternatively, you can select "Centre as Below" in which the X and Y coordinate of the centre must be specified.

X =

This is the X coordinate of the centre of the circle. It may be specified in Ends, Feet/ins or Mtrs/cm. It represents the distance from the left edge of the design.

X coordinate Units

This is the units for the X coordinate. You may select Ends, Feet/ins or Mtrs/cm.

Y =

This is the Y coordinate of the centre of the circle. It may be specified in Rows(Picks), Feet/ins or Mtrs/cm. It represents the distance from the bottom edge of the design.

Y coordinate Units

This is the units for the Y coordinate. You may select Rows(Picks), Feet/ins or Mtrs/cm.

 

Draw Button

Select this to cause the circular border to be drawn.

Undo Button

Select this to undo the circular border. Note that it has the same effect as the normal undo button on the main design window and therefore you can undo as many drawing operations as you like even those not done using the Circular Border Dialog.

Close Button

Use this to close the Dialog.

 

Ideal Data

Three separate "Ideal" figures are shown as described below. These figures are only a guide and do not have to be used.

You would in any case not use all of them simultaneously. You may for example adjust the radius so that the brush is not compressed or stretched quite so much. Or, you may adjust the number of repeats per quarter to a more suitable figure.

Lastly you may decide to keep the latter two figures the same but close the dialog, redesign the brush to a more suitable size and then create the border at a later time.

Ideal Radius

This will change if the 'Number of Repeats per Quarter' changes.

The figure is calculated so that exactly the correct number of repeats would fit along the outer edge in each quarter.

Ideal Repeats

This figure is calculated using the brush size and the current radius. It will change if the radius is changed.

Two figures are always shown. The first figure is the number of repeats which fit along the inner edge of the border in each quarter. The second figure is the number of repeats that fit along the outer edge of the border in each quarter.

Since these figures will rarely be a whole number, they are shown to 1 decimal place.

For example if the text shows "Ideal Repeats 4.5 - 4.9" it means that the number of repeats of the border element which fit along the inner edge of the border for one quarter is 4.5 and along the outer edge 4.9. There is a difference of course because the inner edge is shorter than the outer edge. Thus if you set the number of repeats to 4, the brush will be stretched by about 12.5% on the inner edge but only 2% on the outer edge. If you set the repeats to 5 then it would be compressed by 10% on the inner edge and compressed by 2% on the outer edge.

Sometimes the inner edge can be compressed whilst the outer edge is stretched.

The figures give you an idea of how much the software will have to stretch or compress the border element in order to fit it into the specified circular figure.

 

Ideal Brush

This will show the ideal width or length of the brush depending on whether the border element is a top, bottom, left or right border section.

If the border element (i.e. the brush) is a left or right edge then the ideal length will be shown.

If the border element is a top or bottom border then the ideal width is shown.

The figure is calculated using the specified radius (to outer edge) and the repeats per quarter.
