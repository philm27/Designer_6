Specification Dialog

 

This dialog displays the specifications of your design and allows you to make changes.

You can open the specification window in two ways:

  • Select menu option Design->Specification.

  • Click with the right-hand mouse button outside the normal working area of your design. A Pop-up menu will be displayed. Select Specification.

 

The following is an example of the specification dialog. You can click on it to find out more.

 

 

When the dialog is first displayed, the Repeat Type will be the "active" field. This is the field which will be changed if the keyboard is used. Only one field will be active at any one time.

To move to the next field you can use the tab key and in the case of "edit controls" this will also select all the text. Also, you can make any field the active one by clicking on it with the mouse (left button).

Repeat Type

Select a repeat type of Normal, Horizontal, Vertical, Rug or Brick

Match

If you have set the repeat type to Rug, this field will be disabled as it is not required. However, for other types you can set the match type to straight match, half, quarter or third "drop". See the topic Selecting a Match Type.

Design Size

Select the units for Width and Length from the corresponding drop down list on the right. Then enter the width and length of the design in either Ends (or rows/picks), Feet and inches or Metres and Centimetres.

See the topic Setting the Size of a New Design for more information.

Ends x Rows (quality or setting)

Type in the values for the ends and rows (picks) per unit measurement and then select the units from the options in the drop down list. See the topic Setting the Quality.

Grid

This information controls how often a bold line occurs when a grid is drawn. You may want a bold line every inch, every centimetre or at some other interval like 8 ends. See the topic Bold Grid Lines.

Notes

You can enter any information you like into these two fields. This information will be saved with the design and recalled every time the design is re-loaded.

Repeat Size

This shows the design size in both Imperial (Feet and Inches) and Metric units. It is recalculated every time changes are made to the size or to the Quality (Setting).

Set Default

Select this button in order to set the current size, setting and grid information as the default for a new design. The default will remain until it is reset at a future date by the same method. See the topic Design Size Default.

Numeric Information

You can edit the numeric information as follows:

  • Use the up and down arrows at the right-hand side of the boxes to increase/decrease details. if you hold the button down the value will repeatedly increase or decrease until the button is released (or until the value hits its maximum or minimum).

  • Swipe the number with the left mouse button to highlight it and then type a new value using the keyboard.

  • You can also tab from one field to the next which will automatically highlight the next field so you can immediately re-type it if you wish.

 

Drop Down Lists

When an entry box has a drop down list you can click on the down arrow at the side of the entry box to display the list of possible values. Simply click on the required value with the left hand mouse button.

Alternatively you can use the keyboard. Simply make sure the field is "active" by clicking on it with the mouse or by "tabbing" to it using the tab key. Then use the up and down arrows to select one of the options and finally press tab to move to the next field.

Notes

You can edit these fields as follows:

  • Select the text to be changed with the left mouse button to highlight it and then type new text using the keyboard. The new text will replace the old.

  • Simply click at the location at which you wish to insert one or more words. The flashing "caret" will be displayed at that position. You can then type the new information.
