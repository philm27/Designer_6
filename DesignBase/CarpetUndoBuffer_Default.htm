Undo Size Warnings

 

You will find the Undo Size Warning option in Undo Preferences (menu option Edit->Undo Preferences->Warnings) on the Warnings tab.

A drawing operation such as a short line or short freehand operation with a small brush does not take up very much memory in the undo buffer. However, if the design is very large, some actions such as fill, global colour change or large circles for example, can use up large amounts of memory.

For this reason, the undo buffer warning gives you the opportunity not to commit a design alteration to the undo memory and you can set the design section size that will cause a warning to appear.

Note that you may obviate this problem to a large extent by requesting that undo history items are compressed - see the Advanced tab in undo preferences.

In the Undo tab, select a default Undo Buffer Warning Size  the options are

  • No warnings  not normally recommended as you may run out of undo memory without prior warning

  • Large (20MB)  display a warning if the operation would use 20MB or more of memory.

  • Medium (10MB)  display a warning if the operation would use 10MB or more of memory.

  • Small (5MB) - display a warning if the operation would use 5MB or more of memory.

 

If a warning does appear whilst you are working, it means that the alteration you have just made to the design will take up more undo memory than your default selection.

 

  • YES - If you want to be able to undo the operation - click on Yes.

  • NO - If you do NOT want to be able to undo the operation, thereby saving memory - select No.

  • CANCEL - If you would rather abandon the drawing operation - select Cancel.

 

The CANCEL option is useful if you would like to save the design to disk before performing the drawing operation.
