Generate Random Pattern Multi Color

This feature allows the user to create a random pattern using multiple color choices

You may choose a dot size and a number of colors which may be mixed in any proportion.

If the dot size width or length exceeds one you may choose to have the dots aligned.

To use this function choose the menu option Power Tools->Generate Random Pattern->Multi-Color.

Details on how to use the Dot Size page can be found in the 2 Color page

Note that the algorithm used for multi-color is different to that for the 2 color random pattern so you may get a different "feel" to the patterns produced by this method when used for 2 color patterns.

 

A An example of the Multi Color Page - the resulting image will be color 4 - 30 parts, colors 6, 7, 8 - 10 parts each

 

The Color List

  • You can click anywhere in the list on an existing % value and it will be highlighted, ready for typing a new value.

  • You can move up and down the list using the keyboard arrow keys and begin typing a new value on any line.

  • Press <return> on any line to begin or end typing.

The actual values do not need to be percentages - it is only the relative values that matter so in the example above the percentages will actually be 60, 20, 20, 20 as the values given are 30, 10, 10, 10 which add up to 50. So for 3 equal colors you can enter 1 against each of those 3 colors to get 3 equal proportions.

Dot Size

Click on the Dot Size tab and either type a dot size or use the Up/Down arrows to modify the values. You can use values up to 99 at present. For more details of the Dot Size Page see the Topic Generate Random Pattern 2 Color.

OK Button

When this is clicked, the software will generate the image. You can click it several times in succession to try and get an ideal result. If you decide to go back to a previous image you can click the Undo button.

Undo

Use this to go back to a previous image. You do not need to click on Undo in order to generate a new pattern to replace the previous one as the prevous one will simple be overwritten anyway. However, this operates like a normal Undo, so you can go back several steps whether they are drawing operations or Random Patterns.

Close

Use this to close the Property Sheet. If you later decide to undo the Pattern you have created you can use the normal undo procedure.
