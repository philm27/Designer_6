Duplicating Ends/Rows

 

Use duplicate ends or rows (picks) to copy a full length, or full width section of a design a specified number of times.

For details of the Duplicate Full Width or Full Length option click here.

Procedure

To use this feature:

  1. Select Design->Duplicate->Ends. The mouse pointer will change to an upward pointing arrow.

  2. To duplicate a single end, point at the end in the design to be duplicated and press and release the left mouse button. The dialog below will be displayed.

  3. To duplicate a block of several ends, point at the first end (left or right), press and hold the mouse button and move left or right to the opposite end of the block and release the mouse button. The following dialog will be displayed.

  5. Simply enter the number of repeats to be added and click on OK.

  6. To exit the Duplicate Ends function simply select another drawing function.

 

Duplicate Rows

Proceed as above as described for Ends except that the menu option is of course Design->Duplicate->Rows (Picks). The mouse pointer will change to a leftward pointing arrow to make it easier to select a row in the design.

When selecting a series of rows for duplication you should press and hold he mouse button whilst pointing at the first row in the block then move the pointer to the last row in the block and release the mouse button.

A similar dialog to the one above will be displayed. Simply enter the number of repeats and click on OK.

Creating a Large Rug from a Basic Rug

This is a particular technique which uses Duplicate Ends and Rows to good advantage to build a large rug of any size from a basic rug.

Many rug designs consist of a repeating border element and a repeating field of the same width and length. For example, the following is a basic rug having a single repeat in the centre panel.

Basic Rug having one repeat of the centre panel and a border (excluding corner element) which has the same width (at top and bottom edges) and length (at left and right edges)

Let us say for example that we want to make a rug having 4 repeats in the width and 6 in the length, we can proceed as follows:

First select Duplicate Ends and select the width of the centre panel. The following dialog is shown allowing 3 additional repeats to be specified widthwise.

Example shows the repeating ends selected for the Duplicate Ends operation

Click on OK to complete the operation and the rug will now look like this:

The same rug (at a different zoom level) after the above Duplicate Ends operation. There are now 4 repeats of the centre panel across (and 4 corresponding repeats of the top and bottom border element).

Then all we need to do is repeat the panel lengthwise. To do this select Duplicate Rows (Picks) and again select the rows where the centre panel occurs. The rows will be highlighted and the following dialog shown allowing specification of 5 additional repeats.

This again shows the centre panel repeat selected together with the border element in the left and right border. The number of additional repeats has been set to 5 i.e. there will be 6 repeats in total.

Click on OK to complete the operation and the final rug will look like this.

The final version having 4 repeats widthwise and 6 repeats lengthwise. (Shown at a different zoom level).
