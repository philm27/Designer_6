Frame Options

These details are only applicable to users of Carpet Designer.

 

Design Load Options

The two options shown are mutually exclusive:

Load the frame... This option will cause the program to always load the frame plan, if present, when loading a design. Any existing frame plan in the current project will always be removed.

Offer Selection.... This option, will cause the software to offer the option, when loading a design, of using the incoming frame plan or keeping the current one, if any, in the current project. (For this to work properly the two palettes should be identical).

It is recommended that "Load the Frame" is selected as there are comprehensive features for loading existing frame plans (menu option Frames->Load Frame Plan).

 

Locked Frame Plans Options

Again, the three options shown are mutually exclusive. They allow frame plans to be locked immediately on loading and are therefore useful when creating a range of designs using the same frame plan.

Never Lock... Frame plans will always be unlocked on loading.

Always Lock... Frame plans will always be locked on loading.

Offer the Option... If selected, the software will offer the user the option of locking or unlocking the frame plan (if there is one) when loading a design.
