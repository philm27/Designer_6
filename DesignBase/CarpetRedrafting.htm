Re-draft

 

Use redraft to change the quality (Ends and/or Rows (Picks)) of your design whilst keeping it the same physical size in real measurements (inches or cm).

To access this feature select menu option Design->Redraft.

The following dialog will be displayed.

 

An example of the Redraft dialog. Note that the "Smart Draw" option will always be selected by default.

Enter your new design quality in the boxes provided. The old figures are shown only for reference. Click on OK to redraft the design.

The "Per" can be changed simultaneously as well as the ends and rows (picks) values e.g. from 8 by 8 per inch to, say, 3.2 per cm by 4.2 per cm.

Smart Draw

Note that you should normally select Smart Draw as it gives a better result when increasing Ends or Rows (picks). Remember that the new drafting is computer-generated and may still need a designers' eye to perfect it, (see the example below).

This option is ignored if both figures are changing to smaller values. (i.e. there will be less ends and rows (picks) in the resulting design).

Insert Blank Lines

This does just what it says the lines being inserted in the background palette colour. If, for example, you double the number of rows, a blank line will be inserted after every row.

Again, this option is ignored if there are less rows (picks) in the resulting design.

Keep Proportions Correct

You can check this option in order to automatically keep ends and rows (picks) in the same proportions as the original figures.

Change "Per" Only

You can check this option in order to simply do a conversion from one system to another. In this case the ends and rows values will be calculated automatically and their edit boxes will be disabled. Since this is a conversion of units - no actual changes will be made to the design.

 

Note that it is perfectly valid to increase one figure whilst decreasing the other.

The new design quality will automatically be updated in the specification window . However, the grid settings will remain unchanged.

 

Smart Draw Example

The following illustrates the effect of Smart Draw. Remember that it only has an effect when the number of ends and/or rows (picks) is being increased.

Original design section before Redraft used. The number of ends and rows (picks) per inch is to be doubled.

Same section after Re-size with double the ends and rows (picks) per inch but Smart Draw not used. Note the 'blockiness' due to doubling of each end and row (pick).

Same section after Re-size with double the ends and rows (picks) per inch. In this case Smart Draw HAS been used. Note the smoother edges to the motifs when compared with the version above.
