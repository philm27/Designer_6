Create new Shapes - General

This is the General tab on the Create New Shape Property Sheet. Use it to specify the name of the shape. Also on the tab you can choose whether to add the polygons to  the current "collection" of polygons for this design and also whether or not to render the shape into the design.

For more general information see the topic Polygons - Standard Shapes.

An example of the general tab - the user has changed the name of the shape.

Name

Enter any suitable name. If this is omitted, the software will create a suitable name, when and if, the shape is created such as "Rectangle with Borders" for example.

Add to Polygons

Check this box, as in the illustration above, in order to cause the polygon group to be added to the list of all current polygons when the OK button is clicked.

Draw into current design

Check this box, as in the illustration above, in order to cause the shape to be rendered into the current design when the OK button is clicked.

Preview

The polygon preview, which is present on every page, will always show the current shape as defined on the various tabs in this property sheet. The four registration marks (if shown) indicate the edges of the repeat.
