Polygons - Create new Shapes - Size

This is the Size tab on the "Create New Shape" Property Sheet. Use it to specify the size of the shape (before any rotation takes place).

For more general information see the topic Polygons - Standard Shapes and for some examples of different shapes see the topic Shapes.

An example of the Size tab - the user has selected "Width as Design" and "Length as Design" as this is the simplest way to make the shape fit to the design size. Some shapes cannot always fill the design size such as regular polygons and circles.

Units

Choose the units for the figures in the dialog from the drop down list (Mtrs, cm, mm, Yds, ft, ins).

Radius

For circles and Ellipses (and arcs and chords) you can specify the radius of the figure (only one axis for a circle). This figure will automatically change in tandem with the Size figure. For regular polygons you can specify the length of the radial lines here. For Arcs and Chords the figure is the radius of the circle or ellipse on which the arc or chord is based.

Size

For circles, only the width of the figure needs to be entered. For all other figures, both a width and a length can be specified although for some shapes they are tied together such as a regular pentagon 5 sided figure. Sizes are always specified before rotation. For Arcs and Chords the figure is the size of the circle or ellipse on which the arc or chord is based.

Width as Design

Check this box to change the width of the figure to that of the design. For some figures such as Circles and Regular Polygons, this will also change the length.

Length as Design

Check this box to change the length of the figure to that of the design. For some figures such as Circles and Regular Polygons, this will also change the width
