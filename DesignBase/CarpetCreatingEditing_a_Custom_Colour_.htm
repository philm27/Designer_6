Palette Change

 

When you create a new design it will be set up with a default color palette. The initial palette can be modified using this dialog, or by adding shades from a shade file or data base.

Access the Palette Change window using one of the following methods:

 

  • Select Palette Change from the Palette menu

  • Double click anywhere in the palette window

  • Click on the palette button, which is located in the toolbox or toolbar

  • Use the keyboard shortcut key F6

Note that the palette can also be modified using the Shade Library.

Sliding Controls - General

The sliding controls will show the current color value as either an RGB value or HSV value depending on the setting of the HSV check box.

They may also show the average value of several colors if you have selected either "All" or "Selected" modes and in this case any changes will affect all colors (or selected colors) i.e. they will all get darker, lighter or more saturated etc.

The background colors for each slider, whether in RGB mode (as above) or HSV mode, will always show the color that will be created if the sliding control is moved to that position. For example if the Red slider in the above example is moved to the right, the color will tend to a pink, if the Green slider is moved to the right, the color will tend to a Cyan.

Using the Sliding Controls

To change a color value using a sliding control, do any of the following

  • Click and hold the left mouse button on the slider "button" and move left or right. Release the mouse button to set the value.

  • Click on the left arrow at the left end of the slider or the right arrow at the right hand end to move the value one position down or up respectively.

  • Click to one side of a slider button to move it several positions towards the cursor.

Color Swatches

The upper swatch will show the selected colour before any changes using the sliders, the lower swatch is the new color. These will show the average color if either All or Selected colors is selected. These swatches also show the palette number (3 in the above image).

Note that you can also click in the upper or lower swatch, to get a standard RGB selector which also allows the RGB value to be typed.

Mode Single/All/Selected

You can choose to modify several colors at once by choosing All or Selected and you can use RGB or HSV mode for this. In Single mode the colour is chosen in the main palette as normal. However, in Selected mode, the colors are chosen in the Palette list control displayed by clicking on the More (>>>") button. Note that when you click on "Selected", the palette will automatically be shown.

For more details on changing multiple colours see the topic Multi-Color Palette Changes.

Example of the dialog in HSV mode

HSV

Use this check box to switch between RGB mode and HSV mode (see the topic RGB and HSV Color Values for more information about using RGB and HSV values).

In HSV mode the top slider controls Hue and shows the various colors that can be achieved at the current saturation and value settings. The second slider shows Saturation and the third Value (i.e. Brightness or Luminence). In each case the value of the slider is shown as a number in the range 0 to 1.0 to 3 decimal places of accuracy.

The remaining buttons are given here in the order they appear in the dialog...

Normal Button

Use this to cancel any current action like Merge, Move or Copy etc. The software will revert to the "Eye-dropper" function if this button is used.

Add

Use this button to simply add another color to the palette. The maximum palette size is 255 colors The initial value of the added color will be the same as the current foreground color After the color has been added it will automatically be made the current foreground color so it can immediately be edited.

Merge

Use this button if you wish to merge two colors. To use this feature, first select the Merge button then select color 1 in the palette or in the design using the left mouse button. A special cursor will be displayed indicating that color 1 is being selected (the "from" color). After color 1 is selected, the cursor will indicate that color 2 is being selected (the "to" color). Again, use the left mouse button to select a color. After this the software will change all occurrences of the first color in the main design to the second color.

 

 Whilst using this function a special Merge 1 or Merge 2 pointer will be displayed. Note also if choosing colors from the design itself, the software will automatically show the "Eye-dropper" display i.e. the palette number, RGB value and shade code (if any).

After the second color has been selected, the following dialog will be displayed.

Example of the Merge Color Options Dialog. This allows you to control whether or not colors are to be merged in the main design, brush or shelf items.

The dialog above will not be displayed if there is no custom brush and no additional shelf items.

After the merge has been completed, the pointer will change back to a "From" pointer so that another 2 colors can be merged.

If you do not want to merge any more colors click the "Normal" button, press the <esc> key on the keyboard or select a different palette change button or a drawing function etc in order to exit the Merge function.

 

Move

Select this button in order to move a palette color from one position to another. This does not change the design but instead allows a manual re-ordering of the palette. The color at position 1 will be moved to position 2.

As for the Merge function, the cursor will indicate which color of the pair is being selected.

Also as before, click the "Normal" button, press the <esc> key on the keyboard or select a different palette or drawing function when you have finished moving colors.

 

Swap

Select this button in order to swap the location of two palette colors They will also be swapped in the design so, for instance, if the background color in the design is color 1 which is white and it is swapped with color 2 which is red then the background color after the swap will be red (which is now color 1 in the palette). See also "Swap and Keep" below.

As for the Merge function, the pointer will indicate which color of the pair is being selected.

Also as before, click the "Normal" button, press the <esc> key on the keyboard or select a different palette or drawing function when you have finished.

 

Swap and Keep

To do Swap and Keep, proceed as for Swap above but hold the <ctrl> key whilst selecting the second color. This will modify the design itself but will keep the same palette order.

Note that in order to undo this operation you should use the Design Undo option rather than a palette undo as the software will have effectively done two Global Color changes on the design.

Also note that neither the brush nor other shelf items will be modified. However, for Carpet Designer users, the Frame plan, if any, WILL be modified.

 

Range

Select this button in order to create a range or "run" of gradually changing colors in the Palette.

For more information see the topic Color Range.

 

Copy

Select this button to copy an RGB value from one color to another. Note that the shade code, if any, will not be copied.

As for the Merge function, the pointer will indicate which color of the pair is being selected.

Also as before, click the "Normal" button, press the <esc> key on the keyboard or select a different palette or drawing function when you have finished.

 

Del

Select this button if you wish to delete a palette color

The pointer will change to a special Delete cursor whilst colors are being deleted. Simply click on each color to be deleted in either the palette or the design.

Note that the software will not allow you to delete a color which is used in the main design nor in a custom brush or shelf item.

To remove all unused colors in your color palette automatically you can use the menu item "Palette->Remove Unused Colors".

 

>>> (More)

This button will cause the Palette list control to be shown as an extension to this dialog. You should use this to select the colors to be modified when using Selected mode. In other cases it is only for information. It will be shown automatically when you click on Selected if it is not already being displayed.

 

The Palette List

The palette list must be used to select colors for the Selected mode. See the topic Multi-Color Palette Changes.
