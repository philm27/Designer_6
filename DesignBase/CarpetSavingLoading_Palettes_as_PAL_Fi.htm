Saving/Loading Palettes as PAL Files

 

PAL files

This special file type has been provided to hold palette information. One palette can be held in each PAL file.

Although you can load a palette from IFF PAT and BMP files, PAL files are the most efficient way of saving palettes as there is no design included in the file. This keeps the file size as small as possible (usually 5KB - 8KB).

Note that a PAL file also contains details of any shade numbers.

 

Loading a Palette

You can load a palette from an existing design file (formats IFF/PRJ, PAT, BMP) or from the special PAL palette file type.

To load a palette into a design:

  1. Select menu option Palette->Load. The Load Palette dialog will be displayed.

  2. Navigate to the appropriate folder in the usual way

  3. Select the file type from the drop down list if necessary.

  4. Select the file

  5. Click on the Open button or double-click on the file name.

  6. For more information about the "Open File" Dialog see the topic Open Design.

Saving a Palette

To save the current palette and shade information:

  1. Select menu item Palette->Save. The "Save As" dialog will be displayed.

  2. Navigate to the appropriate folder in the usual way.

  3. Type a name for the file. You do not need to add the file extension (.PAL) as it will be added automatically.

  4. Click on the Save button.

It may be useful to create a special folder for palette files if they are to be kept permanently and/or used on a regular basis.
