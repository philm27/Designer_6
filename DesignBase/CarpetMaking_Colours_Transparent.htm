Transparency

 

Normally, the program allows only one transparent colour in a custom brush - the right hand button colour. However, the Transparency feature described here allows for any number of colors to be transparent.

Example of a palette showing transparency buttons (bottom row of buttons). In this example colors 1 and 3 will be visible in a brush the other colors marked with a diagonal hatch pattern will be transparent. (The top row of ticks and crosses are the stencil buttons).

 

Procedure

To make colors transparent in a custom brush:-

  1. Select menu item Palette->Transparency->On. Transparency buttons will be displayed beneath the colour swatches in the palette.

  2. Select or de-select transparency buttons as required (button "up" is opaque, "down" is transparent). You will see the effect of any changes immediately if you have a custom brush selected.

 

Other Transparency Menu options

Having turned the transparency feature on, you can make the following selections from the Transparency menu,

  • Select All  makes all palette colours transparent. Use this function if you have a large colour palette and want most of the colours to be transparent. Simply Select All and then de-select the colours you want to be opaque.

  • De-select All  makes all colours opaque.

  • Invert  all previously selected colours will be de-selected and vice-versa.

 

Turning Transparency Feature Off

To turn the transparency feature off, re-select the menu item Palette->Transparency->On - the transparency buttons will be removed from the palette window.

If you re-select transparency at a later time, your previously selected colours will be remembered.

 

Notes

Colours marked as being transparent will not be painted onto the main design when the brush is used. This applies not only to freehand drawing but to any other drawing function, for example Fill or Global Change.

The transparent colours in the brush are not lost when they are made transparent and can be reintroduced by making them opaque again. In fact a brush could be used several times in a design with different transparent colours or with no transparency at all simply by changing the transparent colour selections.

You can select the transparent colours before or after you capture a brush from the design.

 

When to use Transparency

This feature is particularly useful if you want to capture a motif from the design which has a multi-coloured background. Assuming that the background colours are not used in the motif itself you can simply make the background colours transparent before applying the brush to the design in another location.

Note that if transparency is not being used, the current background colour is always transparent when you cut a brush.
