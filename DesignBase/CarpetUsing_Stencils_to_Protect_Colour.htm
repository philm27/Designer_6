Stencil

 

It is possible to mask off colours within a design so that those areas are protected. For example, you may want to apply an effect to the background in a design (such as a spray). By applying a stencil to all colours except the background colour, you can apply the spray at random, without having to worry about the spray colour damaging the main motif.

Example palette showing stencil buttons. Color 1 is unprotected. Other colors are protected.

Create a Stencil

  1. Display the palette as a palette strip not as a panel, as only the strip displays stencils.

  2. Select menu option Palette -> Stencil -> On. Small buttons will appear next to each colour in the palette initially showing a cross (protected).

  3. A tick in a stencil box means that you CAN draw on that colour. A cross means that you CANNOT draw on the colour as a stencil is protecting it. You can click on the stencil button to switch from protected to un-protected and vice-versa.

 

Stencil Menu Options

Whilst the stencil feature is on, you can make the following selections from the Palette -> Stencil sub-menu:

  • Select All - will automatically set all colours to un-protected which means you can draw on all palette colours. You can de-select individual colours in the usual way if you wish.

  • De-select All - all colours will be made protected. You can then select individual colours that you wish to be able to draw on.

  • Invert - all previously selected colours will be de-selected and vice-versa.

 

Turning Stencil Off

To turn the stencil facility off, de-select menu item Palette -> Stencil -> On - it should now not be ticked. If you re-select stencil at a later time, your previously selected colours will be remembered.

 

Stencil Regenerate

When you apply a stencil to a design, the stencil is created at that particular time. If you then draw with a colour with a protected colour then that area of colour will not be protected unless the palette is regenerated.

You can choose to automatically regenerate the stencil every time the design is changed by selecting the option in preferences. Alternatively, if you do not select the automatic option, you can either select the menu option Palette -> Stencil -> Regenerate Stencil or you can simply press letter 'P' on the keyboard whenever you wish to regenerate the stencil.

New Design

If a stencil exists for the existing design when you create a new design, you have the option of turning the stencil area from the old design into a protected area in the new design.
