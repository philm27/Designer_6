Polygons - Standard Shapes

You can use this feature at any time to create a wide variety of shapes as listed below. You can choose to add the polygons created to the list of those that already exist, or to simply render them into the current design.

All polygons, if retained, are stored in the design file.

As with all polygons - they do not have to be rendered into the current design. They can simply be used as Guide Lines for drawing.

Also, all polygons, once created, can be re-sized, moved, rotated dynamically and can also be copied using the polygon toolbar and menu.

The shapes created will be grouped together and can optionally be given a name. If no name is give (on the General tab) the software will automatically give it a name such as "Rectangle with Borders" for example.

To use this feature select menu option Polygons->New->Standard Shape, or select the "New Standard Shape" button on the Polygon Tool bar.

An example of the Create Standard Shape Property Sheet. The user can display and modify any tab as many times as required and then click the OK button to create the polygon group and/or render it into the current design.

Shapes

The shapes that can be created automatically include:

Triangles

Rectangles

Regular Pentagons (5-sided), Hexagon (6-sided), Heptagon (7-sided) etc

Stretched versions of above i.e. variations of a lozenge or a "stretch to fit" figure.

Circles

Ellipses

Arcs

Chords

Borders

Borders can be specified for any shape and each can have a different color. The centre panel, for closed shapes, can be filled or not filled as required.

Radial Lines

Radial Lines can be added to any figure which is why, the Shape tab allows the number of "sides" to be specified for all shapes even circles and ellipses (and arcs/chords).

Size and Location

The shape created can easily be made to fill the current design and to be centered. Alternatively, it can be made any size and placed accurately at any location.

Rotation

Any shape can be rotated by any amount making it easy to make, for example, an ellipse at 45 degrees with borders.
