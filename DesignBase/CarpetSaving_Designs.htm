Saving Designs

 

It is always advisable to save work frequently not just when you have completed a design to avoid loss of work and effort in case of computer system failure. It is recommended that you save your work to hard disk at least every hour.

 

IFF and PRJ file types

If you want to save your main design, together with weaves (if any) and frame plan (if any), you should save it as a file with an IFF file extension. This will NOT save any associated shelf items.

In order to additionally save the shelf items, save as a PRJ file type. The software will normally default to this type.

 

Save versus "Save As"

The Save option accessed from the menu option File->Save or from the Toolbar Save button, simply re-saves the design using the same file type as the previous save (or Open if no prior Save). If the design is a new one then the software will automatically display the "Save As" dialog.

The Save As option, accessed from the menu option File->Save As will display the "Save As" dialog and allows you to:

(a) Specify (or navigate to) a specific directory

(b) Specify the file name you wish to use.

(c) Specify the file type you wish to use.

 

Save as Procedure

Select menu option File->Save As. The following dialog will be displayed.

 

Example of the "Save As" dialog

 

  1. Find the folder in which you want to save the design. To do this, navigate through the folder hierarchy.

  2. Go to the "parent" folder by clicking on the button, or to the "offspring" folder by double-clicking on a folder in the main list.

  3. Select IFF files (or whatever file type is required) from the Save as Type drop-down list. The software will automatically show all files of the specified type in the current directory.

  4. Enter your design file name in the File Name box. You do not need to type the file extension as that will be added automatically.

  5. Click on Save to save the file in the folder. The Save As dialog will be closed.

For more details about the icon size in the above list and in the use of the favourites feature - see the topic Open File.

 

Save Procedure

You can simply overwrite the previously saved or opened design by selecting Save.

You would do this only if you are happy to overwrite it of course and you do not want to change the file name nor type.

This option is only available if you have previously saved your design or if you have previously opened the design. The old design file will be overwritten and you will not be able to retrieve it.

As mentioned above, if you select Save for a new design which has not previously been saved the software will display the "Save As" dialog so that you can specify the name and file type.

  • Select menu option File->Save, or

  • Click on the Save button in the toolbar

  • Use the keyboard shortcut <Ctrl> + S

 

Overwriting an Existing File

If you wish to overwrite an existing file use the Save As procedure as detailed above but find the required folder and select the file type (if necessary) and then click on the file you wish to overwrite in the list of files.

The file name will be transferred into the File name box.

You could alternatively simply type in the name of the file you wish to overwrite.

Save Shelf Items for PRJ files

If the project contains shelf items then they will also be saved in a .prj file. This will be the case whether or not the current design (the one being edited) is the main design (shelf item 1) or not.

Save Shelf Items for non-PRJ files

File types other than PRJ such as IFF BMP PCX etc do not save shelf items - they only hold a single "image" therefore the software will proceed as follows when additional shelf items exist.

  1. Current Design is Main Design

If the current design is the main design the following warning will be displayed:

If you wish to cancel the save in order to start again, this time saving as a prj file, simply click on CANCEL otherwise click on OK

  1. Current Design is shelf item 2 or more

In this case the software will display the following message allowing user to choose whether to save the main design in box 1 or the current shelf item.

Select Yes to save the main design in box 1 or No to save the current shelf item (the one being edited).

PAT File Type

This file type contains the design and palette information but does not contain the frame plan nor weaves (if any). If there is a frame plan it will be automatically saved separately as a CPD file having the same name.

BMP PCX File Types

These file types are industry standard graphic files which may be required to transfer images to other software products. These files should not be used for the permanent saving of designs as technical information is not included in these files nor weaves and frame plans if applicable.

VCF KNF/DNF DES File Types

These file types are proprietary file types for use with various types of production and sampling equipment and should not be used for permanent storage of designs.
