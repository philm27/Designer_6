Copy Brush to Shelf

 

In order to store a brush on the shelf:

  1. If you haven't already done so, capture the required motif as a brush(Custom Brushes).

  2. Click in an empty "box" on the shelf with the left mouse button. This will automatically store it for you*.

 

*If the box already contains a motif that you wish to replace you must delete it first using the pop-up menu:

To delete a shelf item:

  1. Right click on the shelf item to display the pop-up menu

  2. Select Delete.
