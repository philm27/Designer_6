Rotating a Brush

 

Simple 90 degree rotation

A quick and simple method of rotating brushes by 90 degrees is provided in the Toolbox.

To use this method select from the toolbox for a clockwise 90 degree rotation, (or alternatively use the keyboard shortcut >) or for an anti-clockwise 90 degree rotation (keyboard shortcut <).

The brush will be rotated automatically, but will not be redrafted if the quality (setting) is not "square".

 

Rotate Any Angle

This menu option allows a brush to be rotated to any angle and will take the quality (setting) into account to keep the brush proportionally correct.

To use this option select menu option Brush->Rotate to display the Rotate Brush dialog as shown below.

 

Example Brush Rotate by 45 degrees

 

There are a number of ways to use this dialog:

  • Enter the number of degrees in the edit box at the top of the dialog. For an anticlockwise turn, subtract the angle from 360. For example a 90 degree ant-clockwise turn will be achieved by entering 270 degrees.

  • You can click on the large left arrow and right arrow button to nudge the brush round a degree at a time. After each click the brush will be redisplayed rotated to the new angle.

  • These buttons can also be "held" down using the mouse to cause the brush to be rotated continuously. Release the button when the required angle is reached.

  • If you want the brush to be redrafted as it rotates, tick the Quality box before entering an angle of rotation. You should normally leave this item selected.

  • When you are happy with your rotated brush, click on done.
