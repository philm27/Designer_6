Polygon List

The Polygon List allows you to inspect all current polygons and to drill down into each group and to modify the properties and easily delete those no longer required.

To display the Polygon list select menu option Polygon->List or use the appropriate button on the toolbar.

Introduction

The Polygon List is used to inspect and if necessary modify the list and grouping of all current polygons. The list is split into 3 areas - the Tree View the Detail List and the Preview. Each of these 3 can be removed from the display using the Check Boxes.

Note that the dialog is re-sizeable by dragging any of the four edges

The Tree View

The tree view is used to select and inspect any current polygon. There may be several group levels. The example shows some text where the letter S has been selected. You can use the list to:

Name - click on a name (or the space where the name would be) and type a new name or edit the original. This is simply a useful reference when there are a large number of items.

Regroup - using Drag and Drop you can drag a polygon or group to a new group

New Group - you can select (so they are checked) several items and click on the "Group" button to make a new group from those items.

Re-Order - using Drag and Drop you can drag a group or primitive to a new location in the list.

Delete... - select (so they are checked) one or more items and click on the "Delete" button  - a warning message will first be displayed.

Delete All... - click on this button to delete all polygons - a warning message will first be displayed.

The Detail List

This will list the group selected and all its sub items, or, if a primitive is selected only that primitive is shown. The selected group is shown at the top of the list highlighted with a grey background. The columns are as follows:

Name - the polygon name allocated by the user

Type - either "Grp", "Pol" or "Ell" for Group, Polygon or Ellipse

Lines - the number of lines in the Polygon or the total in the group

Guide - G for a guide line otherwise blank. Guide lines are switched on or off using the menu option View->Guide Lines.

Line - the line color to be used when the line is drawn (i.e. rendered) into the design.

Fill - the fill color to be used when the shape is drawn (i.e. rendered) into the design.

Border - a B to indicate a border line. When doing a Fill, a B will cause the software to fill only up to the next Border Line so is useful for rugs. A T indicates a transparent area and this is used for example for the inside shape of a letter O or B etc.

Dist - The distance to the next border line - this is for information only and is only relevant for rug shapes.

Lock - will show as an L if the polygon is locked

Some items in his grid such as Name, Guide Line, Line color and Fill color can be modified using the keyboard. If a group item is modified, the software will query whether to modify all items in the group.

Note that you can right click on an item in the  list to display a short pop-up menu allowing Properties to be accessed and modified for both a group item or a primitive.

The Preview

This will show the selected item in red at a zoom level that will show the whole shape. Any other adjacent polygons are shown in grey lines. This image is for reference and cannot be edited.
