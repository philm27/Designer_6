Select Section Dialog

To display this dialog use menu option Power Tools->Select Section

Example of the Select Section dialog showing a floor plan. Note the selection rectangle at the bottom left.

 

Selecting a Section

Simply make a rectangle using the left mouse button around the section of the design that you want to edit.

You can also use the "tabs" on the rectangle to stretch it in the usual way using the left mouse button. To do this move the mouse over one of the tabs then press and hold the left mouse button whilst moving the mouse in the appropriate direction.

You can also drag the rectangle to another part of the main design using the left mouse button. To do this move the cursor to a position over the centre of the selection rectangle then press and hold the left mouse button whilst moving the mouse.

Next Submenu

Use this to move to the adjacent section. The section will be the same size as the previous one and will not have an overlap nor will there be a gap between this section and the last. Selection using the mouse can be very imprecise since there can be a lot of fine detail packed into a very small area in the image shown in the dialog.

Move To Submenu

You can use this to move the selection rectangle quickly to one of 11 positions within the design. Also you can use the numeric keypad to select these positions and they correspond to the numbers used when designing for example 1 represents the bottom left.

Zero and <.> move to the corners when the design is a repeating design. Zero moves to the bottom left corners and <.> moves to the bottom right corners (i.e. in both cases the section will cross the edges of the main design ). These positions are the same for a straight match but are different ift here is a drop match or brick shift.

Example showing a large persion design in which 2x2 repeats are shown and a section has been chosen which crosses the edges of a half drop repeat

1 Repeat and 2x2 Repeat

You can switch between showing 1 repeat and showing 2x2 repeats using these buttons. The main purpose of this is so that you can select a section that crosses the edges of a repeating design.

You cannot use the 2x2 button for a rug and if the design is a Horizontal or Vertical runner, this button will cause the program to show 1 repeat by 2 repeats (or 2x1 for horizontals).

Reset

Use this to go back to the original size and position when the dialog was first displayed.

OK and CANCEL

Use OK to return to the main design window using the selected section. Use CANCEL to return without changing the section

Design Section

These two figures show the current design section size in ends and rows. YOu can alternatively use units as shown on the drop down list - inches, feet, yards, cm, mtrs, mm.

In each case, if you change the units, the physical size will stay the same but the program will convert the figures for you.

Design Offset

These figures specify the offset of the design section from the bottom left of the design. When the section is at the bottom left, these figures will show zero.

Design Size and Section Size

These two sets of figures which cannot be modified, show for the users information, the design size and the current section size in both "pixels", feet and inches and metres.
