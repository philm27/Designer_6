Elliptical Borders

 

To access the Elliptical Borders window, select the menu option Power Tools->Border->Elliptical.

The dialog looks like this:

 

The dialog and the procedure is almost exactly the same as for circular borders with three exceptions:

  • The x and y radius work independently of each other, allowing you to create an ellipse.

  • You can choose whether or not to Keep Border Depth Constant.

If this option is selected, then the border will be a constant depth around the ellipse though this means that the inner edge of the border will not be a true ellipse (though the outer edge will be a true ellipse).

If this option is not selected then the inner edge of the border will be a true ellipse which will mean that the border will not be a constant depth.

Normally, you would select this option as designs generally require a constant border width. This option is provided since if the inner edge of the border is a true ellipse then the border will not be a constant width.

  • Only two "ideal" calculations are shown - the Ideal Repeats and the Ideal Brush.

For other details of the procedure and the data items see the topic Circular Borders.
