Keyboard Shortcuts

 

Using keyboard shortcuts can save time and almost all of the functions accessed via the menus, toolbox and toolbar have a keyboard shortcut.

Where possible, the key selected to activate a function has been chosen so that it is easy to remember - usually by using the first letter of the feature it represents.

However, note that you can create your own keyboard shortcuts to replace or add to the existing list - see the topic Customised Keyboard Shortcuts.

Numeric Keypad

The following keys /, *, 1-9, 0 and . are those on the numeric keypad on the right of the keyboard. The action performed for keys 1-9 corresponds to their position on a standard keypad, i.e. 7 is the top left key, 1 is the bottom left key etc. This makes them very easy to remember.

Key

Command

/

Moves to Next error in design (See Note 3).

*

Moves to Previous error in design (See Note 3).

  1. * Keys 1-9 can be used as follows:

  2. <key> Pan to that location in design e.g. <7> to pan to top left

  3. <ctrl>+<key> Change Brush handle e.g. <ctrl>+<7> to change brush handle to top left

  4. <shift>+<key> Draw the brush e.g. <shift>+<7> to draw brush at top left of design

Note 1:

Carpet Designer Only

Note 2:

Controls brush size for Freehand Draw and controls Line Thickness for all Line Drawing functions, Circle, Ellipse, Square and Rectangle functions

Note 3:

Errors in this case are yarn not in frame plan (Carpet Designer) or Float Error (Jacquard Designer). Has no effect in Textile Designer.

Note 4:

These two operations will produce exactly the same result if the repeat type is a straight match

 

<Home> and <End> keys

These keys can be used to assist in the drawing of large centred circles or in drawing large diagonal lines. For more information see the topic <Home> and <End>.
