Dot Removal

 

Dot removal can be useful if you have been creating spray effects and want to edit out any stray dots. Also, a scanned image will often contain unwanted dots. To use this feature, select Dot Removal on the Design menu.

 

An example of the Dot Removal property sheet. In this example, the user is removing a dot size of 3 or less and is selecting the dot colors to be removed.

  1. Select menu option Design->Dot Removal. The above dialog will be displayed.

  2. Make sure that the "Do Dot Removal" option is selected.

  1. Enter the size of dots to be removed. This is the number of points in the dot rather than the width or length of a dot (see examples below). It will initially default to 2.

  2. Select Diagonal Connections if you wish to include diagonally connected points in the dot size (see examples below)

  3. Select either All Colors or Selected Colors.

  4. In the latter case select the Colors tab and check that the correct colors are selected.

  5. Click on OK.

Note that you can also remove lines at the same time by clicking on the Line Removal Tab and selecting "Do Line Removal".

Color Tab

If you have selected the "Selected Colors" option, you will also need to use the Colors Tab to specify what those colors are.

Example of the Color Tab. Here the user has selected color 5 and 6 from the palette - these are the only dot colors that will be removed.

To Add a color to the Selected List:

Drag From the palette to the Selected List.

You can also multi-select in the palette by holding the <ctrl> key while selecting colors then dragging them to the selected list.

Or, click on a color in the full palette and then click on the Add Button

To remove a color from the list:

Click on the color or colors and press the <delete> key on keyboard, or, click on the delete button with the mouse.

You can also multi-select by holding the <ctrl> key while selecting colors to delete several at once.

To delete all colors:

Click on the "Del.All" button
