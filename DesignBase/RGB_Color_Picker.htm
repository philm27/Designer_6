RGB Color Picker

This dialog will be displayed if either of the swatches in the Palette Change dialog is clicked. it offers an alternative way to select a color as well as an opportunity to key in an RGB value.

It also provides a way to have a set of 16 standard custom colors that will be stored from one session to another.

An example of the color picker - user has selected a shade of blue - the RGB value may have been typed in or the user may have used the mouse to drag the hue/saturation value and the "Luminosity" (or Value) to new settings.

Basic colors

Click on any of these swatches to select the RGB value for that color.

Custom Colors

This is a list of 16 colors that can be modified and remembered by the system from one session to another. Click on any one of them to select that RGB value. Also, if you want to modify an existing custom color, or create a new one, you should first of all click on that swatch (see below).

Hue and Saturation Selector

Click anywhere in the large rectangle to select that Hue/Saturation. The bottom row is a neutral grey.

Luminosity Selector

Click anywhere in the color bar on the right to select that value, or, click and drag pointer on the extreme right then drag it up or down as appropriate.

HSV and RGB values

Either click on a value with the left mouse button or <tab> to that value then key in a new value. You can use the <tab> key t go to the next value.

Add To Custom Colors

Click on this button to add the current color to the set  of 16 Custom Colors on the left. You should already have clicked on the Custom Color that you want to replace.

OK and Cancel

Click on OK when finished or Cancel as appropriate.
