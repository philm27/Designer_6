Add Borders

You can add borders to the design visually using this dialog. To access it select Design->Add Borders.

 

Example of the Add Borders dialog. The user has added a 5 inch border all round in the light green

Main Design Preview

There is no selection rectangle as there is with similar dialogs.

Border Units

Select the units from this drop down list. Available Units are Ends/Rows/Inches/Feet/Yards/Centimetres/Metres/Millimetres. These units apply to all four sides and when units are changed the software will convert all four sides from the old units to he new.

All Border Same

Check this box to cause the software to make all 4 edges the same border value. When checked, any changes to the Top border will also be made to the other 3 sides.

Border Sizes

These four figures permit a different size to be specified for each border. If "All Border Same" is checked then the left, right and bottom figure swill be protected.

Border Colour

This small panel shows the border colour. It will initially default to the Right hand button colour but can be clicked to allow a change (see below).

Original Size

These "Read Only" figures give the size of the original design.

New Size

These "Read Only" figures give the new size of the design repeat including the borders.

Reset

Click this button to reset the design to its original size and to set the borders to zero.

1 Repeat

Click on this button to show only one repeat.

2x2 Repeat

Click on this button to show 2 repeats across by 2 repeats lengthwise.

3x3 Repeat

Click on this button to show 3 repeats across by 3 repeats lengthwise.

OK

Click OK to add borders to the design as specified and to close the dialog. For Carpet Designer users, any current frame plan will also be re-centred if necessary. For Jacquard Designer users, the weave, if any, will also be re-centred.

The locations of any polygons will be adjusted, and, any regions and their inlays and out-fills will be adjusted so that they relate to the same figure in the design.

Note that the design itself is not modified until OK is clicked. The operation can then be undone if required.

Cancel

This will cause a return to the main design window without modifying the design in any way.

Changing the Border Colour

Click on the small colour panel in the centre of the border sizes and a list of palette colours will be displayed as shown in the illustration below. Simply click on a colour to highlight it and click on OK. The borders if any will immediately change colour.
