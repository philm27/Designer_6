Default Directory

 

You can specify whether you want the software to use the same default foler for both load and save operations.

To do this select the menu option Edit->Preferences and select the General Tab.

You have the option of selecting or deselecting the directory default as "Same for Load and Save". If this is selected Open and Save As will use the same default directory otherwise they will have separate defaults.

The default directory is the last one used thus if you open a design file from a directory it will always default to the same directory for the next Open operation. However it will also default to the same directory for "Save As" if this option is selected.

It can be useful to switch this option off if you are doing a lot of work where you are loading from one directory but saving to another.
