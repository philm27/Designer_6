Design Preview

 

This function enables you to see your design in a secondary window either as one full repeat or in many repeats. You can also view a section by selecting a rectangular area whilst in display 1 or display many.

Example of the preview window in "Display 1" mode. The user has drawn the selection rectangle using the left hand mouse button. The button has not yet been released.

The same window after the mouse button has been released . The viewing mode is now "Display Section".

Conveniently, whilst in view section mode, you can push the design around to view other parts of the design at the same zoom level. Whilst the cursor is over the Preview Window, the cursor will be shown as a "Hand" icon.

You can also zoom in and out and move the display using the keyboard or mouse wheel whilst the cursor is over the preview window. See below for more information.

You can leave the window on the screen whilst you continue to work on a section of the design. The preview window updates automatically as you work (if you are displaying the current design), allowing you to see the effect your alterations make to the design as a whole.

Alternatively, you can load a different (reference) design into the window and this can be a 24 bit image if necessary Â– to compare it with your current design or to view design ideas.

To display the preview window on screen:

  • Select menu item View -> Design Preview or...

  • Use the keyboard shortcut <F7>

Preview Menu Options

To display the Preview Menu - click on the menu title "Preview" in the menu bar as above.

Display One

One full repeat of your current/reference design will be displayed. You can change the size of the window and therefore the design by dragging any edge or any corner of the window.

You can use the left mouse button to make a rectangle around a section which you would like to examine in more detail in Display Section mode.

Display Many

Several repeats of your current/reference design will be displayed. The software will show a minimum of 3 repeats in both directions. You cannot use this option for rugs.

As for Display one, you can use the left mouse button to make a rectangle around a section with which you would like to enter Display Section mode.

Display Section

A section of your current/reference design will be displayed at actual size. The cursor will change to a "Hand" symbol when hovering over the Preview Window allowing you to "push" the design around.

Zoom Level

This option is only available in Display Section Mode. The % values shown are % of real size so 100% is real size. The normal percentages available are 100%, 75%, 50%, 25% 10%, 5%, 1% and another may be added by the software in order to be able to show the selected section (in the example image above, 12.85% has been added).

See also "Zoom using Keyboard" below

Edit This Section

Again this option is only available in Display Section Mode. It allows you make the main design window "jump" to the same section as the preview window.

Copy

This will copy the current section to the clipboard, making it easy to copy a section of the design to the shelf. After using Copy you can Paste the section to the shelf or to another project, (or, in fact to any other software product).

Open Reference Design...

This allows you to load any existing design as a Reference Design to be shown in the Preview Window instead of the current one. When you select this menu item an Open file dialog will be displayed. Simply select a design in the normal way.

There is only one reference design shared between all projects but you can load a new one at any time.

Open From Arc...

Select this option to load a reference design from the Arc database

Recent Files

Select this to select a reference design from one of the list of 10 recent designs. This is the same list as will appear on the File menu in the main design window.

Show Main Design

Select this to show the main design from the current project. You can swap between the Main Design and the Reference Design at any time.

Show Reference Design

Select this to show the Reference Design.

 

Close the preview window using the same method as for opening it, or alternatively click on the close button in the top right-hand corner of the window.

 Minimise the preview window to the taskbar by clicking on the minimise button in the top right-hand corner.

 

Zoom using Keyboard

Instead of using the menu to select a zoom level, you can use + and - on the keyboard to zoom in and out. The rule is that if the cursor is over the preview window, then the preview window will be zoomed. Otherwise the main design will be zoomed.

Also note that you can zoom in from display many to display 1 to display section. Display section will display several zoom levels before reaching its maximum.

When zooming out from the maximum zoom, the software will eventually revert to display 1 and then, if you zoom out again, the software will revert to display many.

Zoom using Mouse Wheel

Whilst the cursor is over the preview window you can use the mouse wheel to zoom in (push wheel forward) or out (pull wheel back).

Location Using Keyboard

As with the main design you can select a location when in section mode using the numeric keypad. As with zooming using + and -, if the cursor is over the preview window then the preview will be re-positioned.

For example you can use 1 to move to the bottom left corner, or 5 to move to the centre of the design, or 9 to move to the top right etc.

Zero and "." (dot) move the corners of a repeating design to the centre.

Location using Mouse

You can draw a rectangle to indicate the required section for the preview window. Simply hold the <Ctrl> key whilst pressing and holding the left mouse button. You can use this feature in Display 1, Display Many or Display Section Preview modes. (It is not necessary to hold the <ctrl> key in Display 1 and Display Many modes).

Choose main design section

You can cause the main design to move to the same section as the preview window simply by double clicking it. Alternatively you can select "Edit this section" on the Pop-up menu.

In SAS mode this is equivalent to selecting a new section using the menu option Power Tools->Select Section.
