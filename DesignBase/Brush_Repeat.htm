Brush Repeat

 

The brush repeat can easily be set to one of the standard settings by using the menu options Brush->Repeat and Brush->Match. In both cases a sub menu is displayed as shown below:

An example of the Brush Repeat Menu. Brick repeat is currently selected.

 

An example of the Brush Match Menu. A half drop (or half brick shift) is currently selected.

 

The above settings can be useful when used with Flood Fill, Global Colour change, Filled Rectangle, Square, Circle and Ellipse. In all these cases a custom brush can be given a normal drop or a brick shift type drop.

Note that whilst Rug, Horizontal and Vertical repeat types are shown on the menu, these are only for completeness. Rug and Horizontal will at present behave like normal drop and Vertical will behave like Brick shift. This does not apply to the main design which will treat each of these type differently when displaying the main design see the topic Selecting a Repeat Type.
