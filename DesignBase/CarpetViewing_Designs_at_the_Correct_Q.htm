Aspect Correct

 

To view your designs at their correct quality i.e. as they would look in production as a woven fabric, you must ensure that the aspect correct function is selected. Without it, your work will be displayed as a square quality (7 x 7, 8 x 8, 20x20 etc.)

To select this feature:

  • Select View->Aspect Correct on the menu. You can select the same option to turn it off.

  • Alternatively, select the Aspect Correct button on the toolbar. You can select the same button to turn it off. The button will be "down" if the feature is currently selected.

 

Examples

Example design in a 24x16 per inch quality (setting) with Aspect Correct selected. The design is shown in the correct proportions. Each design point is an oblong shape (with an aspect ratio of 3:2) which can be seen if the design is zoomed in and the grid on.

This is a zoomed in view of the above image and shows the shape of the design "points" each of which has an aspect ratio of 16:24 (i.e. 2:3) in this example.

Same design as above with Aspect Correct not selected. The design is shown distorted since there are less rows (picks) than ends per inch. Each design point is square in this view.
