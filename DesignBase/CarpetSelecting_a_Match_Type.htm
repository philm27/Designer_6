Match Type

 

There are four standard match types to select from:

  • Straight

  • Quarter

  • Third

  • Half

Additionally, you can create your own match type using the "Any" option.

You can set the Match Type (i.e. the drop) by selecting menu option Design->Match->type where type is one of the above options.

You can also set the match type in the specification dialog.

Drop Any

In order to set the drop to a non-standard value. Select the menu option Design->Match->Any. The following dialog will be displayed.

The "Drop Any" dialog. You can use any value from 1 row (pick) up to the length of the repeat.

Enter the required drop value and lick on OK. If the value entered is actually equivalent to a standard drop then that standard value will be shown as selected in both the menu and in the specification. For example if the design length is 600 rows (picks) and the value entered in the "Drop Any" dialog is 150 then the Quarter Drop will be automatically selected.

You can also set the drop value in the Specification dialog.

Brick Repeats

The software can use use brick repeats. In this case the drop value is the horizontal shift in the next repeat vertically (i.e. next repeat above when viewed on the screen).
