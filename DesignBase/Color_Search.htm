Color Search and Find and Replace

This option is found on the Power Tools Menu Power Tools->Color Search.

Color Search Features

This feature allows you to search for one or more colors and to have them outlined (highlighted) on the design display. Once you have done a search, you can zoom, move the display and edit any part of the design in the normal way. The highlights will not be removed until you start a new search or until the dialog is closed.

The color regions found will be highlighted by outlining them in black and you can move from one region to another using the Next and Prev. button options. The "current" region will be highlighted in red and will be moved to the centre of the screen if possible at the current zoom level.

You can also switch the highlighting on or off using the "Show Highlights" check box - you may find it useful to switch the highlighting off while an area is edited. You can also switch the highlights off by closing the dialog altogether - the same regions will be highlighted if you re-display the dialog for the same project by selecting it on the menu.

The design can be edited while the highlights are being displayed but the region outline will not be modified until a new search is performed.

Color Search Dialog

When Color Search is selected, the following dialog is displayed:

Example showing two selected palette colors for the search.

Find Colors

This is a list of colors for the search, the user may add to it by selecting from the full palette list below and clicking on the Add button. Similarly, colors may be removed by selecting them in the Find List and clicking on the "Delete" button.

As this uses the same code as the color list in the Palette Change dialog you can right click the list and choose a different format such as List View.

Note that the sequence of these colors does not make any difference for this feature.

Search Button

Clicking on this will start a search in the current design. Any previous highlights will be removed. The software will stop if more than 100 occurrences are found.

Next Button

Click on this to move to the next color Region. It will return to the first one if the current one is the last.

Prev. Button

Click on this to move to the previous color Region. It will return the first last one if the current one is the first.

Highlights Button

Use this to modify the highlighting. You can select the colors used for both normal and selected regions and you can also switch the Outline and the Cross Hatch on or off independently. For more information se the topic Region Highlights.

More (>>>) Button

Use this button to display or hide the palette list in the lower part of the dialog. This allows more of the design to be shown whilst working with a set of colors as in the example below. The button can be used at any time.

Show Highlights

Use this button to show or remove the highlights around all of colors. If the colors are to be edited, it may be useful to switch the highlight off. It can be used any time - it does not cause a new search.

Palette List

This shows the current palette. Use it to select the search colors and to add or delete them from the Find Colors list (the search list).

As for the Find List above, you can right click the list and choose a different format such as Report View.

 

Add Button

Use this button to add one or more selected colors from the Palette List to the Find List at the top.

Delete Button

Use this button to delete one or more selected colors from the Find List at the top.

Delete All Button

Use this button to delete all the colors in the Find List at the top.

Close Button

Use this button to close the dialog. This has the same effect as using the standard close window button at the top right of the window.
