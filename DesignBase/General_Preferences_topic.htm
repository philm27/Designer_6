Spacebar

You can set the behaviour of the spacebar (Pipette) keyboard shortcut as described below.

Spacebar Activates Pipette

If this is selected then whenever the spacebar is pressed, the pipette feature will be activated. After you have selected either a foreground or background color (using the left or right mouse button respectively), the previous drawing function will be re-activated.

 

Selects Color.

If this option is selected, then the color under the cursor will be selected as the new foreground color without actually going into "pipette" mode. The current drawing function will remain selected.

The disadvantage of this option is that you cannot select the background color other than by selecting it in the palette with the right hand button.

 

Grid

Grids are drawn and numbered from either the left or the right according to a preference.

Select the "Grid Drawn from Left" option to cause the grid to be drawn from the left.

If not selected, grids will be drawn from the right when printing designs with a grid and also when drawing grids on the screen. (This may make a difference if the bold line interval does not fit into the width a whole number of times).

Also, the end number, as displayed in the status bar and in the Information Panel and "Tape Measure" will be affected by this option. If selected, End 1 will be on the left otherwise it will be on the right.

 

Zoom to Cursor

Select this option to cause the design to be zoomed so that the same design design point stays under the cursor when using "+" or "-" keys or the mouse wheel.

See the topic Zoom to Cursor for more information.
