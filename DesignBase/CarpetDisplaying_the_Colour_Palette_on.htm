Palette Window

 

The colour palette can be displayed as either a palette strip or a panel.

In either case, to display the palette window, use one of the following methods:-

  • Select menu option Palette->Palette Window.

  • Click on the button in the toolbox or toolbar

  • Use the keyboard shortcut key F5

Remove the palette window by repeating any of the above operations. Alternatively, click on the X in the top right-hand corner of the window.

 

Palette Strip

Normal, horizontal palette showing a "Tool Tip" for color #1 as the cursor is hovering over that color. The "P" in square brackets indicates that the item is a Plain Shade.

The tucks at the top right indicate those colors are used in the design and the tuck at the bottom left indicates that color is the left button color. Similarly a tuck at the bottom right indicates the right button color.

You can scroll the list using the Mouse Wheel whilst hovering over the Palette window.

Vertical palette

 

Note that in both cases a scroll bar will appear if there is insufficient space to display all palette colors and in this case you can also scroll using the Mouse Wheel.

Change from Horizontal view to Vertical by dragging one edge or corner of the palette window.

Palette Numbers

If there is a "Zero Color" then color numbering will begin at "0" as shown in the example above, otherwise it will begin at "1".

Minimise

Minimise the palette to the taskbar by clicking on the minimise button in the top right-hand corner of the window.

Tool Tip

A "Tool Tip" will be displayed when the mouse hovers over a palette swatch as n the example above. This will always show the palette number,

Foreground, Background and Used Color Indicators

Each colour swatch in all 3 types of display can have up to 3 "tucks" in the corner.

The tuck at the top right indicates a colour used in the design. (In the above examples colors 1-6 are used in the design). This is only shown if the "Automatic usage" update is selected in preferences.

The tuck at the lower left indicates the current foreground or left hand button colour.

The tuck at the lower right indicates the current background or right hand button colour.

Note that you can select left and right button colors in the Palette Information window in which shade codes and descriptions can also be seen.

Reorder the Palette

The palette can be re-sequenced simply by dragging a color to a new location and releasing the button. This can be done using either the vertical or horizontal layout.
