Color Range

 

Use this feature to generate a 'run' or 'set' of colors - you just nominate the first and last color of the set.

The colors don't need to be a light and dark of the same colour - you can create a range between any two colors. For example, you can have a 'cool' first colour and a 'warm' last colour - so that the set twists.

 

Example

Example Palette before the Range operation. Color 5 has been made a light red and colour 10 a dark red. The RGB values of the colors in between do not matter.

The same palette after the Range operation has been used from colour 5 to colour 10.

 

Procedure

To use the Range feature you may need to add colors to the palette in order that there will be sufficient for the range you are trying to create. (The Range function does not add colors to the palette for you). Also, since the colors have to be consecutive, you may need to use the move option so that the new spare colors are in the correct place. Then proceed as follows:

  1. Set the colour at the start of the range using the Palette Change dialog.

  2. Set the colour at the end of the range using the Palette Change dialog.

  3. Click the 'Range' button. The pointer will change to a 'From' pointer.

  4. Select the first colour in the range. The pointer will change to a 'To' pointer

  5. Select colour 6 to generate the set.

  6. Start again at step 4 to create another range, or...

  7. Select the 'Cancel' button in the Palette Change window or press <esc> on the keyboard to exit.
