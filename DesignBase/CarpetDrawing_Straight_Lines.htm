Drawing Straight Lines

 

You can draw straight lines in any thickness as described below. There are two main options. The standard line allows you to draw lines at any angle. The "Snap" line option only allows you to draw lines at 15 degree increments i.e. zero, 15, 30, 45, 60, 75, 90 degrees etc. This latter function takes into account the ends and rows (picks) so that the line will be correct in cloth.

A slight variation of the normal line drawing function is the Catch Cords system which simply involves holding the <shift> key whilst drawing the line.

Note: You can draw lines covering the entire design accurately whilst zoomed in and without scrolling across the entire design - see the Spacebar Toggle facility.

 

Drawing a straight line between any two points at any angle

To select this option either:

  • Select the Normal Line button in the toolbox.

  • Use the keyboard shortcut <L>.

 

Drawing a "Snap" Line

To draw a line at an angle of 0, 15, 30, 45, 60, 75 or 90 degrees. The line will "snap" to a multiple of 15 degrees. The start point will stay fixed but the end point may be moved to the nearest point which will give an angle which is a multiple of 15 degrees.

To select this option either:

  • Select the Snap Line button in the toolbox.

  • Use the keyboard shortcut <K>.

 

Line Thickness

You can use the toolbox or the keyboard to select line thickness:

  • Move the mouse over the "Pen Selector" at the bottom of the toolbox and click on one of the 6 preset pen sizes. Alternatively double click this area to display a dialog permitting a larger size to be set.

  • While in Rectangle Drawing mode press one of the keys 1, 2, 3, 4, 5, 6, 7, 8, 9 to select line thickness.

 

 

Drawing the Line

Click on the start point and then drag the mouse to the finish point before releasing the mouse button. A straight line will be created between the two points. If using a digitiser with a pressure control option you can determine the line thickness using pen pressure control rather than brush thickness Â– do this just before releasing the pressure completely at the line finish point.

When using the mouse you can use either the left or right button - left to draw the foreground colour, right to draw the background colour as normal.

 

Fan Effect

 

 

Having dragged the mouse to the finish point, hold down the <Ctrl> key on the keyboard and sweep in an arc with the cursor to create the fan. Only let go of the <Ctrl> key after you have let go of the mouse button.

Catch Chord Lines

There is a setting in the tool box that allows you to draw all lines in Catch Cord mode, or, to invoke it when the <shift> key is pressed. See the topic Line Properties.

If the latter option is selected, you can then draw a "Catch Chord" line, when necessary, by simply holding the <shift> key and drawing the line as normal.

The designer has additional control in that the left <shift> key will add additional points on the left of the line, the right <shift> key will add additional points on the right of the line. (Left and Right here are relative to the direction in which the line is drawn).
