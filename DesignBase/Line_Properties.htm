Line Properties

Line Properties can be displayed by right clicking any of the following buttons in the toolbox: Line Draw (any of the 3), Circle, Ellipse, Square or Rectangle (empty or filled) or any of the line thickness tools when they are visible.

These options are used for all types of line drawing including circles, ellipses, squares and rectangles.

 

Example shows a line thickness of 1 inch, with Catch Cord Lines drawn on <shift> and with thick lines being started and "held" by their left corner.

Line Width

Either type the line width or use the up/down spin control to modify.

Units

Choose units from the drop down list - these can be Pixels (i.e. ends or rows), Inch,Foot, Yard, Metre Centimetre or Millimetre.

Catch Cord (or Running Cord)

Choose either Always, Never or "when <shift> key down". Note that this option is ignored if line width exceeds one pixel.

Start Point

When drawing lines thicker than one pixel especially when drawing for example border lines which may be one or more inches wide, it is useful to know exactly where the line will be drawn. This option allows a user to specify that the mouse will indicate for example the left edge of the line.
