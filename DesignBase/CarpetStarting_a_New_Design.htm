Starting a New Design

 

You can start a new design or a new project.

New Design

The main design is the one currently in the main display area it is normally shelf item 1 although it may be any shelf item.

To start a new design within the existing project either:

  • select menu option File->New or

  • select the New button in the toolbar or

  • press <Ctrl> + N on the keyboard

A message box will be displayed warning that the design (and the specific shelf item number) is to be cleared. Only the current shelf item will be cleared. All other shelf items and the palette will be retained.

See also Starting a New Design from a Shelf

 

New Project

To start a new project select menu item File->New Project. A message box will be displayed warning that the entire project is to be cleared. All shelf items will be deleted. A new main design will be created using the default size information.

In both the above cases the palette will be retained.

 

Starting a separate Project

To start a new project whilst retaining the current one, select an unused project from the list of 5 in the projects menu and start a new design as described above.

In this case, the new design will be created with the default colour palette, using your default specification details. All shelf items will be clear.
