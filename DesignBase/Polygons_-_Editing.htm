Polygons - Editing

Once you have created a polygon figure, either open or closed, you can edit it and create new figures in many different ways:

 

 

Selecting a Polygon

To select a polygon in order to copy it, delete it or display its properties etc. click on the "Polygon Select" button in the Polygon Tool Bar then click on one of the points in the polygon. This can also be used on a group. This allows you to select a polygon without fear of moving the polygon or moving an individual point slightly in error.

Note that you can make a rectangle on screen by clicking and dragging the cursor in order to select all the objects that have a point within the rectangle.

You can also then hold the <ctrl> key in order to add polygons to those selected or to remove  any that are already selected.

Undo and Redo

Simply click on the Undo and Redo buttons to undo or redo a polygon edit or draw operation.

These operations are held separately to the normal design history and palette history. As a result you can render a newly created polygon into the design then undo the operations that created the polygon. This will leave the rendered version in the design itself and if you need to remove that you must use the normal undo button.

Add Straight Lines or Add Curves

While creating polygons or adding lines to existing ones, you can select either of these buttons which are mutually exclusive in order to control the type of line being added.

Close or Open a Polygon

Clicking one of these buttons will immediately Close the current figure by joining the first point the last one, or, for Open will split a polygon at the current selected point, deleting the previous line.

Edit a Single Point

Select the Edit Point button in the Polygon Tool Bar then click on one of the points in the polygon and drag it to a new location.

Move a Polygon

Select the Move button in the Polygon Tool Bar then click on one of the points in the polygon and drag it to a new location. This can also be used on a group.

Rotate a Polygon

Select the Rotate button in the Polygon Tool Bar then click on one of the points in a polygon or in a group and drag it in a circular motion (clockwise or anti-clockwise) around the centre of the figure.

Re-size a Polygon

Select the Re-Size button in the Polygon Tool Bar then click on one of the points in the polygon and drag it to a new location. The polygon, or group, can then be enlarged or made smaller by dragging the mouse towards or away from the centre of the figure.

The figure or group will normally retain its aspect ratio but if you wish to distort one axis more than the other you can hold the <shift> key and aspect will no longer be locked.

Duplicate a Polygon

Clicking this button will immediately make an exact copy of the current figure and will insert it below and to the right of the first figure.

Flip and Mirror

Clicking one of these buttons will immediately mirror horizontally or flip vertically the  current figure.

Group and Ungroup

Clicking one of these buttons will immediately group or ungroup the currently selected figure or figures. Note that a group will consist of more than one object and those objects can be polygons (primitives) as well as existing groups. Thus an "Ungroup" operation will only separate off the top layer of objects in the group - it will not separate any lower level groups.

Draw Lines, Fill, Brush Fill

These buttons will "render" the polygon into the actual design at the current design ends per inch and rows/picks per inch. You can display the polygon properties (see below) in order to change the design color and line thickness to be used. The brush fill will use the current brush.

Obviously a "Fill" will not be possible if the polygon is not closed.

Note that it does not matter what is in the design within the bounds of the polygon figure. The "Fill" operations will always fill the whole figure regardless of any existing colors in that area.

Bring into View

In certain circumstances, possibly due to design size changes and repeat match changes, it is possible to have a figure that is outside the design repeat. This button should move any such figures back into the scope of the repeat.

Lock/Unlock

Select this button to either lock or unlock the currently selected objects. The software will always ask you to confirm the action required.

Guide Lines

Select this button to flag the currently selected objects as Guide Lines. Or, if the selected objects are already Guide Lines, it will remove the Guide Line flag. The software will always ask you to confirm the action required.

Properties

Select this button to display the Properties of the currently selected object. You can modify the object Name, Guide Line flag and the Guide Line color (which is also used to draw the line when in Polygon Mode), and you can also modify the rendering details such as Line Thickness, Render Line Color and Fill color etc.

These details can also be modified using the Polygon List...

Polygon List

Select this button to display a list of all current objects. The list allows you to see a hierarchical list and also to modify polygon and rendering details.
