Display Preferences

Select Preferences using menu option Edit->Preferences then select the Display Tab.

Example of the Display Preferences - this shows the default settings when the software is first installed which will use a diagonal Cross Hatch starting at Top Left and will use two greys.

This tab controls the area outside of the actual design but within the document window. It does not affect the shelf in any way. By selecting solid in the list above, the area will be displayed as a plain color, or you may select a Cross Hatch pattern from the list.

To change the colors, click on one of the color buttons and use the standard color picker to select a new color for either the cross hatch lines or for the background. See the example below.
