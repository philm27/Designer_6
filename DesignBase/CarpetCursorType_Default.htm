Cursor Preferences

 

Select a cursor-type default by going to preferences (edit->preferences). You should select the Cursor tab this is normally displayed first by default.

 

Mouse Wheel

You can control  the speed that he mouse wheel affects the zoom function my modifying this value. It will initially default to 1. It is recommended that you only use values of 1 2 or 3. this value represents the number of steps that the program will zoom in or out by for each "notch" on the wheel. The maximum value is set to 5.

Cross Hair

If checked then a cross hair will be displayed whilst using free hand draw and line draw. If the cross hair is on, it is visible on top of the brush. If it is de-selected, you just see the brush.

Lock in Centre of Tuft

This option will cause a cross hair (if being used) to be always displayed in the centre of the grid point (or tuft for carpet users). When the mouse or pen is moved to another grid point, the cross hair will jump to the centre of that grid point. If the lock is de-selected, the cross hair will move smoothly to any pixel on the display.

Cursor Style

  • Normal - the brush is seen as a block of colour (size as selected).

  • Outline - the brush is seen as a block of colour with a fine outline (size as selected).

  • Transparent - only the outline of the brush can be seen until it is placed.

The above only apply to the standard brushes available in the control panel. A custom brush never has an oultline.

This option can be used with or without the cross hair option and the Lock in Centre option above.

Experiment with different cursor types to find what suits your working method best. Once you have selected your default, it will remain that way until you alter it.

Cross Hair Style

You can select a cross hair from the list shown. Scroll the list to see other options and select the one you would like to use. The selected cross hair will be marked with a check mark.

The Dark Medium Light options are simply so that you can view the chosen cross hair against different background colours. These settings do not in any way have a permanent effect on the cursor.

Some cross hairs are more solid than others and may be easier to see.
