Colorways Update

A typical scenario during a Colorway update. the user is about to save a new colorway code 320 to the range code P515. It will be immediately added to the colorways in the Ranges tab on the left.

For Jacquard Designer users the dialog will have a warp and weft tab and additional buttons in the "Fetch From" group

Introduction

The colorway update dialog (shown on the right in the above image) can be used to view individual color sets and to create and update them in the Colors and Yarns database. The main purpose of the dialog is to allow creation of new colorways and changes to existing ones but you can also recolor the current design from here using some of the same features as in the Shade Library as described below.

To Display an existing Color Set

Select a range from the drop down list.

Select the Color set from the "Colorway" drop down list

The software will ask if you would like to fetch that color set - click on Yes or No.

The shades in that color set will be displayed allowing you to view and/or change the color set and possibly to add one or more of the colors to the current palette.

Note that the color set in the data base can only be changed using the Save command (see below) or by using the Delete All and Delete Color Set buttons.

 

To Add/Change Colors in a Color Set

Colors can be changed in the dialog in various ways. Normally you will Drag and Drop colors from the Shade Library. To do this it may be helpful to position the Shade Library window next to the Update dialog as shown in the illustration above. Then you can easily switch tabs in the Shade Library and drag colors into the Update window. You can even drag a whole color set from the Range tab in the Shade Library.

 

To Delete entries in a Color Set

User has right clicked on color 5 to display the pop-up menu

Right click on an entry and select Remove Shade from the pop-up menu.

 

To Re-Order a Color Set

The simplest way to re-order the color set is to use Drag and Drop within the list to move a shade to a new position

You can also right click on a shade and select Move Up or Move Down from the pop-up menu.

 

Fetch from Design

This can be useful if you wish to save the current design palette as a new color set (or as the basis for a new color set).

Simply click on the Fetch from Design button and the current palette will be copied into the Update window.

 

Fetch from Design Warp (or Weft) (Jacquard Designers only)

Similarly, if you wish to use the warp (weft) colors in the current design as a new color set (or as the basis for a new color set), simply click on Fetch from Design Warp (or weft as appropriate).

 

Fetch from C+Y

To retrieve a color set from  the Colors and Yarns data base, either type the Range and Color Set or select from the drop down lists and click on "Fetch from C+Y". For Jacquard Designers this will fetch either a Warp or Weft color set depending on which tab is being used.

 

Fetch From Warp Tab (or Weft tab) (Jacquard Designers only)

If you have set up a color set in the Warp tab, you may wish to quickly copy that to the Weft tab where it can be modified if necessary and saved as a Weft color set. To do this, display the Weft tab and click on the Fetch from Warp Tab button.

Obviously, the reverse applies, in the Warp tab click on "Fetch from Weft Tab" to copy a Weft color set to the Warp tab.

Note, as in all cases, that the database itself is not updated until you click on Save.

 

Delete All

This button will cause the entire Range to be deleted from the database. For Jacquard users it will delete either all warp or all weft depending on which tab is being used.

This is obviously a powerful feature and a warning message will be displayed as illustrated above.

 

Delete Set

Similarly, this button will cause the color set to be deleted from the range and a warning message will again be displayed as shown above.

 

Save Set

Example after a complete new color set has been saved

Example of the warning displayed when saving a color set that already exists - the software will display the number of changed colors, the number that are the same and the number of additional colors

To save the current set - make sure the Range number is correct and the color set is correct and click on the Save button. A confirmation will be displayed as shown above

If you are saving over a color set that already exists, the old color set will be deleted and the new one saved. The new color set will be exactly as seen on the screen. No old colors will be retained. However, in this case, a warning will be displayed before the save is started as shown in the example message on the right here which will summarise the changes that are to be made.

After a save has been completed (or a delete) the Shade Library display will be updated.

 

Clear Tab

To clear all colors in the list in order to start again with a blank list simply click on the Clear Tab button

 

Copying Colors from the dialog to the Palette

(1) Hold <ctrl>+<shift> and click on color set title to recolor the artwork (for Jacquard Designers will recolor warp or weft as appropriate)

(2) Drag and Drop an entire color set by "dragging"  the color set title to the palette.

(3) Drag and Drop an individual Color to the palette

Note in this dialog you cannot double click an individual color, nor hold ctrl+shift and click on an individual color.
