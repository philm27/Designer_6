Saving Projects

 

If you want to save a project - your main design together with all its associated shelf items, palette, frame plan (if applicable) and weaves (if applicable) then save it as a PRJ file.

The layouts must be saved separately but note that if the project is closed, the software will ask if you would like to save any modified Print layouts as these will also need to be closed.

If you do not want to save the shelf items you can save the design as an IFF file.

For more information about Save and Save As see the topic Saving Designs.
