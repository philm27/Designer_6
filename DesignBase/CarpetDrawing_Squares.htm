Drawing Squares

 

You can use this feature to draw a square taking into account the current ends and rows(picks). You can draw a normal unfilled or filled square in any line thickness.

Note: You can draw rectangles covering the entire design whilst zoomed in on the corners - see the Spacebar Toggle facility.

If you want to blank out the whole design, use the menu option Edit->Clear.

 

Normal Square

  • Click on the Normal Square button in the toolbox.

  • Use the keyboard shortcut Q

 

Filled Square

To select a filled square, either:-

  • Click on the Filled Square button in the toolbox

  • Use the keyboard shortcut keys <shift>+Q

 

Line Thickness

You can use the toolbox or the keyboard to select line thickness:

  • Move the mouse over the "Pen Selector" at the bottom of the toolbox and click on one of the 6 preset pen sizes. Alternatively double click this area to display a dialog permitting a larger size to be set.

  • While in Rectangle Drawing mode press one of the keys 1, 2, 3, 4, 5 or 6 to select line thickness.

 

Custom Brush

It is also possible to create a filled rectangle made from a custom brush. Cut a brush (or select one from a shelf) and then create a rectangle as before. You will notice that the rectangle is made up purely of repeats of your brush  it doesnt have a different outline colour.

The Brush will fill the rectangle with additional repeats and partial repeats if necessary. The fill is performed according to the current brush handle i.e. from the centre outwards if handle is Centre or from the corresponding corner outwards in other cases.

The current transparency setting will be used.

 

Line Thickness Greater than 1

When using a line thickness other than 1 the software will use the points defined by the mouse as the OUTER edge of the square rather than the centre of the line drawn. This makes it particularly easy to draw a border of a defined width (double click any of the rectangle or line buttons or the line icons themselves to get the line thickness dialog box).

Also, of course, if the square drawn is not large enough for the required line thickness it will simply be a solid square in the current left hand button colour. For example a line 1 inch thick would require a square more than 2 inches in width and height otherwise it will simply be a solid square.
