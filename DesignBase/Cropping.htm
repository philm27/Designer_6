Crop

You can crop ends or rows from the design visually using this dialog. To access it select Design->Crop.

 

Example of the Cropping dialog. In this example the user has modified the cropping rectangle to an 18 inch square and has moved it so that the main motif is central within that 18 inch square

 

Main Design Preview

Use the mouse to drag the rectangle around when the mouse is pointing anywhere inside of the cropping rectangle. When pointing at the tabs you can make the rectangle larger or smaller.

Design Section

These figures give the size of the current rectangle. In the example above, the user has modified figures using the keyboard, to make the selection rectangle an 18 inch square by first of all changing width and length to inches, and then changing the width itself to 18 in each case. The selection rectangle can then be moved to a suitable location in the preview.

Design Offset

This gives the offset of the bottom left of the selected section from the bottom left of the original design which in this case is 62 ends and 44 rows. Again, this can be modified manually if necessary.

Design Size

These "Read Only" figures give the original design size in ends and rows, feet and inches, and metres.

Section Size

These "Read Only" figures give the selected section size, again figures are shown in ends and rows, feet and inches, and metres.

Reset

Click this button to reset the cropping rectangle to the whole design

OK

Click OK to crop the design according to the current selections. The dialog will be closed. For Carpet Designer users, any current frame plan will be modified by removing any left and right sides as necessary. For Jacquard Designer users, the weave, if any, will also be cropped.

The locations of any polygons will be adjusted if necessary (i.e. if their distance from the bottom left of the design has changed). Also, any regions and their inlays and out-fills will be adjusted so that they relate to the same figure in the design.

Note that the design itself is not modified until OK is clicked and the cropping operation can then be undone if required.

Cancel

This will cause a return to the main design window without modifying the design in any way.
