Paths Folders and File Names

 

Designs, projects, palettes, exported files and frame plant guides are all saved as files within directories, also known as folders.

Use folders to keep your work ordered and easy to find.

You can find out more about creating folders here.

Path

The full folder and file name is often called the path and may look something like this:

c:\Jacquard\production\AA5378.PRJ

c:\Jacquard\palettes\Spring1.PAL

In the first example above, the file AA5378.PRJ is located in the "production" folder which is contained within the "Jacquard" folder which in turn is contained within drive "c:".

Similarly the file Spring1.PAL is in the "palettes" folder which is also contained within the "Jacquard" folder which in turn is contained within drive "c:".

File Extension

The file extension - a dot followed normally by a 3 character code - follows the main part of the file name and identifies the type of file. Thus project files for example have the .PRJ file extension and palette files have the .PAL extension. However, the file extension is strictly speaking part of the file name.

Limits

There is no limit to the number of directory levels although the full path cannot exceed 260 characters.

Also, there is no limit to the number of folders and files within a folder. Each folder and file name must of course be unique within the parent folder.
