Placing a Brush using the Mouse

 

You can place single copies of a custom brush into the design as many times as you wish.

 

Capture a brush

There are several ways to capture a custom brush so that you can draw with it:

  • Cutting a Rectangular Brush - The simplest and quickest method

  • Cutting a Freehand Brush

  • Cutting a Polygon Brush

  • Cutting an Elliptical Brush

  • Select it from the shelf using the left hand button

 

Paste a brush

Having captured a brush you can place it into the design as described below.

  1. If you want to make any colours in your brush transparent (so that they will not be placed in your design) in addition to the background colour, do this now.

  2. Choose the appropriate Brush Handle .

  3. Zoom in to the area in which you want to place the brush.

  4. Float the brush over the design until it is in the correct place (either by visual judgement or by checking the cursorÂ’s co-ordinates in the Status Bar ).

  5. Click with the left-hand mouse button to place the design.

  6. Place the brush in a shelf box (if it is not already there) if you think that you may want to use it again.

You can also press the button and draw with it, dragging the pointer across the screen. However, because of the time taken to draw a custom brush, this may not produce smooth "brush stroke".

 

Right Hand Button

If you draw the brush using the right hand button, then a solid colour version of the brush is drawn using the current right hand button colour.

This can be useful for certain design effects such as creating a drop shadow (select black as the right hand button colour then draw a black version of the brush using the right hand button then the normal version on top offset slightly).

Many designs also feature a pale version of the main motif in the background in a solid colour. You can create this type of background by using the main motif as your brush and selecting a pale colour for the right hand button colour, then drawing the brush using the right hand button. (Obviously if you have already drawn foreground motifs you would have to use the stencil feature so that only the background colour is overdrawn).
