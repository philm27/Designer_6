Drawing in Freehand

 

Freehand Draw enables you to draw freehand either just a single point or a continuous brush "stroke" using the current brush.

Choose the brush shape and size from the toolbox , or alternatively you can draw with a custom brush.

Draw with the left-hand mouse button to draw in your foreground colour . Use the right-hand mouse to draw with your background colour .

 

To select Freehand Draw either:-

  •   Select the freehand draw button from the toolbox, or,

  • Use the keyboard shortcut D

 

Brush Size

You can either:

  • Select a brush from the selection in the Toolbox. There are 8 standard square brushes from 1x1 up to 8x8 together with 4 round brushes, moresque, a standard size and a custom size. For more information click here.

  • Or, simply press one of the keyboard keys 1, 2, 3, 4, 5 or 6 to select a square brush size

 

Digitiser

If using a pen and digitiser, you can draw in freehand with a pressure sensitive pen to get lines gradating from thin to thick, rather like using a paintbrush.
