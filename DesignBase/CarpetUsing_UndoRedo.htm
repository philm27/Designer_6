Using Undo/Redo

 

Use the "Undo" feature to undo the last drawing operation. If you make an alteration to your design that you are not happy with, you can "undo" it, and get back to the stage you were at before the alteration.

For more information see the topic Undo Types.

Note that once you do another drawing operation (or something else that needs to be written to the history), the new data will replace any history that has been undone but not redone, though you can undo the new operation and any previous ones still in the history..

For example if you do 7 drawing operations which we will number 1 to 7 for convenience. Then undo all 7 and redo operations 1 and 2 and start drawing again, then drawing operations 3 to 7 will be permanently lost and cannot be redone.

Number of Undos

The number of drawing operations that can be stored is limited but can be set using preferences - see the topic Undo Preferences. The default is 20 but you can increase it to 100 or more if you wish.

Undo Preferences

Undo has a set of preferences - see the topic Undo Preferences.

Undo in Carpet Designer

Any frame plan edits and swap operations can be undone in the usual way using "Normal Undo".

Undo in Jacquard Designer

For Jacquard Designer users, the weave can be modified and undone separately so if you edit floats in the weave then return to the artwork you can undo artwork changes and if necessary return to the weave and undo any previous weave changes.

The weave history is cleared when weaves are applied.
