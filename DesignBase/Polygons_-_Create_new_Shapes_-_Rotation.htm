Polygons - Create new Shapes - Rotation

This is the Rotation tab on the Create New Shape Property Sheet. Use it to specify the amount of rotation of the shape.

For more general information see the topic Polygons - Standard Shapes.

Example of the Location tab showing an ellipse that was initially the same width and length as the design repeat but has been rotated clockwise by 45 degrees.

The Angle of Rotation

If this is zero then there is no rotation. A negative value is anti-clockwise and a positive value is clockwise.

Setting the Angle of rotation

You can modify the angle of rotation by one of the following methods:

  • Click in the "clock" face to cause the rotation angle to change by 5 degrees towards the cursor

  • Click on and drag the pointer round the "clock" face

  • Click in the Text Edit box and type a new value.
