Creating a Peg Section

 

This feature is only available to Carpet Designer users. It effectively creates a Brush containing a colour palette.

It is the quickest and easiest way to create a Peg Section for a sampling machine.

You can select a rectangular section of the design and a colour strip will be added automatically (if required).

For details of how to set the colour strip size see the topic Peg Section Preferences.

Procedure

To use this function select the menu option Brush->Create Peg Section and then select a size from the Sub Menu from the following list (all sizes in inches):

4 x 4, 6 x 6, 9 x 9, 12 x 12, 18 x 18, 24 x 24, 27 x 27, 36 x 36, Custom

If you select the last option "Custom" then the following dialog will be displayed:

 

An example peg section custom size request of 10 x 6 inches

After you have either selected the required size from the sub-menu or specified the size in the Custom peg Section dialog and clicked on the OK button, a rectangle will be displayed over the main design as shown below allowing you to select a suitable area for the Peg Board Section.

Note that whilst you are choosing the required section, you can zoom and move the design as usual and you can cancel the Peg Board Section by selecting a drawing function.

Simply click the left mouse button to select the required rectangle and the peg section complete with colour strip will be captured as the current brush.

Once selected, if your section does not contain all of your used palette colours, a warning will appear on the screen.

Example showing the selection rectangle for a 9 inch x 9 inch peg section. Simply click the left mouse button to capture the specified area. The colour strip (if any) is not included in the rectangle

Saving the Peg Section

The fastest way to save a peg section is to save it whilst it is a brush as follows:

  1. Select menu option Brush->Save. The Save dialog will be displayed.

  2. Save as the file type you need for your sampling machine (E.g. select .DES file from the File Type list)

You could also save it by placing it on the shelf and saving from there as described in the topic Saving a Shelf Item.
