Jacquard Preferences

Note this page is only relevant to Jacquard Designer users and will not be displayed for other users

Select Preferences using menu option Edit->Preferences then select the Jacquard Tab.

Jacquard preferences showing the Scottish system selected (warp threading sequence is right-to-left). None of the Carpet options are selected in this case. The weave search directory is \\CommonFiles\Weaves.

 

Scottish System

This preference means that the Warp Threading dialog must represent the sequence of the threads starting at the right hand side of the loom and working across to the left.

The Warp Threading dialog still reads from left to right however i.e. the first value in the first column represents the first stripe on the right of the fabric.

Also, when producing a cloth having Warp Extras, the software generates the Warp Threading automatically and it will also take account of this preference.

Another way in which this preference is used is when a design is opened. The software will check the threading sequence in the design file and if it is stored the opposite way to your preference it will display the following dialog:

 

The above would mean that your preference was left-to-right but the stored sequence was right-to-left or vice-versa. This may happen if you were sent a design file from a mill that had the opposite preference to your own.

Selecting "Yes" in the above dialog gives the user a simple way to reverse the warp sequence to match your own preference. (Note this is not changing the design - just the sequence as shown in the Mill Ticket or in the Threading Sequence dialog). You should re-save the design to permanently store the Warp Threading to your own preference.

Bracket Levels

The default bracket levels is the number of levels that will be visible on the threading dialogs (warp and weft). Designs loaded from disk will also show this number of levels or the number used whichever is the greater.

Users will only be able to add bracket levels up to the maximum specified here which itself must be in the range 1-3.

If a design loaded from disk uses more bracket levels than the maximum figure, a warning will be displayed.

Apply Weaves

Darkest is Weft - by checking this the software will assume that dark colors are weft i.e. drops instead of the normal lifts. This will be  important when editing the weave, when viewing cloth simulations and when making loom files.

Always Reset Weft Plan when Cramming  - This option only applies when using cramming in Simple Weaves. If it is checked then the software will automatically reset the weft plan after weaves are applied to a crammed design without asking the user. If not checked the software will always ask the user whether or not to reset the Weft Plan.

Carpet Users

The following options apply only to carpet and pile fabric users and should not be selected by Flat Cloth designers.

Wilton Dialogs and Messages

Selecting this option causes certain dialogs and messages to be oriented towards carpet users. It does not, however, cause any change in functionality. It simply invokes "cosmetic" changes.

Disable "Apply Simple Weaves"

Carpet users should use "Apply Composite Weaves" not "Apply Simple Weaves" in Jacquard Designer. However, the two options look very similar and therefore to prevent mistakes happening, this option allows the Simple Weaves option to be disabled so that it cannot be used.

Align Single Dent Weaves with Start of Motif

Some single dent Wilton effects like loops over 2 or more wires have a different weave on the first pick to that on the 3rd 5th 7th pick and thus the first pick of the weave needs to be aligned with the start of the motif. This enables the user to simply draw a motif 4 rows long for example anywhere in the design in order to get a loop over 4 wires or two loops over two wires after suitable weaves are applied. For more information see the topic Weave Alignment Preference.

 

Weave Search Directory

You can use this to set the location of your weave directories when they have been changed. This is quite common when weaves are held on a machine on the network as the network set up may change from time to time.

For example, if you move your weaves from a local directory c:\projects\apso\weaves to a new network location \\CommonFiles\Weaves you can set this preference to the new location as shown in the example above and the software will look in this location when and if a weave is not found.

Note also that the location specified can be the location of a series of directories and sub directories containing the weaves. In this example, an original weave path could be c:\ projects\apso \weaves\doublecloth\31.iff. The software will still find the weave providing the directory structure is the same i.e. the new weave path is \\CommonFiles\Weaves\doublecloth\31.iff.

The directory names used here are only for illustration. You can use any directory names and any number of directory levels.
