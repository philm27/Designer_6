Viewing With Texture

 

It is possible to view a section of your design at actual size with textured cut or loop simulation. Other simulations may be available.

The textured image is displayed in a secondary window that you can leave on your screen whilst you continue to work in your main design area. The texture window shows a snapshot of your design at the time you request it  it does not always update as you alter your design (for example whilst you change a palette colour). If you want to update the image, simply click in the texture window.

Texture files are stored by default in a directory called c:\apso\cadvtextures but they can in fact be stored anywhere on your hard drive or network.

 

Example Texture Window dragged out to a larger size showing the textured motif next to the untextured version in the main design window.

 

Follow these steps to view a texture:

  1. Select menu option View -> Texture.

  2. Select menu item Texture -> Load on the Texture Window menu or select Load on the Popup menu. A standard Open File dialog will be displayed.

  3. Texture files are by default installed in the APSO folder on drive c: so navigate to the APSO folder and then within this find the CADVTEXTURES folder

  4. Open the CADVTEXTURES folder  it normally contains two texture files CUT.BMP and LOOP.BMP

  5. Select CUT or LOOP as appropriate for the pile effect you require, and then click open. A section of the design will be displayed in the texture window.

 

Texture Properties

Display the texture properties by selecting Properties on the Popup menu or on the Texture window menu. This allows you to make the texture darker or lighter by using the sliding control to modify the threshold value in the texture image at which a yarn colour will become lighter.

You can select Auto to select the threshold value automatically.

Centering of Textured Image

The textured image is shown real size and will show as much of the main image as possible. The textured image will have the same centre point as the main display. So to view a particular motif, move it to the centre of the main display then click in the Texture window to update it.

Changing Window Size

If you want to see a larger sample, simply drag the texture window out to your required size (dont worry if the sample is distorted at this stage) and then click once anywhere within the window  your textured sample will be re-displayed correctly.

Scrolling the Display

To move the texture display simply press and hold the mouse button in the texture window and drag in any direction.

Texture Files

Texture files can be IFF, BMP, JPG, TIF, PCX and can be 8 bit or 24 bit. They should normally be in shades of grey but there is nothing to stop them being in colour however, they will be interpreted as a grey-scale image. Any colour in the image will not cause a change of tone in the design.

Pop-up Menu

You can display the Popup menu by clicking in the texture window with the right mouse button.
