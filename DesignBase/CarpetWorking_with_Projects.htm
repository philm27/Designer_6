Projects

 

A project consists of your main design and all its shelf items and all its print layouts.

The project is held in a .prj file which holds the main design and all its shelf items. The print layouts are file type .lyt and will be named for example project1_1.lyt, project1_2.lyt etc for a project file having a name project1.prj. The lyt files do not need to end with a number but they must start with the project name and the "_" (underscore) character.

There can be any number of projects loaded simultaneously and each project can have a maximum of 999 shelf items. This enables you to refer to other designs and copy sections from them to your current design.

The shelf items share the palette and will normally have the same quality as the main design which is in box 1 on the shelf.

When you initially start the software, the first "New Design" will be project 1, the second project2 etc.

To copy brushes or shelf items to another project see the topic Copying Brushes and Shelf Items between Projects.

 

Starting a New Project

Select menu item File -> New Project. This will create a new design with default specifications , using the palette from the project that you are replacing.

 

Clear Project

To clear a project, select menu option File -> Clear Project. You can then either open an existing design or create a new one. In the latter case, the new design will be created with your default specifications and default colour palette.

 

Saving Projects

If you save the project as file type PRJ then the shelf items will also be saved in the same file with the main design - see the topic Saving Projects.
