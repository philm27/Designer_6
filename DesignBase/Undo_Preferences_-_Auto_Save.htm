Undo Preferences - Auto Save

Undo Preferences are displayed by selecting menu option Edit->Undo Preferences. The Auto Save Tab looks like this...

Do Auto Save

Check this box to enable the Auto Save feature. You cannot modify the remaining items unless this box is checked.

At least once every

This is the number of minutes between one Auto Save and the next.

Note that you can force an Auto Save for the current project using keyboard shortcut <Ctrl>+\ or use the menu option. If you force an Auto Save, the timer will restart from zero.

In normal circumstances the timer will start on completion of an Auto Save. This may mean you do not get a save exactly after 10 minutes as the software will wait for completion of a drawing operation before doing an Auto Save, this could be a matter of minutes in some cases like a very large Circular Border.

Number of Files to Retain

You can specify as many files as you like here. The minimum is 1 but you could use 6, 10 or even more.

This number does not impact on software performance or memory usage. It does however, use more disk space of course.

Progress Bar

It is recommended that you check this box in order to show a progress bar whilst an Auto Save occurs. For large designs, the save may take a few moments and you will not be able to design whilst it takes place.

Auto Save Folder

This will initially default to c:\autosave. The folder will be automatically created if it does not exist when the dialog is closed or when and Auto Save takes place.

You can browse for a folder using the "..." button.

It is recommended that you choose a simple path such as the default.

If you change the folder whilst using the software, any files in the old folder will still be deleted when a new file is saved. However, they will not be "adopted" as history when the program is restarted in a new session.

Show warning when program starts if files exist

The software can be made to display a warning when the program starts if Auto Save files exist in the specified folder. The message looks like this.

Clicking Yes will display the list of auto save files. Clicking No will start the software as normal.

It is a good idea to review the list of Auto Save files when starting the program and deleting any files that are no longer required, however any files that relate to a current project that are working on can be left in place and will be removed automatically as they are replaced by new ones.
