Outline

 

This useful and versatile tool allows you to automatically outline one or more design colours. You can vary the outline thickness, restrict it to one or more sides of a motif and/or change the corner style as described below.

Procedure

First of all select the menu option Design->Outline. The following dialog will be displayed

Preview Tab

Initially a small part of your design (or the whole thing if it is quite small) will appear on the left hand side as a preview of the current outline properties. The whole design will appear on the right-hand side. Select a section from the right hand side to use as a preview. This will then appear in the left-hand side of the window.

You can re-select a different rectangular area as many times as you like and you can return to this tab later and select a different area. This can be useful as you can see the effect of the settings you have made in different areas of the design.

The preview area should be as small as possible, whilst still containing samples of the colour(s) you want to outline the smaller the preview area, the quicker the application works.

Note that later, when you click on the OK button, the software will apply the outline to the whole of your design, not just the preview area.

Here, a small section from the centre of  the design is being used as a preview area

Color to Outline Tab

This tab will display the palette allowing you to select one or more colors to be outlined.

You can remove a tick by clicking on that colour a second time.

Note that if you move the mouse over the design preview on the left, you can select colors from the design preview.

Here, the green, color 11 has been selected to be outlined and the preview has shown the result although we are going to modify the thickness and the outline color.

Outline Color

Select, either in the list of colors or in the preview, the color that is to be used for the outline.

  2. In this case the blue, color 15, has been selected and this is reflected in the preview

 

Outline Thickness

Specify the width of the outline in either ends/rows/picks or in inches or millimetres.

 

In this example, a one inch wide outline has been set and this is immediately shown in the preview in light blue.

Direction

This allows the outline to create highlights or shadows.

For example, to create a highlight with the light source coming from the top left click on Top and Left and de-select Bottom and Right (you would probably use a light colour for the outline in his case).

To create a shadow from the same light source, click Bottom and Right and de-select Top and Left (normally outlining in a dark colour).

To create a constant outline around a shape (as in this example) select all 4 corners.

Corners

You can choose one of None, Square, Round or Cut. An example of the chosen style is shown in the preview on the right of the dialog.

The difference between the styles is more apparent when the outline is several pixels thick as in the example on the right of this tab.
