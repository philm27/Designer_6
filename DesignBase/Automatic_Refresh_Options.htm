Automatic Refresh Options

 

Sometimes, when editing a very large design you may find that the software is not quite as responsive as you would like. This may be because it is automatically updating things like the palette usage after every brush stroke. These "Automatic Refresh" items can be switched off in preferences as described below.

Example of General Preferences note that the "Frame Plan" and Bobbin Count options apply only to Carpet Designer.

 

Each of these options causes additional processing time after each brush stroke which may be noticeable if the design is very large. It may therefore be of some benefit to de-select these options in this case.

 

Palette Usage

If this is selected the software will update the palette usage after every brush stroke. The items updated include the "tuck" at the top right of each palette swatch as well as the percentage figure in the palette information.

Bobbin Count (Carpet Designer only)

If this item is selected, the software will update the Bobbin Count in the Palette information on each and every design change. This only applies to Carpet Designer.

Palette Stencil

If this is checked, then the stencil (based on the palette) is updated after every brush stroke.

If it is not selected then you must force the stencil to be regenerated yourself by pressing "P" on the keyboard or by selecting the regenerate stencil option on the menu (Palette > Stencil > Regenerate).

Frame Plan (Carpet Designer only)

If this is checked, then the frame plan will be updated after every brush stroke.

Suppress Auto-Refresh

If this is checked then the software will not perform Auto-Refresh if the design size (i.e. width x length in ends and rows) exceeds the specified figure. This is very useful for designers who use the software to create floor plans as tuft counts and bobbin counts on extremely large designs can be quite time consuming and is not normally required.

For example you may choose to suppress Auto-Refresh on all designs exceeding 32MB as in the example above.

 

For details of other options on this tab see the topic General Preferences.
