Polygons - Create new Shapes - Colors

This is the Colors tab on the Create New Shape Property Sheet. Use it to control the Guide Line color of the shape together with the centre panel color.

For more general information see the topic Polygons - Standard Shapes.

 

 

Guide Line

Use this to both indicate that the shape is a guide line and also to set the guide line color. The Guide Line color is used both to draw the line when editing Polygons and to draw the line over the design when being used purely as a Guide Line.

The Guide Line setting can be changed at any time while in Polygon edit mode and Guide Lines can be switched on or off at any time while designing. For more information see the topic Guide Lines.

To change the Guide Line color simply click in the color swatch (yellow in the example above) and a standard color selector will be displayed.

Centre Panel Draw Outline

Check this box to cause the outline of the centre panel to be drawn using the specified color in the adjacent color swatch.

Fill Panel

Check this box to cause the centre panel to be filled using the specified color in the adjacent color swatch.

Outline Color and Centre panel Color

To choose a color for the either the outline of the centre panel or the fill color, click in the appropriate color swatch to display the design color selector. If the swatch shows an x in a white background as in the example above it means that the item is not to be drawn
