<Home> and <End> Keyboard Shortcuts

These work with certain drawing functions in a special way in order to simplify some common drawing operations. To use them simply press <Home> then <End> (or vice versa) to simulate pressing the left mouse button in a corner or in the centre of the design as appropriate after selecting the drawing function. There is no need to use the mouse.

* Note that these operations use the "Brush Handle" (see the topic ) so if the handle is bottom left then the <Home> key will place the brush so that the bottom left corner of the brush is at the bottom left of the design - useful for the corner of a rug. If the handle is centre, the <Home> key will place the centre of the brush at end 1 row(pick) 1.

In the above table BL is Bottom Left, TR is Top Right, TL is Top Left, BR is Bottom Right.

Note that the points can be done in reverse i.e. <End> then <Home>.

<Ctrl> Key with <Home and <End>

<Ctrl> can be used together with a key to switch top and bottom so that <ctrl> + <home> then <ctrl> + <end> will draw a line from TL to BR.

 

Drawing Lines using <Home> and <End>

One of the most useful functions provided by this feature is to be able to draw diagonal lines simply by selecting line draw, then the appropriate left hand button colour then pressing <Home> and <End> on the keyboard. For the opposite diagonal simply press <Ctrl>+<Home> then <Ctrl>+<End>.

For example a line can be drawn up the left edge of the design by pressing <Home> then <Ctrl>+<Home> while using line draw.

Using Either <Home> or <End> with the Mouse

Another way in which these keys can be used is to draw a figure using the <Home> or <End> key together with the mouse. Using <Home> or <End> can be a substitute for locating the start or end of a figure using the mouse.

For example, to draw a line from the centre of the design to the bottom left corner, select line thickness and colour as normal then click in the centre of the design, this will start a line there as normal. Then press <home>, the line will be drawn using the bottom left as the end point.
