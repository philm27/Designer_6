Protected Areas

 

It is possible to protect areas in the design. Once an area of your design is protected, any work you do will not affect the protected area. For instance, if you draw or spray on your design, any color that lands on your protected area will not "stick" as it is masked off.

Note that you can undo drawing operations in the Protected Areas Mask in the normal way.

Also note that the Protected Area masks, if present, are saved for all shelf items.

Procedure

To crate protected and working areas similar to the above, follow these steps, (there are menu equivalents of all these options).

  1. Activate protected areas by clicking on the left most button in the Protected Areas Tool bar.

  2. Click on the large red cross button to make the entire design protected.

  3. Click on the "Working Ellipse" button and draw an ellipse in the design*

  4. Use normal drawing operations to draw into the design

 

The Toolbar buttons

Protected Areas On/Off

Click on this button to switch protected areas on or off. When switched off the mask is retained and can be used again later. In fact it will even be saved with the design and can be used again at a later date.

Make all Working

Will cause the entire design to be made into a Working Wrea - effectively deletes the mask (note that undo is available).

Make all Protected

Will cause the entire design to be made into a Protected Area - effectively deletes the mask (note that undo is available).

Make Rectangular Working Area

This drawing function will allow one or more rectangular Working Areas to be created. Draw them in the usual way by pressing the left mouse button and dragging out to the opposite corner of a rectangle. You can cross the edges of the repeat when you do this.

Make Elliptical Working Area

This drawing function will allow one or more elliptical or circular Working Areas to be created. Draw them in the usual way by pressing the left mouse button and dragging out to a corner of the containing rectangle. Alternatively, as for  normal ellipses, you can hold the <shift> key and click and hold the left button on one corner of the containing rectangle and drag the mouse to the other before releasing. You can cross the edges of the repeat when you do this.

Make Irregular Shaped Working Area

This drawing function will allow you to press and hold the left mouse button while drawing an irregular shape. You can cross the edges of the repeat when you do this.

Fill Working Area

This drawing function will allow you to click in a figure in the main design in order to make that figure into a Working Area

Make Rectangular Protected Area

This is used in the same way as for the Working Area above except that a Protected area is created.

Make Elliptical Protected Area

This is used in the same way as for the Working Area above except that a Protected area is created.

Make Irregular Shaped Protected Area

This is used in the same way as for the Working Area above except that a Protected area is created.

Fill Protected Area

This is used in the same way as for the Working Area above except that a Protected area is created.

Invert Mask

Click on this to make Working Areas into Protected Areas and vice versa.

Grow Working Area

Click on this to "grow" the working area. For more details see below.

Grow Protected Area

Click on this to "grow" the protected area. For more details see below.

Create Mask From Stencil

Click on this to replace the current Mask with one derived from the current Stencil Selections after this has been done the mask can be edited in the usual way as described here.

Add/Subtract Colors

Displays the "Add/Subtract Color" dialog allowing you to protect or make working, all occurrences of one or more colors.

 

Add/Subtract Colors Dialog

The list shows the current status of each color and allows the user to change a color to all protected or all working without affecting other areas of the mask.

 

This example might follow on from the example design above in which the user would like to make the ground color a working color but only within the central ellipse that has been drawn in the mask. Here, all colors are selected except for color 1 which is the ground color in this case.

If the user clicks on the Protect button, all selected colors will be made protected as shown here and color 1 will remain partially protected.

Selecting items in the list

You can select items in the list in the usual way by clicking on them and by holding the <ctrl> button to select a series of individual entries, or by holding he <shift> key to select the first and last in a consecutive sequence of entries. You can hold the <ctrl> key whilst selecting an item to toggle it between selected and not-selected whilst keeping all other selections.

Pop-up menu

Having made some selections, you can right click in the list to select one of 3 options: Protect, Unprotect, Leave. These are also explained below.

Protect

This will protect all selected colors

Unprotect

This will unprotect, i.e. make into Working Areas,  all selected colors.

Leave

This will remove a new Protected or Working designation. It is only required when a color has been switched to Working or Protected by mistake.

Select All

This will select all colors.

DeSelect All

This will unselect all colors.

Invert Sel.

This will select all colors that are not currently selected and vice versa.

Grow Working or Protected Area

The dialog will be displayed if either of the "Grow" functions is selected. Note that these operations can be undone using the normal design undo/redo methods. You can use the dialog several times to Grow or Shrink the designated area before closing it.

An example of the Grow Protected Area dialog with a "grow" value of 2 ends/rows The user has already clicked grow. All colors except the background were initially protected and this will prevent any background fill from touching the motif

Grow By

Set this value using the up or down arrows or by using the keyboard as usual.

Units

Select the units Ends/Rows or In/cm etc from the drop down list

Grow/Shrink

Click on either the Grow or Shrink buttons to Grow or Shrink the Area. The title bar of the window will indicate the currently selected function "Working" or "Protected".

Close

The dialog can be used for several successive Grow and/or Shrink operations before it is closed using either the Close button or the standard X button in the title bar.
