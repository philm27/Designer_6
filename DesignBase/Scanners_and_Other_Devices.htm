Scanners and Other Devices

 

This option allows direct access to a scanner or other image capture device.

The device must be TWAIN compatible.

The Import from Twain device on the File Menu

If these options are disabled on your menu see the notes below regarding additional software.

To "acquire" an image from a scanner or other device select menu option File->Import->Twain Acquire. The device must be connected and switched on.

If you have several devices you may need to use "Twain Source..." to select the appropriate device - the newly selected device will be the new default when the software is next started.

Twain Source

If there is more than one device, you can select one or the other using the "Twain Source…" option which will allow selection from a list of available devices.

 

Performing a Scan

  1. To perform a scan, make sure the device is attached and switched on.

  2. Select Menu option Twain Acquire – there will be a short pause then the device driver will display its own dialog such as the one shown below.

  3. It will normally be necessary to perform a preview scan

  4. Drag a selection rectangle round the area to be scanned.

  5. Change any settings such as dpi  as necessary.

  6. Click on Scan.

  7. When an image has been acquired the software will ask whether to keep as 24 bit or reduce to a set number of colors.

  8. After the scan is complete, the image will be written to a new project and you can save, edit, color reduce and use all normal design tools as required.

Installing Scanner and Twain Software

In order to import an image from a scanner, the following file is required

C:\Windows\System32\TwainDSM.dll

If this file is not present, the Twain Acquire and Twain Select options will be ghosted out and not selectable.

Warning - there is likely to be a different version of this file for 32 bit applications in the folder C:\Windows\SysWOW64. This is a different file and should not be copied to C:\Windows\System32.

A copy of the file can be obtained using the link //https://github.com/twain/twain-dsm but be careful to access the 64 bit version not the 32 bit version.

If your system does not have this file installed you can contact your sales representative for assistance.

You will also need a 64 bit driver for your scanner and this should be provided by the scanner supplier or available for download from the manufacturers website.
