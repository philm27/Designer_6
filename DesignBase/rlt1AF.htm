Using the Toolbox

 

The toolbox allows you to have easy access to the most commonly used functions whilst you are designing. You can remove from the display by selecting View->Toolbox on the menu and you can display it again by selecting the samem option.

Alternatively, try using the keyboard shortcut F4.

Familiarise yourself with the toolbox by clicking on the icons within it.
