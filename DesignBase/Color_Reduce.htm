Color Reduce

This option will reduce the number of colors in the image to a specified value.

(The alternative function Reduce To Palette allows a color reduction to a specific palette).

When reducing an 8 bit image (i.e. 255 colors or less), the software will choose which colors to use from the existing palette.

When reducing a 24 bit image the colors chosen may not be present in the current image.

To do Color Reduction use the main menu option Palette->Reduce Colors. The following dialog will be displayed.

Color Reduce Dialog

In this example the current design has 24 colors and the user has specified a reduction to 16 colors.

Current Design Colors

This will show the number of colors currently being used by the design. It can be several thousand if the image is 24 bit. It does not indicate the current palette size

Number of Colors

The user should enter the required number of colors in this field then press OK or Cancel as appropriate. If clicking on OK, the following dialog will be displayed.

Approve Changes Dialog

This will be displayed after a color reduction.

An example of a reduction from 24 colors to 16. The user can click on OK to approve the change or Cancel to return to the Color Reduce Dialog above.

Undo

Note that even after clicking on OK in the Approve Changes dialog, the user can Undo in the normal way using Design Undo (not palette undo).
