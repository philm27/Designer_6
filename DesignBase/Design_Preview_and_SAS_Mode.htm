Design Preview and SAS Mode

 

The preview window, selected using function key <f7> or selecting View->Design Preview, can be very useful whilst designing in SAS mode.

Example showing the Preview window (reduced in size) and the user about to select "Edit this Section from the Popup menu

The Preview window always shows the whole design either 1 repeat or several (Display Many) rather than the current section therefore you can see how you changes affect the whole design.

Also the Preview window can show a section and by changing the zoom level and/or moving the design using the cursor, you can check any section of the design.

If you find a section that you would like to modify, simply select "Edit this Section" on the Popup menu. If you are using SAS mode, the currently visible section will be selected even if it crosses the edges of the repeat. (Note that the Preview widow will not allow the preview to cross the edges of a rug nor the sides of a runner).

For more details about the Preview Window, see the topic Viewing a Design Preview.

Twin Monitors

You may find that if you are working on very large designs for an extended period, a twin monitor set-up is useful. This allows the Preview window to be placed, full size, on the second monitor whilst the main design and other windows are on the main monitor.
