Easily Resolved Problems

 

Screen Mode:-

Carpet Designer should always be run using a screen mode of True Colour.

To check what screen mode you are currently running in, follow these steps:-

 

  1. Click with the right-hand mouse button on the desk top this is the screen which displays "My Computer" and other shortcut icons.

  2. A pop-up menu will appear select Display Settings (Windows 10) or Properties.

  3. A Display Properties window will appear click on the Display option if not already selected.

  4. You can modify the resolution here if you wish

  5. Click on "Display Adaptor Properties" at the bottom.

  6. Click on "List All Modes" and make sure that the selected option is a "True Colour(32 bit)" and if not - select a True Colour(32 bit) option and click on OK.

  7. Click on OK in the "Display Adaptor Properties" dialog.

  8. Close the Settings window.

Screen Refreshing:-

If your design and shelf box turn into blank white rectangles, you need to refresh your screen.

Simply hit the <shift>+<F5> keys on the keyboard this will recover the picture.

 

Task Manager:-

If the software locks, and you cannot use any function on screen, you will need to close the program down. You will be unable to do this in the usual way as the cursor will not respond - you need to use Task Manager.

To access Task Manager, follow these steps:-

  1. Press <Ctrl> + <Alt> + <Del> keys the Windows Security window will appear on screen. ( In Windows XP the Task Manager may be displayed immediately)

  2. Click on Task Manager if not already displayed

  3. Select the Processes tab - all programs currently being used will be listed here

  4. Select the program you are using so that it is highlighted (it will probably have the status "Not Responding").

  5. Click on the "End Task" button.

  6. Close the Task Manager window.

Once the program has closed, you can start it again and continue working. Unfortunately, any unsaved work will have been lost. It is always advisable to save work frequently to avoid loss of work and effort in case of computer system failure for any reason.
