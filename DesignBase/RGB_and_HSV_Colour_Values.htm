RGB and HSV Colour Values

 

When using the Palette Change Dialog you can switch between the RGB system and the HSV system at any time.

 

RGB

While in RGB mode, the 3 controls on the Palette Change Dialog allow the Red, Green and Blue values to be changed. In this case when all 3 controls are fully to the right (all 3 values 255), the resulting colour is Pure White. When they are fully to the left (all 3 values are 0), the resulting colour is Black.

When all 3 values are equal the colour is a neutral colour. There are thus 256 possible shades of grey from black 0, 0, 0 to white 255, 255, 255.

 

HSV

When in HSV mode, the 3 controls on the Palette Change Dialog represent Hue, Saturation and Value. Each value is on a scale from 0.0 to 1.0:

H

Hue

Varies the colour from red through orange, yellow, green, blue indigo and violet and back to red again i.e. red has the value 0.0 and also the value 1.0

S

Saturation

Saturation - Increases or decreases the colour intensity from 0 (no colour) to 1.0 fully saturated

V

Value

Value - lightens/darkens the colour from 0 (black) to 1.0 (brightest)

Note that this system does not give any additional colours to the RGB system, it is simply a different way of selecting a colour.
