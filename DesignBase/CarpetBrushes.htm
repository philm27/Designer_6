Brushes

This panel in the toolbox shows a selection of standard brushes. The first 8 are simple solid colour squares from a size of 1x1 up to a size of 8x8.

These are followed by 4 small circles having diameters of 3, 4, 5 and 8 pixels.

The final row features 2 moresque patterns, a solid square* and a solid custom size square which you can set to any size you like.

* The third brush on the bottom row has a size related to the specification ends. For example the size will be a 1 inch square if specification uses "ends per inch" or 1 cm if specification uses "ends per cm" etc..
