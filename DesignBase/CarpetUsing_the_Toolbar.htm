The Toolbar

 

The software uses several different tool bars. These are normally situated at the top of the screen but can be moved to any convenient position and their location will be remembered from one session to the next. It contains shortcut icons to a number of commonly used functions.

You can turn the tool bars on and off using the options on the View menu.

When the pointer hovers over a button it will change from it's "Flat State" to a raised button as in the case of the Save button above.

When a button is selected it will change to it's "depressed" state as shown by the View Many, Autoscroll and Aspect Correct buttons above.
