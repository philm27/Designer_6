Mitre Brush

 

This feature allows you to mitre a brush to any angle as shown below.

Select the menu option Brush->Mitre. This will open the Brush Mitre dialog. Enter the angle in degrees for the mitre. You can use any angle between 0 and 90 degrees.

The mitre facility always mitres the brush down the vertical axis in the centre of the brush. If you want to mitre a brush in the horizontal axis you will first need to rotate it by 90 degrees.

The mitering facility is useful for running a border around a corner.

To use this feature you will normally capture a brush representing 2 sides of a polygon as they would look straightened out such as in the example below. This can then be 'bent' to produce the same two sides as they would look in the design.
