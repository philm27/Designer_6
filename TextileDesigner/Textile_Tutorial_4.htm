Tutorial 4

 

This tutorial continues with the design created in Tutorial 2 and shows how a basic design can be adapted and developed into a rug.

The tutorial covers the following topics:

  • Creating a Border

  • Mitre Corners

  • Constructing a finished Rug

  • Copying shelf items from one project to another

  • Auto-scroll and Custom Brush Filled Rectangles

 

Before we Start - Correcting mistakes

As before, remember that if you make any mistakes you can use the "Undo" feature and if you undo too many times you can use the "Redo" feature.

Border Design

We can now create a matching border design. We are going to do this by editing the previous shelf item so that the border design will be a shelf item in the project.

  1. So begin by clicking on the 4 inch by 4 inch shelf item with the left button (shelf item 4 in the example above). This will capture it as a brush and click immediately in the next empty shelf item - again with the left button. This will effectively copy the shelf item from one box to another.

  2. Then, as we are going to edit the new shelf item (to convert it to a border design), right click it and select Edit from the Pop-up menu.

  3. The new shelf item should be visible as the main design and it should be outlined in red on the shelf indicating that it is the current design. The current shelf item number should also be shown at the extreme right of the status bar.

  1. Change the repeat type to Horizontal by selecting menu item Design->Repeat->Horizontal and also select menu item Design->Match->Straight. (The software allows Horizontal repeats to have a drop so we must set it to straight match in this case).

  2. Make sure grid is on (press G on the keyboard).

  3. Make sure dark blue is the current right hand button colour.

  4. Now we need to remove the light blue pin-dot. To do this select Global Colour Change in the toolbox and, using the right button, click on light blue in the design itself. All occurrences of light blue should now be changed to dark blue.

  8. Using Black draw a horizontal line across row 26.

  9. Using Light Blue, Fill above row 26.

Rug Design

 

To construct the rug from the elements created above, we will use a separate project:

  1. Select menu item Project->2

  2. Select menu item File->New to create a new design.

  3. Select menu item Palette->load and select MEDALLION.PAL which was saved earlier. (Alternatively you can load a palette from one of the previously saved IFF files if you select file type IFF from the drop down list).

  4. Select menu item Design->Specification and change the repeat type to "Rug", the size to 308 ends x 392 rows and the quality to 7 ends by 7 rows per inch. Make sure "Grid as Quality" is ticked. Click on OK to close the specification dialog, and click again on OK when the dimensions dialog appears. Your design area is now set up to enable you to design a rug size 3'8" x 4'8" including a 4" border.

  5. Fill your design area with dark blue.

  6. Import your border and field designs by right clicking an empty shelf item and selecting Import from the pop-up menu. You should put the field design (Medallion1.PRJ) in shelf item 2 and the border design (Medallion Border.iff) in shelf item 3. Note that a dialog will appear giving the option of (YES) adding all used colours to the colour palette or (NO) using the existing design palette where possible. Click on NO as that will force the imported items to use the same colours as the main design when the RGB values are the same so that in this case there should be no need for the software to add additional colours to the palette

  7. Click on the border design in shelf item 3 in order to select it as a brush.

  8. Click on the Rotate 90 degrees clockwise button on the toolbox so that the brush is aligned as for a left edge border. The blue/black edge should now be on the right and the green/beige edge should be on the left.

  9. Select menu item Brush->Numeric Paste. This will allow us to accurately and easily paste the borders on to the design.

  10. You only need to change the two drop down lists at the bottom of the dialog. Change the first one to "At Left Edge" and the second to "Full Height". Click on Apply and the left border should appear in the design. If you make a mistake you can use the Undo button in the dialog - it performs the same purpose as the Undo button in the toolbar or toolbox.

  11. Now, with the Numeric Paste Box still visible, click on Rotate Brush Clockwise twice so that it is now oriented as for a right edge. Now change the left hand drop down list to "At Right Edge" (other should still read "Full Height"). Then click on apply - you will see that your border now also forms the right edge of the rug.

  12. Click on Rotate Brush Clockwise once so that it is now oriented as for a bottom edge. Specify that you want to place the brush "Full Width" and "At Bottom Edge". Then apply - you will see that your border now forms the bottom edge of your rug. (partially drawing over the left and right edges already applied).

  13. Click on Rotate Brush Clockwise twice so that it is now oriented as for a top edge edge. Specify that you want to place the brush "Full Width" and "At Top Edge". Then apply - you will see that your border now forms the top edge of your rug (again, partially drawing over the left and right edges already applied).

  14. You can now close the Numeric Paste dialog.

Top section of rug showing top border after Numeric Paste

Bottom section showing bottom border after Numeric Paste

Now, in order to mitre the corners, select Design->Mitre and click on each of the four main buttons once so that the top two are pointing downwards and outwards and the bottom two point upwards and outwards. If you make a mistake you can keep clicking until the chosen setting is displayed. These buttons control the direction of the diagonal mirror.   1. Close the mitre window.

  2. Go to HANDLE in the BRUSH menu and select a central handle. (This means that when you fill the field, the centre of your brush will be placed in the centre of the rug - and will repeat outwards from there.)

  3. Pick up your field design as a brush from the shelf box.

  4. Select Fill in the toolbox (or F on the keyboard) and click anywhere in the field of the rug.

Insert Medallion Into Field

 

Lastly, try inserting your alternative medallion design into the field area of the rug. We will use an alternative technique for this - the filled rectangle - which can use repeats of the current brush, as can Filled Circle, Filled Ellipse and Filled Square. When using a Custom brush, these functions do not draw an outline round the overall shape.

We will also learn how to copy a shelf item from one project to another.

A version of the Rug using alternative field design

  1. The alternative medallion is a 4 inch square with the medallion in the centre and a half drop repeat. This should still be on the shelf in Project 1. To copy it, first select menu item Project->1. Then right click the alternative field design (which should be shelf item 4) and select Copy. This will copy the design to the clipboard.

  2. Move back to project 2 by selecting menu item Project->2, right click an empty shelf item and select Paste. Select "No" in the resulting dialog to use the same palette colours. the alternative field design should now be visible on the shelf in project 2.

  3. Click on the shelf item with the left hand button to select it as the current Custom Brush.

  4. Menu item Brush->Match->Half should be checked. If it is not, select it. Note that repeat drop and match and "handle" information cannot be passed via the clipboard hence the need to reset it as described here.

  5. Make sure that the brush handle is set to centre (menu item Brush->Handle->Centre) this will cause a medallion to be placed in the centre of the field when drawn.

  6. Select drawing function Filled Rectangle (keyboard shortcut <Shift> + R).

  7. Now, to draw over the field of the rug press 1 on the numeric keypad to take us to the bottom left of the rug. Click and hold the left mouse button in the bottom left of the field area (end 29 row 29). Now, while still holding the left mouse button, press 9 on the numeric keypad to take us to the top right of the design. Move the mouse to the top right of the field (end 280 row 364) and release the mouse button.

  8. An alternative way of doing the previous operation is to use auto-scroll. This is switched on using the button in the toolbar or by using keyboard shortcut A. The design will automatically scroll when the pointer moves off the edge of a design when drawing large lines or rectangles. After pressing and holding the mouse button on the first point, move the mouse towards the top right. You will find that the design will scroll when you move off the edge of the display area. With a little practice you will be able to control it quite easily. The top right of the field area will eventually appear enabling you to release the mouse button at the required point.

Note that the software does not use transparency when drawing Filled Rectangles or Squares. It is possible that a future version may offer this option and thus you would need to first clear the area or make sure that no colour within the brush is currently transparent in order to over-paint the original.
