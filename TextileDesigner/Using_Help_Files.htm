Using Help Files

 

 

  • When viewing Help, you can maximize the Help window or resize it to suit your personal preference.

  • Click Back to return to the previously viewed topic.

  • Click Print to print the current topic.

  • Use the tabs at the top left of the Help window to select the Table of Contents (TOC), Index, Search or Favourites.

  • To get a general picture of the facilities available to you as a designer, browse through the Contents, and read topics that are of interest.

  • Use the Index to find help related to a particular key word.

  • You can also search and list all topics which contain a certain keyword using the Search tab.

  • Assemble your own list of favourite topics using the Favourites tab.

  • If a topic is longer than the window size, it will have a scroll bar on its right-hand edge. You can use the mouse to adjust the scroll bar up or down to read the whole topic. You can also use the up and down arrows on the keyboard.

  • If a word or words in the text are shown in a different colour and underlined, like this: link to another topic, then you can click on it to jump to that topic. Try clicking on the link in the previous sentence to see how it works and how you can easily return to the original topic.

  • Many of the graphics within the Help File hold hidden information Â– by clicking on areas within the graphics, this information will be disclosed.

  • When explaining how to use the tools for designing, the Help Files always refer to the left and right-hand buttons on the "mouse". If you are working with a pen and digitiser, then for "left-hand mouse button" read "nib of pen" and for "right-hand mouse button" read "side button on pen".

  • Having displayed a Help topic, you can move to another topic using the buttons at the top of the Help topic window in three ways:-

 

    1. Click on CONTENTS to return to the Contents Library.

    2. Click on INDEX to return to the Index.

    3. Click on BACK to return to the Help topic that you had displayed before the present one.

 

  • Having read a Help topic, you can minimise it to the taskbar when you return to your design work - so that it is readily available to bring back up on screen if you want to refer to it.
