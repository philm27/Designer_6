Tutorial 3

 

This tutorial introduces some of the basic techniques used when producing a traditional carpet design though the techniques used here are equally applicable to fabric design.

The tutorial covers the following topics:

  • Starting a new design

  • Editing a Palette

  • Creating a custom brush

  • Brush Mirror and Rotate

  • Working with Shelf items within a Project

  • Exporting from the shelf

  • Saving Projects

 

Before we Start - Correcting mistakes

Remember that if you make any mistakes you can use the "Undo" button, and you can undo several times. If you undo too many times you can use the "Redo" button as many times as you like. You can undo palette changes as well as drawing operations.

Field Design

Create a 2" repeating pin-dot throughout the design by following these steps:

  1. Press G on the keyboard to draw the grid.

  2. Zoom in to a convenient magnification similar to that in the image above using the + key on the numeric keypad.

  3. Press 1 on the numeric keypad to move to the bottom left of the design. In this case since it is a normal repeat (as opposed to a rug), the software will show a small part of the adjacent repeats at that corner.

  4. Press D on the keyboard to select Freehand Draw.

  5. Select light blue (colour 2) as the left hand button colour by clicking on that colour in the palette using the left mouse button.

  6. If you have not already done so, select dark blue (colour 1) as the right hand button colour. We do this so that if we make any mistakes with the foreground colour we can easily paint over them with the background colour. Also, this will make the background colour transparent when you pick up a brush.

  7. Draw using a single click, the two blue pin-dots as shown above at end 1 row 1 and at end 8 row 8.

  8. If you accidentally draw light blue in the wrong position you can simply use the right hand button to over-paint any mistakes in dark blue.

A useful tip here is to press <F11> key on the keyboard which will switch on the information panel as shown above. (You can switch the panel off at any time by pressing <F11> again.) For freehand draw this will show the current end, row and distance from the bottom left corner. The end and row is also shown in the status bar at the foot of the main design window. The information shown varies slightly with the function being used.

  1. Select Rectangular cut (keyboard shortcut <Ctrl> + R) in the toolbox (see the topic Cutting a Rectangular Brush) and pick up a brush exactly 2 inches square i.e. 14 ends by 14 rows from end 1 row 1 to end 14 row 14. To do this click the left mouse button on end 1 row 1 and, holding the mouse button down, move to end 14 row 14 and release the button. The two dots should now be seen floating over the design as the mouse moves around.

This shows the rectangle just before the mouse button is released. Again, the <F11> feature can be very useful as the information panel now specifies the size and location of the current rectangle - the position where the left button was first clicked, (1,1) the current position (14,14) and the size of the rectangle selected (14 x 14).

  1. It is good practice to place a new brush on the shelf in case it is needed again later. To do this simply click on an empty shelf item with the left button.

  2. Now click on the Fill button in the toolbox (or press F on the keyboard) and click anywhere in the background of the main design. The pin-dot effect should be repeated throughout the main design.

  3. To get rid of the brush attached to the mouse button simply click on any standard brush at the bottom of the toolbox.

 

Drawing the Medallion

  1. You can centralise the design on the screen by pressing 5 on the numeric keypad, and you can easily locate the centre of the design is as it will be marked in red whilst the grid is on as shown below.

  2. If we are not already using freehand draw - press D on the keyboard. Also, if we are not using the smallest brush size select the single point in the panel at the bottom of the toolbox.

  3. Now draw the figure shown below using two beiges and two greens. This will overwrite the light blue pin-dot in the centre. It is probably easiest to keep dark blue as the right hand (or background) colour so that mistakes can easily be erased as before.

 

  1. Now capture the figure we have drawn as a brush so select rectangular cut again (<Ctrl> + R) and draw a rectangle round the shape as we did previously for the pin-dots. Since the figure is central within the pin-dots immediately surrounding it, we can also pick up the pin-dots to help us locate the figure when mirrored and rotated. Note that the background colour is transparent due to the fact that dark blue is the right hand button colour. If you make a mistake, for example letting go of the mouse button too soon and only capturing part of the figure, simply re-select rectangular brush cut and start again.

  2. Now click on the Horizontal Brush mirror button near the bottom of the toolbox (or use keyboard shortcut H) and the brush will flip over left to right.

  3. Place the brush on the design by moving it into position then pressing the left mouse button. You should now have the shape shown in image A below.

  1. As described above, select the new image as a brush in the same way as before, flip it vertically using the vertical flip button near the bottom of the toolbox (keyboard shortcut V) and place it to complete the top and bottom sections as shown in image B above.

  2. Then capture the figure as shown at image B as a new brush, rotate it using the 90 degree rotate button on the toolbox (or use > on the keyboard) and place it to complete the shape as shown in image C above.

  3. In the final version of the medallion, each section is coloured differently therefore we now need to edit the figure using freehand draw. So, select single point brush in the toolbox and freehand draw (D on the keyboard) then over-paint the figure using various colours to complete the colour changes as shown below.

 

  1. We now need to place a copy of the medallion over the corners of our design repeat. To do this, capture the medallion and a few surrounding pin-dots as a brush using rectangular brush cut (remembering to select the dark blue as the background colour in order to make it transparent).

  2. Zoom in to the corners of the design (by pressing 0 or . on the numeric keypad).

  1. Place the medallion in the corner of the design (with end 1 row 1 as the centre of the medallion). Note that the software allows the medallion to be drawn over the edges of the repeat - in fact in this case into 3 adjacent repeats. Our image is now complete.

  1. In order to now see the full design select zoom to fit on the toolbar (or use the keyboard shortcut <Shift> + Z). You can also select View Many on the toolbar and zoom out a little using the - key on the numeric keypad. This allows us to check that the design repeats properly. You should now see an image similar to the one above.

  2. Your design should be saved to a design file using menu option File->Save As as described for the previous tutorial. You can either overwrite the original (blank background) version if that has been saved or give the design a new name such as MEDALLION1.PRJ. You can use either .IFF or .PRJ for the file extension. (The latter will save the shelf items as well as the main design).

  3. You may also like to try saving the palette on its own for use in other projects at a later date by saving it as a .PAL file. This is not essential and in fact you can load palettes from any design file. To save a palette, use menu option Palette->Save and save the palette in the appropriate folder as MEDALLION.PAL

     

Alternative Repeat Size

 

Now that your design is finished, try altering the design size and repeat to get a different effect like the one shown below.

  1. Put the grid on the screen and zoom in to the centre of the design (using the numeric keypad shortcuts 5 to centralise and + and - to zoom in/out).

  2. Using rectangular cut, pick up a brush from co-ordinate 29/29 to 56/56 (a 4" * 4" brush). Again the Information panel will be useful for this (<F11> key). The brush size should be 28 ends by 28 rows.

  3. Place this brush on the shelf by clicking in an empty shelf item.

  4. Pointing at the shelf item, click the right-hand mouse button and select Edit Box from the Pop-up menu. This will bring the brush on to the screen as the current design.

  5. Select View Many to show the new medallion design in repeat.

  6. Select menu item Design->Match->Half Drop to see the medallion in a half drop

  7. To save this shelf item as a design in its own right click on the shelf item with the right mouse button and select Export. Make sure the directory is correct and enter a file name such as MEDALLION_ALT1.IFF and click on Save.

 

Save Project

Now we need to save the project i.e. the main design and the shelf items. To do this:

  1. First switch back to the main design so right click on shelf item 1 and select "Edit Box".

This is necessary as we cannot "Save Project" while editing a shelf item other than the main design.

  1. Select menu item File->Save As.

  2. Using the standard dialog, select a folder to save the project and then use a file name such as MEDALLION2.PRJ. If you do not see PRJ as one of the available file types in the drop down list then you have probably not switched back to the main design as described in step 1 above.
