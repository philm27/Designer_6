Saving Layouts as Bitmaps

 

It is possible to save your layout as a .BMP file, for importing into other Windows software.

To save a layout as a .BMP file, follow these steps:-

 

  1. Select OUTPUT TO BITMAP from the FILE drop-down menu . This will bring the Output to Bitmap window on screen.

 

  1.  

  1. Type in the path for the folder you want to save to  including a filename with the file extension .BMP

 

If you do not know the path, click on the button at the right-hand side of the window. This will open another window to enable you to navigate to the required folder :-

 

 

Type inyour filename (remembering to include the file extension .BMP) and then click on Save to return to the initial window  with your designated path now showing in the Output Filename box.

 

  1. Click on OK to save the file.
