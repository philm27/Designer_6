Standard Templates List

 

You can select any .lyt file as a Template and add it to the list of 10 Standard Templates.

 

To do this:-

 

  1. Select menu option Templates and Layouts->Change Standard Template->option where option is the position in the list where you would like the layout to appear. This will display a File Open dialog.

  2. Find and select the .lyt file that you want to use as a Standard Template.

  3. Click on Open. The selected .lyt file will now be nominated as a Standard Template and will appear in the Standard Template list at the appropriate position.

For example, to set a previously saved layout "c:\layouts\projectA.lyt" as number 5 in the list.

  1. Select option "Change Template Layout" and select entry number 5 - it does not matter whether it is currently blank or not.

  2. This will display a "Choose Template Layout" dialog which is identical to a "File Open" dialog.

  3. Select the required layout file and click the Open button.

The chosen layout will replace whatever was there previously in the list and can be used by selecting menu option

Templates and Layouts->Standard Template->5.c:\layouts\projectA.lyt
