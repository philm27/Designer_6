Quick Title Properties

Example showing a title to be located at top centre using Pristina 36 point

 

The text for the title will be set at the time the layout is generated and will be taken from the project name. After the layout the object can be modified in the same way as any other text object.

As for other properties, the text and the font can be modified after the layout has been generated by accessing the object properties (double click the object in the layout).
