Frame Information

 

You can insert a short report detailing the colour usage by frame. To do this proceed as follows:

 

  1. Right click the design on the layout to select the design and display the pop-up menu.

  2. Select menu option Insert->Frame Information.

  3. Edit he information as required. Note that you can amend the width so that the report is given for a full loom width based on a smaller design repeat width.

  4. Click on Ok to proceed.

  5. The outline of the report will be attached to the cursor and you can place it anywhere on the layout by clicking the left mouse button.

 

As an alternative to step 1 and 2 you can select the design then select main menu option Insert->Frame Information.
