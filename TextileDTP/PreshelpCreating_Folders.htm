Creating Folders

 

Before creating new folders, make sure that you understand the concept of folders.

To create a New Folder for Textile DTP, select SAVE LAYOUT AS in the FILE drop-down menu.

 

 

Find the folder in which you want to create the new folder. To do this, work your way through the folder family tree.

Go to the parent folder by clicking on the icon, or to the offspring folder by double-clicking on a folder within the currently selected folder. When you find the required folder make sure that it is open in the Save In box.

 

When you have got the required folder open in the Save In box, click on to create a new folder within it. The new folder will appear in the list of sub-folders  replace the words New Folder with the title of your choice.

You can now open your new folder by double-clicking on it, ready to save your files into it.
