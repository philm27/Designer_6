Palette Style

 

Note: Before inserting a color palette, you must first select the design for which you want the palette inserted.

 

Having done this, click on the icon in the toolbox or alternatively select Insert->Palette on the pop-up menu . The Current Palette Properties window will appear on screen.

Use the Style tab to specify the overall look of the palette as described below.

 

 

Swatch Size

Set the swatch size in millimetres and the gap between swatches. The gap is the horizontal distance for a Horizontal palette and conversely the vertical distance between swatches for a Vertical palette.

 

Direction

Select either a Vertical list or  Horizontal List.

 

Shape

Select one of the standard shapes.

 

Other Options

Show Used Only - Select this to only show colors that are used in the design. De-select it to show the full palette.
