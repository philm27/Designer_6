Saving Layouts in Folders

 

This is a general topic outlining the process of saving files. For more specific Help, see:-

 

  • Saving/Loading Layouts

  • Saving/Loading Templates

 

To save a .lyt file, access the Save As window by selecting SAVE LAYOUT AS from the FILE drop-down menu.

This will display the Save As window on screen:-

 

 

Find the folder in which you want to save the layout.

Enter your layout file name in the File Name box ensuring that the file is saving as a Layout File type (which will put the extension .lyt after the name if it is not there already).

Click on Save. The Save As window will close, taking you back to the layout screen.

If you have already saved your layout to a folder, it is possible to update the file with more recently done work. The old file will be overwritten and you will not be able to retrieve it, so only use this function if you are sure you do not want to keep the previous version.

 

Overwrite a previously saved .lyt file in one of the following ways:-

 

  • Select SAVE from the FILE drop-down menu.

  • Click on the button on the toolbar

 

The file will keep the same name.

It is always advisable to save work frequently to avoid loss of work and effort in case of computer system failure for any reason.

 

Warnings During Save

You may encounter the following message during the save operation.

 

May occur when saving a new layout for a new design that has not yet been saved. In this instance the layout includes shelf items from a second project that has also not yet been saved.

 

Because the lyt file contains details of the file path for each image in the layout, this information will not be complete if the design has not yet been saved. This will only apply to designs inserted using "Insert Design" or Import from Project/Shelf.
