Draw all Components

This feature relates to the Tufted and FRS software.

When selected and saved as part of a Layout, the swatches for any special yarns (twist, mix, space-dyed, knop, slub or air-entangle) will be displayed on the layout print from FRS or Tufted with the all their components - the swatch will be split into sections to show how the yarn is made up.

Note if you are printing straight from Textile DTP then selecting this option will have no effect - it only operates when saved as a Layout file which is then opened in the Tufted or FRS packages.
