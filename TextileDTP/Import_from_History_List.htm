Import from History List

 

The History List is shown on the File Menu under Import and also on the Pop-up menu (right click anywhere in the layout).

This allows easy access to the last 10 designs opened or saved. You can also access recently used Arc designs designs in this way.

Example History List - note items 3 and 10 which are Arc designs. All other items are on drive C: in this case.

 

Simply select the required item in the normal way to insert it into the design.

For more information about the history list see the topic Arc General.
