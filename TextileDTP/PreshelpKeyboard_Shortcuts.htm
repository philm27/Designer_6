Keyboard Shortcuts

 

+

Zoom in

-

Zoom out

<ctrl>+C

Copy

<ctrl>+V

Paste

<ctrl>+X

Cut

<ctrl>+Y

Redo

<ctrl>+Z

Undo

<shift>+Z

Zoom to fit

<Tab>

Selects the next object after currently selected object - useful if some objects are hidden.

<Tab>+<shift>

Selects the previous object

<Del>

Deletes all currently selected objects

<f5>

Refreshes the page

<ï> <ð><ñ> <ò>

Moves the selected object(s) slowly around the page in the direction of the arrow.

<Shift>+<ï> <ð> <ñ> <ò>

As above but faster.

<Ctrl>+<1>...<ctrl>+<0>

Changes the default scale for new designs. (Used when a design inserted and <Ctrl> key held). The new value is shown on the status bar. (e.g. <Ctrl>+<1> for 1:1 scale, <Ctrl>+<2> for 1:2 scale etc, <Ctrl>+<0> for 1:10 scale).

<Alt>+<Return>

Displays the properties for the current object

<PageUp>

Scroll Up one screen full

<PageDown>

Scroll Down one screen full

<ctrl>+<PageUp>

Scroll Left one screen full

<ctrl>+<PageDown>

Scroll Right one screen full
