Inserting the Design in Cloth Mode (Jacquard & Dobby only)

 

To insert a jacquard design in cloth mode, make sure that it is displayed in Cloth Mode in Jacquard Designer. This will make the cloth mode your current design when you Insert Design in Textile-DTP.

Alternatively, if you have previously saved your jacquard design in Cloth Mode, you could then Import it into the working area.

To show cloth and artwork in one layout:-

 

  1. Save the design as artwork in Jacquard Designer.

  2. Display the design in cloth mode in Jacquard Designer.

  3. Open Textile-DTP and Insert the design in cloth mode.

  4. Import the previously saved artwork file.

 

To insert a dobby design in cloth (textured) mode, you must first export the texture mode as a bitmap file in Dobby Designer, and then import the file in Textile-DTP.

When you save a dobby design in Texture mode, because the bitmap is made up of so many colours (to simulate the texturing), it doesnt have the normal design colour palette. If you want to display a colour palette with a textured dobby design in Textile-DTP, in addition to the texture .BMP file, you must also save the design in Design mode as a .BMP or .IFF file  these files will contain the palette for your design. Follow these steps to display a textured dobby design with the design colour palette:-

 

  1. Import the textured design into the working area .

  2. Import the design in Design mode into the working area.

  3. Select the Design mode design and insert its colour palette into the working area.

  4. Arrange your layout so that the Textured design and the colour palette are placed on the layout page, whilst the Design mode design is left off the page, and therefore not included in the printed presentation. (Refer to Understanding Working Boundaries )
