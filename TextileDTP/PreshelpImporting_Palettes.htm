Importing Palettes

 

You can only use this facility if you have previously saved palettes as .PAL files, either in Textile DTP or whilst using APSOs design software.

To import a palette into a design within your layout:-

 

  1. Select the design within the layout.

  2. Select LOAD DESIGN PALETTE from the PALETTE drop-down menu  the Load Palette window will appear on screen

  3. Navigate to the required folder and select the .PAL file.

  4. Click on OK. The design will now be displayed in the new palette colours.

 

If your imported palette has more colours than your selected design, only the used colour numbers will be inserted.

If your imported palette has less colours than your selected design, the available colours will be replaced whilst any others will remain in the original colours.
