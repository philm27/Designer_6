General

To display the current print preferences select menu option Edit->Preferences then click on the "General" tab if not already displayed.

 

The General tab in Preferences. In this example the page background color is white.

 

Page Background Color

This can be used to modify the page background color both on screen and in print. To change it simply click on the color swatch and the software will display the Color Selector dialog in which any RGB value can be chosen (i.e. this is not restricted to design colors).

 

Page Print Order

This is only relevant when printing multi-page documents. Select "Left to Right" in order to print the top row of pages then the second row etc. Select "Top to Bottom" to print  the left column of pages first then the second column etc.. If printing page numbers (see below), these will also follow this sequence.
