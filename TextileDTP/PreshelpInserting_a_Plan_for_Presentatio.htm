Inserting a Frame Plan

 

If you want to include a frame plan in a layout with a design, you would normally display the frame plan at the same scale as the design  so that you can locate the plan directly beneath the design.

If you do not wish the design to appear on the printout you can insert a scaled down version of the design off the page so that it is not printed.

Proceed as follows:

  1. Right click the design in the layout (for which a frame plan is required) to display the pop-up menu and select menu item Insert->Frame Plan. This will display the dialog shown below.

  2. Enter your required details and click on OK.

As an alternative to step 1 you can first select the design, then select main menu option Insert->Frame Plan. If there is only one design in the layout, you do not need to select it first.

  1. An outline of the frame plan will be attached to the cursor  place it on the page and click with the left-hand mouse button to insert. If you have requested numbers on the frame plan, these may not be visible on screen, but they will appear on the print out.

 

Printing a frame plan for Production Purposes

If you need a frame plan for production purposes, you are likely to need it at full scale on its own  without printing the design. To print the plan without the design, simply move the design off the page as described above.

Here are some guidelines if printing very large plans:-

  • If you are using a sheet-feed printer, minimise the amount of joins that you will have to stick together by printing your plan vertically down the pages.

  • If you are working on banner paper, print your plan vertically to eradicate the need for joining sections. Select Multiple Pages for printing and make the pages about 15 long - your plan will still print out in one continuous strip but it will be spooled in sections, which reduces printing time.

  • If you are printing to a wide plotter, again select Multiple Pages as above. To eradicate the need for joining sections, print the plan vertically. To minimise paper wastage, print several vertical frame plans across the page.

  • Always use the printer drivers as recommended by APSO.
