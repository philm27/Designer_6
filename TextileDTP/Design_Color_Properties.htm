Design Color Properties

 

Use this page of the Design Properties to adjust the RGB values of palette colors prior to printing or to import shades previously saved in your Yarns Database.

It is also possible to make colors within a design transparent.

 

Example showing color 2 overridden by Shade U079 and color 3 overridden by RGB value 160,160,160

 

Used Only

Change the list from a "Used Only" list to a "List All" list by changing this check mark.

 

Reset

This button, when selected, will reset only the currently selected color to the way it was when the properties were displayed. It does not clear the override - use the Clear button for that.

 

Reset all

This button, when selected, will reset all palette colors to the way they were when the properties were displayed. It does not clear the overrides - use the Clear All button for that.

 

Clear

Use this button to clear the override, if any, for the currently selected color.

 

Clear All

Use this button to clear all overrides for all colors.

 

Overrride Using an RGB

Click on the color in the list if not already selected.

Then either

Click on the Spin controls (Up/Down arrows) to modify the RGB values then click on next palette color to be adjusted if any

Or

Click in the Red box and type a new value, press <tab> and type a new green value, press <tab> and type a new blue value, press <tab> again. The software will automatically move to the next color in the palette.

 

RGB Color Picker

As an alternative to typing the RGB values as described above, click the "..." button. THis will display a standard Color Picker dialog from which the override color can be selected.

 

Override Using a Shade Code

Click on the color in the list if not already selected, then click in the Shade box and type the required code and click on Search. Assuming that the Shade Code exists, it will then appear in the Override column and the selection will automatically move to the next color in the palette.

An example of the Shade Library window (Yarn Page shown) using the "Swatches" option for the display

 

Override by browsing the Shade Library

Click on the color in the list if not already selected, then click on the Shade button. The software will display a window similar to that in the above example.

Select a shade from a list by double clicking on it, or, hold the <ctrl> key and click once. The shade will appear in the Override column and the selection will automatically move to the next color in the palette.

Or

While displaying the list of shades, type in a shade code in the box at the bottom of the Library List and press return - it will override the current palette color and, as above,  the selection will automatically move to the next palette color making it easy to do a series of overrides.

You do not need to close the Shade Library in order to select a different color from the palette list.

You can choose a color from several different yarn types in the Shade Library such as Mixtures, Twists, Space Dyed etc as well as from the Yarn tab itself. For more details on using the Shade Library see the Help File for Jacquard Designer or Carpet Designer.

 

Transparency

To make a color transparent simply click on the palette color in the list and then place a check mark in the Transparent box. A "T" for transparent should appear against the color in the list.

You can clear a Transparency flag in the same way and you can clear all Transparency flags by clicking the Clear All button in the Transparency group.
