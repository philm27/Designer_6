Cut, Copy Paste

You can use the normal Cut, Copy and Paste procedure to copy objects from one location to another in the layout and from one layout to another.

 

To do this you can use the standard keyboard shortcuts as follows:

 

Cut

  • Select the Object to be Cut
  • Press <ctrl>+X on the keyboard - the object is now copied onto the clipboard and the object is deleted

 

Copy

  • Select the Object to be Copied
  • Press <ctrl>+C on the keyboard - the object is now copied onto the clipboard

 

Paste

  • Do a Cut or Copy as described above
  • Select the Layout in which you wish to place the object if it is not the current one
  • Press <ctrl>+V on the keyboard - the object should now be attached to the cursor so that you can move it to the required location using the mouse.
  • Click the Left mouse button to place the object an appropriate location

 

Note that there are also menu options for Cut, Copy and Paste on the Edit menu.

 

Also note that you may be able to paste complicated text and design objects from other software though the repeats and print size will need to be adjusted for design objects and the Font will need to be set for text.
