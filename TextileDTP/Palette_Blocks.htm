Palette Blocks

 

See the topic Inserting Palette for more information about how to insert a palette into a layout.

This dialog is slightly different when using Jacquard Designer - see the topic Palette Blocks - Jacquard.

 

The Blocks tab allows you to select how many swatches appear in each line (or column) of a Palette.

 

 

Example of the Blocks tab in which the user has specified row sizes of 4, 3, 2, 2 - in fact the final 2 is not required as any further rows (assuming a Horizontal palette) would have 2 swatches anyway if the user specified "4,3,2" . Also a simple space can be used as a separator, the commas are not necessary. The gap between rows in this case is specified as 12mm.

 

Palette Blocks

Type just one or a list of numbers here to specify the number of swatches in each row of a Horizontal palette. For a Vertical palette these values give the number of swatches in each column. For example you could simply enter a value of 6 here to specify that each row (or column) will have 6 swatches (except possibly the final one). The final number in the list will be used for any additional rows after the specified rows have been printed.

 

If the list is empty the software will simply print all swatches in a single row.

 

Gap Between Rows

This is the gap between rows in mm and you may need to experiment with this to get a suitable value. For example too small value may mean that text from one swatch overwrites some text or the swatch itself from another.
