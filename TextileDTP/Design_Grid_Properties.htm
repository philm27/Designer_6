Design Grid Properties

Example in which a grid has been selected. Grid Color is Black, Color Numbers are not shown, Bold lines are printed every 7 ends and 15 rows and Grid Origin is bottom left (i.e. numbering starts at bottom left)

 

Print Grid

Check this box to cause a grid to be printed

 

Grid Color

Select either black lines, white lines or "White on Dark". In the latter case, lines will be black except on dark colors.

 

Color Numbers

Color Numbers (i.e. Palette numbers) can be added to the image as shown in the small image above. Alternatively, the numbers printed can be frame numbers (Carpet Designer only) and in the latter case, the numbers can be rotated 180 degrees.

 

Grid Lines

Grid Lines will normally be a single pixel width with Bold lines (2 pixel width) to coincide with the quality/sett (as in the example above) i.e. if the quality is 8 per inch then bold lines will be drawn every inch. Alternatively there may be no bold lines, or, all lines printed will be bold lines.

 

Bold Line Frequency

If bold lines are being printed normally, then the frequency can be adjusted here. For example if the quality is 7x8.5 per inch, the user may want to change the bold line frequency to one every 17 rows whilst leaving it as every 7 ends.

 

Grid Origin

This will normally be Left and Bottom but Bottom Right is fairly common and any other permutation can be selected.
