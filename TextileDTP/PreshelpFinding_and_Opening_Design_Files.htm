Finding and Opening Design Files

 

If you are importing a design for use in a layout, you will be presented with the Open window.

 

An example of the file open dialog

 

Eitehr double-click on the file that you want to open from the list, or select it and click on the Open button.

 

File of Type

Select a file type from the drop downlist in order to restrict the list to the type of file in which you are interested. Alternatively select "All" to list all file types.

 

The Design Preview

A design preview is shown (if the software recognises the file type and the file contents) when you select a file so you can easily click on different files to show the design they contain.

You can also zoom in and out on the design by clicking on it then using the + and - keys as you would on the main design.

You can also move the design around by pressing and holding the left mouse button on the design itself and moving the mouse.

Some brief details about the design size and ends and rows/picks per inch are available if you right click the design preview.
