Design Texture Properties

 

An example in which texture has been selected and the chosen texture file name has been specified (..../cut.bmp). The specified texture file represents a 3 inch square of the design in this case.

 

Show Texture

Check this box to cause the design object to be shown textured.

 

Texture File Name

Either type the path to the texture file or select browse and navigate to the required file.

 

Texture File Size

In order to use a texture file, the software needs to know how the size of the texture file relates to the size of the design. In all cases, the texture file image will be repeated over the surface of the design and blended with it.

 

Lock to Design

Select this option and specify a number of design ends and rows to lock the texture image to the specified size in ends and rows

 

Lock to Size

Select this option and specify a size in one of the standard units in the drop down list (normally inches or centimetres) to lock the texture image to the specified size.

 

Threshold Value

This can be used to adjust the brightness of the final image. It must be in the range 0 to 255. A lower value will cause the image to be brighter and a higher value will cause it to be darker. As a general rule a value in the range 150-200 will probably work best.
