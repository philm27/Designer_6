Palette Swatch Text

 

See the topic Inserting Palette for more information about how to insert a palette into a layout.

 

The Swatch Text tab allows you to select which of the several text items is to appear for each swatch and in what order. You can also control whether to combine items into one piece of text.

 

 

An example of the Swatch Text page - in this case the user has selected Palette Number, Shade Code and Usage% to be shown above a Horizontal palette (or to left of a Vertical). These 3 items will appear on separate lines as they are not combined.

 

Display Headings

Check this to display headings for the text items - the headings will appear to the left of the first swatch on each row for a Horizontal palette and at the top of each column for a Vertical Palette.

 

Unused Items

This is a list of the various possible items that can be printed alongside a palette swatch. To use them click on one (or more) and then click on one of the ">" buttons which will move them to either the "above" or "below" list.

 

Above Swatch

This is a list of the items that are to be printed above each palette swatch (or to left for a Vertical palette).  The items are printed in the order that they are listed here.

 

Below Swatch

This is a list of the items that are to be printed below each palette swatch (or to right for a Vertical palette). The items are printed in the order that they are listed here.

 

">" Buttons

This button will move the selected items in the unused list to the appropriate list of items to be printed. Since the sequence is also important you may need to remove some items and start again to get the correct sequence.

 

"<" Buttons

This button will remove the selected items back to the unused list.

 

"+" Buttons

This will add a plus sign to the Text item as in the example below and will cause the item to be combined with the next item listed into one text string with a comma separator. This is a toggle switch so if the selected items already have a "plus" sign, they will be removed.
