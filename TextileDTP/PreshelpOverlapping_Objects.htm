Overlapping Objects

 

Every time you insert an object onto the page, an invisible layer is added to the layout  rather like laying transparencies over one another. If you move an object into the space occupied by another object, it will go behind the object if it was inserted onto the page before the stationary object, or it will overlap the object if it was inserted after the stationary object.

It is possible to swap the layers around, so that, for instance, a design that is overlapped by everything (i.e. the bottom layer  the object that was inserted first) can be made to overlap other objects (by bringing it forwards  towards the top layer).

To swap the layer position of an object:-

 

  1. Select the object that you want to adjust.

  2. Click on one of the overlapping tools on the toolbar :-

 

Brings the object in front of all other objects (it becomes the top layer of the layout).

Sends the object behind all other objects (it becomes the bottom layer of the layout).

Brings the object forward by one position (the layer moves up one place).

Sends the object back by one position (the layer moves down one place).
