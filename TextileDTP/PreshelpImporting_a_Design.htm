Importing a Design

 

If you want to include designs other than your current design in your layout, you can import them from folders or discs.

To import a design:-

 

  1. Click on the icon in the toolbox or, alternatively, select IMPORT  DESIGN from the FILE drop-down menu or from the pop-up menu (right click in the layout). This will bring the Open window on screen so that you can navigate to the required folder and select your design.

  2. Click on Open to load the design and close the window.

 

If you want to import a design that is open in another project , simply select IMPORT  PROJECT DESIGN from the FILE drop-down menu, and select the required design.

 

  1. Click and drag using the left-hand mouse button to create a rectangle within the working area. The size of the rectangle will determine the scale at which your design will be inserted.

  2. When you release the mouse button, the design will be inserted.

 

In addition to designs, you can also import many other files  room sets (if you have 3Di), scans etc.
