Design Size Properties

An example of the design size property page. The whole design is being shown starting at end 1 row 1.

 

Start End/Row

If a design repeats you can use this to change the end and row/pick at the bottom left of the printed image. This can also be changed using the Cropping feature.

 

Width in Ends/Rows

Change these figures to modify the size of the printed image in terms of Ends and Rows/Picks. Again, this may have been modified using the Cropping feature.

 

Rugs/Runners

Normally, if a design is a rug, you cannot "overcrop" i.e. show part of the design across the edges of a repeat. However, if this feature is checked, the software will treat the design as a straight repeat. Similarly with Runners (i.e. Horizontal or Vertical repeats), the software will allow over-cropping in both axis, rather than just one.
