Re-sizing Objects

 

Any object in the layout (text, logo, design, palette etc.) can be re-sized by double clicking it to display its properties (see below) which can then be modified to show the object smaller or larger and in the case of designs you can modify Quality or Setting and the number of repeats.

 

However, a quicker way is  to simply adjust the size interactively using the mouse. To do this:-

 

  1. Click on the icon in the toolbox .

  2. Click on the object or group of objects that you want to re-size. "Handles" (small black rectangles) will be shown at each corner and each side of the object or group of objects and you can click on these handles then drag in them in an appropriate direction while holding the mouse button down.

See also the topic Cropping which is a way of increasing or decreasing the number of repeats for a Design object.

 

Object Properties

 

All object types can be re-sized using the Object Properties which can be accessed by double-clicking the object in the layout.

 

For example, for Design Objects, the scale can be set numerically using the Design Properties.
