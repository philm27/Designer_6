Easily Resolved Problems

 

Screen Refreshing:-

 

If your screen colours get confused, you may need to refresh your screen.

Simply hit the F5 key whilst holding down the SHIFT key on the keyboard this will recover the picture.

 

Task Manager:-

 

If Textile-DTP locks, and you cannot use any function on screen, you will need to close the program down. You will be unable to do this in the usual way as the cursor will not respond - you need to use Task Manager.

To access Task Manager, follow these steps:-

 

  1. Hit the <delete> key whilst holding down the <ctrl> and <alt> keys - the Windows Security window will appear on screen.

  2. Click on TASK MANAGER

  3. Select the APPLICATIONS tab - all programs currently being used will be listed here

  4. Select Textile DTP, Carpet Designer or Jacquard Designer as appropriate to highlight it

  5. Click on END TASK

 

Once the program has closed, you can relaunch it and continue working. Unfortunately, any unsaved work will have been lost. It is always advisable to save work frequently to avoid loss of work and effort in case of computer system failure for any reason.
