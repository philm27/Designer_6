Saving Palettes

 

You may want to save a palette so that you can load it for use with another design. To do this:-

 

  1. Select the design whose palette you want to save.

  2. Select SAVE DESIGN PALETTE from the PALETTE drop-down menu  the Save As window will appear on screen.

  3. Name your palette file  it will be given the file extension .PAL

  4. Save to an appropriate folder.

 

It is recommended that you create a palette folder in which to keep all your .PAL files.
