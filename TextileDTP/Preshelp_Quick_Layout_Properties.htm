Quick Layout Properties

 

The Quick Layout can be modified using the menu option "Quick Layout Setup..." and a "Quick" layout is created by selecting menu option "Generate Quick Layout".

 

There is only one "Quick Layout" unlike the Template feature.

 

To view or edit the current Quick Layout Properties, select menu option Templates and Layouts->Quick Layout Setup.
