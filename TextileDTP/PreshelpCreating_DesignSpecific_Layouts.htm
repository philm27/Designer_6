Creating a Layout

 

There are basically 3 ways to create a layout as explained here:

 

Create Layout from Scratch

To create a layout from scratch it should be built up step by step until you are happy with it - it can be as simple or as complex as you wish. Here is a rough guide to building up your layout:-

 

  1. Set your layout area - paper size , number of pages .

  2. Insert the current design into the working area .

  3. Insert associated objects into the working area: colour palettes, specification details , frame plans(Carpet Designer only)

  4. Import other designs if required, and insert their associated objects.

  5. Import bitmaps of scanned room sets, textured designs etc.

  6. Insert customised text .

  7. Insert a company logo .

  8. Manipulate the objects - scale them, expand designs into repeat, crop designs, texture designs (Carpet Designer only).

  9. Arrange the objects on the page(s) using aligning , spacing and overlapping tools.

  10. Save your layout.

 

Create Layout from Template

Note that Any existing layout file can be used as a template. If you want to create a layout for use time and again with different designs, you should keep in mind that all occurrences of the "Current Design" will be replaced when the layout is loaded as a Template.
