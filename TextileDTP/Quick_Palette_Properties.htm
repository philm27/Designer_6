Quick Palette Properties

 

 

This example will print a palette showing only used colors with rectangular swatches and will only show Shade ID's from the available information. The palette will be positioned centrally in the layout

 

The font used for any text shown with the palette can be modified by clicking on the "Modify Font..." button. If this is not done a default font will be used.

 

The shape, along with all the other settings can be modified in the normal way after the layout has been generated (by double clicking on the object to display the object properties).
