Quick Watermark Properties

 

 

In this example, a watermark has been selected and the path is as quoted in the "Image:" box. The repeats (4x4) will be stretched (or shrunk) to fill the page. The lightness is 88% and the first repeat will be located centrally in the layout.

 

Any watermark created when a Quick Layout is generated will be placed at the bottom of the z-order.

For more information about the meaning of these values see the similar topic Design Watermark Properties.
