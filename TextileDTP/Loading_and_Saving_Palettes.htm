Loading and Saving Palettes

You can load a .PAL file or any other image file in order to recolor a design object.

 

To Load a Palette

  • Select the Design object to be recolored
  • Select Menu option Palette->Load Design Palette
  • Use the Open file dialog to select the appropriate file (you can select a .PAL or any other file type).

 

To Save a Palette

  • Select the Design for which you would like to save the palette
  • Select Menu option Palette->Save Design Palette
  • Use the Save file dialog to select a folder and give the file a name - it must be a .PAL file
