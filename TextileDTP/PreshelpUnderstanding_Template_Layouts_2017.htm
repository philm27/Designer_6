Understanding Template Layouts

 

Textile DTP allows you to use any existing .lyt file as a template for any other print layout.

 

Furthermore if there are one or more templates that you wish to set up as standard layouts, you can create a list of up to 10 which are instantly accessible via the Templates and Layouts menu.

 

Use any lyt file as a Template

To use a .lyt file as a Template on a one-off basis, simply select Menu option "Open as Template...". This will display an "Open File" dialog so you can access any .lyt file in the system. The software will substitute the current project design for what was the main design when the layout was created. Any Palettes and Specification text in the layout will also be amended  if necessary.

 

Standard Templates List

Select one of the items on the "Standard Templates" sub-menu - there may be up to 10 additional Standard Template layouts available.

 

Selecting either of these options will replace the current layout with the one chosen and the Current Design will be substituted for the original. Also, the palette and specification details will be changed to reflect the changed design. Text and logos will remain unchanged.

You can edit the resulting layout in the usual way.

 

You can change the Standard Templates list at any time - it is not automatically updated by the software.

 

If Textile DTP software is a stand-alone program, rather than part of Carpet Designer, Jacquard Designer etc., please read The Current Design topic.
