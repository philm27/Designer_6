Extra Print Options

These are print options that are additional to the standard ones and will be specific to the software that you are using and possibly specific to the software options that you have purchased.

 

For a Carpet Designer and Presentation Designer there are currently no Extra Print options.

 

For a Jacquard Designer and Dobby Designer there are additional options as shown below.

 

Jacquard Designer

 

Additional print options menu will be as follows:

 

Designer Source and General Text

These items are taken from the Design Specification (Summary Page)

Cloth

The Cloth Image from the Cloth Document if any

Weave

The Weave - this will always be available if weaves have been applied

Designer Source and General Text

These items are taken from the Design Specification (Summary Page)

Designer Source and General Text

These items are taken from the Design Specification (Summary Page)

 

Inserting text

When inserting text, the print object will initially be attached to the cursor and can be placed anywhere in the layout and then resized in the normal way.

Any further text items will default to the same text size as the first one when being placed.

 

Inserting Images

When inserting images, the software will use the normal "Insert Design" procedure so you can press the left mouse button on one corner of the image location and drag the mouse to the other corner to size it in the normal way.

You can hold the <ctrl> key while doing this to cause the software to snap the size to a whole number of repeats at the preferred magnification.
