Using Menus

 

The Pop-up menu

Example Pop-up menu

The simplest way to access the menu items for print layout creation is to right click on the layout. This will cause the Pop-up menu to be displayed. This contains the most commonly used menu items such as Insert, import, align, space evenly, order (depth) etc

In order to select a menu item from this menu, simply float your cursor over the desired item and select it with the left hand mouse button. If it has a sub-menu it will be opened and you can select a sub menu item.

Note that the initial right click simultaneously displays the pop-up menu and selects the object pointed at, if any, so it is a quick way of, for example, inserting a palette or specification on a page which has more than one design.

All the Pop-up menu options are duplicated on the main menu. However there are many software features such as saving and loading layouts that are not included on the Pop-up menu and so the main menu must occasionally be used.

 

Main Menu

An example of the main menu showing the selection of Tools->Space Evenly->Vertically

To display the main menu, click on a menu heading in the menu-bar with the left mouse button (this will display the required menu) and then click on the required menu item. If there is an arrow to the right of an item, this means that there is a sub-menu to select from. In this case you can move the mouse across and select the appropriate sub-menu item.

In the example above the user has clicked on "Tools" then "Space Evenly" then "Vertically"

Note that in the text of this help file menu option selection is sometimes written as for example Tools->Space Evenly->Vertically to show the selection hierarchy.
