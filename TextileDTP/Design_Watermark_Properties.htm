Design Watermark Properties

Any design image including 24 bit images can be used as a watermark.

 

An example in which the object is selected to behave as a watermark with the software automatically filling the page with multiple repeats at a scale of 1:1 with a lightness value of 80%

 

Behave as Watermark

Select this check box to cause the design object to automatically go to the back of the Z order and to fill the page either by repeating it at a specified scale or by stretching a specified number of repeats as defined in the remaining settings here.

 

Stretch Specified Repeats to Fill

Select this to cause the specified number of repeats to be stretched to fill the page. The first repeat can be located as per the final option below.

 

Multiple Repeats at Specified Scale

Select this to cause the design to be printed at the specified scale but additional repeats to be added if necessary to fill the page. Again, the first repeat can be located as per the final option below.

 

Lightness

Use the sliding scale (using the mouse or left right arrow keys) to modify the brightness of the image. A value of 0% will cause the image to be printed in its normal colors whereas a value of 100% will be pure white so about 80% should work well.

 

Scale

If using the "Multiple repeats at Specified Scale" option, the required scale can be specified here. It is not possible to edit this value if Stretching to Fill.

 

Repeats

If using the "Stretch Specified Repeats to Fill" option, the required repeats must be specified here. It will not always be possible to show the specified number of repeats in both axis as the software will calculate the required scale for both axis and will use the smaller of the two.

 

Design Alignment

This option allows the watermark to be located in the centre, at one edge or at one corner. Other repeats, if required, will work outwards form there to fill the page.
