Undo

To display the current print preferences select menu option Edit->Preferences then click on the Undo Tab.

 

An example of the Undo tab in Preferences. in this example the user may undo up to 64 steps of editing in each layout.

 

Maximum Undo History

 

Change this figure to reduce or increase the length of the edit history retained for each layout. A lower value will save memory.

 

Notice that simply moving or re-sizing an object  or changing the number of repeats in a design uses very little memory.
