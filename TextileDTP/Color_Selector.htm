Color Selector

This is a standard dialog and is used in several different situations (e.g Palette and Text objects) - it can be used to set an RGB value and to set up a list of 16 "standard" RGB values.

An example of the Color Selector Dialog. The yellow in the second row is currently selected.

Using the dialog

You can select any one of the standard colors shown in the upper part of the dialog if they are suitable. After selecting a color, click on OK in order to use that color.

Custom Colors

You can also select any one of the Custom Colors shown in the lower part of the dialog if they are suitable. As above, after selecting a color, click on OK in order to use it.

The Custom Colors are a set of up to 16 colors that can be modified and are saved by the system from one session to another.

Making a New Color

In order to create a new color that is not in the list, click on Define Custom Colors and the dialog will change to a format similar to the following. You can type in Hue, Saturation and Luminance ("Value"), or, RGB values, but also, you can click in the color chart to set the Hue and Saturation and then further modify the color Luminance using the sliding control on the right of the dialog.

Adding to Custom Colors

In order to add a new colors to Custom Colors, click on the "Add to Custom Colors" button. In order to modify an existing Custom Color you must click on the color first in order to select it so that it is marked by a color outline. If you make several new colors and click on "Add to Custom Colors" for each one, they will be added to consecutive locations in the list.
