The Current Design

 

If your Textile DTP software is accessed through another APSO software package (i.e. Jacquard Designer, Carpet Designer, Textile Designer), you will have a CURRENT DESIGN to use in your layouts. This is the main design that you are currently working on. If you have several projects open, you will have a current design in Textile DTP for each of those projects.

If your Textile DTP software is accessed directly, you will not have a current design that will automatically be used in your layouts, as you will not have a design file open. In order for you to open a file for use in your layouts, the Open window will appear when you select the following options:-

 

  • Insert Design

  • Generate Template Layout

  • Quick Layout
