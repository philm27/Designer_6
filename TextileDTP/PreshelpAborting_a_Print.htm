Aborting a Print

 

If you want to cancel a print that is either currently printing, or in a print queue, follow these steps:-

 

  1. Click with the left-hand mouse button on the START button on the task bar

  2. Click on SETTINGS to reveal a sub-menu

  3. Click on PRINTERS to open the PRINTERS window.

 

  1.  

  2. Double-click with the left-hand mouse button on the printer that you sent the print to  this will bring up a list of all items in the print queue.

  3. Click once on the print item that you want to cancel  it will become highlighted in blue.

  4. Hit the DELETE key on the keyboard  the print will now be cancelled.
