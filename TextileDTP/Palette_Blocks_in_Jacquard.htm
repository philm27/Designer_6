Warp/Weft Blocks in Jacquard

When using Jacquard Designer, this page of the Palette Properties is slightly different. It includes a check box "Block as Warp and Weft".

 

.

An example in which "Block as Warp and Weft" has been selected. In this case the normal Palette Blocks option is disabled except for the Gap between rows.

 

Block as Warp and Weft

When this is selected the software will automatically block the warp and weft colors (according to the Selectors dialog and the threading dialog in the main program) as two rows showing a row (or column) of warp colors and a row (or column) of weft colors. Each is shown in selector order which may in some cases result in a palette color being shown twice if it appears in both warp and weft.

 

All other aspects of this dialog are as described in the topic Palette Blocks.

 

Note also that when using Jacquard Designer, it is possible to add Selector Number to the swatch text - see the topic Palette Swatch Text
