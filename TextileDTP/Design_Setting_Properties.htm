Use Override Values

Check this box in order to allow the figures to be modified

 

Ends/Rows/Per

Change these figures to override the normal Quality/Sett for the design. These new figures are only used for the print layout and do not modify the design itself in any way. However, they will be saved with the layout and will be applied again if the layout is subsequently re-loaded.
