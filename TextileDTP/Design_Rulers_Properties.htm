Design Rulers Properties

.

In this example a ruler will be printed in black at all 4 sides. Each of the 4 sides will have markings every foot. An outline will be displayed around each ruler and the bottom left will be used as a start point for measurements.

 

Ruler Color

The drop down list can be used to select one of 16 standard colors

 

Ruler Required

Select either All 4 sides or select individual sides.

 

Ruler Properties

It is possible for each of the four sides to be a different style, so one side could be ends and rows, another could be feet and inches and another could be centimetres and metres.

 

To make all four sides the same click on the "All" radio button, otherwise to modify an individual side, click on the appropriate radio button.

 

(Though the sides to be actually printed is dictated by the "Ruler Required" option as stated above).

 

Style

Select one of:

 

End/Row

Row with Drop

Inch/Foot/Yd

Cm/Mtr

End/Row with Mirror

 

Row with Drop

This option will print two columns of row numbers, the second of which which will correspond to the next repeat so that in many cases it is not necessary to print two repeats. For example if the design is 18inches square, 144x144 ends and rows with a half drop, the first column of row numbers will be 1-144 and the corresponding row numbers in the second column starting from the bottom will be 73-144 then 1-72. If the design does is straight match the additional column will be the same as the first. This is useful for anyone preparing jacquard cards.

Example of the End/Row with Mirror in the Left hand ruler - the total rug length in this case would be 1221 rows so the bottom row is the same as row 1221. This example also illustrates the printing of a grid with color numbers

End/Row with Mirror

Similar to the drop option above, this enables the user to print half of a design that mirrors in either axis and to prnt a second column (or row) of numbers that correspond to the second repeat in that axis. For example if a rug mirrors vertically, it is only necessary to print the bottom half and the row numbers for the top half will be printed alongside the row numbers for the bottom half in the left and right rulers. This option can also be used for horizontal mirrors to print two rows of end numbers in the top and bottom rulers.

 

Ruler Title

If checked this will cause for example "FT" to be printed in the ruler when using Feet or "Mtrs" when using metres.

 

Ruler Outline

If checked will cause the ruler to be outlined with a black line.

 

Rotate Text 180 degrees

If checked this will cause text in that ruler to be rotated 180 degrees. It will most often be used in top and bottom rulers so that they read from the left as they will normally read from the right as in the small illustration above.

 

Ruler Start Position

Select either Bottom Left or Bottom Right in order to start measurements in the rulers from that position.
