Inserting the Design in Weave Mode (Jacquard & Dobby only)

 

To insert a jacquard design in weave mode, make sure that it is displayed in Weave Mode in Jacquard Designer. This will make the weave mode your current design when you Insert Design in Textile-DTP.

Alternatively, if you have previously saved your jacquard design in Weave Mode, you could then Import it into the working area.

To show weave and artwork in one layout:-

 

  1. Save the design as artwork in Jacquard Designer.

  2. Display the design in weave mode in Jacquard Designer.

  3. Open Textile-DTP and Insert the design in weave mode.

  4. Import the previously saved artwork file.

 

To insert a dobby design in weave mode, you must first export the weave mode as a bitmap file in Dobby Designer, and then Import the file in Textile-DTP.
