Copy and Paste Design Objects

You can copy and paste using standard keyboard shortcuts Ctrl+C and Ctrl+V. Both Designs and Palette objects can be copied (as well as Text objects)

 

Note that when copying a palette all details are copied so a copied palette will still be associated with the same design object.

 

You can also copy images from a separate software product, the resulting design or logo object will be treated as an imported bitmap image.

 

To Copy a Design Object

 

Simply select it and type Ctrl+C on the keyboard or select Copy on the Edit menu.

 

To Paste a Design Object

 

(1) Type Ctrl+V on the keyboard.

(2) Select Paste->Design on the pop-up menu

 

Pasting from an external software source

 

The software will offer a choice of "Paste as Design" or "Paste as Logo" if the object is from an external source. If pasted as a design, the software will treat the image as an imported bitmap image since it does not know it's source or file location.

 

Saving a Layout Containing a Pasted Image

If a layout is saved that contains images from an external source, the software will not simply be able to save the location of the file as is normally the case. .There are two ways to counteract this.

(1) You can save the image to a file after it has been placed in the layout. To do this, right click the object in the layout and select Save on the Pop-up menu. You can then save the image in any location as any standard file type. The software will remember this location in the .lyt file if it is subsequently saved.

(2) When saving the layout, the software will detect a pasted image and will offer the chance to embed the image (or images) in the layout file.
