Setting Printer Defaults

 

It is a good idea to set the printer that you use most often as a default  so that whenever you go to print a layout, that printer is selected to print to. If you want to print occasionally to a different printer, you can just change the selected printer for that print  after which the selected printer will revert back to the default.

Similarly, it is a good idea to set your most frequently used paper size as a default. If you want to print occasionally on a different size of paper, you can just change the paper size for that print  after which the selected paper size will revert back to the default.

To set these defaults, follow these steps:-

 

  1. Click with the left-hand mouse button on the START button on the task bar

  2. Click on SETTINGS to reveal a sub-menu

  3. Click on PRINTERS to open the PRINTERS window.

 

  1.  

    1. Click on the printer that you have chosen as your default  it will become highlighted in blue.

    2. Select SET AS DEFAULT from the FILE drop-down menu at the top of the window. Once selected, a will show that the printer is now set as the default.

    3. DOCUMENT DEFAULTS from the FILE drop-down menu  this will open a Document Defaults Window for your selected printer.

    4. Select your chosen paper size from the drop-down menu.

    5. Click on OK. Your paper size default will now be set.

    6. Select CLOSE from the Printers window FILE drop-down menu.

 

If you want to alter other printer defaults (e.g. Colour Resolution, Paper Source) you alter them in the Document Defaults window  refer to the printer manual for more information.
