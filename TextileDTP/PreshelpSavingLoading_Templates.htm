Saving/Loading Templates

 

If you create a layout that you want to use several times for different designs, you can simply save it. You do not need to save it in a special way.

To save a template layout, select Save Layout from the file menu. Navigate through the folders until you find the one you want to save your template in.

 

At a later time, if you want to place your current design into a previously saved template layout, simply select Open As Template from the Templates and Layouts menu..

 

If your Textile DTP software is a stand-alone package, please read The Current Design topic.
