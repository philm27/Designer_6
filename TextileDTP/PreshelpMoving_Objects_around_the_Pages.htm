Moving Objects around the Page(s)

 

Any object can be moved  text, palettes, specification details, designs etc. To move objects around the page:-

 

  1. Select the object or group to be moved.

  2. Whilst still using the (select, move and resize) tool, float the cursor within the confines of the objects/groups handles  so that the cursor becomes a .

  3. Click and drag the object/group around the page.

  4. When it is in the corect position, release the mouse button.

 

If you have overlapping objects, you may find that you want to change the layering sequence when you move things around  to bring objects to the foreground or send them behind other objects.
