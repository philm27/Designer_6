Scrolling the List

  • Hold the shift key and press and hold the left mouse button anywhere in the list and drag it left or right or up or down as appropriate, or,

  • Use the scroll bar in the normal way, or,

  • Use the mouse wheel whilst hovering the cursor over the list

Selecting from the List

You can replace a color in the current design and/or palette by...

  • Double-Click on a color, or

  • Click on a color while holding the <shift>+<ctrl> keys on the keyboard

  • Click on a color so that its name goes into the Shade Text Box then press <return> on keyboard

  • Drag and Drop a color from the list to a location in the palette or design in the print layout.

Using the first 3 methods, the color will replace the current selected color.

 

Drag and Drop

You can Drag and Drop shades from any list including Collections and Range tabs into a palette or onto a design as an alternative to above selection method (or by typing the shade number as described below).

 

Shade

This edit box can be used to key in a shade or yarn code which will be selected when you press the <return> key on the keyboard. The shade does not have to be visible but should be from the current tab. Also, as noted above, if you click on a shade in the list, it's code will be placed into this box automatically.

 

Add, Replace, Replace + Next

These are not selectable when designing Print Layouts.

Options

The Options menu - in this example the list is Horizontal, Plain shades are in File Order (i.e. shade code order) and colors are shown as Swatches

Horizontal/Vertical List - choose a horizontal or a vertical list. This is ignored on the Range tab.

File/Card Order - this only applies to the plain page. The position on the card is as specified in the Colors and Yarns database. Any colors that do not have a card will be listed in shade code order on a card titled "<none>".

Normal View - Shows a list using the traditional circular shape

Cut - is intended to resemble yarn ends cut

Loop - is intended to resemble yarn looped

Swatch - is intended to resemble the yarn used in a woven swatch
