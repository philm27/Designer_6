Make Same Width or Height

You can make several objects the same size by using the Make Same Width or Make Same Height Buttons. This feature can be used on a mixture of object types - such as palettes, text and existing groups.

To do this

  1. Decide which object you wish to align to. This object will provide the size information for the other objects.

  2. Click on the icon in the toolbox.

  3. Hold down the SHIFT key on the keyboard and, keeping the key held down, click once on each object that you wish to re-size  make sure that you click on the object you wish to align to last.

  4. Release the SHIFT key. All of the objects that you have selected will have for handles, (drag points) except for the object that is being aligned to  which will have for handles.

  1. Select either the "Make Same Width" or the "Make Same Height" button.

    • Makes the objects the same width.

    •   Makes all the objects the same height.

Whilst the objects remain selected, you can align them in different ways as often as you wish. You can also space them evenly whilst they remain selected as a group as in the example above.

If you want to move the objects, they will move as a group whilst they remain selected.

De-select the objects by clicking once outside the group  the handles will disappear.
