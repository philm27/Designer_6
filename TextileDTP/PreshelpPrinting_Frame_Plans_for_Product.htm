Printing Frame Plans for Production Purposes

 

If you need a frame plan for production purposes, you are likely to need it at full scale on its own  without printing the design. To print the plan without the design, please refer to the Understanding Working Boundaries topic.

Before creating a layout for your frame plan, you must first select your printer and paper size . Here are some guidelines if printing very large plans:-

 

  • If you are using a sheet-feed printer, minimise the amount of joins that you will have to stick together by printing your plan vertically down the pages.

  • If you are working on banner paper, print your plan vertically to eradicate the need for joining sections. Select Multiple Pages for printing and make the pages about 15 long - your plan will still print out in one continuous strip but it will be spooled in sections, which reduces printing time.

  • If you are printing to a wide plotter, again select Multiple Pages as above. To eradicate the need for joining sections, print the plan vertically. To minimise paper wastage, print several vertical frame plans across the page.

  • Always use the printer drivers as recommended by APSO.

 

To create a frame plan layout for production purposes, follow this guide:-

 

  1. Set your Page Setup to a single page.

  2. Insert the design to the side of the page at small scale.

  3. Right Click the design to display the pop-up menu and select menu item Insert->Frame Plan.

  4. Enter your required details into the Frame Plan Information dialog.

As an alternative to step 3, select the design and then select main menu item Insert->Frame Plan.

 

  1. Example "frame plan insert" dialog

     

    1. If your plan is large, a warning will appear on screen to say that the plan will not fit on to the page. Click on YES to add the required pages.

    2. An outline of the frame plan will be atached to the cursor  place it on the page and click with the left-hand mouse button to insert. If you have requested numbers on the frame plan, these may not be visible on screen if the plan is being viewed at small scale, but they will print out.

    3. The required number of pages will now be displayed on screen. Move your frame plan into the required position within the layout.

    4. If you wish to keepthe layout, Save it using using Save As.

 

If you want to amend any information in the Frame Plan window, double click on the plan to recall the window. Having made your alterations, click on OK to return to the layout. The frame plan will be automaticaly updated.
