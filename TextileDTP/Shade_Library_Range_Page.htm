Range

Select any range from the drop down list. These range codes are taken from the Colors and Yarns data base.

 

The Colorways List

The software will list all colorways in the above format in which the Colorway is shown as a heading at the top a column of swatches forming the colorway. If there are a large number of colors in one or more colorways, the software will display a vertical scroll bar and if there are a large number of colorways (7 or more) there will be a horizontal scroll bar as in the example above.

The Range list is used in exactly the same way as a normal list as described for the Color Tab. Thus the list can be scrolled by holding <Shift> and dragging it left right or up and down. Also you can select colors by

 

  1. Double-Click

  2. Ctrl+Shift and a click

  3. Click and press <return>

  4. Type the code and press <return>

  5. Drag and Drop to a palette or the design in the layout

You can also select a whole colorway at a time as described below.

 

Selecting a Complete Colorway

This is similar to selecting an individual shade...

  1. Double Click on the Colorway Title (i.e. the Colorway Number), or,
  1. Hold <Ctrl>+<Shift> and click on the Colorway Title, or,
  1. Drag and Drop from the Colorway Title to the Palette.

Note that in all cases the color replacement is based on the palette sequence and on consecutive palette numbers except that a zero color if any is ignored. This means for example that if all the used colors are at the end of a large palette you may not see any change in the design itself.

Methods 1+2 here will do a strict color replacement in the palette starting at color 1 (zero color if any is ignored).

Method 3, since it is controlled by the mouse gives an extra level of control. You can drop on color 1 in the palette to do the same as options 1+2 but you can also replace colors further down the palette.
