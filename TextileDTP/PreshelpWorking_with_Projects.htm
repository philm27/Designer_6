Working with Projects

 

Projects work differently depending on the way in which you access your Textile-DTP software.

 

For every project that you are using for designing, you can create a layout in Textile-DTP. When you open a project in Textile-DTP, your current design to be inserted into your layout will be the design that is on screen in that project (in the Design package). You cannot create new projects in Textile-DTP.

If you want to print several designs using the same template layout:-

 

  1. Open each design in a different project in your Design package.

  2. Nominate your template as the Quick Layout.

  3. Select the projects one by one, each time generating a Quick Layout.
