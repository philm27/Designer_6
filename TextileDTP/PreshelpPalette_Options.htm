Palette Text

 

See the topic Inserting Palette for more information about how to insert a palette into a layout.

 

The Text tab allows you to select a different font, font size, style, alignment, color, transparency and background color as detailed below.

 

 

Font

This box simply displays the name of the current font, to change it click on the Change button.

 

Height

This is the point size to be used.

 

Justify

For a Horizontal palette you can select Left, Right or Center justification for alignment with the edges of the swatch. For Vertical palettes this is ignored (text of the left is right aligned, text on the right is left aligned).

 

Text Col.

Background

To change the background color, double-click in the box (grey in the above example) and use the standard color selector to choose a new color. (not restricted to the current palette colors). This is only used if the Transparency option is not selected.

 

Change

Click this button to display the Text Properties shown below.

 

 

Text Properties

This will default to a text string ABCabc123 you can modify this if you wish but the characters used will not affect the actual text being shown.

 

True Type Font

Select a font style from the drop down list. All fonts installed on your PC will be listed so the content may vary from one PC to another though certain ones are standard.

 

Size

Select the point size - this will normally vary between about 7 and 12 for a palette.

 

Justification

As noted above, his is ignored for a Vertical palette but affects the alignment with the left and right edges of a swatch for a Horizontal palette.

 

Color

Select a standard color from the drop down list, or, select "more..." at the bottom of the list to access the full range of RGB values in the standard color selector.

 

Transparent Background

Check this for a transparent background. If it is unchecked, the specified background color will be used to paint a large background rectangle behind the entire palette.

 

Background Color

As described above this will form the background rectangle for the entire palette if Transparency is not checked. To change it select a color from the drop down list or select "more..." and then any RGB value from the standard selector.

 

Text

Type any text into this box using the keyboard - this will be shown in the large box at he bottom of the dialog in the chosen font style and size  but will not affect the printed palette  in any way.

 

OK

Click OK to accept the changes

 

Reset

Click Reset to revert all settings  to the way they were when the dialog was first displayed.

 

Cancel

Click Cancel to close the dialog and ignore all changes.
