Cloning Layout

This feature provides a quick and easy way to copy a layout within the current project so that it can possibly be modified.

 

To create a "Cloned" layout simply...

 

  • Create or load the layout that you wish to copy
  • Select the menu option "Templates and Layouts->Clone this Layout".

 

The software will create a new document with a new document name and will create copies of all the objects in the current document. After the clone has been created you can edit the new document in the normal way.
