Example shows no changes to the current print scale or number of repeats

 

Scale

You can very quickly select any of the common scales 1:1, 1:2 etc by clicking on the radio buttons along the top of the page.

If the required scale is not shown as a radio button, you can use a longer list by clicking on "Other" and selecting from the drop down list.

If the required scale is not in the drop down list, you can select "Any" and type in the required figure.

The scale may have been set automatically by stretching or shrinking the design image using the mouse
