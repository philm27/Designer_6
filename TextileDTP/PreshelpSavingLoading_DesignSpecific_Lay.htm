Saving/Loading Layouts

 

There are two ways to save layout files (.lyt) Â– either:-

 

  • With the same name (with the exception of the file extension which will change to .lyt) and in the same folder as the current design . To do this, select Save on the File menu or, alternatively, simply click on the icon on the toolbar .

  • With a name and folder of your choice. To do this, select SAVE LAYOUT AS from the FILE drop-down menu. This will take you into the Save As window , where you can name the file and place it in the folder of your choice.

 

If your Textile-DTP software is accessed directly rather than through one of the Designer software packages, you will not have a current design and so must always save layouts using the Save Layout As option.

 

When you save a layout to a folder, the name and location of the design(s), logos and bitmap files used within the layout are saved within the .lyt file. If you move the design(s), logos or bitmap files to another folder, rename or delete them altogether, the computer will not be able to recreate the layout and you will lose your layout. The .lyt file is meaningless to the computer without the associated design files and cannot be opened.
