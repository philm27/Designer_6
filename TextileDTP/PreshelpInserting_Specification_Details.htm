Specification Details

 

Use this option to insert a series of text items giving various design details.

Example Specification Design tab. The text on the right hand side shows how the item will appear in print. These will be ghosted out if the item is not selected.

 

Example Specification Print details tab. Again the text on the right hand side shows how the item will appear in print.

To Insert Specification Details

  1. The simplest method is to right click the required design. This will simultaneously select it and show the pop-up menu. Select Insert Spec....

  1. Select the details that you want included and un-select those that you do not. The Design tab includes all the information that relates to the design, the Print tab refers to how it is to be printed.

  2. Also, several items have format options such as the date format on the print tab so select the format that you require. Your choices will be remembered and used as defaults next time you insert a specification.

  3. Click on OK.

  1. An outline of the rectangle containing the specification text will now be attached to the cursor.

  2. Position the rectangle as required in your print layout, and then click with the left-hand mouse button to insert the details.

  3. The specifications can now be re-scaled and repositioned as normal.

 

Alternatively for step 1 and 2:

  1. First select the design for which you require specification details to be printed. If there is only one design on the page this does not matter.

  2. Then select the icon from the toolbox or, alternatively, select menu option Insert->Spec. The Specification Properties will be shown. (See above).

 

Printed Headings

Each printed item has a "heading" or "title" which will be printed to the left of the item in question. These can be edited. For example the first item on the first tab is "Design ID:". You could change this to "Design Name" if you wished or you could even change it to a different language if the print is to be sent to a customer abroad. The colon ":" after the heading is entirely optional and could be removed or changed to dots for example "Design Name..."

The Titles are remembered by the software when a specification is inserted and are used as defaults next time. Simply recalling an existing print layout and reprinting it does not affect these defaults.

The headings can even be removed completely if required.

De-select All/Select All

You can use this button to remove the check mark from all items. Immediately after doing this however, the button title will change to Select All and alternate clicks will change the button title and its effect. You can always select all or de-select all options by one or two clicks of this button.

Amendments

Once specification text has been created, you cannot change it other than modifying the font by accessing the object properties. (The quickest way to do this is to select the object then double click it). If you wish to change the details shown e.g. add the metric size as well as feet and inches, you must delete the old specifications and insert a new one showing the required information. (Though see the note below titled "Removing Items from Specification").

However, if you alter the design to which the specification relates, (e.g. change the scale or number of repeats) the specification details in the print layout will update automatically.

Fonts

You can change the font that the system uses by selecting the font button. However, this can also be changed afterwards by simply stretching the text box containing the specification in order to change the font size, or, by double clicking the text to show the font selection dialog to change the font type or size. See the topic Text for more information.

The Group Box

The specification is actually placed into the print layout as a group box containing several text items. Each title is a separate text item as well as the specification detail itself. Therefore if you wish to simply remove one or more specific items you could do this as described below, rather than recreating the specification through the "Insert Specification" option.

Removing Items from Specification

In many cases it will be quite easy to recreate a specification by deleting the old one in its entirety and re-selecting Insert Specification for the appropriate design and inserting amended specification text. However, there may be occasions where it is useful to simply remove the items using the following technique.

  1. Select the specification text

  2. Ungroup it by selecting menu item Tools->Ungroup or using the "Ungroup" button on the toolbar.

  3. Click on the item to be removed. If necessary hold <shift> and select additional items to be deleted.

  4. Press the <Delete> key.

  5. You may now need to reposition other items in the specification to allow for the ones you have removed.

  6. It is usually a good idea to regroup the remaining items so select them either individually while holding the <shift> key, or, draw a rectangle round them. Then select menu item Tools->Group or select the "Group" button on the toolbar.
