Text

 

It is easy to insert text into a presentation, with a wide variety of font-types (type-faces) to choose from.

The simplest way is to right click the layout to display the pop-up menu then select menu option Insert text. The following dialog will be displayed.

Alternatively, click on the icon in the toolbox, or, select menu option Insert->Text.

 

 

To insert text into the layout proceed as follows.

  1. Place your cursor in the "Text" box and type your required text.

  2. Open the drop-down list for True Type Font by clicking on the down-arrow. You will find a whole range of font types. These are standard system fonts and you can add new ones in the usual way.

  3. Experiment with different font types by selecting them from the drop-down menu, and seeing their effect on your text in the "Preview" box. In this example, Trebuchet MS has been selected.

  4. Experiment with different font sizes by selecting them from the Size drop-down menu, and seeing their effect on your text in the Preview window. The font size in the example is 20.

  5. When you are happy with your font-type and size, choose a justification (left, centre or right). If you have more than one row of text it will align to your chosen justification. In the example, the text is aligned to the centre.

  6. Your design colour palette will be found in the Palette drop-down menu  select your text colour from this palette.

  1. When you are happy with your text, click OK to exit the Text Properties window. A rectangle will be attached to the cursor allowing you to place you text on the print layout.

  2. Click the left mouse button to insert the text.

  3. You can now move and resize the text if required.

  4. If you want to amend your text specifications, double-click on the text within the layout to return to the Text Properties window. Any adjustments you make will be automatically amended in the layout.

 

Copying Text

If you need to use the same text several times within a presentation, with minor alterations (for example, your text may include the words DESIGN 100/BK454 and you need text for designs 200/BK454, 300/BK454 etc), it may be useful to copy and paste the text. Having typed your initial text in the Text box, highlight it and then use the keyboard shortcut keys <Ctrl> + C to make a copy of the text on the clipboard. When you go back into the Text Properties window, simply position the cursor in the Text box and then hit the keyboard shortcut keys <Ctrl> + V to paste the copied text back in the box. You can then make your wording alteration.

Using the same method as above, it is also possible to paste text from other Windows applications such as Word.

Background Colour

You can now put a background colour behind the text or specify the background as being transparent.

On the line that says Transparent or Coloured Background either tick the box to select a transparent background or leave the box unmarked and select a colour from the pull-down menu on the right.

The font-type and size of any text within the layout (Specification Details, Customised Text, Legend) can be altered by simply double-clicking on the text  this will display the text properties so that amendments can be made.
