Finding & Loading Layout Files (.lyt) within Folders

 

This is a general topic outlining the process of loading files. For more specific Help, see:-

 

  • Saving/Loading Design-Specific Layouts

  • Saving/Loading Templates

 

Before loading a layout, make sure that the layout currently on screen is saved (if you want to keep it).

Access the folders, in order to look for your layout, in one of the following ways:-

 

  • Select LOAD LAYOUT from the FILE drop-down menu (for design-specific layouts)

  • Select LOAD TEMPLATE LAYOUT from the TEMPLATES AND LAYOUTS drop-down menu (for customised templates)

 

The software will display an Open Layout dialog like this...

 

Example of the Open Layout dialog

 

Either double-click on the file that you want to open from the list or select it and click on Open.
