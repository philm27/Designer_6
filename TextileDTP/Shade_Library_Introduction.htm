Shade Library Introduction

Access the Shade Library by selecting Shade Library on the Pop-up menu or on the main Edit menu or by clicking on the Shade Library button in the Print Tool Bar

 

.

 

The Shade Library allows users to access shades, Color Ranges and Collections in the Colors and Yarns database and/or to access Plain shades in a text file held anywhere in the system or network.

Note that when this dialog is being displayed,  the software will behave in a slightly different way and some features may not be usable.

 

When displaying Shade Library

 

You can recolor a design and /or palette

 

Example shows user pointing at a location in a design. A color sample is shown together with its shade code. After clicking in that location the color is selected as the "current color" as indicated at the bottom right of the design image.

Whilst hovering over a design or a palette, a tool tip will show a color sample and the shade code as illustrated above.

 

You can select - i.e. click on - a color in a design or in a palette and this will be indicated in a small rectangle of color at the bottom right of the image also as illustrated above.

 

You can select a shade from anywhere in the Shade Library. The shade will replace the current selected shade in the selected design or palette or you can replace several colors at once with a Colorway (or Color Set) from the Range tab. For more information see the topic Range Tab.

 

You can drag shades onto a design (except for 24 bit images) or onto a palette.

 

If you recolor a palette it will automatically recolor the associated design and vice versa.

 

To access any of the Tabs Plain, Mix, Twist, Ranges etc simply check those boxes in the list and the Tabs should appear ready to be viewed.
