Inserting a Color Palette

 

The simplest way to do this is to right click on the required design. This will simultaneously select the design and show the pop up menu. Then select Insert->Palette.

Alternatively, select the appropriate design (for which a palette is required), then click on the icon in the toolbox or select main menu option Insert->Palette.

Note that if there is only one design in the layout you need not select it first.

The palette properties will be displayed - you can either click on OK to proceed using the current defaults, or you can make changes such as Horizontal or Vertical palette. For more details about this see the Property page topics:

Style

Text

After clicking OK the palette will be shown but attached to the cursor so you can place it wherever you wish. Position the palette as required in your presentation, and then click with the left-hand mouse button to insert the palette.

Once inserted, you can move and scale the palette within the working area.

Making Amendments to your Palette:

If you wish to amend any of the information (e.g. change text size, add or remove items of palette information), double-click on the palette in your layout to access the properties window. When you have made your changes, click on OK. The palette will be automatically updated.
