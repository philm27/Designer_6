The Default Scale

 

The default scale is set in the Current Design Properties window. Alternatively, it can be set whilst you are working on the layout by holding down the CTRL key on the keyboard and typing in the required default scale (although using this method only whole numbers can be used).

The current default scale can be seen in the status bar .

To insert the current design at the default scale:-

 

  1. Select the icon from the toolbox .

  2. Hold down the CTRL key on the keyboard whilst clicking and dragging using the left-hand mouse button to create a rectangle within the working area. The size of the rectangle will determine the amount of your design to be inserted at the default scale. (If the rectangle is large and the design is small, it will be inserted in multiple repeats. If the design is larger than the rectangle, a partial repeat will be inserted)

  3. When you release the mouse button, the design will be inserted.

 

Import a design at the default scale in the same way.

To expand a design into full repeats at the default scale, hold down the CTRL key whilst following the procedure for rescaling a design. Instead of increasing or decreasing the scale of the design, the design will be rescaled to the default scale and expanded into repeat (or reduced into lesser repeats) all in one action.
