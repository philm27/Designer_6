Editing the Design in Merge View

 

Editing the Weave - Single Level

You can edit the weave in the normal way and on picks where only one color is shown, such as picks 1 to 3 in the example above, the warp at that point is either lifted or dropped.

Editing the Weave - Multi-Level

In this case all weft colors at the edited point are either lifted or dropped. For example:

(a) If the user clicks on the first warp thread in row 4 above, (initially blue weft is above the warp and grey is below), with the left hand button, then the warp will be lifted over both colors at that point.

(b) Conversely, if the user clicks on the first warp thread in row 4 above, with the right hand button, then both weft colors at that point will be lifted over the warp

(b) If the user clicks on the second warp end in row 4 above, (where both weft colors are "up"), with the left hand button, this will lift the warp over both weft colors.

 

Editing Colors

To edit colors, first select "Allow Edits Colors" in the Information Panel at the top right. Then press D on the keyboard or select Edit Mode on the pop-up menu. You can display the palette window using function key F5 on the keyboard or clicking the Palette button or selecting Palette Window on the main menu.

You can then change a color using the left or right hand button in the usual way. You can change a color even if it is not being displayed i.e. if the color has a cross through it.

Editing Cramming Marks

Similarly, to edit cramming marks, first select "Allow Edits Controls" in the Information Panel at the top right. Then press C on the keyboard or select Controls on the pop-up menu.

You can then place a mark or remove a mark using the left or right mouse button respectively, in the usual way.
