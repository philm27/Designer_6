Composite Weave Generator

 

You can use the "Apply Simple Weaves" dialog as a generator for multi-level cloth weaves. The procedure for this is described here.

For more information about that dialog click here and for information about editing weaves see the topic Weave Edit.

Note that you do not normally need to generate composite weaves unless you are using the Apply Composite Weaves dialog. They are not required if you use the "Apply Simple Weaves" dialog.

Procedure

You can use your existing project if you wish but it may be more convenient to use a new design as described here.

  1. Select a new project from the project menu.

  2. Select "New" from the file menu. This will create a new empty design. You do not need to draw anything - just leave it plain.

  3. Select "Apply Weaves Simple" from the Jacquard menu (Jacquard->Apply Weaves->Simple). You should see a single line schema.

  4. Select the "Reset Schema" button and you will see the following dialog.

  6. Enter the number of levels e.g. 2 Warp and 2 weft for double cloth and select OK.

Above example shows the dialog after weaves have been loaded and a stitching point added also pairings changed using "Properties" on the Pop-up menu.

  1. Simply use the editing facilities to set the weaves (either edit interactively using left mouse button or Open existing weaves).

  2. Set the stitching points if required (click in the weave using right hand mouse button and select "Insert Stitching Point" from the Pop-up menu).

  3. Set the warp/weft pairings in each level.

  4. Move levels up or down as appropriate.

  5. Display the Pop-up menu by clicking the right hand mouse button in the schema.

  6. Select "Display Composite". You should see the Composite Weave preview dialog something like this:

The example above shows a double cloth weave having 1+4 sateen in the face using A/B pairing and a plain weave in the back using B/A (warp/weft). Also a stitching point (V) was specified resulting in a lifter (/) being changed to a drop.

  1. Assuming the result appears to be correct, click on the weave with the right hand mouse button to display the Pop-up menu.

  2. Select "Save as..." from the menu and use the standard dialog to choose a name for the file and save it. It will default to file type IFF.

  3. Close the Composite Display Dialog using the "x" button at the top right corner.

  4. If you have another Composite Weave to generate for the same type of Schema - simply start again at step 10, otherwise click on Exit to return to the main design window.

 

The weave that you have saved above can now be used as a weave when using "Apply Weaves Composite" and should appear as shown above with stitching points marked as shown.
