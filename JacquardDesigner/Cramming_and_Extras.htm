Cramming and Extras

 

The procedures for production of these two types of design are very similar. This topic describes the procedure for Weft Cramming. For Warp Extras simply proceed as below but for warp read weft and vice versa.

The procedure, in outline, is as follows:

  1. Complete the artwork using the FACE ends and picks per inch (or cm etc.) so that the artwork has the correct number of total face ends and picks.

  2. Assign Colors to Weft Selectors.

  3. Use Apply Weaves Simple to create the Schema and select and apply weaves. This will also set the weft threading sequence automatically and will mark crammed picks with a '+' sign in the threading.

  4. Make any required manual changes to warp or weft threading sequences. (e.g. set warp threading sequence to solid black).

The design can now be saved, viewed in Cloth mode etc like a normal design.

The following sections of this topic give more detail of the procedure.

 

Artwork

First of all the artwork should be produced and finalised. You should aim to use a different colour to show where each weft colour is used in the face of the cloth.

In, more complex cases you may wish to use more than one weave for a particular selector and you should use additional colours to indicate this.

In other words when the artwork is complete, a given palette colour indicates an area or areas where the weaves applied to each selector are the same.

Loading Face Weaves and Back Weaves

The following assumes that the weaves have already been designed and saved to disk. If they have not simply create new weave, edit it and save it as described in the topic Weave Edit. You can do this within the "Apply Weaves Simple" dialog.

In many cases only two weaves are used in this type of cloth - a face weave and a back weave. Though this is not true in this example, it may be a good way to start since it is very easy to do. We can then change the pink and dark red artwork colours. To do this proceed as follows:

  1. Click on the Schema with the right mouse button to display the pop-up menu.

  2. Select "Select Face Weaves". The software will select all lines where the artwork colour matches the selector colour (including selector 2 for palette colour 6 since it knows they are actually the same from the Selector dialog).

  3. Click on the Schema again with the right mouse button to display the pop-up menu.

  4. This time select "Open..." to display an open file dialog and locate the weave (sati5 in this case) and either double click it or click it once then select the Open button.

  5. The weave will be recorded against all selected items (in this case where yarns are used in the face).

The above shows the last few lines of the schema after face weaves have been selected.

You can now proceed to load the back weaves:

  1. Click on the Schema with the right mouse button to display the pop-up menu.

  2. Select "Select Back Weaves". The software will select all lines where the artwork colour does not match the selector colour.

  3. Click on the Schema again with the right mouse button to display the pop-up menu.

  4. This time select "Open..." to display an open file dialog and locate the weave (satiback10 in this case) and either double click it or click it once then select the Open button.

  5. The weave will be recorded against all selected items (in this case where yarns are NOT used in the face).

Multi-Line Selection

We now need to load satiback32 for selectors 2, 3 and 4 for the pink artwork colour. This will illustrate another technique - the ability to multi-select lines in the schema.

There are two ways to do this in this instance since the items are consecutive, either:

  1. Select First item (selector 2 in this case) using the left mouse button.

  2. Hold <Ctrl> key down and select the other two items in turn using the left mouse button.

  3. If you select an item by mistake and want to un-select it, simply click on it a second time whilst keeping the <Ctrl> key down.

  4. Release the <Ctrl> key when all items have been selected.

If after that you decide you want to add another item you can press <Ctrl> again and hold it down while you select further items.

OR, you can use the <Shift> key to select a consecutive set of lines as follows:

  1. Select First item (selector 2 in this case).

  2. Hold <Shift> key down and click on the LAST item in the range to be selected (selector 4 in this case). All items between the first selection and the last will be selected. (i.e. Selectors 2, 3 and 4). You can now release the <Shift> key.

  3. If you want to de-select some items or add other items you can now hold the <Ctrl> key and click on other lines to select or de-select them whilst keeping the <Ctrl> key down. Release the <Ctrl> key when the correct lines are selected.

When the correct set of lines is highlighted you can open the weave using the Pop-up menu as described previously, i.e. click on the schema with the right mouse button then select the "Open..." option to select the appropriate weave (satiback32 in this case).

Using the Weave Invert and Shift options

The Apply Weaves dialog has features which allow us to edit the weave. These edits are retained within the IFF or PRJ file when the design is saved. They do not cause the individual weave to be changed in its disk file (unless it is resaved of course). For more information about this see the topic Weave Edit.

The dialog not only allows us to edit individual points in a weave but we can make changes which affect the whole weave such as the shift ends, shift picks, invert and horizontal flip options.

Original version of sati4

We have currently loaded the sati4 weave against all 3 selectors for the dark red artwork colour but in fact only selector 2 (red) will use it as it is (weft faced). The other two selectors should use a backbinding version of it so we need to change them

We can achieve this very easily as follows:

Select "Selector 1" (pink) this should be the only line selected (edits do not work on all lines if multi-select is used - only on the one having the "caret" or dotted line outline as there is only one of these).

Click on Invert so it is checked.

Change the "Shift Ends" figure to 1.

You should now see the following:

Above example shows the weave for selector 1 for the dark red palette colour shifted 1 position to the right and inverted. This means that the stitching points are 1 end to the right of the warp floats in the face weave on selector 2.

If we repeat this procedure for selector 4 (last line above) then all changes are complete.

Note that we do not need to reverse and shift weaves in every case - it was simply used here to illustrate a technique. Backbinding weaves can be stored on disk and used in designs without editing as in the case of the other palette colours Black, Green, Pink, Red.

Edge Protection

Example shows edge Protection switched on and a "float" of 1 minimum in face and back at the edge of a motif.

In most cases you will probably want to use edge protection in order to prevent the outlines of motifs looking "ragged". You can compare the results of applying weaves with this feature on and off in the topic Edge Protection.

To use edge protection click on the Options button and make sure that the feature is selected and that both figures are set to 1.
