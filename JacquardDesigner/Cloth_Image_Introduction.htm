Cloth Image Introduction

The Cloth Image will appear in its own document which is part of the Design Project in a similar way to the Weave Editor and Print Layout Documents.

It can be displayed (after you have Applied Weaves and there are valid warp and weft threading plans) by selecting "Cloth Mode" on the Jacquard menu or by selecting the "Cloth Mode" button on the Jacquard tool bar.

While displaying the cloth image you can modify the threading plan in the warp and weft bars but, as for the Weave Editor, there are safety procedures built in so that it should not be possible to make changes by mistake.
