Open Weave

 

When a line in the schema is double clicked or when Open button or menu item is selected, the following "Open Weave" dialog will be displayed.

The weave chosen will replace all the selected lines assuming the "Open" button is selected or the file name is double-clicked.

An example of the "Open Weave" dialog. Note that the window can be resized enabling you to see more files listed and a larger preview.

 

Some old weave files may not have a .iff file extension. For this reason the file type defaults to all files though most should have the .iff file extension.

Default Directory

When a weave is double clicked or you select "Open" on the Pop-up menu in Apply Weaves, the software normally defaults to the last directory used.

However, if there is already a weave present it will default to the same directory as the one from which that weave was loaded.
