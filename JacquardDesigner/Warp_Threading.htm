Threading Sequence

 

Note that the Warp Threading can work left-to-right or right-to-left (when viewed from the front of the loom) according to the Jacquard Preference. For more information see the topic Jacquard Preferences.

You can access either the Warp Threading or Weft Threading from the Jacquard menu (Jacquard->Threading->Warp and Jacquard->Threading->Weft) and you can also switch from one to the other using the OK/Switch button. The procedure and options for both warp and weft are exactly the same therefore the following applies equally to both.

Up to 3 levels of bracketing can be used.

This dialog allows you to change the selector colours (see Selector Colours below) as well as the threading sequence.

If you wish to edit only the A or B threads from a double cloth or perhaps the A or B weft threads in a 1:2 (1 warp to 2 weft) weave you can use the Edit Layer feature.

Note also the letter A in the first column which indicates to the loom to use "Option A" for those 73 picks this might be a different loom speed or other option. Note also the plus signs ('+') which indicate crammed picks.

The above also shows the powerful Pop-up menu available by clicking the right hand mouse button and the multi-select feature allowing several cells to be selected at once.

 

Using the Dialog

Simply make your edits to the threading sequence then click on OK or on OK/Switch. The latter will switch to warp or weft as appropriate.

The software will compare the total threads against the ends or picks in the weave as appropriate and will display a warning if there is a discrepancy.

Any changes will cause the thread totals on the right to be updated (you cannot edit the totals column directly).

 

Undo/Redo

If you make any mistakes you can use the undo feature (<Ctrl> + Z) or the redo feature (<Ctrl> + Y). All operations can be undone including mirror, paste, delete columns and rows etc etc.

The undo information is cleared as soon as you exit the dialog using either CANCEL or OK. This means that the next time you display the dialog you will not be able to undo the data to a previous state.

Both Undo and Redo are available at the click of a button in the toolbar.

Cut Copy and Paste

You can select a block of one or more cells and use Cut (<Ctrl> + X) Copy (<Ctrl> + C) and Paste (<Ctrl> + V).

Cut will clear the block at the same time as copying it. The copied block can subsequently be pasted to a different location.

The cut copy paste feature uses the same internal format as the MicrosoftÂ™ XL spreadsheet program therefore you could prepare a threading sequence in XL, use copy in XL and then paste the result into a threading sequence in Jacquard Designer. This may be of some interest if a design has a warp or weft threading consisting wholly or partly of a complex numeric progression.

Note that when pasting you do not need to highlight the all the destination cells. You simply need to highlight the top left cell.

You can copy bracketing from one level to another. If you try to copy bracketing to selector information, the numbers will be copied but the brackets will be ignored. if you try to copy selector information to brackets then new brackets will be created where a number occurs.

Note that you can also cut and paste to or from Dobby Designer (version 6.1.1.1 or later) threading sequences.

Threading Sequence Grid

There are two types of value in the dialog - thread counts and times values:

Thread Counts

These represent a number of consecutive occurrences of a particular colour. If a design has "solid" colour i.e. it only uses one colour, then the whole dialog need only contain a single figure, probably in the first column on the first line (excluding brackets).

The count may be followed by a letter and or plus sign. Letters are used to indicate a loom option which is to be operated over those picks for example a fringing mechanism or a different pick rate, and a '+' sign is used to indicate a crammed pick (or an extra end).

Times Values

These give the number of times that a bracket repeats. The bracket will cover one or more columns and is indicated by a line starting underneath the Times Value. A maximum of two levels of brackets which may be used.

 

Moving around the grid

You can move around the grid either by clicking in a cell using the left mouse button OR you can use the up/down left/right cursor keys on the keyboard.

 

Multi-Select

You can multi-select a rectangular block of cells by either:

  • Using the mouse - press the left mouse button on a cell and, keeping the button down, move the mouse whilst keeping the button down to "drag" out a rectangular area.

  • Using the keyboard - move to a cell that is at one corner of the block and press the shift key. Keep the shift key down whilst you move to the cell at the opposite corner of the block using the cursor keys then release the shift key.

Various operations as described below use the columns or rows defined by the block, or use the block itself.

Entering Values

To type a new value in the grid simply move to the required cell as described above (i.e. either click in that position with the left mouse button or move to it using the cursor keys) then simply type the figure using the keyboard. If you make a mistake you can press the <Back> key to remove the last character.

The new value will replace the original - you cannot simply add a zero to make a 20 into 200 for example, you must type all 3 digits.

Examples of valid figures are:

20

200F

1Z

1+

12ABCD

1Z+

20A+

6

6

As can be seen it is possible to combine several letters representing loom options with a cramming ('+') (or warp extra) symbol. There is no limit to the number of letters that can be used in one cell though, of course, if there are more than about 6 characters in total, they would not all be visible at once.

Note that if the figure is a "times" value in a bracket you do not need to type the "x" - the software will automatically add it for you.

Deleting a Value

If there is a value in a cell which you wish to remove then first of all select the cell then use the <Del> on the keyboard to remove the figure.

You can delete a whole block of figures and brackets by using multi-select then pressing <Del> to delete the entire block. Alternatively you can select "Clear Selection" on the Pop-up menu.

Note that only brackets which are entirely contained within the selected block will be deleted but any that are partially within the selected area will be shortened as shown below

Example showing a multi-select block ready for deletion note the bracket on the right (x16) is only partially within the selected area.

The same cells after the <Del> key has been pressed. Note how the x16 bracket on the right has been shortened as it was not contained entirely within the block. The x2 bracket at the top level has not been affected as it started before the block and finished after it.

Editing Values

You can only retype a figure so simply click on it or move to the cell using the cursor keys then retype the figure. The old figure will be deleted as soon as you start typing. Note that you can use the <Back> key to erase a character at a time during editing.

To remove the old figure completely simply use <Delete>.

 

To Create a New Bracket

The simplest way to do this is to multi-select the empty cells where the bracket is to be placed as shown below. In fact it does not matter whether you select both rows or only one of the two rows which make up the bracket. Then simply type the "times" value.

Example showing multi-selected empty cells at level 2 where the bracket is to be placed.

The same cells after the digit 5 is typed on the keyboard to complete the bracket. You can continue typing if the "times" value is more than one digit.

 

If you make a mistake you can, as usual, use the <Back> key.

As an alternative to this you could select the single cell where the "times" value is to go then after typing the times value, the software will create a single column bracket for you. You can extend the bracket as described below.

To Extend a Bracket Using the Mouse

If a bracket needs to be extended, the simplest way is to use the mouse by clicking on the appropriate end of the bracket and "dragging it to the left or right as appropriate. If you make a mistake you can move back in the opposite direction whilst keeping the left mouse button down.

Example of a single column bracket after the "5" has been keyed in. To extend the bracket simply click on the bracket below the "x5" (highlighted).

The same cells after the bracket is "dragged" out to make a 4 column bracket

Note that whilst you do this, when dragging out a bracket at level 2 or higher, there may be invalid situations arising as it is not possible for a level 2 bracket to start or finish in the middle of a level 1 bracket. This may mean that the software will "jump" over several columns at once.

To Extend a Bracket Using the Keyboard

Either click on one end of the bracket or move the selected cell to one end of the bracket using the cursor keys. This method does not work if the selected cell is in the middle of the bracket.

Note that if you use the mouse, the whole bracket will be highlighted but the software knows which individual cell you have clicked on and that cell should be outlined by a dotted line.

Press the '>' key on the keyboard to move the selected end of the bracket to the right (you do not need to use the <Shift> key). Press the '<' key to move it to the left. Each time you press the key, the bracket will be extended or retracted by one column unless there is a multi-column bracket at a lower level in which case it will skip over the entire lower level bracket.

To Delete a Bracket

The simplest way to do this is to click on any position in the bracket itself using the mouse. This will cause the whole bracket to be highlighted then press the <Del> key on the keyboard. The whole bracket and the "times" value will be deleted.

Alternatively, if you only select the times value and press the <Del> key then the value will initially be reduced to zero. At this point you could type a new "times" value as described above. However, if you now press <Del> a second time, while the value is zero, the whole bracket will be deleted.

Optimise

This menu option will attempt to find additional brackets and will combine brackets together to get a simpler threading sequence. It does this in many ways from simple situations to more complex ones where a figure can be removed from one end of a bracket and another figure moved into the other end of the bracket to make a larger repeat as shown below:

Example bracketing before Optimise used. Note the inefficient bracketing in columns 2 to 5.

Same section of threading sequence after Optimise. Note how the figure of 14 in column 2 has been changed to 13, allowing 1 thread to be moved into the bracket, a similar thread dropped off the end of the bracket allowing the occurrences to be made up to x25 with the inclusion of the 1+ in the old column 5.

Using Optimise can often have an effect right across the threading sequence since more efficient bracketing in one set of columns can often lead to more efficient bracketing in the next set of columns.

At the same time the software will check for any additional bracketing that can be added or columns which can be included in existing brackets.

The Optimise operation will only use the number of visible levels so if you want it to use 3 levels (the maximum) you should use the Bracket Levels option described below to increase the levels to 3 before using Optimise.

 

Compress

This will force the threading to occupy the least number of columns so that several figures will be placed in one column. This can be useful after a large amount of editing has taken place which has left empty columns etc.

It can also be useful to compress an expanded threading sequence (see below) i.e. one that only has a single figure in each column.

This Pop-up menu option achieves exactly the opposite of the Compress function above, i.e. it expands the threading sequence by making sure that there is only one figure in each column. Note that any blank columns will also be removed.

Bracket Levels

You can select this pop-up menu option in order to change the number of bracket levels which can be set to 1 , 2 or 3. If the design is saved and reloaded, the visible levels will be either the preferences default or the number of levels used in the design whichever is the least.

Note that operations like Optimise will only use the number of visible levels so if you want an Optimise operation to be able to use 3 levels you should first increase the levels to 3 using this option.

Note also that warp and weft can have a different number of levels.
