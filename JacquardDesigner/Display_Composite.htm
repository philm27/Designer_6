Display Composite

 

This feature will display the composite weave for any given set of weave levels in a schema.

The composite weave is the combination of all the weaves defined by a particular artwork colour and may, in theory consist of up to 16 separate weaves (for crammed or extras).

Display Composite is accessed from the "Apply Weaves - Simple" dialog by selecting any line in the schema for which you wish to see the composite weave

The display is identical to that for the "Large Display" option - see that topic for further details on how to use it. An example of a Composite Display is shown below

 

Composite Weave Size

The software automatically makes the composite weave size up to an appropriate figure for the weaves at each level. This may mean using several repeats in one or both directions which can lead to a very large composite weave size.

Example 1: A double cloth having a 4x4 weave in the face and a 5x5 weave in the back will result in a composite weave size 40x40 (i.e. 20 face ends/picks and 20 back ends/picks).

Example 2: A double cloth having a 5x5 weave in the face and a 32x32 weave in the back would result in a composite weave size 320x320 (i.e. 160 face ends/picks and 160 back ends/picks).

If you intend to save the Composite Weave into your directory of weaves then you may want it to be as small as possible in which case you should use the smallest individual weaves possible.

For example if a plain weave in the back of a double cloth is not 2x2 but 8x8 (i.e. 4 repeats in both directions) and is used with a 5x5 weave in the face then the composite will be 80x80 NOT 20x20 as might be expected when using the basic size of 2x2 for the plain weave.

However, if you are simply applying weaves without saving any of the composite versions, then the size does not matter as the system will work equally well even if the resulting composite weaves have sizes of several thousand ends or picks.

 

Stitching Points

These are automatically created in the composite weave if marked in the separate weaves for each level.

Note that if the separate "Simple" weaves are saved to disk then stitching points are also saved. Thus if you mark a stitching point in, for example, a 1+3 twill and save it, then the mark will still be there when you select it again at a later date.

Note that when using the same weave in both the face and back of a double cloth, the stitching point will always be suitable (if a position in a long float is chosen). However, in other cases it may well be unsuitable.

It may, therefore be better, for each different design, to always mark stitching points for each weave separately (you only need to mark one level in a double cloth, not both). This would certainly be the case if you routinely use different weaves in face and back when producing double cloths.

Note also that there may be occasions when it is an advantage to make a weave larger than its basic size so that the stitching points will be an appropriate distance apart.

Procedure

From the "Apply Weave Simple" dialog, select one of the lines representing the weave to be displayed then click the right hand button to display the Pop-up menu.

Select "Display Composite" on the Pop-up menu as shown below.

 

The Composite Weave Display

An example composite weave display showing a double cloth having a 1+4 satin face and a 1+4 satin back with a stitching point indicated by the upturned "V".

The Weave Display itself shows the following symbols:

/

Lifters - These mark a position that is always a lift because the warp thread is from a higher level than the weft thread at that point. If a stitching point is placed here (i.e. the weft from the lower level being above the warp thread, then the stitching point symbol will be shown instead.

Stitching point (Downward) Indicates a point at which a warp thread from an upper level goes under a weft thread from a lower level OR a weft thread from an upper level goes under a warp thread from a lower level. These are normally necessary in order to hold the two levels together to prevent large pockets forming.

This symbol indicates that the user marked the stitching point on the upper level weave in the schema.

Stitching Point (Upward) - Indicates a point at which a warp thread from a lower level goes over a weft thread from an upper level OR a weft thread from a lower level goes over a warp thread from an upper level. This symbol indicates that the user marked the stitching point on the lower level weave in the schema as in the example above (note the "(st)" appended to the weave name sati5).

Stitching Point (Combined) - Indicates that a stitching point was marked at the same point in both the upper and lower levels of a double cloth. The resulting Composite Weave is, in fact, the same whether one or both levels are marked in a double cloth.

 

The display shows the following:

Name

Always set to "Composite"

Size

Size as calculated 10 ends and 10 picks in this case

End/Pick

Shows the end and pick number being pointed at (end 3, pick 1 in the above example).

Weave Edit Tools

The composite weave cannot be edited so these are disabled

 

Save Composite Weave

You can save the composite weave into your weave library by simply selecting "Save As..." from the Pop-up menu. In fact this is the only menu option that is available.

See the topic "Large Display" for more details of the Pop-up menu options.
