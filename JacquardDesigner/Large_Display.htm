Large Display

 

This dialog is available for those weaves which are very large and which therefore are difficult to check or edit in the small viewing area in the Apply Weave dialogs (either Apply Simple Weaves or Apply Composite Weaves).

An important feature of this dialog is that you can make the window as big as you like by dragging the bottom right corner down and/or to the right and you can also zoom in or out and scroll the image. All of which make it very easy to handle large weaves.

The dialog also has the same editing features as the main Apply Weave dialogs i.e. shift ends/picks , Horizontal Flip and Invert.

To access the Large Display dialog select it from one of the Pop-up menus in either of the Apply Weave dialogs. You should see an image similar to the following.

 

Example of Large Display showing the dialog at its default size but with the image zoomed in.

The dialog shows the following information:

Title Bar

The title bar shows the full file path for the weave (if any)

Name

The weave name i.e the file name without the file extension (.iff or .prj).

Size

This is the weave size in ends and picks

End/Pick

This is the end and pick number that the mouse is pointing at. If the pointer is out of range then this will be shown as --/--.

In the above example the pointer is at end 1, pick 8. Note that if you have selected the Scottish system (right to left numbering) in Jacquard Preferences then end 1 is on the right.

 

Editing the Weave

The features available for editing the weave are exactly the same as those for the Apply Weave dialogs - see the topic Weave Edit for more information.

Briefly you should use the left mouse button to toggle a point from lift to drop and you can use the Shift Ends, Shift Picks, Horizontal Flip and Invert options on the left hand side.

The right hand button will display the Pop-up menu allowing things like insert stitching point, clear stitching point, remove all edits, remove all stitching points etc.

 

Zoom, Scroll and Re-Size

The Large Display dialog allows the following additional features:

Zoom

Use "+" and "-" on the keyboard to zoom in and out as you would do on the main design screen. If the keyboard beeps it may mean that the image does not have the keyboard focus (such as when the program is waiting for a value to be keyed in to one of the Shift values). In this case simply click anywhere in the main dialog or in an unused area of the weave display. You will then be able to zoom in and out again

Scroll

When the weave cannot all be seen as it is too large at the current zoom level then scroll bars will be displayed allowing you to move the weave up or down, left or right.

Re-Size

You can, at any time, press and hold the mouse button at any edge of the dialog and stretch it upwards, left, right or downwards. If you click on a corner of the dialog you can re-size the corresponding two edges simultaneously.
