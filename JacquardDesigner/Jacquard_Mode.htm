Jacquard Mode

 

There are 3 different modes, selectable on the Jacquard Menu, that can be used to view the design:

  • Artwork

  • Weave

  • Cloth

Artwork Mode

Use this mode for all drawing and painting other than correction of floats in the weave. The size of the artwork can be checked in the Design Specification. The size of the Artwork should correspond to the number of ends and picks in the face of the cloth. In other words it should be half the total ends or picks in a double cloth and one third of the total for a treble cloth. A similar rule applies for crammed and extras, just count the number of ends in the face of the cloth (i.e. in crammed cloths count the ground picks).

For example if a double cloth repeat is 3 inches and it has a total of 120 ends and 120 picks then the face of the cloth has 20 ends per inch and 20 picks per inch. The Artwork size will be 60 ends and 60 picks to correspond to the face of the cloth. (Ends per inch and picks per inch will both be 20, again, use the figure which corresponds to the face of the cloth).

 

Weave Mode

This mode is used to display the weave version of the cloth in black and white (dark colour for a lift). However if you have done a float check and asked for floats to be marked, then coloured floats may be shown (see the topic Float Check).

Weave Mode cannot be accessed until you have applied weaves.

The palette will be changed to a special one for this mode which will include black and white and the colours used to mark floats.

Weave Mode and Undo/Redo

You can pan and zoom as normal and draw using black or white. Any editing you do will become permanent although you can use undo and redo in the normal way if you want to remove your edits. If you reapply weaves any manual edits will of course be lost and the undo history will be cleared i.e. you cannot undo back past the point at which weaves are applied.

Note that the weave and the main artwork each have their own independant undo history and you can switch from one to the other doing undo and redo without affecting the other mode.

 

Cloth Mode

This mode allows you to see the cloth with all the correct colours in the face.

You should nominate colours for the selectors and also, if appropriate, key in a threading sequence.

For Cloths having crammed picks or extra warp threads, the threading sequence is normally set automatically. However, a colour still needs to be allocated to each selector (see the topic Selectors) in order for the cloth view to appear in the correct colours. Note that at the time of writing only plain colours can be used in the Jacquard program itself. However, the design can be imported into APSO Dobby Designer and any yarn type used and a textured view shown.
