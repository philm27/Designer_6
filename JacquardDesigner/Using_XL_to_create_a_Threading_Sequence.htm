Using XL to create a Threading Sequence

 

You can use Microsoft XL to create a threading sequence. This may be useful when the sequence is a particularly long mathematical progression.

In this design there will be a pair of stripes alternating and decreasing gradually in size from 24 down to 8. This kind of progression can be created easily in XL by setting the initial two values to 24 then setting the two figures in the next column to A4-1 and A5-1 respectively.

All the other figures can then be copied from column B by selecting the column B figures and using the Fill/Repeat feature as shown below.

(B) Select the two figures in column B as shown then point at the bottom right corner of the highlight, press the left mouse button and stretch it across to the right. This will result in the image (C) below as the software will repeat the formulae in column B.

(C) This shows the situation after the button has been released.

 

After the above steps you can paste the data into Jacquard Designer:

  1. Highlight the whole set of figures in XL

  2. Use <Ctrl> + C to copy the block of figures

  3. Move to Jacquard Designer

  4. Select warp or weft threading sequence as appropriate

  5. Click on the top left corner of the block which is to receive the data in the threading sequence

  6. Use <Ctrl> + V to paste the data into your design.

Note that you do not need to worry about the number of columns available as the software will add extra columns if necessary. However, the software will overwrite any existing data so you should make sure that you are not going to erase any data that you want to retain. Remember that you can use undo (<Ctrl> + Z) if you make a mistake.
