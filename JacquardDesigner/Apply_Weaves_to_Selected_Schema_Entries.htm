Apply Weaves Selective

You can restrict the application of weaves to one color or even to individual levels in the overall weave. This feature can be used in the Simple Weaves dialog or in the Composite Weaves dialog.

To do this use the pop-up menu option as illustrated here, or click on the button in the toolbar.

After a selective apply weave, the dialog will remain open so that it is easy to repeat the process after making new selections and/or changing weaves.

An example of the Apply weaves dialog. User has selected 4 items for color 10 and is about to apply those weaves without affecting any other area in the overall weave. The user could have selected only one or two of those selectors in this case as it is a crammed fabric (see restrictions below).

Restrictions

When using "Apply Weaves Simple" for a Double Cloth, Treble Cloth or any other structure that has more than one level in warp and weft, then the user must select all schema entries for a color, this is because the software is inserting lifters and possibly stitching points  that affect multiple levels of the cloth.
