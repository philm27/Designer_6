Tutorial 1

 

Introduction

This Tutorial guides you through creating a simple design (artwork only) and saving it to disk.

Before we start to actually create a design - a couple of basics:

 

Using the Keyboard

In the following and throughout the Help Files if you see <> angle brackets like this it means that a particular keyboard key is to be pressed. If this is followed by a plus sign it means press two keys together. For example:

<Ctrl> + G

means press the Control key and letter G key together.

<Shift> + Z

means press the Shift key and letter Z key together

 

Using Menus

To use the mouse to select a menu item, simply click and release the left mouse button on the menu bar. The menu you have clicked on will then "pop up". You can then move left or right and up and down menus until you reach the item you require. Click the left mouse button when the required item is highlighted. Some menu items are sub menus having their own "pop-up" menus so you may need to move to the left or right down several menu levels.

You can also use the keyboard to select a menu item. To do this click on the <ALT> key to make the menu bar "active" (one menu will be selected and one character in each menu name will be underlined to indicate its keyboard shortcut). You can then use the keyboard cursor keys to navigate to a menu item or you can use the keyboard shortcuts indicated by the underlined character in a menu title or a menu item. When you reach the required item simply press the <Return> key to select it.

Press <Esc> to make the menu "inactive" and remove menus.

For example to select menu item Design -> Specification you can type <Alt> D S (not together but one after the other). Or, you could type <Alt> <right> <right> <right> <down> <Return>.

Menu options are written like this:

File -> New

Select File menu then New menu item

Design -> Match -> Half

Select Design Menu then Match sub menu then Half menu item

Repeat Type

There are four repeat types to choose from:-

  • Normal - the design will repeat vertically and horizontally. The "drop match" for the next repeat on the right can be chosen as described below.

  • Rug - the design will not repeat. Choose this if you are designing a rug, wrap or scarf etc.

  • Horizontal - the design will only repeat side to side. Choose this if you are designing a horizontal border or ribbon etc.

  • Vertical - the design will only repeat top to bottom. Choose this if you are designing a vertical border, runner or ribbon etc.

  • Brick - the design will in both X and Y axis but the next repeat above will shift by the amount specified as a brick repeat (see below)

Match

There are four standard match types to select from straight, quarter, third and half. These can be applied to any repeat type except Rug. Note that Vertical runners can have a Brick reepat and Horizontal repeat types can have a "drop match".

Also available is an "Any" option which allows a drop to be specified in Rows (Picks) and a Brick shift to be specified in Ends.

 

Design Width / Length

If you want to change the design size, input the new details in the specification window. You can work in ends and picks or in design measurements (feet/inches or metres/centimetres). If you are working on a large design, you might find it easiest to work in design measurements and let the computer work out your ends and picks (the design size is the "design repeat" this may be smaller than the actual cloth width). These will be displayed in the status bar.

 

Ends & Picks

To set the quality (sett) of your design in the specification window, you can select ends and picks per inch, 3 inch, foot, yard, centimetre, 10 centimetres or per Metre. Within your chosen parameter you can set the ends and picks to one decimal point.

You can see what quality you are working in at any time by glancing at the status bar.

When you have set your design size and setting, click on OK. If the changes mean that the design size has changed, the software will display call up a dimension window, asking where you want to add/cut the existing size to create the new size. When starting a new design, this is unimportant, so just click on OK. The dimension window only becomes important when you alter the size of a design that you have already worked on.

You can immediately see your new design size, both in ends/picks and in design measurements, displayed in the status bar.

 

Grid Setting (Width & Length)

The grid needs to be set so that the software can draw bold lines at specific intervals when the grid is switched on. Usually, the bold lines are required at the same spacing as the design quality. For example, if your design quality is 36 x 40 to the inch, then you may well require bold grid lines every 36 ends and 40 picks. If this is the case, after setting your design quality, just check "Grid as Quality" in the specification window.

If you require a different interval for bold lines, uncheck "Grid as Quality" and enter the required frequency.

Set Default

If the Size, Repeat, Setting and Grid information is likely to be used many times, you can make it a default specification by clicking on the "Set Default" button in the bottom right hand corner of the dialog. Then each time you start a new design, it will automatically default to these figures. You can re-set it at any time.

Drawing Tools and Colors

Before we start to create a design let's look at a few basics.

At the top of the toolbox you will see a rectangle with a circle in it (the currently selected colors). The color of the inner circle (Light Brown in the example below) represents the color associated with the left mouse button. The rectangle (green in the example below) represents the color associated with the right mouse button. Moreover, if you look at the "Palette Window", you will notice that the colors you have selected on each button of the mouse have a "tuck" on the lower left or right corner respectively. This feature serves as a visual reminder of which color is represented by which mouse button.

For the purposes of this introduction, we will stick with the default palette .

To change the selected colors, simply point the cursor over the palette window and click the left button of the mouse over the color you wish to draw with. You can click with the right hand button to change the right hand button (background) color.

When using any drawing function, you should also select a brush-type from the toolbox. There are a variety of different sized square and round shapes. These can be found at the bottom of the toolbox.

There is a different selection (this time just governing line thickness) for the Line and Shape-drawing functions:

 

 

If you want to draw with a larger brush size than those in the toolbox, double-click on the drawing tool buttons (again in the toolbox). You can then select your own size/width of brush.

It is also possible to draw and fill with a Custom Brush (see the Topic Custom Brushes for more details ).

If you are working with a pen and graphics tablet, then you can control brush size by applying pressure with the pen on the tablet. To use this function go to Edit -> Preferences -> Pressure and enable the function.

Try out the "drawing tools" - to familiarise yourself with them - by clicking on the buttons (in the toolbox ) and trying the different methods of drawing (remember you can utilise the two selected colours with the two mouse buttons).

Moving the Toolbox

If you find that the toolbox is obstructing your view you can move it around the screen by clicking and dragging on the top grey bar.

Click and hold down the button on the empty area (the topmost part of the toolbox, above the currently used palette colours)  it turns blue in colour. Now drag and drop the toolbox anywhere on the screen, so that it does not get in the way of your design.

Zoom

You may also want to zoom in on the design area. To do this press the + key on the keyboard. Similarly, to zoom out, press the - key. For both + and - you can use the numeric keypad or the main part of the keyboard.

 

Drawing a Simple Design

You can either create a simple design of your own or do the following to create the simple design shown below.

Undo

If you make a mistake at any stage simply click the undo button. If you click this button several times in succession then a series of drawing operations will be undone.

Select Palette Colours and Set Background to White

In the palette, click on red with the left button and white with the right button. You should see the two colours at the top of the toolbox to indicate that you have selected those two colours. (Red being in the circle and white the background as shown below).

If the current background in the main design is not white, select the fill button and then click anywhere in the design with the right mouse button. This should fill the whole design with the current right hand button colour which is white.

Draw The Circle

Now select the Circle Outline button in the toolbox. Move to the main design and draw a red circle.

Now re-select the fill button and then click anywhere inside the circle with the left mouse button. The circle should now be flood filled with the red colour.

View in Repeat and set the Half Drop Match

Put the design into repeat by clicking the View Many button in the toolbar.

Make the design a half drop match by selecting menu item Design -> Match -> Half.

Zoom in and or out using the plus or minus keys on the keyboard so that there are about 3 - 5 repeats across and down the screen as shown below.

Draw The Blue Lines

Select the blue palette colour with the left mouse button.

Select the line drawing function in the tool box.

We want to draw a much thicker line than the normal single point so either right click the Line button or right click or double click the "Custom Line" at the bottom of the list of line thicknesses in the toolbox. This will display a "Custom Line Width" dialog as shown here. Set the line thickness to 1 inch (or any other value you choose) and click on OK or press the <return> key.

The Line Properties Dialog with a thickness of 1 inch selected.

Draw some lines anywhere in the design by pressing and holding the left mouse button on the start position and then moving to the end point and releasing the mouse button.

Notice that the lines will be immediately repeated in all other visible repeats and that you are able to draw across the edges of the repeat. You can if you wish draw a diagonal line longer than one repeat - it will simply "wrap" around across the actual repeat.

Saving

The normal file type of a design file which will contain not only the design and palette but the specification, frame plan shade numbers etc is the IFF file type. A second file type PRJ can also be used and this will in addition save the Shelf items i.e. the whole project.

To save an IFF file, access the "Save As" dialog by selecting menu item File -> Save As.

Find the folder in which you want to save the design. To do this, work your way through the folder "family tree". Go to the "parent" folder by clicking on the icon, or to the "offspring" folder by double clicking on a folder within the currently selected folder. When you find the required folder make sure that it is open in the "Save In" box.

Select IFF or PRJ files from the "Save as Type" drop-down menu, and then enter your design file name in the "File Name" box.

Click on the Save button to save the file in the folder. The Save As dialog will close, taking you back to the design screen.

If you have already saved your design to a folder, it is possible to update the file with more recently done work. The old design file will be overwritten and you will not be able to retrieve it, so only use this function if you are sure you dont want to keep the previous version.

You can re-save a design without changing its name as follows:

  • Select menu option File -> Save.

OR:

  • Click on the Save button on the toolbar

OR:

  • Use the keyboard shortcut <Ctrl> + S

 

It is always advisable to save work frequently to avoid loss of work and effort in case of computer system failure for any reason.
