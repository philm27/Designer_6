1:2 and 1:3 Ratios

 

Introduction

This type of cloth is one in which some of the benefits of a double cloth can be achieved in that there is a wider range of colours available but this only happens in either the warp or weft direction.

Weaves must be chosen very carefully since each warp thread (in a 1 warp to 2 weft ratio) is used in both levels all the time and not every combination of weaves will work.

A common use of this type of cloth is a 1:2 ratio where two weft colours alternate in the threading sequence so that the A weft is used in the face and the B weft is used in the back. The weaves can be applied in such a way that the colours swap from the back to the face and vice versa in certain areas to form motifs. The cloth can then be weft faced AND weft backed by using, say, a 1+3 satin in the face and a 3+1 satin in the back.

 

Procedure

The procedure for production of one of these two types of design is very similar to that for double cloth or treble cloth. In outline, this is as follows:

  1. Complete the artwork using the FACE ends and picks per inch (or cm etc.) so that the artwork has the correct number of total face ends and picks.

  2. Assign Colours to Warp and Weft Selectors. (Though you can leave this until later if you wish).

  3. Use Apply Weaves Simple to create the Schema (specifying the "ratio" in the "full levels" values) and select and apply weaves. (Stitching Points are not possible as the cloth is Self Stitched).

  4. Make any required manual changes to warp or weft threading sequences. Unlike when using Cramming or Extras, the warp or weft threading is not created automatically.

The design can now be saved, viewed in Cloth mode etc like a normal design.

The following sections of this topic give more detail of the procedure.

 

Artwork

First of all the artwork should be produced and finalised. As usual you should aim to use a different colour to show where different weaves and/or warp/weft pairings are used in any level. It is not just the face that matters of course. An area having a different back weave (or middle level weave in the case of 1:3 ratio) must be represented by a different artwork colour.

Warp/Weft Pairings

In our example design we have used two different coloured areas where we are simply going to use a different weft selector in the face.

In order to change the warp/weft pairing in the red area (line 3 above):

  1. Click on line 3 with the left mouse button to select it.

  2. Click anywhere in the schema using the right hand button to display the Pop-up menu.

  3. Select Properties from the Pop-up menu.

  4. Select Warp A and Weft B as shown below and select OK

Note that when you select OK, the software will automatically make any necessary changes to other levels. In this case the back of the cloth must be Warp A and Weft A and so it is not necessary to make this change manually.

Alternatively you can simply move the bottom layer up to the top using the mouse - see the topic Apply Weave Simple for more information.

 

Loading Face Weaves and Back Weaves

The following assumes that the weaves have already been designed and saved to disk. If they have not simply create new weave, edit it and save it as described in the topic Weave Edit. You can do this within the "Apply Weaves Simple" dialog.

In many cases the face and back weaves will be different. However, in the example described here, they are all the same.

There are several ways of multi-selecting lines in the schema, some of which can be used to advantage when loading weaves as described below.

 

Multi Line selection

We need to set every weave to sati4, to do this:

  1. Click anywhere in the schema with the right hand mouse button to display the Pop-up menu.

  2. Select menu option "Select All". All schema items should now be highlighted.

  3. Display the Pop-up menu again using the right hand mouse button.

  4. This time select "Open..." to display an open file dialog and locate the weave (sati4 in this case) and either double click it or click it once then select the Open button.

  5. The weave will be recorded against all items.

The schema should now look like the example shown above.

 

Using the Weave Invert and Shift options

The back weaves need to be a slightly modified version of the face weaves. First of all they are normally an inverted version of the face weave so that the product is weft faced and weft backed. Also they need to be shifted slightly so that the weft floats in the face are free to roll over the shorter 1 end weft floats in the lower level (see the illustration on the right below).

We can use the weave edit options in the Apply Weave dialog to do this as shown below. Simply select each back weave in turn and make the two simple changes shown here.

Weave sati4 used for face with no edits. This will be placed on picks 1,3,5,7.

Weave sati4 with a shift of 2 picks and inverted used for the back weave (note warp faced so cloth is "weft backed" as well as weft faced). This will be placed on picks 2,4,6,8.

The resulting Composite Weave. Note the positions at which drops occur (marked by a red x) in the back weave (even numbered picks) are "good" because they are hidden by weft floats in previous and in next pick. This does not happen in every variation of the shift value so great care should be taken.

 

The diagram above represents one repeat of the weave and it can clearly be seen here how picks 1 and 3 can slide together and cover pick 2. Similarly all pink weft (the A threads) can cover the red weft (the B threads).
