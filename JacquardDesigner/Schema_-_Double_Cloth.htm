Schema - Double Cloth

Schemas are created, reset and edited using the "Apply Weaves Simple" dialog.

A Double Cloth example in which the first color is to have Warp B and Weft A in the face and is to be a weave called sati7 in both face and back. The second color is to be warp A and weft B in the face and is to be weave sati5 in both face and back. One or more stitching point have been added (denoted by the "(st)"). Note that the order of the schema dictates which is the face and which is the back.

To create a double cloth schema click on the Reset button in The Simple Weaves dialog and enter 2 warp and 2 weft and click on OK.

To modify the warp weft pairings, right click any line and select Properties. You can then modify the Warp letter and/or Weft letter.

You can also use the mouse to "drag" an entry to a lower or higher level.

To modify weaves, you can either create a new weave and edit in situ, or, double click any line and select a weave from disk. For more details on this and other general aspects of Apply Weaves dialog see the topic Schemas - General.
