Reverse Cloth

 

To reverse cloth simply select menu option "Jacquard->Reverse Cloth->Horizontal Flip" or "Jacquard->Reverse Cloth->Vertical Flip". Neither of these options affect the artwork so if you want to reverse the artwork as well you must use the Design->Mirror option.

The Reverse Cloth Menu Options

Horizontal Flip

This effectively flips the design over left to right as viewed by the weaver at loom so that the viewer is shown the back of the cloth.

  1. Warp threading is mirrored left to right, however, any bracketing is retained thus a second "Horizontal Flip" should return the threading sequence to its original state.

  2. The entire weave is flipped left to right and also lifts become drops and vice versa.

Weft threading remains unchanged as does the Artwork.

 

Vertical Flip

This effectively flips the design over from top to bottom as viewed by the weaver at loom so that the viewer is shown the back of the cloth.

  1. Weft threading is mirrored, however, any bracketing is retained thus a second "Vertical Flip" should return the threading sequence to its original state.

  2. The entire weave is flipped top to bottom and also lifts become drops and vice versa.

Warp threading remains unchanged as does the Artwork.

 

Saving and Undo

Note that the design can be saved with the cloth reversed. In this case it will weave reversed if transferred to the loom via an interface file such as Staubli JC5 or BONAS .ep file.

If you have reversed the cloth but then want to save it with the original face, simply reselect reverse cloth. At the time of writing, the UNDO operation does NOT undo a "reverse cloth" operation.
