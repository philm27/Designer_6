Edit Layer

 

This feature, though named "Edit layer" is a method of editing just the A threads or B threads in a double cloth, or the A, B or C threads in a treble cloth.

Since the cloth does not have to actually be a double or treble cloth, it may also have other special uses where it would help to edit for example, the 4th thread in a warp and every 12th or some other unusual requirement.

To use the feature, simply select the Warp (or weft) threading plan from the menu. This may, (after selecting selector colors), initially show a dialog as follows:

Example threading plan for which the total threads is 252 and the user has selected 3 colors and which is currently solid dark grey (selector 3)

In order to edit the A threads for this small double cloth design. Simply click on the "Edit Layer" button on the right. This will display a dialog similar to the following:

Example of the "Edit Layer" dialog, set up to edit the A threads in a double cloth. In order to edit the B threads, simply set the "Edit Layer" value to 2 instead of 1

Click OK in the above dialog to return to the threading plan but it will show a shorter plan in which the specified threads are shown in isolation. Continuing with our example and after a simple edit, we could define stripes of 4 threads in the face of our double cloth, alternating beige and red with, say a dark grey 6 end stripe on the end. The dialog would look like this:

Example of a layer edit defining alternating beige/red stripes and finishing with a wider grey stripe. Note that the Edit Layer button and the OK/Switch button are disabled. Also note that the window title shows Warp Threading Sequence[Layer 1/2].

While editing the layer you can use Undo and redo and all other buttons and pop-up menu as normal except that you cannot undo to a point before the Layer Edit was started and you cannot modify the total number of threads in the layer as this will be invalid.

In order to exit from Layer mode you must either cancel (to forget the layer edit) or click on OK to cause the merge with the other threads to take place.

Obviously the software cannot retain the original bracketing and will generate a new version from the merged data.

In this case the new, merged plan will look like this:

Example threading plan after merging. You can now click on OK or cancel to return to the main design program

Note that if you make further edits or you wish to remove the changes made during the "Layer Edit", you can click on Undo. The "Layer Edit" as a whole will be removed and treated as 1 single operation as far as Undo and Redo are concerned.
