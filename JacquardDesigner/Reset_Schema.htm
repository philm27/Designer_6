Reset Schema

 

This dialog is displayed when you select the "Reset Schema" button on the "Apply Simple Weaves" dialog.

You will be prompted to "Reset the Schema" when selecting the "Apply Simple Weaves" dialog if you have made a change to the design necessitating a schema change. You should use this option when you change the design by adding more colours and in the case of Cramming and Extras, even if part of the design has simply been re-drawn since it may mean that you need to specify more weaves.

Note that using this option does not lose any required weave information as all weaves will be copied to the new schema at the appropriate level or warp/weft selector. The only weave information that will be lost is that for unused Artwork colours or levels.

Any new colours or levels/selectors will be set to "No Weave".

Example "Schema Reset" dialog set for Weft Cramming.

The following rules apply at present:

  • You can set either Extra Ends or Crammed Picks but not both.

  • If you set either Extra Ends or Crammed Picks then Full Levels must both be 1 i.e. a single cloth.

  • Full Levels must be in the range 1 to 3.

  • If Warp and Weft Full Levels are different then one of them must be 1, you cannot have a cloth that is a 2:3 ratio.

Procedure

Simply select the appropriate options and set the number of warp and weft full levels then select OK.

 

Full Levels

This is used to indicate a level which runs the full width or length of the cloth.

In a double cloth, there may for example be 160 ends in the face repeat and 160 ends in the back repeat making 320 ends in total. Both levels are full in this case as there are always 2 ends for each "artwork" end, or, to put it another way, 1 artwork end converts to 2 cloth ends, 1 for each level.

If, there are 160 face ends but only 200 ends in total with an occasional second level (or more) being used according to the design then that requires use of Warp Extras and those levels are not "Full" Levels.
