Double Cloths and Treble Cloths

 

The procedures for production of these two types of design are very similar.

The procedure, in outline, is as follows:

  1. Complete the artwork using the FACE ends and picks per inch (or cm etc.) so that the artwork has the correct number of total face ends and picks.

  2. Assign Colors to Warp and Weft Selectors. (Though you can leave this until later if you wish).

  3. Use Apply Weaves Simple to create the Schema and select and apply weaves. If stitching is required you should also select stitching points.

  4. Make any required manual changes to warp or weft threading sequences. The threading for both warp and weft will automatically be set to an AB threading for a double cloth and an ABC threading for a treble cloth.

The design can now be saved, viewed in Cloth mode etc like a normal design.

The following sections of this topic give more detail of the procedure.

 

Artwork

First of all the artwork should be produced and finalised. You should aim to use a different color to show where different weaves and/or warp/weft pairings are used in any level. It is not just the face that matters of course. An area having a different back weave (or middle level weave in the case of treble cloth) must be represented by a different artwork color.

In other words when the artwork is complete, a given palette color indicates an area or areas where the weaves and parings in each level are the same.

Warp/Weft Pairings

In our example design we have used five different colored areas. In actual fact the green and bright red areas use the same pairings and weaves and so could have been the same artwork color, though that would not have given artwork which was representative of the final product in this case.

In order to change the warp/weft pairing in the black above (line 1):

  1. Click on line 1 with the left mouse button to select it.

  2. Click anywhere in the schema using the right hand button to display the Pop-up menu.

  3. Select Properties from the Pop-up menu.

  4. Select Warp B and Weft A as shown below and select OK

Note that when you select OK, the software will automatically make any necessary changes to other levels. In this case the back of the cloth must be Warp A and Weft B and so it is not necessary to make this change manually.

You now need to change the green and the two reds in a similar way each of which have the A/B paring in the face.

 

Loading Face Weaves and Back Weaves

The following assumes that the weaves have already been designed and saved to disk. If they have not simply create new weave, edit it and save it as described in the topic Weave Edit. You can do this within the "Apply Weaves Simple" dialog.

In many cases the face and back weaves will be different. However, in the example described here, they are all the same.

There are several ways of multi-selecting lines in the schema, some of which can be used to advantage when loading weaves as described below.

 

Multi Line selection

We will begin by setting the weave sati7 into black and dark red.

This will illustrate another technique - the ability to multi-select lines in the schema.

  1. Select the First item (face weave for black) using the left mouse button.

  2. Hold <Ctrl> key down and select the other four items in turn using the left mouse button.

  3. If you select an item by mistake and want to unselect it, simply click on it a second time whilst keeping the <Ctrl> key down.

  4. Release the <Ctrl> key when all items have been selected.

Note that if you accidentally release the <Ctrl> key before you have selected all 4 items you can press it again and hold it down while you complete the selection process.

Now, to load the weave sati7 from disk:

  1. Click on the Schema again with the right mouse button to display the pop-up menu. The four selected items will remain selected.

  2. This time select "Open..." to display an open file dialog and locate the weave (sati7 in this case) and either double click it or click it once then select the Open button.

  3. The weave will be recorded against all selected items.

Example schema after weave sati7 applied to black and dark red face and back.

Using the Weave Invert and Shift options

The Apply Weaves dialog has features which allow us to edit the weave. These edits are retained within the IFF or PRJ file when the design is saved. They do not cause the individual weave to be changed in its disk file (unless it is re-saved of course). For more information about this see the topic Weave Edit.

The dialog not only allows us to edit individual points in a weave but we can make changes which affect the whole weave such as the shift ends, shift picks, invert and horizontal flip options.
