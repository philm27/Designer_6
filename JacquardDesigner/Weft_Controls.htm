Weft Controls

Select "Weft Controls..." on the Pop-up menu. This should display the Weft Controls Dialog.

Note that this option is not available in Merge View.

 

Example of the Weft Controls Dialog. In this example controls ABC and F have been selected.

Use this dialog to nominate which controls are shown and possibly edited in the main Weave Edit window. Click on one or more letters in the left column then ">" to move them across to the selected list on the right. You can also move one or more letters out of the selected list by selecting those letters you wish to remove and clicking the "<" button.

When the OK button is clicked, the software will check if there are any controls used in the design that are not going to be displayed.

After clicking the OK button, the weave edit window will be modified to show the selected controls as in the example below.

Part of a weave edit window showing crammed threads at pick 7, 9, 11 etc (indicated by "+" on top of the thread color) and weft controls ABC and F with F being used on picks 1, 2 and 3 and A being used in pick 4. In this example there is a gap between picks and picks relating to the same artwork row are "Grouped" e.g. picks 1-5 relate to artwork rows 1-5 and picks 6 and 7 relate to artwork row 6. The artwork is shown in the background (a broad grey curve can be seen at the top right surrounding an area of blue weft floats).

Display

The display only shows a subset of the 26 possible as it is highly unlikely that more than 3 or 4 will be required. but note that there could be others that are used but not shown (see the comments for the Controls dialog above).

Edit Control Setting

Simply click on a letter using the Left hand mouse button to set a control. Click using the right hand mouse button to unset a control. Controls that have been selected are shown in red, unused controls are in light blue.

Controls and their Meaning

The meaning of these controls ABC...Z is ultimately dependant on which hooks they lift or electronic signals they set in the loom file created from this data.

For example F might lift hook 12 which, at the loom, might be a fringe mechanism; A, B and C might be changes of pick rate etc.

These "connections" between a letter and a loom function are set up in the Looms Maintenance software and can thus, in theory, be a different hook or electronic signal at each loom or loom group.

Threading Plan

Note that the controls and cramming information may also be set and edited as part of the Threading Plan. See the topic Threading Sequence.
