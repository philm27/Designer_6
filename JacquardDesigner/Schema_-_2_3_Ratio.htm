Schema - 2:3 Ratio

Schemas are created, reset and edited using the "Apply Weaves Simple" dialog.

In this example the user has created a schema having a ratio of 2 warp to 3 weft. One weft in every 3 is used as a "stuffer" between 2 cloths. The "Stuffer" in each case has no warp pairing - just a dash eg -/C. In the orange and yellow areas weft C is the "stuffer" whereas in the blue area weft B is the "stuffer" and weft C is paired with warp A in the face.

 

To create a schema of this type click on the Reset button in The Simple Weaves dialog and enter 2 warp and 3 weft and click on OK. Stuffer threads will be shown, for example, as -/C where C is the weft stuffer thread. A warp stuffer would be shown as A/-. Any weave on these schema entries is ignored.

To modify the warp weft pairings, right click any line and select Properties. You can then modify the Warp letter and/or Weft letter.

Example of the Schema Properties - the user has just selected weft C to be paired with warp A.

Another example of the Schema Properties - the user has just selected weft B and in this case there is no Warp in the pairing - this will be the normal case for a stuffer thread.

You can also use the mouse to "drag" an entry to a lower or higher level.

To modify weaves, you can either create a new weave and edit in situ, or, double click any line and select a weave from disk. For more details on this and other general aspects of Apply Weaves dialog see the topic Schemas - General.
