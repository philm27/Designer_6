Schemas - General

 

Schemas are the means by which you can define what is required in each area of the design. In other words they define the structure of the cloth.

For a simple single cloth the schema consists of one entry per artwork color and that entry defines only the weave to be applied to that color.

For a multi-level cloth however, the schema defines the weave to be applied to each level or selector.

Schemas are created, reset and edited using the "Apply Weaves Simple" dialog from which the examples below have been taken

The Schema has three main columns:

Artwork Color

The list will show both the number of the palette entry and a rectangular swatch of the color. Only colors used in the design are shown.

If successive lines have the same palette color then the color is only shown once (as in the double cloth example above).

Selector

This column shows the Selector information for the various levels within the artwork color.

It is important to note that the sequence of the list (when in Artwork sequence) implies the cloth level (working from the face through to the back). So in the above example, color 1 (black) has weave sati7 using Warp B and Weft A in the face and sati7 in the back using Warp A and Weft B.

Using the various editing features, lines can be moved up or down, pairings can be changed and/or weaves changed. See Apply Weaves Simple for more information about editing the schema.

For more information about this column see the section Selectors and Pairings below.

Weave

This column shows the weave name. See below for more details.

 

 

Selectors and Warp/Weft Pairings

The middle column of the Schema List shows the Selector number if applicable or the Warp/Weft Pairings for each level. The first figure shown is the Warp Letter or "selector", the second is the Weft Letter or selector.

For single cloths this column is irrelevant and will show "-/-" as there is no relevant warp or weft selector information.

For cloths having "full" levels such as double cloth, treble cloth, 1:2 ratios etc., the column will show for example A/B or A/A or B/B etc to indicate the warp/weft pairing that is to be used for the Artwork color at that level. Note that unlike Warp Extras and Weft Cramming this does not imply the actual color to be used since the color of the A or B thread can change according to the threading sequence.

For a double cloth, the valid pairings for the two levels are:

Face

A/A

B/B

A/B

B/A

Back

B/B

A/A

B/A

A/B

For a cloth having 1 warp level but 2 weft levels the valid pairings are:

Face

A/A

A/B

Back

A/B

A/A

For a cloth having Warp Extras or Weft Cramming, the Selector Column will also show an icon representing the yarn color for that selector as well as the selector number (see the example below).

Example of a Schema for a cloth using weft figuring i.e. Cramming

For example a cloth having Warp Extras will show a "selector" number in the form 1/- (warp number 1) or 2/- (warp number 2) etc..

A cloth having Weft Cramming (as above) will show the selector number in the form -/1 (weft "selector" 1) or -/2 (weft "selector" 2) etc..

If a cloth is for example 2:3 there will be 2 cloth levels with one weft forming a "stuffer" thread which will normally be between the two layers. In this case there will be a pairing that is shown as -/C for example. It is not necessary to add a weave to these entries as all "points" on these picks are always lifters or droppers.

Weave Name

This shows the weave name to be applied to the selector (for warp extras or weft cramming) or to the level.

The full file path (if there is one) is not shown but may be viewed for example by selecting "Large Display" from the pop-up menu.

The following may also be shown in this column:

No Weave

No weave has yet been selected nor has the weave been edited.

Edited

A weave has been created by editing

name(st.)

The weave name has been selected and one or more stitching points have been added.

name(mod)

The weave name has been selected and has been edited (modified).

Note (mod) and (st.) may be shown together.
