Weave Browser

 

This dialog allows easy selection of weaves to be applied to a line in a schema. It is accessed by clicking on the Browse... button in the Apply Weaves Dialog or by selecting "Browse Weaves..." in the popup menu.

Note that there are several ways to set structures into the Schema and a useful time saver is to copy from one design to another - see the topic Copying a Schema.

Example of the Apply Weaves and Weave Browser shown together but note that they are actually separate dialogs. Also note that the Browse button is now disabled whilst the browser is displayed.

 

Dialog Size and Position

Note that both dialogs can be re-sized so the layout and sizing is the users choice. The size and position of each dialog will be remembered by the software as will the most recently used weaves folder.

 

Using the Weave Browser

Navigate to a Folder

The Weave Browser is actually a standard Windows Dialog as used for Opening and Saving files so the following features are available:

Use the navigation panel on the left

Click on items in the folder edit box

Click on the Up, Back and Forward arrows at the top left

Type in a folder path or amend the existing one and press return

 

Add a weave from the list to the schema

Again this is done using standard features so..

Double Click on a file, or,

Select it with a single click then click on the "Select and Move" button

Drag and Drop from the Browser to the Schema.

Use <ctrl>+C to Copy from the Browser and <ctrl>+V to Paste in the Schema (click on the target line in the Schema after doing the Copy).

In all cases, the software will set (or reset) the weave for the current line in the schema and will move to the next entry.

You can change the selected line in the schema at any time

 

To Close the Weave Browser

Click on the Close button, or, click on the standard 'x' button at the top right.

If you click on Exit in the Apply Weaves dialog, (or close the dialog using the 'x' button at top right) then the software will also close the Weave Browser.

List Style

You can choose one of several list types: Extra Large Icons, Large Icons, Medium Icons, Small Icons, List, Details, Tiles and Content. In the example above, Medium Icons are being used and this is probably the most useful. The List style is changed using the button having a drop down list (3rd from the right near the top).

Preview Pane

In the example above, a preview pane is shown. If you would prefer more room for the list itself, the preview pane can be switched on or off using the button next to the "?".
