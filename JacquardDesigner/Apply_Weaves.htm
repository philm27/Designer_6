Apply Weaves

 

You can apply weaves in one of two ways.

Note that if the cloth is a single cloth, there is no difference between the two methods.

Simple Weaves

This method is so called because you can regard each layer of the cloth as a separate cloth and apply basic weaves such as plain, twill etc. to each layer.

This method uses the concept of a "Schema" (see below) and is particularly useful for cloths that feature weft figuring or indeed warp figuring since the software can also generate the complex threading plan that might be involved.

Composite Weaves

This method requires the cloth to be regarded essentially as a single cloth. If the cloth is in fact, a multi-level cloth, then you must design and create all the required weaves. In this case the weaves must be a composite of all the cloth levels.

Schema

The Schema defines for each artwork color:

The number of "levels"

The warp/weft pairings

The weaves for each level

The stitching points if any
