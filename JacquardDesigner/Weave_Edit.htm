Weave Edit

 

The notes in this topic apply to three dialogs - the Apply Simple Weaves, Apply Composite Weaves and also the Large Display.

The same tools and the same Pop-up menu are available on the "Large Display" dialog as are available on the two Apply Weaves dialogs as shown below.

Example of Apply Simple Weaves Dialog for the same weave, again showing the Pop-up menu

 

Editing the Weave

The weave may be edited using the left hand mouse button. Simply use the mouse to point at the location to be changed and click the left mouse button. Each time the button is pressed, the point will change from lift to drop (lifts being black, drops being white).

The Large display allows the weave to be zoomed in or out using + or - on the keyboard. The display can also be scrolled using the scroll bar. The weave is a fixed size on the Apply Weaves dialog and cannot be zoomed nor scrolled.

The Large Display dialog can also be increased in size to make the edit area larger by dragging any of the four edges of the dialog in or out.

For more details of the Large Display dialog see the topic Large Display.

Other edit functions are available on the Pop-up menu and also there are the shift, flip and invert functions as described below.

Edit Tools

All the following can be used in any combination.

Shift Ends

Use this feature to shift the footing of the weave to the left or right. Click on the up arrow to increase the shift by 1 this will shift the weave to the right. Click on the down arrow to shift the weave to the left. The shift value will remain positive since a negative value will be replaced by the equivalent positive value. (For example if the weave width is 10 ends then -1 is equivalent to +9).

Note that the original bottom left corner of the weave is marked by a red outline.

Shift Picks

This is used in the same way as the above. Click on the up arrow to shift the weave upwards by one pick. Click on the down arrow to shift the weave downwards by 1 pick.

Horizontal Flip

Select this option to mirror the weave left to right.

Invert

Select this option to cause all lifts to become drops and vice versa.

A good example of the use of these tools is given in the topic 1:2 and 1:3 ratios here.

 

Pop-up Menu

Access the Pop-up menu in the usual way by pointing at the weave and clicking the right mouse button.

Insert Stitching Point

To insert a stitching point use the mouse to point at the required location and click the right hand mouse button to display the Pop-up menu then select "Insert Stitching Point".

The stitching point will be marked with an upturned V whether or not the weave is a face or back weave.

Remove Stitching Point

To remove a stitching point use the mouse to point at the required location and click the right hand mouse button to display the Pop-up menu then select "Remove Stitching Point"

Remove All Edits

Select this option to remove all mouse edits from the weave. It does not affect the Shift, Flip or Invert options.

Remove All Stitching Points

Select this option to remove all stitching points.

New...

Select this to clear the weave and possibly a new size. See below for more details

Open...

Use this to open a new weave from disk.

Save As...

Use this option to save the current weave with a new name. Note that this will cause the Shift, Flip and Invert tools to be consolidated into the weave and they will be reset to their initial states (zero or unchecked as appropriate). The name of the weave internally will then be the same as the file name without the .iff file extension.

Large Display

Select this to display the weave in the Large Display dialog.

This option is not available from the Large display itself.

 

New Weave

Select New Weave from the Pop-up menu to display the following dialog.

Example New Weave dialog

Set the size of the weave (each value should be in the range 2-256) then select OK. A new weave will be displayed. The weave will initially be blank (all drops).

You can then edit the weave using the left mouse button. Each time the button is clicked the point selected will switch from drop to lift or vice versa.

If you intend to use the weave in the future you should also save it (see above for information about the save as option).
