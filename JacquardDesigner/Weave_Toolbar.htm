The Weave Tool Bar

 

If this is not currently displayed - select menu option View->Weave Tool Bar while using either a Weave Edit document or a Cloth document.

Example when user is using Weave Edit and both Artwork and weave are currently being shown

 

Artwork

Select this to switch to the Artwork document

Weave Editor

Switches to the Weave Editor (currently selected in the above example).

Cloth Display

Switches to the Cloth View document

Warp Selectors

Display the Warp Selectors dialog

Weft Selectors

Display the Weft Selectors dialog

Apply Weaves Simple

Display the Apply Weaves Simple dialog

Apply Weaves Composite

Display the Apply Weaves Composite dialog

Warp Threading Plan

View and Edit the Warp Threading Plan

Weft Threading Plan

View and Edit the Weft Threading Plan

Copy Warp to Weft

Copy the Warp Threading Plan to Weft

Copy Weft to Warp

Copy the Weft Threading Plan to Warp

Weave Adjustments

Display the Weave Adjustments Dialog (Wilton Carpet/Plush fabric Designers only)

Float Check

Do a float check

Save Weave

Save the weave only as an image file - either iff or any of the other standard types.

Copy Weave To Artwork

Replace the Artwork with the Weave image

Reverse Cloth Horizontal

Flip the Weave over left to right inverting the weave (i.e. swapping lifts and drops) and reversing the Warp Plan and Denting Plan. The user will then be viewing the reverse of the fabric and this will also be reflected in the cloth view. The weave can be left in this format and subsequently saved in the file, or can be reversed back again. using the same button.

Reverse Cloth Vertical

Flip the Weave over top to bottom inverting the weave (i.e. swapping lifts and drops) and reversing the Weft Plan. The user will then be viewing the reverse of he fabric and this will also be reflected in the cloth view.

Select Rectangle

Enables user to select a rectangle that can be copied to the clipboard and can therefore be pasted into a print layout or into other software products. This only applies to a cloth view.

Cancel Selection

Cancels the selection rectangle (does not modify the image).

Cut

Does nothing at present

Copy

Copies the selected rectangle to the clipboard so it is available for pasting into a print layout or into any other software such as a word document

Paste

Does nothing at present

Rectangular Highlight

After selection this button allows the user to draw, using click and drag a rectangle around a part of the design in either Weave Editor or in Cloth View. The rectangle is automatically drawn in both views, so if drawn in Weave Editor it is automatically drawn in the Cloth View. Highlights can be switched on or off using the popup menu.

Circular Highlight

Similar to above but a circle/ellipse instead of a rectangle.

Delete Highlight

After selection the user can click in or near any highlight to remove it.

Show Artwork

If selected, the artwork is shown behind the weave. This will actually modify the current background preference and if not selected, the background will be a plain color as chosen in the preferences.

Show Weave

This will display or remove the weave. If the weave is removed, only the background will be shown and if that is the Artwork, it enables the user to very quickly check the motif in a particular area. Normally it will be switched off then on again very quickly.
