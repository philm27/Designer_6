Schema - Treble Cloth

Schemas are created, reset and edited using the "Apply Weaves Simple" dialog.

 

Example of the Apply Weaves dialog for Treble Cloth. The orange color uses warp A weft A in the face with BB in the middle and CC in the back. Similarly the 3 levels in the yellow areas are BB, CC and AA with the blue areas being CC, AA and BB. A 2+2 twill is used for every color at every level. No stitching has been added.

To create a treble cloth schema click on the Reset button in The Simple Weaves dialog and enter 3 warp and 3 weft and click on OK.

To modify the Warp and Weft pairings, right click any line and select Properties. You can then modify the Warp letter and/or Weft letter. The software will automatically, where necessary, swap letters between levels to achieve the required pairing.

You can also use the mouse to "drag" an entry to a lower or higher level. For example if you wanted CC in the face of the first color above instead of AA, you can click and drag the third line up to the top, rather than using properties.

To modify weaves, you can either create a new weave and edit in situ, or, double click any line and select a weave from disk. For more details on this and other general aspects of Apply Weaves dialog see the topic Schemas - General.
