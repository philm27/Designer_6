Apply Composite Weaves

 

Introduction

This dialog replaces the "Apply Weaves" in previous versions of Jacquard Designer. Therefore you can use it to modify any existing designs and you can use the same weaves. In fact if you load an old design, all the same weave names should appear.

The dialog has all the facilities of the "Apply Weaves Simple" dialog except that there is always one line in the Schema for each palette colour.

Only used palette colours are shown and if you change the design by adding a new colour, it will automatically be added to the list.

As the name suggests, this dialog only uses composite weaves. For a single cloth this makes no difference whatsoever. However, if you use this dialog to produce a multi-level cloth, you must combine the weaves together. See the double cloth example procedure below.

Note that you cannot use this dialog to produce cloths having Warp Extras or Weft Cramming.

Composite Weave versus Simple Weave

You should always use the Apply Simple Weaves dialog in preference to the Composite Weaves dialog. However, there may be certain circumstances, for example an existing design produced using an older version of Jacquard Designer for which the weaves already exist, when it is easier to use composite weaves.

The following are the main differences between the two dialogs.

  • The following buttons on the Composite dialog are not applicable and are disabled: Reset Schema, Move Up, Move Down.

  • When you select Apply Weaves on the Composite dialog you will be asked to confirm the "Ends/Art Ends" and "Picks/Art picks" i.e. the conversion factor from artwork to weave. For example both values are 1 for a single cloth, 2 for a double cloth. See below for more details.

  • Although you can insert stitching points when using composite weaves they only serve as a visual reminder, they do not change the weave in any way.

  • Certain Pop-up menu options are not applicable and are disabled as shown below:

 

Apply Weaves Button

The Apply Weaves button will display the dialog above in order that you can confirm the following two values:

Ends/Art Ends

The number of warp threads in the cloth for each "end" in the artwork.

Picks/Art Picks

The number of weft threads in the cloth for each "pick" in the artwork.

Note that the above all assume that the Artwork contains only the same number of Ends and Picks as the FACE OF THE CLOTH. This will then in turn mean that the Cloth mode will be representative of the appearance of the woven product since it will show the same number of ends and picks as the artwork i.e. the same number as are in the face of the cloth.

 

Double Cloth Example Procedure using Composite Weaves

Before you can complete the procedure below, the composite weaves should be designed and saved to disk if they have not already been created. You can do this using the "Simple Weave" dialog even if you intend to use the Composite Weave dialog to apply weaves to the main design. See the topic Composite Weave Generator for a description of how to do this.

The procedure is:

  1. Complete the Artwork making it the same size in ends and picks as the FACE OF THE CLOTH i.e. half the total number of ends and picks in a repeat (for a double cloth).

  2. Select Apply Weaves Composite on the menu (Jacquard->Apply Weaves->Composite).

  3. Double click each line in turn and select the weave from disk to be applied to that colour OR.

  4. You can draw the weaves interactively if you wish though this may be time consuming for double cloths.

  5. Click on Apply Weaves

  6. Enter the values Ends/Art Ends = 2 and Picks/Art Picks = 2 (for a double cloth).

  7. Click on OK.

  8. The weaves will be applied, the dialog closed and the weave for the main design displayed.

  9. Enter the Threading sequence if necessary for warp and weft.

  10. Check floats using the "Check" menu option.

  11. View the Cloth to confirm that weaves and selectors are correct.

  12. Save the design.
