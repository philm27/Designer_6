Apply Simple Weaves

 

Introduction

Access this option from the menu by selecting Jacquard->Apply Weaves->Simple.

The word "simple" in the title of this dialog refers to the fact that for multi-level cloths you can use basic weaves such as plain, 2+2 twill, 3+1 twill etc. and apply them to each level separately without having to first create composite weaves (i.e. the weaves from each level merged together).

You can also specify stitching points if necessary (see the topics Weave Edit, Display Composite and Large Display) and the resulting composite weaves can be previewed using the Display Composite option.

To use Cramming and Extras you must use this dialog rather than the Apply Weaves - Composite option.

Schema

Before weaves are applied you must create the appropriate schema. This is simply a plan of which levels are present in each artwork color and the properties of each of those levels.

Note that an important pop-up menu is available by selecting the list with the right hand mouse button.

To find out more about Schemas see the topic Schemas - General.

Once a Schema has been created you can edit it in the following ways. Almost all the following options use the Pop-Up menu which is accessed by clicking on the Schema with the right hand button then selecting a menu item with the left button. The most common functions are also available in the toolbar.

 

  • Open a weave

Either Double Click an item or select "Open" from the Pop-Up menu. Then use the standard navigation facilities to select and Open the weave file.

It is important to note that you can multi-select items in the Schema and there are several ways of doing this (see below). The weave you select will be applied to ALL the selected items.

  • Create a New Weave

Select "New..." from the Pop-Up menu (either in the weave itself or in the schema). A simple dialog will be displayed in which the size can be specified.  You can then edit it as described in the topic Weave Edit.

You do not need to save it to disk (using "Save As...") but it may be advisable if you think you will need it again for another design.

  • Edit the Weave

You can edit the weave itself whether you have loaded it from a disk file or by using the "New..." option from the Pop-Up menu by simply using the left button to toggle any point in the weave from lift to drop and vice versa.

You can also add stitching points.

See the topic Weave Edit for more information about the various ways the weave can be edited.

Again, you do not need to save it, but it may be advisable if you think you will need it again for another design.

 

  • Move levels up or down

The example shows the bottom level (pairing B/A and plain2 weave) being dragged up to the top level (mouse pointer not shown).

That section of the Schema will look like this after the mouse button is released.

You can move a level up or down within a double cloth or treble cloth by simply dragging it using the mouse. For other types of fabric, whilst the software will permit it, the position in the list does not make any difference since the weaves determine what is in the face of the cloth in these cases.

Dragging levels will only work if the Schema is in Artwork sequence and you cannot, of course, drag a line out of its Artwork colour.

To use this feature simply press and hold the left mouse button on a particular item whilst dragging it up or down. A special "caret" will appear whilst this is in progress (see example above). Release the mouse button when the caret is in the required position.

  • Change Warp/Weft pairing

The initial default pairings will be A/A in the face, B/B in the back of a double cloth or C/C in the back of a treble cloth. However these can be changed either by moving a level up or down or by selecting the Properties option on the Pop-Up menu (see below).

 

Item Select

You can multi-select items in the Schema and this can be very useful particularly when selecting weaves. You can select items in several ways:

  • Simply click on an item with the left mouse button.

  • Hold the <ctrl> key on the keyboard whilst selecting an item with the left mouse button. ALL existing selections will be retained. If the item you select is currently selected then it will be de-selected so you can remove an item from a selected group.

  • Select one item normally then hold the <shift> key on the keyboard whilst selecting a second item with the left mouse button. All the items in between the first item and the second item will be selected.

  • Click on an item, hold the mouse button and drag the mouse to another line to select all items in between

  • Use "Select All" on the Pop-up menu.

  • Use "Invert Selection" on the Pop-Up menu.

  • Use "Select Face Weaves" on the Pop-Up menu.

  • Use "Select Back Weaves" on the Pop-Up menu.

For more details of the last four options see the "Pop-Up menu" section below.

You can use any of the options above in any order e.g. use "Select All" from the Pop-Up menu then de-select some items by holding the <ctrl> key whilst selecting with the left mouse button.

Schema Sort

By clicking on one of the 3 buttons in the headings at the top of the list you can sort the schema on that column. This can be very useful for checking weave names for complicated schema.

  • Artwork

This is the default sequence. It groups items into Artwork/Level sequence.

Certain features such as moving weaves up or down a level can only be used whilst the schema is in this sequence.

  • Selector

Use this if you wish to see items grouped together by selector (for weft crammed or warp extras) or by warp/weft pairing for a cloth having "full" levels.

  • Weave

Use this if you wish to see items grouped together by weave name.

 

Buttons

Reset Schema

Use this button to reset the Schema. You always need to use this button the first time you use the Apply Weave Dialog for anything other than a simple single cloth.

If you have changed the design it is safest to reset the schema. Warnings may be given when displaying this dialog or at a later stage if you do not.

Note that if there is an existing schema the program will try to preserve the weave information by palette color and by level/selector.

For more information see the topic Reset Schema.

Clear Schema

Use this button to completely clear all entries in the Schema. There is no real benefit to this other than to tidy up a design that will only be used as Artwork and possibly printing.

Clear Weaves

Use this button to clear all weaves.

Browse

Click this button to display the Browser dialog. This is useful as it allows you can select weaves in the browser and also select the location in which they are to be placed in the schema allowing very rapid selection of weaves. See the topic Weave Browser.

Move Up

Use this button to move a schema entry up one level. The list must be in Artwork sequence.

For example if a double cloth has a combination of A/A in the surface and B/B in the back, then in order to move B/B into the surface you need only select that line in the list then select "Move Up". The two lines will swap places making the face of the cloth B/B and the back A/A.

For details of how to change A/A to for example A/B see Properties below.

Move Down

Use this button to move a schema entry down one level.

Undo

Undo the previous action. The user can go back many steps - currently set at 50 though this may change

Redo

Click on this button to redo a previous operation that has been undone

Options

Use this to specify whether or not you want to use edge protection for crammed or warp extras designs. For more information see the topic Edge Protection.

Print

Use this button to print the schema list. A standard Print options window will be displayed allowing selection of a printer device and paper size etc. This may be useful as complex schemas can consist of well over 100 lines. This option is also accessible from the pop-up menu described below.

Apply Weaves

Use this button to apply all weaves. You may wish to use the Float check facility after this and/or change to Cloth Mode.

The Apply Weaves dialog will stay open in case you wish to change a weave and repeat the process. Note that if you close the dialog, you can return to it at a later time and apply weaves again if you make any changes to the artwork.

Also note that if the cloth contains Crammed Weft (or Warp Extras) you will be asked to confirm that you wish to reset the Weft Threading (or Warp Threading). See the topic Cramming and Extras.

Apply Weaves to Selection(s)

Use this button to apply only the selected weaves.

Again, the Apply Weaves dialog will stay open after this.

There are some restrictions on how many levels can be selected. These restrictions apply to double and treble cloths etc where stitching points and lifters are being inserted and it is not possible to apply only layer at a time.

Undo Apply Weave

Undo an Apply Weave or a partial Apply Weave. The software will store several Apply Weave operations. This is useful especially where manual edits have been done and/or float checking which has been accidentally cleared by an Apply Weaves operation.

The undo will also restore the warp and weft threading sequence and cramming/extras flags as they may have been reset by the Apply Weaves operation.

Redo Apply Weaves

Cancels an Undo operation allowing the user to revert back to the state before an undo,

Exit

Use this button to exit the Apply Weaves dialog and return to the main design window.

If you click on any item in the Schema and press the right hand mouse button, a Pop-up menu having some very powerful features will appear as in the following:

Use this to create a new weave. See the topic Weave Edit for more details.

Use this to Open a weave from a disk. You can multi-select to make the weave replace several items at once (Select All, Invert Selection, Select Face Weaves and Select Back Weaves may be useful here). For more information see the topic Open Weave.

Browse Weaves...

 

Click this button to display the Browser dialog. This is useful as the browser does not close after a weave is selected, thus allowing rapid selection of several weaves. See the topic Weave Browser.

Save As...

 

Save current weave using standard "Save As" dialog.

Note that stitching points are also saved.

Note also that when you save a weave, all edits will be consolidated into the weave and the values of Shift Ends, Shift Picks, Flip Horizontal and Invert will be cleared. The weave will then appear in the Schema under its new saved name.

If you want to save a composite weave you can do this by selecting "Display Composite" then using "Save As..." from the Pop-Up menu for the Composite Weave.

Edit

 

 

 

Cut

Copies the current selection(s), clears the weave names, and places them on the clipboard

 

Copy

Copies the current selection(s) and places them on the clipboard

 

Paste

Places all the previously copied weave names at the current selection. All the weaves from the clipboard will be placed into consecutive entries regardless of how many entries are selected. So, if only one item is selected but there are 4 on the clipboard, all 4 will be pasted. Note that when copying and pasting to/from a Schema list, the "Inverted", "Horizontal Flip"  and Shift x and y settings are also copied.

Note also that you can also Copy files (one or more simultaneously) from any File Explorer and Paste into the Schema

 

Undo

Undo the previous action. The user can go back many steps - currently set at 50 though this may change

 

Redo

Click on this button to redo a previous operation that has been undone

 

Add Entry

Click on this button to add an entry for another selector to an artwork color. The selector will be assumed to work wherever the color occurs. This may involve adding additional "slave" entries to other artwork colors. The"Main" additional entry will be marked with two plus signs "++" and "slave" entries will be marked with a single plus sign "+". For more information see the topic Add Schema Entry.

 

Remove Entry

Allows one or more "additional" entries to be removed. Because they cannot be removed in isolation, the program will remove all occurrences of the additional selector which are marked as "++" or "+" i.e. additional Main or Slave entries. Again, for more information see the topic Add Schema Entry.

Schema Properties

In order to change the warp/weft pairing of a schema entry you should proceed as follows:

  1. Select the appropriate line that you wish to change with the LEFT mouse button.

  2. Use the RIGHT mouse button to display the pop-up menu.

  3. Select Properties.

  4. Make changes as necessary using the drop down lists. You can change one or both of the Warp and Weft letters for example A/A to B/B or A/A to A/B.

  5. Select OK

An example of the Schema Properties in which Warp B and Weft A have been selected

Weave Changes

You can make some of the common weave changes using the four options here.

Shift Ends/Picks

Enter a figure here in order to allow for a change of alignment of the weave. Weaves are normally aligned at the bottom left of the design. This corresponds to a shift of zero in each axis.

However a shift of 1 end means that the weave is shifted right by 1 in the design. Similarly a shift of 1 pick means the weave is moved up one pick. When these figures are modified, the weave display will change so that it shows the weave as it will be aligned at the bottom left of the design and the original end 1, pick 1 will be shown with a red outline like this...

This shows the weave as it will be placed at the bottom left of the design, notice the original end 1 pick 1 with a read outline

Weaves are normally based at the bottom left of the designs and repeated out from there. Any partial weave repeat is lost at the top and/or on the right hand side. For example if your repeat is 9 ends wide and 36 long in a design that is 336 x 1200 then there will be 3 ends ignored on the right and 12 at the top.

If the "Scottish" right to left preference is selected, however, the weaves are based at the bottom right and any partial weave repeat will be on the left.

Horizontal Flip

Select this option to flip the weave from left to right before it is applied.

Invert

Select this to invert the weave i.e. all lifts become drops and vice versa.
