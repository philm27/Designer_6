Adding Weaves to a design

 

In this tutorial we will take the design completed in Tutorial 2 and add weaves to make it into a finished fabric design.

We will create weaves from scratch and add them to a library of weaves.

 

A section of the starting design. Note it has been made into a straight match (see below). You can access this design by clicking on the button above.

A section of the design after weaves have been applied

 

Make Straight Match

It is usually advisable to make fabric designs into a straight match rather than, for example a half drop. This is because you will otherwise have to worry about whether weaves continue properly across the edges of the repeat bearing in mind that the next repeat is dropped. Also, of course, a weft threading sequence can have a different effect in one repeat to the next.

Fortunately there is a menu option specially for this.

First of all open the test design if necessary using the open button or the menu option File->Open

First of all make sure that the design is a half drop by selecting menu option Design->Match->Half

Menu showing the "Set Half Drop" option. Note this is only necessary if the repeat contains only a single medallion and is not currently set as a half drop.

Now, in order to make the design into a "Self Match" or "Straight Match", simply select menu option "Design->Create Self Match". This will automatically (in this case) double the width of the design and place a second repeat, half dropped, to the right of the original.

The design repeat should now look like this.

The design repeat after making into a self match

 

Simple Weaves Dialog

You should now display the simple weaves dialog so that we can apply weaves to the design so select menu item Jacquard->Apply Weaves->Simple. The very first step is to reset the schema to a simple single cloth so click on the "Reset Schema" button and the following dialog will be displayed.

The Reset Schema dialog. Make sure that the figures for warp and weft are both 1 and that neither of the other options are selected and click on OK.

Example "Simple Weaves Dialog" showing the pop-up menu.

If weaves already exist on your hard drive (e.g. 2+2 twill, plain, 1+3 sateen etc), which will normally be the case, you can double click an entry in the list and then browse until you find the required weave. For the purposes of this tutorial we are assuming that all the weaves must be designed from scratch.

The "Schema" is the cloth structure - single cloth, double cloth, crammed etc and the default schema is always a single cloth. In this case the list of colours and weaves, as in the above example, will have space for only one weave against each artwork colour.

In this Tutorial we are only concerned with a single cloth but you can follow these links to see what the schemas look like for a double cloth or for cramming (weft figuring).

 

Creating a New Weave

We are going to create 4 new weaves from scratch for this design for each colour in turn starting with the first - black in this example.

  • First of all make sure that the first line in the list is selected (highlighted in blue as in the example above).

  • Right-click either the list of colours and weaves or the default weave image on the left of the list. A pop-up menu will be displayed.

  • Select the New option. The following dialog will be displayed.

Example of the Create Weave dialog showing a width and length of 6 ends and picks.

  • We are going to create a 5+1 twill so set the two figures to 6 and click on OK. The weave image on the left will be reset to the specified size and will be all drops (white).

  • Draw in the diagonal line from bottom left to top right with single mouse clicks (left button). If you make a mistake you can cancel a lift, again, with the left button. In fact the left button is simply switching a drop to a lift or vice versa. The right button is of course displaying the pop-up menu in this case so is not used for editing.

  • The dialog should now look like this.

Example schema after the first weave has been created and a diagonal line drawn to form a twill.

  • Now to convert this into a 5+1 twill (i.e. warp faced) simply click on the Invert option at the bottom left of the dialog. This will change all lifts to drops and vice versa so that the weave should look like the example below.

Example dialog after the 5+1 twill has been created and inverted so that it is warp faced

 

Save Weave

It is not strictly necessary to save weaves after creating and editing them and this step could therefore be omitted. However, it will be slightly easier when creating designs in the future if weaves are saved to your hard drive.

You can use either of the two pop-up menus. Although they are slightly different, the "Save As" option is available on both.

  • Right click on the weave image (or on the list of items) and a Pop-up menu will be displayed.

  • Select the "Save As" option.

  • A standard "Save" file dialog will be displayed. Select a directory and type in a name for the new weave then click on the SAVE button.

The Apply Simple Weaves dialog after the right mouse button has been clicked over the weave image to display the pop-up menu. The "Save As" option has been selected.

After saving a weave, its new name will appear in the list. For example if you saved this weave as "5+1 twill", then the list would now look like this:

Example weave list after the first weave has been saved and named as "5+1 twill".

Apply Weaves

Now all that remains is to Apply Weaves by clicking on the Apply Weaves button at the lower right of the dialog. The dialog will be automatically closed and the "Mode" will be changed to the Weave view. You can select Weave Mode at any time by selecting menu item Jacquard->Mode->Weave.

The project contains 3 main images (as well as the shelf items) - Artwork, Weave and Cloth. If the file is saved as an IFF or PRJ file then these 3 images are all saved in the file.

 

Warp Colour and Threading

We should now check and edit if necessary, the warp threading sequence and colours as appropriate.

Select Jacquard->Threading->Warp. You should see a dialog as shown below.

Click on the first select row in one of the columns. In the example below, the use has clicked on the first column.

The top 4 rows are used for bracketing (2 levels of bracketing each showing a figure and a bracket).

Example of the Warp threading dialog. Note that the figure for the first selector in the first column is highlighted in blue indicating that it has been selected using the mouse.

  • Notice that the number of threads is 1296 This is because there is no existing threading sequence so on entry to the dialog it defaulted to a solid warp using selector 1 and having the same number of warp threads as the weave. If this is not the case simply type in the correct figure (1296) whilst the old figure is selected (highlighted). The new figure will replace it. For more details about editing threading sequences see the topic Threading Sequence.

  • Now select colour 1 in the palette in order to assign that colour (black) to selector 1 (we call each row a selector even though this is warp and therefore, strictly speaking, may not be "selectors").

  • The highlighted selection will move down to the line below so that several selector colours can be set very rapidly.

  • Click on OK.

The first warp selector after selection of black. The figure 1 above and to the left of the colour is the selector number.

Note that there is an alternative dialog for setting selector colours but we do not need to use that in this case.
