Edge Protection

 

This allows edges to be protected when using warp or weft figuring. This should prevent stitching points causing longer floats at the edge of a motif making them look "ragged".

Note that edge protection will not be applied when adjacent motifs use the same selector in the face with different weaves. However, if a selector is in the back at both sides of an edge, then protection will be applied in order to prevent stitching points occurring and causing undesirable floats in some cases.

Weave Options

The Apply Weave Options Dialog

To display the "Apply Weave Options" dialog select the "Options" button in the Apply Simple Weaves dialog.

If you are creating a cramming design then the weft cramming options will initially be displayed. Similarly if you are creating a design with warp extras, the Warp Extras tab will be displayed. However, you can view both pages if you wish.

In order to use edge protection when applying weaves, simply make sure that the "Use Edge Protection" feature is checked as shown above and set the two values for face and back to the number of ends to "protect" either side of an edge.

Similarly for a design using Warp Extras, the two figures give the number of picks to protect either side of a motif edge.

Make Defaults

If you want to make the current settings into the defaults for a new design, you can select the "Make Default" button. Both tabs are saved separately i.e. selecting Make Default on the Weft Cramming page will not affect the Warp Extras defaults.
