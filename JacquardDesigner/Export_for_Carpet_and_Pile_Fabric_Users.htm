Export for Carpet and Pile Fabric Users

 

Introduction

This topic is intended to be read by carpet designers and pile fabric designers who need to use Jacquard Designer to create files for electronic jacquards or electronic punch machines via the export feature.

An example of this export type is the .bin files for the Staubli punch machines or .ep files for Van de Wiele electronic jacquards. These features are optional extras and may not be enabled in your copy of the software.

 

When to use Jacquard Designer

If a design requires special effects you must use Jacquard Designer. Special effects in this case refer to features like "Keep over 2 wires" etc. If it is simply a case of using "Through to Back" instead of "Not through to Back" or a similar overall change in the weaves then this can be accomplished by creating a separate Format File and then using Carpet Designer to export the files which is slightly easier than applying weaves in the Jacquard program. See the topic Format Files.

It is the "Weave" in the IFF file which is exported to the .bin or .ep file. The exported files do not contain any original artwork.

If a change is made to the design, or if frame numbers have changed, then the weaves must obviously be re-applied, the IFF file re-saved and the .bin or .ep files, re-exported.

 

Outline Procedure

  1. Create the design as normal using a different color for each frame but also for each special effect for each frame.

  2. You must now apply weaves to the design to create the weave pattern.

  3. In some cases you may wish to do the weave adjustments at this stage using the option on the Jacquard menu.

  4. You can now use the menu option Export -> BIN File to create .bin files.

 

Producing the Design

As noted earlier the design i.e. the "Artwork" can be produced in either Jacquard Designer or Carpet Designer as there is no difference between the two as far as drawing features are concerned and designs may be loaded into either program having been created in the other.

However, in order to apply weaves you must, of course, use the Jacquard Designer program.

 

Design Colors

You must use a separate design color for each different weave which equates to having a different color for each effect for each frame.

For example if frame 1 has 3 different effects and each of frames 2 and 3 have 2 effects then 7 design colors must be used. You can of course use similar colors so you could for example use a dark red where frame 1 is lifted over 2 wires and a lighter red where frame 1 is lifted over a single wire.

Of course if planting occurs then the situation will be even more complicated as you may wish to use even more colors in order to illustrate what the design actually looks like. In this case you will probably apply the same weave to more than 1 palette color.

 

Preferences

It is recommended that all the following preferences are set (see the topic Jacquard Preferences).

  • Wilton Dialogs and Messages

  • Disable "Apply Simple Weaves"

  • Align Single Dent Weaves with start of motif.

It is particularly important that you select the Alignment preference when using weaves which are one dent wide and 4 or 6 picks long such as double or treble loops so that they are aligned with the start of the motif.

 

Apply Weaves

The weaves in most cases will have been pre-designed and each weave represents an effect.

Black (or the darkest colour in the weave) represents a "hole" in the card (or a lift in the case of electronic jacquard) and white represents "no hole" (or a drop).

It is recommended that a different directory is used for each type of weave for example you could have a directory for 3 frame Paper looms, 5 frame Paper loom, 3 frame VDW NTTB, 3 frame VDW TW, 3 frame VDW SW etc.

When all weaves have been allocated click on Apply Weaves. The small dialog shown here should be displayed:

Set the two figures in the small dialog to the width of each dent in control points (the width multiplier) and the number of picks in a design row (the length multiplier).

For example if there are two holes per frame (because it is a 3 position selection mechanism) and 3 frames then the width multiplier will be 6.

The length multiplier is normally 2 but may in some cases be only 1.

Click on OK. The weaves will be applied and you will be returned to the main design screen.

 

Mode

Immediately after applying weaves, the program will show the Weave view of the design - expanded and in black and white weave form. You can switch to the design view by selecting menu option Mode->Artwork or back to the weave view by selecting Mode->Weave (both on the Jacquard menu).

 

Correcting Errors

If you have made an earlier mistake such as setting the multiplier values incorrectly (e.g. entering 5 and 1 instead of 6 and 2) or perhaps setting the wrong weave against the wrong color, you can correct your mistake and re-apply weaves at any time.

 

Adjust Weave

For the electronic Jacquards, certain weave adjustments are recommended by Van de Wiele and you may wish to do this at this stage. A useful benefit of using the Adjust Weaves option on the Jacquard menu is that the adjustments can actually be inspected as they will be marked with a different colour.

Alternatively the weave adjustments can be made when the file is exported.

For more information - see the topic Weave Adjustments.

 

Save the Design

Make sure that you save the design after weaves have been applied and adjustments have been made.

This will ensure that if the file needs to be exported again at a later time, the weaves and adjustments will already be incorporated.

You should save the Jacquard version of the design as a different file name to that saved from Carpet Designer if there is one since the Jacquard program saves different information and does not display nor save a frame plan if one is attached to the file.

It is therefore recommended that you attach the word "_Weave" to the filename or just a letter W or J for example so that the file names can be distinguished. See the example below.

Export File

Simply select menu option File -> Export -> BIN File or ->Van De Wiele File and proceed as described in the topic Export where the procedure is described in detail. However, the following applies to the use of Export through the Jacquard program.

Once you have selected a Card Format from the drop down list in the File Export dialog, the software can check that all the mathematics are correct.

The message box shown here will be shown if the weave is not the correct size for the design. For example, if you had applied weaves for a 3 frame loom but accidentally selected a 5 frame format in the Export Dialog rather than the 3 frame. Or, conversely, that you had used the wrong multipliers when weaves were applied.

The above message cannot occur when using Carpet Designer as the correct multiplier for the Card Format is automatically used.

 

Creating a New Weave

More information about the pop-up menu and about create new weave is given in the topic Weave Edit. The following has been written with particular reference to carpet and pile fabric designers.

If the weave you wish to use does not exist in the weaves directory, you will have to design it from scratch so, to design a new weave:

  1. Right Click on the colour in the Apply Weaves Dialog. This will cause the Pop-up menu to be displayed. (You can also right click on the weave itself on the left of the dialog).

  1. Select New. The "Create Weave" dialog will be displayed.

  1. Set the size of the weave required. For example if there are 3 frames and 2 control points per frame and 2 picks per design row then a normal weave (1 dent wide) will have width 6 and length 2.

  2. Click on OK.

  3. The weave will be created to the required size with all grid positions empty.

  4. Click in the weave where you require a mark. To remove a mark simply click it a second time with the left button. (The right button operates the Pop-up menu so does not insert a drop).

The weave will be saved and the SAVE dialog closed.

In fact, it is not absolutely essential to save the weave but if you do not save it you will obviously have to redraw it the next time it is required.

It is recommended that the weaves for the various different looms are kept in separate directories as the list of weaves may eventually become quite complex. Some types of cloth may also warrant their own directory of weaves. For example, for a 3 frame Van de Wiele loom it will be worth having 3 directories "VDW 3frame", "VDW 3frame SW" and "VDW 3frame TW". Each of these may have a number of normal and special effects weaves.
