Weave Adjustments

 

This feature is mainly of interest to Carpet and Pile Fabric designers, it should generally not be used if you are designing flat cloth.

Introduction

Adjustments are advised in certain situations as described below for NTTB and TTB TW weaves for Van De Wiele Looms.

The adjustments can either be done simultaneously as the design is exported or they can be done in the Jacquard Designer program using an option on the Jacquard menu and in this case the results can be inspected.

Procedure

To adjust the weave select menu item Jacquard->Weave Adjustments. The following dialog should be displayed:

The Automatic Weave Adjustments dialog set for NTTB. The other two options are for TTB-TW products.

Select the required adjustment type or types and click on OK. The adjustments will be made and marked in red (lifts inserted) or yellow (lifts cancelled). To see them make sure that Weave mode is selected. Note that if the design is saved and reloaded the adjustments will not be lost but the weave will revert to black and white and therefore the adjustments will also be black and white.

Weave adjustments can be made at the same time as the disk is made by selecting the appropriate options, if available, but using the Jacquard menu option has the advantage that the adjustments will be saved into the design file if it is subsequently saved.

Note that you can if you wish also make manual adjustments to the weave.

The following gives details of what the software actually does when it adjusts the weave.

No Adjustment

Weave with NO adjustments assuming red is frame 1 and blue is frame 3.

 

When a yarn stops being used as in the case of the red above, it forms a double point and must be taken under the 3rd pick above with the blue as below.

 

With Option (1) Adjustment

The red yarn has been "dropped" under the weft in pick 3. In the weave this point is marked with a dot.

 

With Option (2) Adjustment

(In this example Option (1) has also been applied as in the previous example).

The new yarn is lifted over the first pick. However in some designs this may not be desirable as it may create a double point. Therefore is may be advisable to use only Option (1). However, in some cases Option (1) on its own creates a long loop in the back as shown below.

 

TTB-TW Adjustments Isolated Tuft

If option (1) is applied when an isolated tuft of colour occurs in one dent between two areas of the same colour then an undesirable result may occur as illustrated below. You should therefore avoid this situation or use Option (1) and (2) together although this is also not a perfect solution.

 

No Adjustment

Weave with NO adjustments assuming red is frame 1 and blue is frame 3. (Illustration for 3 frame Isolated Tuft of Blue between two or more tufts of red).

 

With Option (1) Adjustment

 The points marked with a dot have been "dropped" under the weft in picks 3 and 5. Although this is correct in the face - a long loop of red has been made in the back from picks 3 to 5. This only occurs when a single tuft of another color occurs between 2 of the same color.

 

With Option (1) + (2)

Option (1) and (2) together also lifts any new color OVER the first weft in a pair. This avoids the float in the back from picks 3 to 5 but also creates a slight double point in the face.

This is virtually the same problem as when there are no adjustments at all and is therefore not ideal.
