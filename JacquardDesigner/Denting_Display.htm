Denting Display

In order to display the Denting Plan in the main display (where it can also be edited) - select "Denting On" on the pop-up menu (right click in an empty area in the Weave Editor).

A small example taken from a larger display showing the denting plan in the warp bar. The product is sleyed in 2's in this case.

Whilst the denting is displayed, it can be edited using the mouse in the usual way though brackets cannot be created of course.

To create and edit brackets in a more complicated case you can edit the Denting Plan.
