Weave Editor Introduction

The Weave Editor will appear in its own document but which is part of the Design Project in a similar way to Print Layout Documents.

It is automatically displayed after you have Applied Weaves, or, it can be displayed on request, if weaves have previously been applied, by selecting "Weave Mode" on the Jacquard menu or by selecting the "Weave Mode" button on the Jacquard tool bar.

Though the Weave Editor allows simple modification of the weave, there are safety procedures built in so that it should not be possible to make changes by mistake.

The Weave

The weave display shows each warp and weft thread allowing crossings to be changed from lift to drop or vice versa. If the repeat type is Rug, it will not be possible to scroll past the edges of the repeat. However, if the design is a normal repeat, the software will allow scrolling past the edges.

If the repeat type is Horizontal, the top and bottom edges of the design repeat are the scroll limit in Y axis.

If the repeat type is Vertical, the left and right edges of the design repeat are the scroll limit in X axis.

The software does not allow drops i.e. all matching must be straight match.

There are various display options:

  • Black and white

  • Color

  • Show gaps between threads

  • Vary the gap between threads

  • Group together threads that relate to one artwork end or pick

  • Show the Artwork behind the weave or show a plain color

Warp Bar

The warp bar is shown here at the top but can be shown below the weave if preferred, or, it can be switched off altogether to make more room for the weave.

The warp bar will show end numbers for both the Artwork and the actual Weave. The colors and Extras flags can be edited if necessary.

Whilst in Move Mode (cursor is the Hand symbol), the bar can be dragged left or right

Weft Bar

This can also be switched on or off and will show pick numbers for both the Artwork and the actual Weave.

Colors can be modified and Cramming Flags can be edited.

Whilst in Move Mode, the bar can be dragged up or down.

Scroll Bars

Use these to scroll the display when it is not all shown at once. If you click on an arrow (left mouse button) the display will move by only one thread. If you click to one side of the scroll box, the display will move a whole screen full of threads in appropriate direction.

You can also of course drag the scroll box to scroll up to the maximum allowed for the repeat type.

Control Panel

This shows a thumbnail of the artwork and indicates the location of the display using both a red rectangle and black and white lines at the side of the thumbnail as shown above so that the user can always check which part of the design is being displayed in the Weave Window.

The red rectangle can be dragged around to scroll the main weave display to a new area in the design.

The control panel also displays a summary of the total Artwork and Weave sizes and the repeat type for easy reference.

Also, at the bottom of the Control Panel there are buttons giving quick access to the commonly used functions Design Profile, Surface Effects, Float Check, Preferences and Hide (hides the control panel).

The control panel can be re-sized by dragging it's left edge. Once it is about double the original width, it will change the layout slightly so that more height can be used for the preview as shown below.
