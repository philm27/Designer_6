Float Check

 

The Float Check option allows you to check for a given maximum length of float. There are four separate maximums for warp face and back and weft face and back.

You can selectively check any of these four options.

You can either mark the errors so that you can correct them yourself after inspection or you can get the software to correct the errors for you.

Select the main menu option Jacquard->Float Check (after you have applied weaves of course), or, click on the Float Check button at the bottom right of the Information Panel.

The Float Check button in the Information Panel

 

Example which will check warp face for a maximum of 4 and weft face for a maximum of 5

Maximums

There are four maximums, one for each of the warp/weft face/back permutations so that they can all be different

Do Check

Each of the four options has a "check box" so that they can easily be switched on or off individually.

Mark

Use this button to mark the floats as opposed to actually automatically correcting them.

Correct

Use this button to cause the software to automatically correct the floats as specified by the check boxes. The software

Close

Use this button to close the dialog.

 

 

Procedure

The dialog will initially be displayed using the figures and check boxes used last time so in many cases no changes will be required

  • For each type of float that you wish to check, simply enter or adjust the maximum float length and make sure that it's check box is "checked".

  • Click either the "Mark" or the "Correct" button. The following dialog will be displayed in both cases, though in the case of "Mark", all floats will, of course, be in the Remaining column rather than the corrected column.

Example after Automatic Float correction

 

  • If you select Close the dialog will be closed and the floats either marked or corrected as appropriate.

  • If you select Review/Edit you can check the floats (even the corrected ones) and make changes as shown below.

Example of Float Check Navigation dialog - only one float has been fixed so far (the one outlined in blue). The Float outlined in red is the "current" float. Floats outlined in yellow are not yet fixed.

Review/Edit the Floats

The image above shows is a small part of the resulting image after a float check has been performed. Floats are outlined in one of 4 colors.

Red - the current float and it is unfixed

Green - the current float and it is fixed

Yellow - an unfixed float

Blue - a fixed float

Note you may need to press D on the keyboard or select "Edit Mode" on the pop-up menu in order to be able to edit weave points (if you are not already in Edit mode)

When editing you should use the left mouse button to cause a warp lift and the right hand mouse button to cause a drop as usual. The outline color will be modified immediately and the figures in the dialog will be modified. For example if the warp float above, outlined in red, is corrected, the outline color will immediately change to green.

Float Check Navigation Dialog

Float Summary - This will show a summary of the number of floats and how many have been fixed. For example Float 33 of 656 (1 fixed).

Fixed, Not Fixed, All - click on one of these buttons to filter  the search when using the Next and Previous buttons.

Next, Previous - click on one of these buttons to move the display to the next or previous float. This can be useful when there is a small number of floats in a large design - the user can easly move from one to the other.

You can use the Next and Previous buttons to navigate to floats and you can filter the search by clicking on the Fixed, Not Fixed or All radio buttons. However, you do not need to use the Next and Previous buttons. Or, you can use a mixture of scroll bars, using the "Move by hand" feature and these buttons.

Also, floats do not have to be fixed in the order presented by these Next and Previous buttons. They can be reviewed and fixed in any order. For example in the image above, all four visible floats could conveniently be fixed while they are on the screen.
