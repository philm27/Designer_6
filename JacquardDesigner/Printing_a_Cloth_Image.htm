Printing a Cloth Image

There are 3 ways to incorporate a cloth image into a print layout:

The "Extra Print Options" sub-menu in Textile DTP. For more information about the other items on this menu see the Help file available from a Print Layout.

(1) The quickest and easiest way is to us the "Extra Print Options" feature in Textile DTP i.e. a print layout. This will be available if a cloth document exists. To start a new Print Layout select menu option "File->New Print Layout".

(2) Copy the cloth image (or a section of it) to the clipboard and paste that into a layout.

(3) Do a "Jacquard->Save Cloth As" operation from the Jacquard menu and then, in a print layout, "Import Design" to import the saved file. The image should probably be saved as .tif or .bmp or .png, though .jpg may also be used.

Once inserted, the image can be enhanced using the Pinked or Frayed edges option and, if a whole repeat has been inserted, the image can be modified to show more than one repeat in the usual way.

 

Using the Extra Prints Feature

Once the cloth image has been prepared switch windows to the main design and click on "File->New Print Layout" or right-click outside the design and select New Print Layout on the pop-up menu.

In any print layout right click and select "Extra Print Options" on the pop-up menu and Cloth Image on the sub-menu.

This option will be disabled if there is no cloth document for this project.

Then while pressing and holding the left mouse button you can hold the <ctrl> key to snap the image size to one whole repeat.

For more information about manipulating images in TextileDTP see the help file available in that software.

 

Using the Clipboard

Copy Entire Image to the Clipboard

To do this simply select "Copy" from the Edit menu, or, select the "Copy" button in the Jacquard tool bar. There must not be a current selection rectangle at this time otherwise the software will only copy that section. So if in any doubt click on the "Cancel Selection" button in the tool bar.

Copying the entire image allows several repeats to be subsequently shown in the print layout by cropping outwards or by using the object properties to specify additional repeats.

Copy Selection to the Clipboard

To do this click on the "Select Rectangle" button on the Jacquard tool bar and draw a rectangle in the cloth. The selection rectangle can cross the edges of a repeat.

Then click on the "Copy" button in the Jacquard tool bar to copy the selected image to the Clipboard

To Insert the Cloth Image in Layout

In a print layout, use the standard keyboard shortcut <ctrl>+V, or, select Paste->Design from the pop-up menu or main menu. in the former case you will be asked whether to paste as a design or a logo. You should probably always choose "Design" as that gives you the flexibility to show additional repeats or parts of a repeat, and to show a pinked or frayed edge.

Note

Because the image has not been saved to disk and, at present, these images will not be incorporated in the .lyt file, they should be saved from the print layout by right clicking the cloth image and selecting "Save...".

 

Saving the Cloth Image

To save the cloth image to disk, simply select "Save Cloth As..." on the Jacquard menu. The resulting saved file can be imported into a layout for the current or any other project in the normal way (i.e. Import->External Design).
