Selectors

 

Use this menu option to view or edit the colors assigned to each selector. There is a separate dialog for Warp and Weft and you can switch between the two using the Switch button.

For a simple single cloth, double cloth etc you do not necessarily need to assign selector colors before you apply weaves or set up threading sequences - you can do it at any stage of the design process though you should do it before you try to display the design in cloth mode.

For Weft Crammed or Warp Extras you need to assign selector colors before applying weaves since the colors are used in creating the schema (see Cramming and Extras and Apply Weaves Simple).

Note that simple changes to selector colors can be made using the Threading Sequence Dialog.

The Selector dialog can also be displayed whilst editing a threading sequence by clicking on the "Selectors" button.

Cramming and Extras

When creating designs using cramming or warp extras, a slightly different technique is used see the topic Cramming and Extras.

This type of cloth may, in some cases, require that more than one palette color is assigned to a selector. This permits different weaves to be used in different areas of the design for the same yarn color. You can use up to 32 colors (including the "main" one) for each selector.

Using the Selector Dialog

You can use the following features to allocate colors to selectors:

  • You can point and click on any position in the Selector List to select that position whether or not there is a color there. The dark rectangle will move to that position to show it is selected (at selector 1, color 1 in the above example).

  • Click on a color in the palette at the bottom of the dialog. That color will be placed in the current selected position. Any existing colors at that position will be moved to the right to make room.

  • A palette color can only be used once so if it is already used in a different selector it will be removed from that position and placed in its new position.

  • You can also move around the list using the cursor keys.

  • Use the <DEL> key on the keyboard to delete a color from a selector.

 

Buttons

 

  • OK

Save the changes and close the dialog

  • OK/Switch

Save the changes and switch to warp or weft as appropriate

  • CANCEL

Exit the dialog but do not save the changes

  • Move Up

Move the colors from the current Selector "up" the list and also move the selection "up". Useful for moving complex color lists to another Selector. For example if you have 6 reds on selector 6 and you wish to move them to selector 2 simply select any color on selector 6 and click "Move Up" four times and the 6 reds will now be on Selector 2.

  • Move Down

Similar to "Move Up" but the colors are moved down the list thus for example to move the colors on selector 2 back down to selector 6 simply select any color on Selector 2 and click on "Move Down" 4 times.

  • Insert Selector

Makes the current Selector blank by moving all colors on the current and subsequent selectors down one position.

  • Delete Selector

Removes all colors on the current Selector and moves all colors on all subsequent selectors up one position to "close the gap".

  • Delete color

Removes the color at the current position. Any subsequent colors on the same selector are moved one position to the left to "close the gap".

  • Clear Selector

Clear the current selector.

  • Clear All

Clear all colors from all selectors.
