Creating a Peg Section

 

This feature is designed for those who need to make "peg board" sections for sampling machines. You can select the peg board size required and automatically attach a color strip.

Note that there are preference options which affect this feature.

If the selected section does not contain all used palette colors, a warning will be displayed.

To create a peg section:

  1. Select menu item Brush->Create Peg Section->size where size is a selection from the sub menu. Size can be one of the standard sizes (4x4 6x6 9x9 etc. in inches) or the custom size option.

  2. If you now move the pointer over the design, a highlighted rectangle of the appropriate size will be displayed. Simply move the rectangle over the required section of the design. In order to help you locate the best section for the Peg Board you can zoom in and out, scroll using the cursor keys or numeric keypad, change the drop match, select display many etc. etc.

 

Example of the Peg Section highlight rectangle. The rectangle is under mouse control and can be fixed by clicking the left mouse button.

  1. The peg section will now be the current brush, with a color strip along the bottom edge (used colors only).

  2. Save the Peg Section by selecting menu item Brush->Save making sure that you select the file type appropriate to your sampling machine.

  3. Alternatively, place this brush in the shelf by simply clicking on an empty shelf item then Save the shelf item.

Custom Size Option

If this menu option is selected, the following dialog will be displayed allowing a special size to be specified. You should enter a size and click on OK. You can then proceed as at step 2 above for standard sizes.
