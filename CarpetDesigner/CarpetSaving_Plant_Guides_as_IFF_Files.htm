Saving Plant Guides as IFF Files

 

Some carpet manufacturers may find it useful to save the frame plan as a design so that it can be transferred to the loom as an Electronic Jacquard file or as cards. It can then be used to check that the creel has been set up and planted correctly by running this design.

To save a Plant Guide simply save the frame plan using menu option Frames->Save Frame Plan. For further information see the topic Saving the Frame Plan.
