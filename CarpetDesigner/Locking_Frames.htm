Locking Frames

 

A frame plan may be locked either when loaded (depending on preference setting) or by explicitly locking the frame using menu item Frames->Frame locking->Lock Frame. Once locked the frame will not be updated when designing though you may still manually edit it.

To Unlock a Frame Plan simply re-select the same menu option. When you do this the frame plan will be immediately updated and errors frames added if necessary.

Whilst the frame plan is locked the following features are available on the Frame Locking Menu:

 

Check Frame Plan

This will check the frame plan against the design and report, using a simple message, if there are any instances of colours used in the design which are not in the plan or if there are any colours in the frame plan which are not in the design. In both cases the software will offer the option of marking the errors (equivalent to selecting one or both of the following two options).

 

Show Design Errors

This option will mark all design errors with diagonal shading. See the topic Checking for Errors for more information. Select the same menu option to remove the marks.

 

Show Unused Frames

This option will mark all frame plan errors (colour not used in design) with diagonal shading. Again see the topic Checking for Errors for more information. Select the same menu option to remove the marks.

 

Remove Unused Bobbins

Use this option if you wish to remove all unused "bobbins" from the frame plan whilst keeping the frame plan locked.
