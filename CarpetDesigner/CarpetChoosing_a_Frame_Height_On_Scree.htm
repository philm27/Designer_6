Choosing Frame Height

 

Having created your frame plan , you can adjust its height on screen.

Select menu item Frames->Frame Height and enter the required height into the dialog.

Example of the Frame Plan Height Dialog

The maximum value is actually 32 but you will probably find that a good range of values is from 6 to 16.

Alternatively, try using the keyboard shortcuts <[> (left square bracket) to reduce frame height or <]> (right square bracket) to increase it.
