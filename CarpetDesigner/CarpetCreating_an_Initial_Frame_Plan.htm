Creating a Frame Plan

 

You can create a frame plan at any time in the design process. If you are intending to make extensive use of planting, it is a good idea to create your initial frame plan at an early stage. The frames will update with every alteration you make to your design (assuming automatic refresh is selected), allowing you to see exactly where there is space to plant colour.

To create a frame plan, simply select menu option Frames->Create. (All other options in the Frames menu, other than Load Frame plan, will be disabled if a frame plan has not yet been created).

A Frame Plan dialog like the one below will be displayed.

The Create Frame Plan Dialog showing that 7 frames are required without planting (i.e. 1 color per frame) and a minimum of 5 with planting. The user has selected Color Usage Order (i.e. Frame 1 is most used) and the option "Smart Planting". Frame 8 will be used for planting wherever possible.

 

Frames required

These figures are for information only and show the number of frames required without planting (i.e. 1 color per frame) and the absolute minimum number of frames required using planting as much as is possible.

 

Frame order

If Color Usage order is selected, frame 1 will contain the most used color, frame 2 the next most used color etc.

If Palette Order is selected, frame 1 will contain the first color from the palette which is actually used, frame 2 will contain the next used color from the palette etc.

 

Planting None

Selecting this option will cause the frame plan to be created with only one color per frame. Also, during any subsequent design editing whilst the frames are unlocked, the software will not use color planting - any new colors added will go into error frames.

However, you can still plant colors manually by using Frame Edit.

 

Smart Plant

Selecting this option will cause the frame plan to be created with planting wherever necessary.

The nominated planting frame will be used (see below) wherever possible.

Also, during any subsequent design editing whilst the frames are unlocked, the software use color planting when necessary.

 

Custom Plant

This option gives you complete flexibility over which color goes into which frame and allows you to set a priority sequence for the placing of each color. In other words you can nominate frame 3 for a particular color and then nominate frame 4 as the second choice and frame 5 as the third choice etc.

See the topic Custom Planting for more details.

 

Edit Custom

Click on this button to display the Custom Planting dialog.

 

Frames on Loom

Enter the number of frames on your loom. If you want to use less frames than your loom actually has (for example, you may want to use only 6 frames even though you have a 12 frame loom), then enter this number in the Frames on Loom box.

 

Nominated Plant Frame

This allows you to choose a preferred frame to be used for planting. Colors will be put into frames in palette or usage order and planted in the nominated plant frame if there are more used colors than there are frames.

 

OK

Click on OK to create or re-create the frame plan or click on Cancel to leave the frame plan as it is. In either case you will be returned to the design screen.

 

Notes

  • Having created an initial frame plan, you can re-create it from scratch at any time.

  • You can change the planting designation (None, Smart, Custom) as well as the number of frames, Nominated Planting Frame and custom parameters at any time using Modify Frames.

  • If the Frame Plan window tells you that you require more frames than you actually have, then there must be design errors. The best thing to do is to go ahead and create the frame plan so that the ends which are in error will show up in error frames and can easily be corrected.

  • If you want to delete the frame plan, select menu option Frames -> Delete. If you want to delete your existing plan and create a new one  just go through the Create Frame Plan procedure again - you do not need to delete it first.

  • If you just want to remove the frame plan from your screen, but dont want to delete it  simply select menu option Frames->Frame Window. You can re-select this item to display the Frame plan again at a later time.

  • If cross planting is inevitable (a color has to be planted in more than one frame) then the software will do this intelligently so that the minimum number of colors are cross planted

  • The Zero Color, if any, will not appear in the frame plan.

 

Error Frames

When the computer generates the plan it will add "Error Frames" in red beneath your bottom frame. These will contain the areas of color that will not fit into the plan, enabling you to pinpoint exactly where the planting problem is. You can then edit your design until the error frames have gone. Alternatively, you can use the Frame Edit feature to change the design by editing the frame plan.
