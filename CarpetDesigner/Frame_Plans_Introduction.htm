Frame Plans Introduction

 

Frame Plans can be created at any stage of the design process. They can be developed for a specific design or for a series of designs or colourways by using the Locking feature.

If frame plans are not locked they will normally automatically update as changes are made to the design and if there are more colours used in an end than there are frames available then error frames are created to hold the colours which cannot be planted. It is thus easy to check for errors visually.

The preference Automatic Update Frame plans must be selected in order for frame plans to automatically update as the design is edited.

 

Planting

Planting refers to the placing of more than one colour in a frame. There are 3 different strategies for planting:

None: i.e. full frames (one colour per frame).

Smart Plant allows specification of one preferred frame for planting and is therefore suitable for most woven carpet designs.

Custom Planting allows specification of priorities for placement of Colour in Frames

Frame Plans can always be edited manually - this is not affected by the option chosen.

 

Custom Planting

Each color can have a priority number for each frame. Thus, for example, a color in an 8 frame design could have priority number 1 as Frame 4, priority number 2 as frame 8 etc. Obviously the maximum length of the sequence, for each color, is the same as the number of frames.

See the topic Custom Planting for more information.

 

Checking Frames

You can confirm that a Frame Plan is correct by using the Frame Check option.

If you want to delete the frame plan, select Frames->Delete. If you want to delete your existing plan and create a new one  just select Create from the Frames menu. There is no need to delete your current plan first as it will be deleted when you select Create.

If you just want to remove the frame plan from your screen, but dont want to delete it  simply de-select Frames->Frame Window.

See the topic Checking for Errors for more information.

Locked Frames

This is normally used where a frame plan is common to several different designs or colourways. A special checking system allows errors (for a particular design) to be marked in the design (i.e. colours not present in the frame plan in a particular end) and can also be marked in the frame plan (colours not used in the design).

See the topic Locking Frames for more information.
