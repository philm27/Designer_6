Using the Keyboard

In the following and throughout the Help Files if you see <> angle brackets like this it means that a keyboard key is to be pressed. If this is followed by a plus sign it means press two keys together. For example:

<Ctrl> + R

means press the Control key and letter R key together (cut rectangular brush)

<Shift> + Z

means press the Shift key and letter Z key together (zoom to fit)

 

Using Menus

To use the mouse to select a menu item, simply click and release the left mouse button on the menu bar. The menu you have clicked on will then "pop up". You can then move left or right and up and down menus until you reach the item you require. Click the left mouse button when the required item is highlighted. Some menu items are sub menus having their own "pop-up" menus so you may need to move to the left or right down several menu levels.

To use the keyboard to select a menu item click on the <ALT> key. The menu bar will then become "active" (one menu will be selected and one character in each menu name will be underlined to indicate its keyboard shortcut). You can then use the keyboard cursor keys to navigate to a menu item or you can use the keyboard shortcuts indicated by the underlined character in a menu title or a menu item. When you reach the required item simply press the <Return> key to select it.

You can press <Esc> to remove menus.

For example to select Specification you can type <Alt> D S (not together but one after the other). Or, you could type <Alt> <right> <right> <right> <down> <Return>.

Menu options are written like this:

File->New

Select File menu then New menu item

Design->Match->Half

Select Design Menu then Match sub menu then Half menu item

Starting a New Design

Now that you are familiar Carpet Designer layout, the next step is to start a completely new design.

To create a new blank design use one of the following methods:

  • Select Menu option File->New (see above for alternative ways of selecting menu items)

OR:

  • Click on the "New Design" button on the toolbar.

OR:

  • Use the keyboard shortcut <Ctrl> + N.

You will now see the following:

  1. A blank white square. This is the design area.

  2. The default "Palette" as shown below.

The default palette. Note that in preferences you can select a different palette to be used as the default. Note also that the corner "tucks" at the bottom of a swatch indicate that the left mouse button colour (foreground colour) is black and the right mouse button colour (background colour) is white. The tuck at the top right of the white swatch indicates that white is the only used colour in the design at present.

Repeat Information: Repeat Type

There are four repeat types to choose from:-

  • Normal  your design will repeat vertically and horizontally. This is the repeat type chosen for most design work. The next column of repeats on the right will "drop" the amount specified in the Match.

  • Vertical  your design will only repeat top to bottom. Choose this if you are designing a vertical border or runner.

  • Horizontal  your design will only repeat side to side. Choose this if you are designing a horizontal border or runner.

  • Rug  your design will not repeat. Choose this if you are designing a rug!

  • Brick - this will repeat rather like bricks in a wall, the row of repeats above will be shifted to the right by the amount specified in the Match.

 

Design Information: Match

There are four standard match types to select from  straight, quarter, third and half. There is also an "Any" option so that a drop or shift of any irregular amount can be specified.

 

Design Information: Design Width / Length

If you want to change the design size, input the new details in the specification window. You can work in ends and picks or in design measurements (feet/inches or metres/centimetres). If you are working on a large design, you might find it easiest to work in design measurements  and let the computer work out your ends and picks (the design size is the "design repeat"  this may differ from the actual loom width). These will be displayed in the status bar.

 

Design Information: Ends & Picks

To set the quality (sett) of your design in the specification window, you can select ends and picks to the inch; 3 inch; foot; yard; centimetre or 10 centimetre. Within your chosen parameter you can set the ends and picks to one decimal point.

You can see what quality you are using at any time by glancing at the status bar.

 

Grid Information: Grid Setting (Width & Length)

The grid quality is set in the specification window  you will find a section of the window called "Grid Information".

The grid should be set so that bold lines are inserted at intervals to help the design process. Usually, the bold lines are required at the same spacing as the design quality. For example, if your design quality is 8x9 to the inch, then you may well require bold grid lines every 8 ends and 9 picks. If this is the case, after setting your design quality, just tick "Grid as Quality" in the specification window.

 

Repeat Size: Default

You can re-set the default specification at any time by clicking on the "Set Default" button in the bottom right hand corner of the dialog.

Each time you start a new design, it will initially use the default values.

 

Other Information:

Save your own notes along with the design, by writing them in this box.

 

Drawing Tools and Colours

To see more of the design, press - (the minus key) to zoom out, to see an area in greater detail press + (the plus key).

At the top of the toolbox you will see a rectangle with a circle in it (the currently selected colours). The colour of the inner circle represents the colour you can use with your mouses left button. The outer (the rest of the rectangle) represents the colour you can use with the right button. Moreover, if you look at the "Palette Window", you will notice that the colours you have selected on each button of the mouse have "tucked" corners on the bottom left or right respectively. This feature serves as a visual reminder of which colour is represented by which button of the mouse.

For the purposes of this introduction, we will stick with the default palette .

The colours currently in use will probably be black and white.

To change the left hand button colour, simply point the cursor over the palette window and click the left button of the mouse over the colour you wish to draw with.

When using any drawing function, you must always select a brush-type from the toolbox. There are a variety of different sized square and round forms. These can be found at the bottom of the toolbox.

 

 

There is a different selection (this time just governing line thickness) for the Line and Shape-drawing functions:

 

 

If you want to draw with a larger brush size than those in the toolbox, double-click on the drawing tool icons (again in the toolbox). You can then select your own size/width of brush.

It is also possible to draw and fill with a Custom Brush (see the Topic Custom Brushes for more details).

If you are working with a pen and graphics tablet, then you can control brush size by applying pressure with the pen on the tablet. To use this function go to Edit->Preferences-> Pressure and enable the function.

Try out the "drawing tools" - to familiarise yourself with them - by clicking on the buttons (in the toolbox) and trying the different methods of drawing (remember you can utilise the two selected colours with the two mouse buttons).

You may also want to zoom in or out on the design area. To do this press + or - respectively on the keyboard.

 

Drawing a Simple Design

You can either create a simple design of your own or do the following to create the simple design shown below.

Undo

If you make a mistake at any stage simply click the undo button. If you click this button several times in succession then a series of drawing operations will be undone.

Select Palette Colours and Set Background to White

In the palette, click on red with the left button and white with the right button. You should see the two colours at the top of the toolbox to indicate that you have selected those two colours. (Red being in the circle and white the background as shown below).

If the current background in the main design is not white, select menu item Edit->Clear. This should fill the whole design with the current right hand button colour which is white. (This function will ask you to confirm that you wish to clear the design before proceeding).

Draw The Circle

Now select the Circle Outline button in the toolbox. Move to the main design and draw a red circle.

Now select the fill button and then click anywhere inside the circle with the left mouse button. The circle should now be flood filled with the red colour.

View in Repeat and set the Half Drop Match

Put the design into repeat by clicking the View Many button in the toolbar.

Make the design a half drop match by selecting menu item Design->Match->Half.

Zoom in and or out using the plus or minus keys on the keyboard so that there are about 3 - 5 repeats across and down the screen as shown below.

Draw The Blue Lines

Select the blue palette colour with the left mouse button.

Select the line drawing function in the tool box.

We want to draw a much thicker line than the normal single point so double click the line drawing button in the toolbox. This will display a "Custom Line Width" dialog. Set the line thickness to 16 (or any other value you choose) and click on OK or press the <return> key.

The Custom Line Width Dialog with a thickness of 16 selected.

Draw some lines anywhere in the design by pressing and holding the left mouse button on the start position and then moving to the end point and releasing the mouse button.

Notice that the lines will be immediately repeated in all other visible repeats and that you are able to draw across the edges of the repeat. You can if you wish draw a diagonal line longer than one repeat - it will simply "wrap" around across the actual repeat.

The resulting design in repeat after the drawing operations described above. Notice that the design is a half drop match and that the blue lines cross the edges of the repeat.

 

Saving

The normal file type for use with the APSO software is IFF or PRJ. The IFF file type will contain not only the design and palette but the specification, frame plan and shade numbers etc. The PRJ (short for project) type is just the same but will also contain all the shelf items.

To save an IFF or PRJ file, access the "Save As" dialog by selecting menu item File->Save As.

Find the folder in which you want to save the design. To do this, work your way through the folder "family tree". Go to the "parent" folder by clicking on the icon, or to the "offspring" folder by double clicking on a folder within the currently selected folder. When you find the required folder make sure that it is open in the "Save In" box.

Select IFF or PRJ files from the "Save as Type" drop-down menu, and then enter your design file name in the "File Name" box.

Click on the Save button to save the file in the folder. The Save As dialog will close, taking you back to the design screen.

If you have already saved your design to a folder, it is possible to update the file with more recently done work. The old design file will be overwritten and you will not be able to retrieve it, so only use this function if you are sure you dont want to keep the previous version.

You can re-save a design without changing its name as follows:

  • Select menu option File->Save.

OR:

  • Click on the Save button on the toolbar

OR:

  • Use the keyboard shortcut <Ctrl> + S

 

It is always advisable to save work frequently to avoid loss of work and effort in case of computer system failure for any reason.

 

 

Creating a Frame Plan

You can create a frame plan at any time in the design process. If you are intending to plant a lot of colours within the frames, it is a good idea to create your initial frame plan early on. The frames update with every alteration you make to your design*, allowing you to see exactly where there is space to plant colour.

*Note that the preference "Automatic Update" for Frame Plans must be set for this to happen and also the frame plan must not be locked.

You can also do clever tricks like re-colouring the design by changing colours in the frame plan.

To create a frame plan for your design, simply select menu option Frames->Create. (All other options in the Frames menu are disabled out at this stage).

A Frame Plan dialog like the one below will appear on screen:

 

The Create Frame Plan Dialog showing that 8 frames are required without planting and a minimum of 8 with planting. The user has selected Colour Usage Order (i.e. Frame 1 is most used) and the option "Smart Planting". As it stands frame 8 will be used for planting.

Note:

  • Having created an initial frame plan, you can repeat this process to re-create it from scratch at any time.

  • Smart Plant will only plant your colours if there are more used colours than there are frames on loom.

For more information about this dialog see the topic Creating a Frame Plan.

Key in the number of frames you have available on loom and select OK.

When the computer generates the plan, if some colours cannot be planted into the given number of frames, the software will add "Error Frames" in red beneath your bottom frame. These will contain the colours that will not fit into the plan, enabling you to pinpoint exactly where the problem is. You can then edit your design until the error frames have gone. Alternatively, you can edit the design from within the plan.

To delete the frame plan, select menu option Frames->Delete.

If you just want to remove the frame plan from your screen without deleting it  simply de-select menu option Frames->Frame Window. You can re-select this option at any time to show it again.

To save the design, now with a frame plan, simply click on the Save button in the toolbar.
