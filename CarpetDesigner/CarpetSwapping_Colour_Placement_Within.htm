Swapping Colour Placement in Frame Plan

 

It is possible to swap positions of colours within the plan for a given end or group of ends. This is a useful function if you are trying to plant certain colours together, or are trying to avoid or tidy up cross-planting.

 

Example of swap feature. TO select the whole block of colour hold the <shift> key and point and click on the brown in frame 2. This will select the whole block as indicated by the dotted lines. The cursor will change to a "TO" arrow and you may now lick anywhere in frame 3 for example which will initiate the change.

Procedure

  1. Make sure that you can see your frame plan clearly enough to work within it. If necessary, increase the frame height (see the topic Choosing Frame Height).

  2. Select menu item Frames->Swap.

  3. As you float the cursor over the frame plan, it will change to a "From" arrow.

  4. Either click and release the left mouse button on a single "bobbin", or, click and hold the left mouse button and move it left or right to highlight a block of colour (and dead positions possibly).

  5. Alternatively you can hold <shift> and click a block of consecutive "bobbins" of the same colour to select the whole block. If you hold <shift> and <ctrl> the whole frame will be selected.

  6. When you release the mouse button the cursor will change to the "To" arrow.

  1. Now click in the frame to which you wish to move that block. The highlighted block and the corresponding block in the target frame will swap positions.

  2. The software will remain in Swap mode allowing you to swap several blocks one after the other.

 

<Shift> Selecting a Block of Colour

If you hold the shift key whilst clicking in the frame plan you can select a whole block of consecutive "bobbins" of the same colour. Thus you only need to press and release either mouse button whilst holding <shift> in order to select that block.

<Shift> + <Ctrl> Selecting Full Frame

Similarly, by holding both <ctrl> and <shift> keys whilst pointing at a frame and clicking the mouse button you can select an entire frame.

Undo

You can undo any frame plan swaps in the usual way.

Locked and Unlocked Frames

You can swap frames whether or not the frame plan is locked.
