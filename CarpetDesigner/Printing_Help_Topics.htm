Printing Help Topics

 

Printing a Topic

Select the required topic from the contents tab or find it using the Search, Index or Favourites tabs.

Click on the topic with the right hand mouse button to display the Pop-up menu.

Select the Print option in the Pop-up menu. This will display the standard Print dialog on which you can set print preferences and request a print.

Alternatively, proceed as below but select "Print the selected topic" in the dialog.

 

To Print a complete Section

To print the Tutorials or any other section:

Select the required section (or "book") in the Contents tab then:

Click on the Print button in the toolbar.

Or, in the Contents tab click with the right hand mouse button to display a Pop-up menu and select Print in the Pop-up menu.

In both the above cases, the following dialog will be displayed allowing selection of the whole "book" (the selected heading and all sub-topics) or just the current topic.
