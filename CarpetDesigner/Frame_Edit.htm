Frame Edit

 

The frame edit option allows you to make manual edits to to the frame plan using the selected colour with either the left or right mouse button.

To select this feature, simply select menu item Frames->Edit. You will find that the pointer will change to a paint brush when it moves over the frame plan allowing painting of colour. This does not affect the normal drawing functions for the main design which can still be used when the pointer is over the design itself.

Example frame edit showing the paint brush cursor in the frame plan. Five beige "bobbins" have just been changed to pink which has automatically changed the colour of the beige design points in the motif above. Note that the positions to be changed in the frame plan are highlighted by a dotted line.

<Shift> Selecting a Block of Colour

If you hold the shift key whilst painting into the frame plan you can select and paint over a whole block of consecutive "bobbins" of the same colour. Thus you only need to press and release either mouse button whilst holding <shift> in order to repaint that block.

<Shift> + <Ctrl> Selecting Full Frame

Similarly, by holding both <ctrl> and <shift> keys whilst pointing at a frame and clicking the mouse button you can repaint all used positions in a frame with the currently selected colour. If the frame plan is locked (see below), the whole frame will be changed to the specified colour. However, if the frame plan is unlocked, any unused positions will remain unused.

Undo

You can undo any frame plan edits in the usual way.

Frames Unlocked

Whilst frames are unlocked you can edit existing colours in the frame plan to change them to another colour. All corresponding design points using the old colour will be automatically changed to the new colour.

You cannot move a colour by drawing it into a different frame nor can you add unused colour points (i.e. unused "bobbins") into the frame plan.

Frames Locked

Whilst frames are locked you can edit existing colours in the frame plan as described above for unlocked frames. Also, however, you can do the following:

  • Add unused colours to "dead" positions in any frame.

  • Move a bobbin from one frame to another by simply repainting it into a different frame.

It may seem that, conversely, more editing features are available when the frame plan is locked. However, this is because whilst unlocked, the software will automatically correct the frame plan (assuming automatic refresh is selected) resulting in any unused bobbins being removed

Remember you can use the left or right hand mouse button to draw into the frame plan.
