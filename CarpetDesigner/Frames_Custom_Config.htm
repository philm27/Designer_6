Frames Custom Configuration

This dialog allows you to control how the Frames will be planted when using the Custom Planting option.

It is only relevant to Custom Frame Plans.

The dialog here is displayed when the creating or modifying a Frame Plan using the Custom Option and clicking on the "Edit Custom" button.

Basic Creating and Modifying a Custom Configuration

Insert a Color

To use the dialog simply drag colors from the palette at the bottom of the dialog, scrolling it if necessary to display all colors.

Or, select a location in the Configuration (indicated by a blue rectangle), then select a color in the palette and click on "Ins. Col" to insert a color.

Or, right-click on a location, having first selected a palette color, then select "Insert Color" on the pop-up menu.

Delete a Color

Select a color in the Configuration (indicated by a blue rectangle), then click on "Del. Color".

Or, right-click on a location, then select "Delete Col." on the pop-up menu.

Move a Color

A color that is already in a configuration can be moved to a new location by dragging it.

The pop-up menu

The Pop-up Menu

The pop-up menu provides an easy way to edit the configuration. Display it in the usual way by right-clicking on a particular location.

Insert Color - inserts the currently selected palette color at the current location, moving any existing entries to the right to make space.

Insert Empty - can be used to make sure that a particular color has particular priority or simply to organise the display in a logical way. For more information about priorities see below.

Delete Col. - deletes the selected color.

Insert Frame - moves everything including the selected frame down the list i.e. to a higher frame number, leaving the selected frame empty. Nothing is deleted.

Delete Frame - the frame holding the selected color is cleared and the data in the frames below is moved up one frame.

Move Up - moves the set of colors from the selected frame up to the frame above and the colors in the frame above down one to replace them. Nothing is deleted.

Move Down - moves the set of colors from the selected frame down to the frame below and the colors in that frame are moved up to replace them. Nothing is deleted.

Buttons

Most of the buttons have equivalents on the pop-up menu but there are two additional simple functions:

Clear Frame - clear all the entries on the selected frame without affecting any other frame.

Clear All - delete all colors from all frames.

Similarly color 7 is planted in frame 3 if possible, otherwise in frame 4.

Planting Priority

The sequence of the colors in each frame also indicates the priority. thus if there is a conflict in any frame at a given end, the color that comes first in the list reading from left to right will get priority. This means that a color can be placed in two different frames with (possibly) different priorities.

For example you may wish to plant color 3 in frame 3 as much as possible but where this is not possible, plant into frame 4 as a second priority, to do this just make sure that color 3 is further to the right in frame 4. as in the image above.

This may require the addition of empty locations to move colors to a lower priority.
