Convert Design to Frame Numbers

This topic may be important to users who need to convert an Axminster or Wilton design that uses planted colors to a design that only uses the first few palette colors and these palette colors will thus indicate the frame number and therefore the design file, when subsequently saved, could be used as a production file when using appropriate plant and equipment.

This option can be found on the Frames menu as "Design to Frame Colors" - it will only be available if there is a current valid frame plan.

The operation can be undone using the normal design undo/redo operations.
