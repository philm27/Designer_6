Sorting Frames by Usage

 

Having created a frame plan, you may need to sort your frames into usage order for manufacturing purposes. Even if you chose to create a frame order by colour usage for your initial plan, if more than one colour is planted on a frame you may still need to do a sort. This is because the percentage usage of each frame is calculated from the combined percentages of all the colours planted within the frame.

To sort your frames into usage order, simply select menu item Frames->Sort by % usage.
