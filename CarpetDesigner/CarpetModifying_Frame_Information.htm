Modifying Frame Information

 

In order to change the number of frames or the frame planting strategy without recreating the frame plan select Frames->Modify.

Example of the Frame Modify Dialog

 

You can select/de-select Smart Plant or Custom Plant and, if using Custom Planting, you can click on the button to edit the priorities. This will not affect the existing frame plan, but will effect any subsequent planting.

If you increase the number of frames in your plan, they will be inserted at the bottom of the plan. If you currently have error frames they will automatically be planted into the new frames (assuming as usual that Locking is not active and that Automatic Refresh is selected in preferences).

If you reduce the number of frames then any color in the frames removed will be placed into error frames.

You can also change the nominated planting frame for the Smart Plant option.
