Export BIN files

 

To export a BIN file select menu item File -> Export -> BIN File.

If you are using Carpet Designer and if there is no frame plan, the message box shown here will be displayed. Click OK to use the palette positions as frame numbers and continue or click on CANCEL to return to the main program.

 

Main Dialog

The image below shows the main Export Dialog with the normal settings. It will normally only be necessary to select the required format from the drop down list and select the OK button.

Example BIN file export dialog. a 3frame fromat has been selected from the drop down list. The punching is to be in normal sequence and all cards and all sets are to be punched.

Select the OK button to proceed with the export of the .bin files or select Close to close the dialog and return to the main design program.

Format

All format files in the current "format file directory" will be listed. There may be only 2 entries for example 3 frame and 5 frame or several variations such as "3 frame TTB" or "3 frame NOT TTB". The names shown are simply the file names excluding the .txt file extension.

Browse

Press this button to browse to a different "format file directory". The last directory used will be remembered and the default directory is c:\program files\common files\apso\WeaveFormats.

Reverse Sequence

If this is unchecked then the first pick in the file will be pick 1 which will therefore be punched first. If this option is checked, the last pick in the pattern will be at the beginning of the file and pick 1 at the end of the file. The header if being punched will always be in front of pick1.

All cards

If selected all cards will be punched. If there are 2 cards per design row there will be twice as many cards are there are rows in the design. Also the number of cards may be increased due to the minimum number of cards specified in the Format File.

Selected Cards

If this is checked you may specify a "From" and "To" card number. This feature may be useful if a small number of cards is to be produced to replace damaged ones. This feature may be used in conjunction with "Selected Sets" to produce a single file.

From and To Card Numbers

These are card numbers not design row numbers and may therefore be higher than the actual number of rows in the design. For example if a design has 20 rows and there are 2 cards per row and a minimum of 48 cards then the software will produce a total of 80 picks for each set (40 for the basic design increased to 80 due to the minimum of 48). In this example card numbers may be in the range 1-80.

All Sets

If selected all sets will be produced. The number of sets is specified in the Format File. There will normally be 6 sets each of which is written to a separate file. The letter A, B, C, D, E or F will be added to the file name.

Selected Sets

If this is checked you may specify a "From" and "To" set number. This feature may be useful if a small number of cards is to be produced to replace damaged ones. This feature may be used in conjunction with "Selected Cards" to produce a single file containing a restricted number of cards.

From and To Set Number

Simply enter the From and To set number. You may specify a single set by entering for example F to F to produce a single file for set F or you may specify a range of set numbers such as B to D to create files for the 3 sets B, C and D.

 

Partial Repeats

If the design does not fit into the loom width a whole number of times, the software will give the option of adding partial repeats at one or both edges as described in the topic Export - Partial Repeats.

 

Save BIN Files

The software will always display the following dialog allowing you to specify the name for the .bin files. You may navigate to a different directory in the usual way.

You may wish to write the files to hard disk and then copy them to floppy disk when required, or you can write them directly to floppy disk.

Note that if there is normally more than 1 set according to the Format File the appropriate letter will be appended to the file name. An example of this is shown above as a previous design has been saved as D5896A.bin etc.

The software will initially default to the current design name as the file name for the .bin files. You can change this if you wish. You do not need to add the letter for the set even if you are only exporting one set out of 6, the letter will be added automatically.

Also note that the software will initially check that the file name without the set letter does not already exist in the target directory (i.e. D5901.bin in this example) but as each file is saved a warning will be displayed for each individual file if it's file name already exists.

 

Disk Full Situation

This may often occur when saving to floppy disk. If a file will not fit in the remaining space on a disk. The following message box will be displayed.

 

You can use the "My Computer" icon to access the disk and delete some files. Or, you can insert a different floppy disk and use that. This may mean that some files in a set are on one disk and some on another. In fact, if necessary, each file could be on a separate disk.
