Export - Van De Wiele Files

 

This option will export an EP file suitable for use on the Van De Wiele electronic Jacquard. An appropriate Format File must exist in order to use this export method.

To export a Van De Wiele file select menu item File -> Export -> Van De Wiele.

If you are using Carpet Designer and if there is no frame plan, the message box shown here will be displayed. Click OK to use the palette positions as frame numbers and continue or click on CANCEL to return to the main program.

 

 

Main Dialog

The image below shows the main Export Dialog for Van De Wiele files. This example shows "a 12 ft Not Through to Back" format selected with the NTTB weave adjustments also selected.

Simply select the appropriate format type, Weave Adjustments if any and other settings then select the OK button to proceed with the export or select Close to close the dialog and return to the main design program.

Format

All format files in the current "format file directory" will be listed. The names shown are simply the file names excluding the .txt file extension.

 

Browse

Press this button to browse to a different "format file directory". The last directory used will be remembered and the default directory is c:\program files\common files\apso\WeaveFormats.

 

Weave Adjustments

Select the appropriate check box. See below*.

 

Cutting Zone Front

This figure is the number of empty picks to be woven at the front of the design and is useful for rug products.

 

Cutting Zone Back

This figure is the number of empty picks to be woven after each repeat, and again, is useful for rug products.

 

 

Weave Adjustments in Export Dialog *

You can select NTTB, or, either one or both of the TTB adjustments. You cannot select NTTB as well as one of the TTB adjustments.

If you are using Jacquard Designer and have already done the specified weave adjustments using the menu option Jacquard->Weave Adjustments then selecting adjustments on this dialog will have no effect.

It is recommended that Option (1) is always used for TTB weaves and option (2) possibly only if the design includes a pin dot (see the topic Weave Adjustments for more details).

 

Define Partial Repeats Dialog

This dialog will ONLY be displayed when the design does not fit onto the loom a whole number of times. It allows partial repeats to be specified at left and/or right edges in order to make up a 12 foot (or loom width) cloth.

For more details see the topic Export - Partial Repeats.

 

Save EP Files

The software will always display the following dialog allowing you to specify the name for the .ep files. You may navigate to a different directory in the usual way.

You may wish to write the files to hard disk and then copy them to floppy disk when required, or you can write them directly to floppy disk.

Unlike BIN files, there will normally be only one file to save.

The software will initially default to the current design name as the file name for the .ep files. You can change this if you wish.

The software will check that the file name does not already exist in the target directory.
