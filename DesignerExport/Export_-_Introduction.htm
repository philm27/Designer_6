Export - Overview

The Export feature allows both Carpet Designer and Jacquard Designer to create files suitable for use by various loom types. These include the BIN files for certain punch machines and the EP files for Van de Wiele electronic jacquards.

These features are an optional extra and may be disabled or not visible on your menus. To find out more contact your sales representative.

Files can be created by both the CarpetDesigner and the JacquardDesigner programs but not by the Textile Designer program.

 

Format Files

These give the software important information for Carpet users such as the number of frames, the number of holes per frame, the number of picks (cards) per design row and the actual positions of each end and frame in the card. More details are given at the end of this document.

 

Weaves

Weave files are only required if Jacquard Designer is being used. If Carpet Designer is to be used then the weaves are determined from information in the Format File.

 

Features - General

Format Files - The System uses Format Files which are simple .txt files that may be edited or created using Notepad. All details of the punching or electronic jacquard file requirements are contained within these files.

Punch Code - The Format file specifies the punch code (or lifts or electronic code) for selected and non selected frames on each of the 2 possible cards (or picks) therefore it is easy to create a Format for variations such as "Through to Back" or "Not through to Back" etc.

Auto-Repeat - The design itself need not be made up to the width of the loom. The software will automatically repeat it out widthwise and will automatically take account of a repeat drop match. (However, if the design uses the new "Brick" repeat feature it must be converted to a self match (i.e. straight match) before the Export option is used).

Partial Repeats - If the design does not fit into the width of the loom a whole number of times, the software will offer the option of starting the output file at any design end therefore creating a partial repeat at the left and/or right hand side.

Zero Colour - In Carpet Designer, the Zero Colour may be used to indicate where all yarn is to be woven in the back.

 

Special Features - Verdol Papers

Header Cards - A header card or cards can be punched in which the design name, Card Set number and card count will be punched in a readable form. The third header card onwards, if punched, will have every frame selected so that these cards can be made to overlap the last few cards and glued to make a card loop.

Card Numbers - may be punched, again in readable form, in unused card columns. The software automatically selects unused columns in which to punch the card numbers

Reverse Sequence - Cards can be written in reverse sequence or normal sequence.

Selected Range - Specific cards can be written to the export files making it easy to re-punch a small number of cards in a particular set if they have been damaged.

Selected Sets - All sets may be written to the exported file or files or a specific range of sets. For example if there are normally 6 sets A - F, each set representing a 2 foot width of carpet, you could request only set D or perhaps set D-F to be exported (and subsequently punched).

Minimum Cards - A minimum number of cards can be specified. This can be useful since a certain number of cards may be required to fit around the drum on the loom.
