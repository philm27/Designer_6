Export - Partial Repeats

 

This dialog will ONLY be displayed when the design does not fit onto the loom a whole number of times. It allows partial repeats to be specified at left and/or right edges in order to make up a 12 foot (or loom width) cloth.

Define Partial Repeats Dialog

 

Example Partial Repeats Dialog showing the "even" option selected which will place an equal sized partial repeat at both the left and right hand side of the cloth. The layout that this produces can be seen at the bottom of the dialog as a partial repeat of 26 design ends on the left and right with 11 repeats of 100 ends in the centre.

 

It may be that the wrong version of a design has been selected in which case select CANCEL to return to the main dialog which can then be closed allowing the correct design to be loaded using the main menus.

On the other hand if you wish to proceed, having made sure that the settings are all as required, click on OK.

All Left

Select this to create a partial repeat width on the left hand side. The right edge of the cloth will align with the right edge of a design repeat (i.e. no partial repeat on the right).

All Right

Select this option to create a partial repeat width on the right hand side. The left edge of the cloth will align with the left edge of the design repeat (i.e. no partial repeat on the left).

Even

Select this option to create an equal sized partial repeat on both the left and right hand sides of the cloth.

As Specified

Select this option in order to specify the start end in the design at the left edge of the cloth.

Layout

This will always show the layout for the current settings in the format shown i.e. the width of the left hand border, the number of repeats and the right hand border width.
