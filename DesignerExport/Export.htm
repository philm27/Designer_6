Export - Carpet Designer Vs Jacquard Designer

 

Using Carpet Designer

If the design has no special effects you can use Carpet Designer to produce the .bin files required for punching the VERDOL cards.

If the frame plan contains planted frames (i.e. frames containing more than one color) then a frame plan MUST be created before the design is exported as a BIN file.

On the other hand, if the frames are all single colour then the frame plan is optional as far as the file export is concerned, as you can use the palette as the frame order.

Special Effects, except for "no yarn" (where a Zero Colour occurs in the design) cannot be created.

 

Using Jacquard Designer

If a design requires special effects you must use Jacquard Designer to produce the .bin or .ep files. Special effects in this case refer to things like "Keep over 2 wires" etc.

You must go through the process of applying weaves to the design before the file can be exported.

If it is simply a case of using "Through to Back" instead of "Not through to Back" or a similar overall change in the weaves then this can be accomplished by creating a separate Format File and then using Carpet Designer to export the files which is slightly easier than applying weaves in the Jacquard program. See the topic Format Files.
